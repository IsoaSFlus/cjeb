                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>芳</strong></p><p><span class="dicpy">fāng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fāng.mp3"></a></span> ㄈㄤˉ</span></p><ol><li>花草的香气：～香。～草。～菲（ａ．花草的香气；ｂ．指花草）。～馥。～馨。芬～。</li><li>喻美好的：～名。～龄（年龄，用于少女）。～姿（美好的姿态，用于少女）。～邻（关系融洽的邻居，用作敬辞）。流～百世。</li><li>花卉：群～竞艳。～时（花盛时节）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">芳</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>芳</strong> <span class="dicpy">fāng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fāng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从艸，方声。本义:花草)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[fragrant grass]</span></p><p><span class="diczx1">野芳发而幽香。——</span><span class="diczx2">宋</span>· <span class="diczx2">欧阳修</span><span class="smcs">《醉翁亭记》</span></p><p><span class="cino">(3)</span> 又如:芳芷(香草名);芳苓(芳荃。香草名);芳草(香草)</p><p><span class="cino">(4)</span> 香气 <span class="encs">[aroma;fragrance]</span></p><p><span class="diczx1">芳，草香也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">芳菲菲其弥章。——<span class="smcs">《楚辞·离骚》</span>。注:<span class="yhcs">“香貌。”</span></span></p><p><span class="diczx1">兰芝以芳未尝见霜。——<span class="smcs">《淮南子·说林》</span></span></p><p><span class="diczx1">芳与泽其杂糅兮。——<span class="smcs">《楚辞·离骚》</span></span></p><p><span class="diczx1">兰有莠兮菊有芳。——</span><span class="diczx2">汉武帝</span><span class="smcs">《秋风辞》</span></p><p><span class="cino">(5)</span> 指春天 <span class="encs">[spring]</span>。如:芳林(春日的树林);芳春(春天);芳信(春暖花开的讯息);芳草(春草，比喻忠贞贤德的人);芳岁(春天)</p><p><span class="cino">(6)</span> 指女子 <span class="encs">[woman]</span>。如:芳卿(对女子的爱称);芳尘(指女子的步履);芳魄(美人的魂魄)</p><p><span class="cino">(7)</span> 比喻美名或美德 <span class="encs">[good name]</span></p><p><span class="diczx1">昭示来世，垂芳后昆。——</span><span class="diczx2">汉</span>· <span class="diczx2">蔡邕</span><span class="smcs">《刘镇南碑》</span></p><p><span class="cino">(8)</span> 又如:芳尘(比喻美好的名声或风气);芳风(美好的风尚和教化;亦指美誉)流芳百世</p><p><span class="cino">(9)</span> 喻有贤德的人 <span class="encs">[sage]</span></p><p><span class="diczx1">腥臊并御，芳不得薄兮。——<span class="smcs">《楚辞·屈原·涉江》</span></span></p><p><span class="cino">(10)</span> 又如:芳躅(前贤的遗迹);芳规(前贤的遗规);芳魂(志行高尚者的灵魂);芳迹(前贤的行迹)</p><p><span class="cino">(11)</span> 通<span class="yhcs">“房”</span>。子房 <span class="encs">[ovary]</span></p><p><span class="diczx1">穗钜而芳夺。——<span class="smcs">《吕氏春秋》</span></span></p><p><span class="cino">(12)</span> 通<span class="yhcs">“旁”</span>( <span class="dicpy">páng</span>)。侧，边 <span class="encs">[side]</span></p><p><span class="diczx1">陕之芳七施，七七四十九尺，而至於泉。——<span class="smcs">《管子·地员》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>芳</strong> <span class="dicpy">fāng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fāng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 香，气味美好 <span class="encs">[aromatic;fragrant]</span></p><p><span class="diczx1">芳草发狂。——<span class="smcs">《素问·腹中论》</span></span></p><p><span class="diczx1">和致芳些。——<span class="smcs">《楚辞·大招》</span></span></p><p><span class="diczx1">鼓芳风以扇游尘。——<span class="smcs">《谷梁传序》</span></span></p><p><span class="diczx1">芳草鲜美，落英缤纷。——</span><span class="diczx2">晋</span>· <span
