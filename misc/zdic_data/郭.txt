						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>郭</strong></p><p><span class="dicpy">guō  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/guō.mp3"></a></span> ㄍㄨㄛˉ</span></p><ol><li>城外围着城的墙：城～。“爷娘闻女来，出～相扶将”。</li><li>物体的外框或外壳。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">郭</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>郭</strong> <span class="dicpy">guō</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/guō.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从邑，从<span class="yhcs">“邑”</span>，表示与城郭有关。本义:在城的外围加筑的一道城墙)</p><p><span class="cino">(2)</span> 同本义。内城叫城，外城叫郭 <span class="encs">[the outer wall of a city]</span></p><p><span class="diczx1">郭，外城也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">城外为之郭。——<span class="smcs">《管子·度地》</span></span></p><p><span class="diczx1">爷娘闻女来，出郭相扶将。——<span class="smcs">《乐府诗集·木兰诗》</span></span></p><p><span class="diczx1">三里之城，七里之郭，环而攻之而不胜。——<span class="smcs">《孟子·公孙丑下》</span></span></p><p><span class="diczx1">门闭，因逾郭而入。——<span class="smcs">《韩非子·外储说左下》</span></span></p><p><span class="cino">(3)</span> 又如:郭围(外城边沿);郭术(城墙和道路);郭郛(外城)</p><p><span class="cino">(4)</span> 泛指城市 <span class="encs">[city]</span></p><p><span class="diczx1">东家有贤女，窈窕艳城郭。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="diczx1">青山横北郭，白水绕东城。——</span><span class="diczx2">李白</span><span class="smcs">《送友人》</span></p><p><span class="cino">(5)</span> 又如:郭邑(城邑);郭外(城外)</p><p><span class="cino">(6)</span> 通<span class="yhcs">“廓”</span>。外部，外周 <span class="encs">[outer frame or sheath]</span></p><p><span class="diczx1">又钱无轮郭文章，不便人用。——<span class="smcs">《后汉书·董卓传》</span></span></p><p><span class="cino">(7)</span> 又如:轮郭(轮廓。物体的外周、边框)</p><p><span class="cino">(8)</span> 春秋国名 <span class="encs">[Guo state]</span>。在今山东省北部</p><p><span class="cino">(9)</span> 皮 <span class="encs">[skin]</span></p><p><span class="diczx1">津液充郭。——<span class="smcs">《素问·汤液醪醴论》</span></span></p><p><span class="cino">(10)</span> 又如:津液充郭(津液使外皮充实、润泽)</p><p><span class="cino">(11)</span> 通<span class="yhcs">“椁”</span>( <span class="dicpy">guǒ</span>)。古代棺材外面套的大棺 <span class="encs">[outer coffin]</span></p><p><span class="diczx1">病甚，临卒，将无棺郭。——</span><span class="diczx2">汉</span><span class="smcs">《楚相孙叔敖碑》</span></p><p><span class="cino">(12)</span> 姓。如:郭申(后汉郭太、申屠蟠的并称);郭汾阳(郭子仪平定安史之乱有功，被封为汾阳王，世称郭汾阳)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>郭</strong> <span class="dicpy">guō</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/guō.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 扩张;扩大 <span class="encs">[expand]</span>。如:郭胸(扩胸)</p><p><span class="cino">(2)</span> 通<span class="yhcs">“廓”</span>( <span class="dicpy">kuò</span>)</p><p><span class="cino">(3)</span> 开拓 <span class="encs">[open up]</span></p><p><span class="diczx1">鼓者，郭也。春分之音也。——<span class="smcs">《风俗通义·鼓》</span></span></p><p><span class="cino">(4)</span> 空 <span class="encs">[empety]</span></p><p><span
