						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>姬</strong></p><p><span class="dicpy">jī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span>  ㄐㄧˉ</span></p><ol>    <li>古代对妇女的美称。</li>    <li>中国汉代宫中的女官。</li>    <li>旧时称妾：～人。～妾。～侍。</li>    <li>旧时称以歌舞为业的女子：歌～。</li>    <li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">姬</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>姬</strong> <span class="dicpy">jī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。本义:上古母系社会流传下来的一种姓氏)</p><p><span class="cino">(2)</span> 水名。相传为黄帝所居 <span class="encs">[Ji River]</span></p><p><span class="diczx2">黄帝</span>以 <span class="diczx2">姬水</span>成， <span class="diczx2">炎帝</span>以 <span class="diczx2">姜水</span>成。——<span class="smcs">《国语·晋语》</span></p><p><span class="cino">(3)</span> 黄帝的姓 <span class="encs">[Ji，surname]</span></p><p><span class="diczx2">黄帝</span>君 <span class="diczx2">姬水</span>，以为姓。——<span class="smcs">《说文》</span></p><p><span class="diczx1">昔少典，取于</span><span class="diczx2">有峤</span>氏，生 <span class="diczx2">黄帝</span><span class="diczx2">炎帝</span>， <span class="diczx2">黄帝</span>以 <span class="diczx2">姬水</span>成， <span class="diczx2">炎帝</span>以 <span class="diczx2">姜水</span>成，成而异德，故 <span class="diczx2">黄帝</span>为姬， <span class="diczx2">炎帝</span>为姜。——<span class="smcs">《国语·晋语》</span>。按，轩辕子二十五人，惟 <span class="diczx2">青阳</span>与 <span class="diczx2">仓林氏</span>得姓姬。</p><p><span class="diczx2">尧</span>立 <span class="diczx2">后稷</span>以为大农，姓之曰姬氏。——<span class="smcs">《史记·三代世表》</span>。按，姬姜为古大姓，姬本 <span class="diczx2">黄帝</span>裔，故 <span class="diczx2">尧</span>赐姓姬也。</p><p><span class="cino">(4)</span> 周人以后稷(黄帝之后)为祖，亦姓姬。又如:姬国(姬姓之国。指周王朝);姬化(周文王姬昌的德化);姬公(周公姬旦;周文王姬昌);姬文(周文王)</p><p><span class="cino">(5)</span> 古代称帝王的妾 <span class="encs">[concubine]</span></p><p><span class="diczx1">毋曰薄姬。——<span class="smcs">《汉书·文帝纪》</span>。注:<span class="yhcs">“内宫也。”</span></span></p><p><span class="diczx2">王</span>姬、 <span class="diczx2">徐</span>姬、 <span class="diczx2">蔡</span>姬。——<span class="smcs">《史记·齐世家》</span>。索隐:<span class="yhcs">“众妾之总称。此名之不正者。”</span></p><p><span class="diczx1">宠姬</span><span class="diczx2">郑袖</span>。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="cino">(6)</span> 又如:侍姬;姬人(妾);姬侍(侍妾);姬妾(妾);姬媵(妾);姬仆(家妓;婢妾)</p><p><span class="cino">(7)</span> 古代中国对妇人的美称 <span class="encs">[lady]</span></p><p><span class="diczx1">妖姬静女。——</span><span class="diczx2">清</span>· <span class="diczx2">侯方域</span><span class="smcs">《壮悔堂文集》</span></p><p><span class="cino">(8)</span> 又如:艳姬;仙姬</p><p><span class="cino">(9)</span> 歌女 <span class="encs">[professional female singer]</span>。如:歌姬;一代名姬</p><p><span class="cino">(10)</span> 美女的代称 <span class="encs">[beauty]</span></p><p><span class="diczx1">于是</span><span class="diczx2">庄王</span>弃其 <span class="diczx2">秦</span>姬 <span class="diczx2">越</span>女，罢钟鼓之乐。——<span class="smcs">《吴越春秋》</span></p><p><span class="cino">(11)</span> 周朝的代称。周人为姬姓，故以之相代 <span class="encs">[Zhou dynasty]</span></p><p><span class="diczx2">姬汉</span>旧邦。—— <span class="diczx2">南朝</span><span class="diczx2">梁</span>· <span class="diczx2">丘迟</span><span class="smcs">《与陈伯之书》</span></p><p><span class="cino">(12)</span> 又如:姬周(周朝);姬汉(周朝和汉朝);姬刘(周朝和汉朝);姬姜(春秋时，周王室姓姬，齐国姓姜，二姓常通婚姻，因以<span class="yhcs">“姬姜”</span>为贵族妇女之称。泛指美女)</p><p><span class="cino">(13)</span> 姓</p>                    <div class="div copyright"> © 汉典 </div>
