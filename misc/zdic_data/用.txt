                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>用</strong></p><p><span class="dicpy">yòng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yòng.mp3"></a></span>  ㄩㄥˋ</span></p><ol><li>使人或物发挥其功能：使～。～心。～兵。～武。</li><li>可供使用的：～品。～具。</li><li>进饭食的婉辞：～饭。</li><li>花费的钱财：费～。～项。～资。</li><li>物质使用的效果：功～。有～之才。</li><li>需要（多为否定）：不～多说。</li><li>因此：～此。</li></ol>

                    <h2 class="h2_entry"><span class="orth">用</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>用</strong> <span class="dicpy">yòng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yòng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (象形。甲骨文字形，象桶形。桶可用，故引申为用。本义:使用，采用)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[use;employ]</span></p><p><span class="diczx1">用，可施行也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">用，以也。——<span class="smcs">《苍颉篇》</span></span></p><p><span class="diczx1">利用为大作。——<span class="smcs">《易·益》</span></span></p><p><span class="diczx1">五行五用哉。——<span class="smcs">《虞书》</span></span></p><p><span class="diczx1">乃攘窃神胝之牺牷牺用。——<span class="smcs">《书·微子》</span></span></p><p><span class="diczx1">乘其财用出入。——<span class="smcs">《周礼·宰夫》</span></span></p><p><span class="diczx1">乃会万民之卒伍而用之。——<span class="smcs">《周礼·小司徒》</span></span></p><p><span class="diczx1">保甲之法，起于</span><span class="diczx2">三代</span>丘甲。 <span class="diczx2">管仲</span>用之 <span class="diczx2">齐</span>， <span class="diczx2">子产</span>用之 <span class="diczx2">郑</span>， <span class="diczx2">商君</span>用之 <span class="diczx2">秦</span>， <span class="diczx2">仲长统</span>言之 <span class="diczx2">汉</span>，而非今日之立异也。—— <span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《上五事札子》</span></p><p><span class="diczx1">皆不足用。——</span><span class="diczx2">明</span>· <span class="diczx2">魏禧</span><span class="smcs">《大铁椎传》</span></p><p><span class="diczx1">用水彩。——</span><span class="diczx2">蔡元培</span><span class="smcs">《图画》</span></p><p><span class="diczx1">兼用激刺。</span></p><p><span class="diczx1">又用篆章。——</span><span class="diczx2">明</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="cino">(3)</span> 又如:用不着(没办法);备用(准备着供随时使用);实用(实际使用);拆用(拆开使用);用功(使用功力);用长(使用长武器);用板(使用诏书);用计(使用计谋);用天因地(利用天时，顺应地利);用钱(使用钱币);用药(使用药物)</p><p><span class="cino">(4)</span> 任用(委派人员担任职务) <span class="encs">[appoint]</span></p><p><span class="diczx1">不用尚书郎。——<span class="smcs">《乐府诗集·木兰诗》</span></span></p><p><span class="diczx1">用于昔日。——</span><span class="diczx2">诸葛亮</span><span
