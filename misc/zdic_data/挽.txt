						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>挽</strong></p><p><span class="dicpy">wǎn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǎn.mp3"></a></span>  ㄨㄢˇ</span></p><ol><li>拉，牵引：～弓。～留。</li><li>设法使局势好转或恢复原状：～救。力～狂澜。</li><li>追悼死人：～词。～联。～幛。</li><li>古同“<span class="diczx3">晚</span>”，后来的。</li><li>同“<span class="diczx3">绾</span>”。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>挽</strong></p><p>輓 <span class="dicpy">wǎn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǎn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，免声。古字从<span class="yhcs">“车”</span>，免声。本义:牵引;拉)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[draw;pull]</span></p><p><span class="diczx1">挽，引也。——<span class="smcs">《小尔雅·广诂》</span></span></p><p><span class="diczx1">輓，引之也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">輓，輓车也。——<span class="smcs">《广韵》</span></span></p><p><span class="diczx1">或輓之，或推之。——<span class="smcs">《左传·襄公十四年》</span>。注:<span class="yhcs">“前牵曰輓。”</span></span></p><p><span class="diczx1">娄敬脱輓辂。——<span class="smcs">《史记·刘敬传》</span></span></p><p><span class="diczx1">转粟輓输以为之备。——<span class="smcs">《汉书·韩安国传》</span></span></p><p><span class="diczx1">挽炮车者。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="cino">(3)</span> 又如:挽郎(出葬时，牵绋唱挽歌的少年);挽强(拉强弓);挽满(拉满弓);挽车(拉车);挽住(憋住);挽手儿(缰绳);挽歌郎(出殡时替丧家在灵前唱挽歌的人);挽强为弱(使强变弱)</p><p><span class="cino">(4)</span> 悼念死者 <span class="encs">[lament sb.'s death]</span></p><p><span class="diczx1">现今</span><span class="diczx2">胡三公子</span>替 <span class="diczx2">湖州</span><span class="diczx2">鲁老先生</span>征挽诗，送了十几个斗方在我那里。——<span class="smcs">《儒林外史》</span></p><p><span class="cino">(5)</span> 又如:挽章(挽词。哀悼死者的词章);挽诗(哀悼死者的诗);挽歌(哀悼死者的丧歌);挽联(哀悼死者的对联);挽辞(哀悼死者的言辞)</p><p><strong>词性变化</strong></p><p>◎ <strong>挽</strong> <span class="dicpy">wǎn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǎn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“绾”</span>。卷起 <span class="encs">[roll up]</span>。如:挽裤腿儿</p><p><span class="cino">(2)</span> 同<span class="yhcs">“绾”</span>。编结 <span class="encs">[coil up]</span>。如:挽手(鞭子);挽角(童年)</p><p><span class="cino">(3)</span> 扭转;挽回 <span class="encs">[reverse]</span></p><p><span class="diczx1">火攻破敌，挽强为弱。——<span class="smcs">《三侠五义》</span></span></p><p><span class="cino">(4)</span> 又如:挽狂澜(扭转局面或风气);挽正(纠正);挽转(挽回);挽通(勾结;买通)</p><p><span class="cino">(5)</span> 荐引 <span class="encs">[recommend]</span>。如:挽推(引荐;扶持)</p><p><span class="cino">(6)</span> 勾;挎 <span class="encs">[carry on the arm]</span>。如:不由分说，拉着文君就走，挽其腰;手挽小篮</p><p><span class="cino">(7)</span> 请，托 <span class="encs">[please]</span></p><p><span class="diczx1">怎奈</span><span class="diczx2">蔡攸</span>处挽心腹催促公人起身。——<span class="smcs">《水浒全传》</span></p><p>◎ <strong>挽</strong> <span class="dicpy">wǎn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
