                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>益</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。小篆字形。象器皿中有水漫出。从皿，从水。<span class="yhcs">“水”</span>已隶变。本义:<span class="yhcs">“溢”</span>的本字。水漫出)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[overflow]</span></p><p><span class="diczx1">益，饶也。——<span class="smcs">《说文》</span>。按，从水，浮于四，会意。字亦作溢。</span></p><p><span class="cino">(3)</span> 引申为水涨</p><p><span class="diczx1">澭水暴益。——<span class="smcs">《吕氏春秋》</span></span></p><p><span class="cino">(4)</span> 在某些方面增加(如体积、数量、程度等) <span class="encs">[increase]</span></p><p><span class="diczx1">损益，盛衰之始也。——<span class="smcs">《易·杂卦传》</span></span></p><p><span class="diczx1">人或益之，人或损之，胡可得而法?——<span class="smcs">《吕氏春秋·察今》</span></span></p><p><span class="diczx1">有所广益。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p><span class="diczx1">斟酌损益。</span></p><p><span class="diczx1">大有所益。——<span class="smcs">《资治通鉴·汉纪》</span></span></p><p><span class="cino">(5)</span> 又如:增益;延年益寿;益甲(增兵);益兵(增加兵力);益封(增加封邑);益军(益兵);益损(增减;兴革);益算(增加岁数);益赋(增加赋税)</p><p><span class="cino">(6)</span> 助;补助 <span class="encs">[help]</span></p><p><span class="diczx1">与我齐者，吾不与处，无益我者也。——<span class="smcs">《吕氏春秋·贵当》</span></span></p><p><span class="cino">(7)</span> 扩大，加大 <span class="encs">[expand]</span></p><p><span class="diczx1">民众城小，则益城。——<span class="smcs">《谷梁传》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>益</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 有益， 有利 <span class="encs">[beneficial]</span>。如:益鸟</p><p><span class="cino">(2)</span> 富裕 <span class="encs">[prosperous]</span></p><p><span class="diczx1">其家必日益。——<span class="smcs">《吕氏春秋·贵当》</span></span></p><p><span class="cino">(3)</span> 骄傲自满 <span class="encs">[conceited and self-satisfied]</span></p><p><span class="diczx1">有貌愿而益， 有长若不肖。——<span class="smcs">《庄子》</span></span></p><p>◎ <strong>益</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 更加 <span class="encs">[all the more]</span></p><p><span class="diczx1">君之疾在肌肤，不治将益深。——<span class="smcs">《韩非子·喻老》</span></span></p><p><span class="diczx1">如水益深，如火益热。——<span class="smcs">《孟子》</span></span></p><p><span class="diczx1">益狎。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《三戒》</span></p><p><span class="diczx1">香远益清。——</span><span class="diczx2">宋</span>· <span
