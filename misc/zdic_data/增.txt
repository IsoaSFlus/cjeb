                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>增</strong> <span class="dicpy">zēng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zēng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从土，曾声。字本作<span class="yhcs">“曾”</span>。本义:增多)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[increase;gain;add]</span></p><p><span class="diczx1">增，益也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">增，加也。——<span class="smcs">《广雅·释诂一》</span></span></p><p><span class="diczx1">鹰鸢犹以山为卑而增巢其上。——<span class="smcs">《荀子·法行》</span></span></p><p><span class="diczx1">残高增下。——<span class="smcs">《淮南子·本经》</span></span></p><p><span class="diczx1">山不加增。——<span class="smcs">《列子·汤问》</span></span></p><p><span class="diczx1">增脚气病。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="diczx1">止增笑耳。——<span class="smcs">《聊斋志异·狼三则》</span></span></p><p><span class="diczx1">乃增损。——</span><span class="diczx2">蔡元培</span><span class="smcs">《图画》</span></p><p><span class="diczx1">旧时栏楯，亦遂增胜。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="diczx1">言其户口，则视三十年以前增五倍焉。——</span><span class="diczx2">清</span>· <span class="diczx2">洪亮吉</span><span class="smcs">《治平篇》</span></p><p><span class="cino">(3)</span> 又如:增年(加寿;年龄增加);增增(众多的样子);增奉(增加俸禄);增拓(增加，扩充);增息(谓增多人丁);增累(增加，累积);增损(增加或减少);增广(增加，扩大);增置(增添，添设);增防(增强防守)</p><p><span class="cino">(4)</span> 通<span class="yhcs">“憎”</span>。厌恶 <span class="encs">[detest]</span></p><p><span class="diczx1">帝式是增。——<span class="smcs">《墨子·非命下》</span>。</span><span class="diczx2">毕沅</span>云:<span class="yhcs">“增、憎字通。”</span></p><p><span class="diczx1">不惧</span><span class="diczx2">季氏</span>增邑不隐讳之害，独畏答 <span class="diczx2">懿子</span>极言之罪，何哉?——<span class="smcs">《论衡》</span></p><p><span class="cino">(5)</span> 扩大 <span class="encs">[extend]</span></p><p><span class="diczx1">增其旧制。——</span><span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="cino">(6)</span> 夸大，夸张 <span class="encs">[exaggerate]</span></p><p><span class="diczx1">患言事增其实。——</span><span class="diczx2">王充</span><span class="smcs">《论衡》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>增</strong> <span class="dicpy">Zēng</span></p><p>〈名〉</p><p><span class="cino">(1)</span> 山名 <span class="encs">[Zeng mountain]</span>。在今内蒙古自治区与陕西省交界处</p><p><span class="diczx2">司马彪</span>曰: <span class="diczx2">增山</span>者， <span class="diczx2">上郡</span>之别名也。——<span class="smcs">《水经注·河水三》</span></p><p><span class="cino">(2)</span> 姓</p><p>◎ <strong>增</strong> <span class="dicpy">zēng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zēng.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 高 <span class="encs">[high]</span></p><p><span class="diczx1">夫鸢鹊遭害，则仁鸟增逝——</span><span class="diczx2">班固</span><span class="smcs">《汉书》</span></p><p><span class="cino">(2)</span> 更加 <span
