                    <h2 class="h2_entry"><span class="orth">抗</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>抗</strong> <span class="dicpy">kàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kàng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，亢声。本义:抵抗，抵御)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[resist;combat;fight]</span></p><p><span class="diczx1">抗，扦也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">抗，御也。——<span class="smcs">《小尔雅·广言》</span></span></p><p><span class="diczx1">抗木横一缩。——<span class="smcs">《仪礼·既夕礼》</span></span></p><p><span class="diczx1">非抗于九国之师也。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《过秦论》</span></p><p><span class="cino">(3)</span> 又如:抗木(旧时所用的葬具。在棺木之上覆盖一席，用以防御尘土);抗力(抵抗力);抗犯(抵抗与进犯);抗玩(玩忽抗命);抗质(谓劫持人质以相抗);抗矫(矫情抗俗，高蹈)</p><p><span class="cino">(4)</span> 抗拒;拒绝 <span class="encs">[defy;refuse]</span></p><p><span class="diczx1">安能抗此难乎。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="diczx1">抗辞慷慨。——</span><span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《指南录后序》</span></p><p><span class="diczx1">欲出力助上以抗之。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《答司马谏议书》</span></p><p><span class="cino">(5)</span> 又如:抗拒(顶撞;违抗并拒绝);抗违(抗拒，违背);抗租(抗缴地租);抗逆(违抗，抗拒);抗斗(反抗斗争)</p><p><span class="cino">(6)</span> 匹敌，抗衡 <span class="encs">[contend with]</span></p><p><span class="diczx1">独莲花与抗耳。——<span class="smcs">《徐霞客游记·游黄山记》</span></span></p><p><span class="cino">(7)</span> 又如:抗横(抗衡);抗行(并行;抗衡);抗美(抗衡媲美);抗库(并立，抗衡)</p><p><span class="cino">(8)</span> 举起 <span class="encs">[raise]</span></p><p><span class="diczx1">抗皓手而清歌。——</span><span class="diczx2">曹植</span><span class="smcs">《七启》</span></p><p><span class="cino">(9)</span> 又如: 抗力(力能举起);抗手(举手为礼);抗足(投足，举足);抗兵(举兵)</p><p><span class="cino">(10)</span> 呈上 <span class="encs">[present]</span>。如:抗表(向皇帝上奏章);抗章(向皇帝上书直言)</p><p><strong>词性变化</strong></p><p>◎ <strong>抗</strong> <span class="dicpy">kàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kàng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 高;大。通<span class="yhcs">“亢”</span> <span class="encs">[high; great]</span></p><p><span class="diczx1">谪戍之众，非抗于几国之师也。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《过秦论》</span></p><p><span class="diczx1">以犯上为亢。——<span class="smcs">《韩非子·问辩》</span></span></p><p><span class="cino">(2)</span> 又如:抗世(高世，超乎世俗);抗殿(谓高筑殿堂或指高大的殿堂)</p><p><span class="cino">(3)</span> 刚正不阿;高尚 <span class="encs">[upright and never stooping to flattery;noble]</span></p><p><span class="diczx1">若贤人之美辞，忠臣之抗直。——</span><span class="diczx2">萧统</span><span class="smcs">《文选序》</span></p><p><span class="cino">(4)</span>
