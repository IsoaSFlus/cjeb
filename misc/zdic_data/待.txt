                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>待</strong> <span class="dicpy">dāi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dāi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> <span class="encs">[口]</span></p><p><span class="cino">(2)</span> 暂时停留 <span class="encs">[stay]</span>。如:在北京只待了一天</p><p><span class="cino">(3)</span> 无目的地消磨时间或在某处逗留 <span class="encs">[hang around]</span>。如:我在家里待一整天</p><p><span class="cino">(4)</span> 另见 <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>待</strong> <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从彳( <span class="dicpy">chì</span>)，寺声。本义:等待，等候)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[wait for]</span></p><p><span class="diczx1">待，逗也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">有待而行也。——<span class="smcs">《易·归妹》</span></span></p><p><span class="diczx1">窃待于下风。——<span class="smcs">《庄子·渔父》</span></span></p><p><span class="diczx1">多行不义，必自毙，子姑待之。——<span class="smcs">《左传·隐公元年》</span></span></p><p><span class="diczx1">吾入关，秋毫不敢有所近，籍吏民，封府库，而待将军。——<span class="smcs">《史记·项羽本纪》</span></span></p><p><span class="diczx1">与其坐而待之，孰若起而拯之。——</span><span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p><span class="diczx1">待君久不至，已去。——<span class="smcs">《世说新语·方正》</span></span></p><p><span class="cino">(3)</span> 又如:待年(等待长大。指女子待嫁);待贤(等候贤人);待诏(等待任职的诏令);待时(等待时机)</p><p><span class="cino">(4)</span> 对待 <span class="encs">[treat;deal with]</span></p><p><span class="diczx1">此贤君子也，君厚待之。——<span class="smcs">《韩非子·喻老》</span></span></p><p><span class="diczx1">古之君子，其责己也重以周，其待人也轻以约。——</span><span class="diczx2">韩愈</span><span class="smcs">《原毁》</span></p><p><span class="diczx1">士别三日，即更刮目相待。——<span class="smcs">《资治通鉴·汉纪》</span></span></p><p><span class="diczx1">荣贼并非推心待慰帅者。——</span><span
