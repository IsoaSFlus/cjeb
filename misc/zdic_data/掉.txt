                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>掉</strong></p><p><span class="dicpy">diào  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/diào.mp3"></a></span>  ㄉㄧㄠˋ</span></p><ol><li>落下：～泪。</li><li>减损，消失：～色。～价儿。</li><li>遗失，遗漏：这一行～了两个字。</li><li>回转（<span class="dicpy">zhuǎn </span>）：～头。～转。</li><li>摇摆，引申为卖弄，耍：～文。～俏。～以轻心。～臂而去。</li><li>对换：～包。～换。</li><li>落在后面：～队。</li><li>用在动词后表示动作完成：改～。戒～。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> turn, move; shake, wag; drop down</p><p><span class="z_ts2">德语</span> fallen, tropfen  (V)</p><p><span class="z_ts2">法语</span> tomber,perdre,tourner,se débarrasser de</p></div>

                    <h2 class="h2_entry"><span class="orth">掉</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>掉</strong> <span class="dicpy">diào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/diào.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，卓声。本义:摇，摆动)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[sway;wag;shake]</span></p><p><span class="diczx1">掉，摇也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">且</span><span class="diczx2">郦生</span>一士，伏轼掉三寸之舌，下 <span class="diczx2">齐</span>七十余城。——<span class="smcs">《史记·淮阴侯列传》</span></p><p><span class="diczx1">末大未折，尾大不掉。——<span class="smcs">《左传·昭公十一年》</span></span></p><p><span class="cino">(3)</span> 又如:掉尾巴;掉荡(摇荡);掉眩(肢体摇动旋转);掉擘(甩着臂膊走，又谓攘臂奋起);掉舌(鼓舌，指游说、谈论);掉嘴弄舌(摇唇鼓舌;搬弄是非);掉栗(战栗)</p><p><span class="cino">(4)</span> 弄，卖弄 <span class="encs">[show off]</span></p><p><span class="diczx1">言必据书史，断章破句，以代常谈，俗谓之掉书袋。——<span class="smcs">《南唐书·彭利用传》</span></span></p><p><span class="cino">(5)</span> 又如:掉罨子(耍花招，卖弄花招)</p><p><span class="cino">(6)</span> 玩弄;搬弄 <span class="encs">[play with;juggle with;dally with]</span>。如:掉鬼(捣鬼;玩弄骗人的把戏);掉弄机权(玩弄权力);掉嘴口(耍贫嘴);掉经儿(瞎吹)</p><p><span class="cino">(7)</span> 落下 <span class="encs">[fall;shed;drop]</span>。如:掉眼泪;熟的苹果从树上掉下来了</p><p><span class="cino">(8)</span> 丢失 <span class="encs">[lose]</span>。如:别把钥匙掉了</p><p><span class="cino">(9)</span> 转回 <span class="encs">[turn]</span>。如:把车头掉过来;掉首;掉背脸(转过脸)</p><p><span class="cino">(10)</span> 调换 <span class="encs">[exchange]</span>。如:掉包儿(暗中偷换);掉忙功夫(方言。忙里偷闲;抽空);掉揽(设法弄走别人的东西);我要大瓶掉小瓶</p><p><span class="cino">(11)</span> 划 <span class="encs">[paddle;row]</span></p><p><span class="diczx1">见前面两个人，掉着一只舡来。——<span class="smcs">《水浒传》</span></span></p><p><span class="cino">(12)</span> 鼓动 <span class="encs">[instigate;abet]</span>。如:掉舌(鼓动其舌，逞口舌之能。指雄辩或游说他人)</p><p><span class="cino">(13)</span> 落在后面 <span class="encs">[drop;fall]</span>。如:掉队</p><p><span class="cino">(14)</span> 减少，降低 <span class="encs">[decrease;lower]</span>。如:体重掉了五斤;掉价儿</p><p><strong>词性变化</strong></p><p>◎ <strong>掉</strong> <span class="dicpy">diào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/diào.mp3"></a></span> </p><p>〈助〉</p><p>用在某些动词后，表示动作的结果。如:丢掉;改掉;打掉</p><p><strong>常用词组</strong></p><hr
