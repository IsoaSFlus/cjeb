                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>像</strong></p><p><span class="dicpy">xiàng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiàng.mp3"></a></span> ㄒㄧㄤˋ</span></p><ol><li>相似：好～。相（<span class="dicpy">xiāng </span>）～。～生（ａ．仿天然产物制成的工艺品；ｂ．中国宋、元两代以说唱为业的女艺人）。</li><li>比照人物做成的图形：画～。塑～。雕～。绣～。肖～。</li><li>比如，比方：～这样的事是值得注意的。</li></ol>

                    <h2 class="h2_entry"><span class="orth">像</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>像</strong> <span class="dicpy">xiàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiàng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从人，从象，象亦声。本义:相貌相似)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[resemble;be like;take after;similar;alike]</span></p><p><span class="diczx1">像，似也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">段</span>注:<span class="yhcs">“然 <span class="diczx2">韩非</span>之前只有象字，无像字。 <span class="diczx2">韩非</span>以后小篆即作像。 <span class="diczx2">许</span>断不以象释似，复以象释像矣。系辞曰，爻也者，效此者也。象也者，像此者也。又曰，象也者，像也。…盖象为古文，圣人以像释之。虽他本像亦作象。然 <span class="diczx2">郑康成</span>、 <span class="diczx2">王辅本</span>非不可信也。凡形像、图像、想像字皆当从人，而学者多作象，象行而像废矣。”</span></p><p><span class="diczx1">影之像形也。——<span class="smcs">《荀子·富国》</span></span></p><p><span class="diczx1">女必像汝，我心甚慰。——</span><span class="diczx2">林觉民</span><span class="smcs">《与妻书》</span></p><p><span class="diczx1">以情乃一人之情，说</span><span class="diczx2">张三</span>要像 <span class="diczx2">张三</span>，难通融于 <span class="diczx2">李四</span>。—— <span class="diczx2">清</span>· <span class="diczx2">李渔</span><span class="smcs">《闲情偶寄》</span></p><p><span class="diczx1">脸上处处像他哥哥，可是那种神气又不完全像他的哥哥。——</span><span class="diczx2">老舍</span><span class="smcs">《黑白李》</span></p><p><span class="cino">(3)</span> 又如:像形(与某事物形状相似);像似(好像);像形奇名(比拟其形状而讹易本名);像煞(活像，好像);像姑(男妓)</p><p><span class="cino">(4)</span> 模拟 <span class="encs">[imitate]</span></p><p><span class="diczx1">于是民人被发文身以像鳞虫。——<span class="smcs">《淮南子·原道训》</span></span></p><p><span class="cino">(5)</span> 仿效 <span class="encs">[follow the example of]</span></p><p><span class="diczx1">中立君树，前植群树，以像朝班。——<span class="smcs">《辽史》</span></span></p><p><span class="diczx1">行比伯夷，置以为像兮。——<span class="smcs">《楚辞·九章》</span></span></p><p><span class="cino">(6)</span> 依随，顺遂 <span class="encs">[comply with]</span></p><p><span class="diczx1">像了他意，再无妒忌。——<span class="smcs">《古今小说》</span></span></p><p><span class="cino">(7)</span> 又如:像心如意(顺心如意);像心称意(称心满意);像心适意(随心所欲);像心像意(顺遂心愿)</p><p><span class="cino">(8)</span> 立像，或比照人物形象绘画或雕塑图像 <span class="encs">[sculpture a statue or draw a portrait]</span></p><p><span class="diczx1">邑先大夫皆像于两傍。——</span><span class="diczx2">宋</span>· <span class="diczx2">叶适</span>文</p><p><span class="diczx1">遂合塑工像之。——<span class="smcs">《茶香室续钞》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>像</strong> <span class="dicpy">xiàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiàng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 形象;容貌 <span class="encs">[appearance;looks]</span>。如:像生儿(模样);像表(指人的容貌、恣态、风度);像赞(为人物画像或人的相貌所作的赞辞);像貌(长相，容貌姿态);像胎(像态。模样儿)</p><p><span class="cino">(2)</span> 雕像 <span class="encs">[statue]</span></p><p><span
