                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>绛</strong></p><p>絳 <span class="dicpy">jiàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiàng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从糸( <span class="dicpy">mì</span>)。本义:大红色) 同本义 <span class="encs">[deep red]</span></p><p><span class="diczx1">绛，大赤也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">纁谓之绛。凡九旗之帛皆用绛。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">绛树在其南。——<span class="smcs">《淮南子·地形》</span></span></p><p><span class="diczx1">纶组紫绛。——</span><span class="diczx2">左思</span><span class="smcs">《吴都赋》</span></p><p><span class="diczx1">绛皓驳色。——</span><span class="diczx2">清</span>· <span class="diczx2">姚鼐</span><span class="smcs">《登泰山记》</span></p><p><span class="cino">(2)</span> 又如:绛帐(绛帏。红色帷帐;对师长之尊称);绛脂(大红胭脂);绛纱(红色的纱帐);绛帻(大红色头巾。代指宫中侍卫人员);绛节(红色符节)</p><p><strong>词性变化</strong></p><p>◎ <strong>绛</strong></p><p>絳 <span class="dicpy">jiàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiàng.mp3"></a></span> </p><p>〈动〉</p><p>染为绛色 <span class="encs">[redden]</span>。如:绛天(把天空染为绛色)</p><p>◎ <strong>绛</strong></p><p>絳 <span class="dicpy">Jiàng</span></p><p>〈名〉</p><p><span class="cino">(1)</span> 古都名。春秋晋地 <span class="encs">[Jiang capital]</span></p><p><span class="diczx1">绛人也。——<span class="smcs">《国语·晋语》</span>。注:<span class="yhcs">“</span><span class="diczx2">晋国</span>都，在今 <span class="diczx2">山西</span><span class="diczx2">绛县</span>。”</span></p><p><span class="cino">(2)</span> 又如:绛人(绛县的老人);绛侯(指汉高祖刘邦的大臣周勃，他以功封绛侯，在铲除诸吕中起了重大作用);绛老(本指春秋晋国绛县的老人，后泛指高寿的老年人)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/绛紫">绛紫</a></span></div>                    <div class="div copyright"> © 汉典 </div>
