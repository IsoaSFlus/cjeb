                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>乍</strong> <span class="dicpy">zhà</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhà.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> (会意。小篆字形，从亡从一。本义:忽然)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[suddenly;abruptly]</span></p><p><span class="diczx1">乍，暂也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">今人乍见孺子。——<span class="smcs">《孟子》</span></span></p><p><span <span class="diczx2">杜牧</span><span class="smcs">《阿房宫赋》</span></p><p><span class="cino">(3)</span> 又如:离乍(分离很骤然);乍乍(忽然);江上渔火乍明乍灭;乍冷乍热，天气很不正常</p><p><span class="cino">(4)</span> 刚;才 <span class="encs">[first;at the beginning]</span></p><p><span class="diczx1">波色乍明。——</span><span class="diczx2">明</span>· <span class="diczx2">袁宏道</span><span class="smcs">《满井游记》</span></p><p><span class="diczx1">乍出于匣。</span></p><p><span class="cino">(5)</span> 又如:春寒乍暖，又下起了毛毛细雨;大雨乍停，球赛又继续进行</p><p><span class="cino">(6)</span> 初;开始 <span class="encs">[for the first time]</span></p><p><span class="diczx1">韶光明媚，乍晴转暖清明后。——</span><span class="diczx2">柳永</span><span class="smcs">《笛家弄词》</span></p><p><span class="cino">(7)</span> 又如:初来乍到，情况又不大了解，怎能乱发议论呢!老战友分别多年，乍一见面内心多高兴!</p><p><span class="cino">(8)</span> 正;恰 <span class="encs">[just]</span></p><p><span class="diczx1">归去潒沉醉，小院新池月乍寒。——</span><span class="diczx2">后周</span>· <span class="diczx2">和凝</span><span class="smcs">《抛球乐》</span></p><p><span class="diczx1">凉透小帘栊，乍夜长迟睡。——</span><span class="diczx2">张翥</span><span class="smcs">《真珠帘》</span></p><p><span class="cino">(9)</span> <span class="encs">[代]</span>∶表示疑问语气，相当于<span class="yhcs">“怎”</span>、<span class="yhcs">“哪”</span> <span class="encs">[what]</span></p><p><span class="diczx1">乍想到了此处，遭逢魔障，又被他遣山压了。——<span class="smcs">《西游记》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>乍</strong> <span class="dicpy">zhà</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhà.mp3"></a></span> </p><p><连></p><p><span class="cino">(1)</span> 或者 <span class="encs">[or]</span></p><p><span class="diczx1">或卷或舒，乍轻乍重。——</span><span class="diczx2">秦</span>· <span class="diczx2">李斯</span><span class="smcs">《用笔法》</span></p><p><span class="cino">(2)</span> 宁可 <span class="encs">[would rather]</span></p><p><span class="diczx1">乍向草中耿介死，不求黄金笼下生。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span>诗</p><p>◎ <strong>乍</strong> <span class="dicpy">zhà</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhà.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 伸开;伸展 <span class="encs">[spread;extend]</span>。如:乍翅</p><p><span class="cino">(2)</span> 因害怕而颤动 <span class="encs">[shake]</span></p><p><span class="diczx1">不由我意张狂，心惊乍。——佚名<span
