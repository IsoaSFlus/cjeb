						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>沃</strong></p><p><span class="dicpy">wò  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wò.mp3"></a></span>  ㄨㄛˋ</span></p><ol><li>土地肥：～土。～饶（土地肥沃，物产丰富）。～腴。肥～。～野。～壤。</li><li>灌溉，浇：～田。～盥。如汤～雪。</li><li>光盛，丰美：～～。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">沃</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>沃</strong> <span class="dicpy">wò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wò.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水，夭( <span class="dicpy">yāo</span>)声。本义:把水从上浇下)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[water]</span></p><p><span class="diczx1">使火燃，以水沃之，可谓水贼火。——<span class="smcs">《论衡·偶然》</span></span></p><p><span class="cino">(3)</span> 又如:沃盥(浇水洗手);沃酹(以酒浇地而祭奠);沃漏(以沸水浇灌漏壶);沃醊(以酒浇地而祭祀);沃雪(以热水浇雪。比喻事情极易解决);沃馈(以酒食祭奠);沃洒(浇洒)</p><p><span class="cino">(4)</span> 大水量的灌溉 <span class="encs">[irrigate]</span></p><p><span class="diczx1">沃，溉灌也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">决</span><span class="diczx2">白马</span>之口以沃 <span class="diczx2">魏氏</span>。——<span class="smcs">《韩非子·初见秦》</span></p><p><span class="cino">(5)</span> 又如:沃濯(浇灌);沃灌(浇灌);沃流(可供灌溉的水流)</p><p><span class="cino">(6)</span> 启沃，竭诚忠告 <span class="encs">[enlighten]</span>。如:沃心(使内心受启发);沃言(沃心之言);沃然(受启发而领悟貌)</p><p><span class="cino">(7)</span> 荡涤;洗濯 <span class="encs">[wash]</span>。如:沃日(冲荡日头。形容浪大);沃洗(洗涤);沃荡(涤灌)</p><p><span class="cino">(8)</span> 饮，喝 <span class="encs">[drink]</span></p><p><span class="diczx2">载</span>( <span class="diczx2">扈载</span>)连沃六七巨觥，吐呕淋漓。—— <span class="diczx2">宋</span>· <span class="diczx2">陶谷</span><span class="smcs">《清异录》</span></p><p><span class="cino">(9)</span> 淹 <span class="encs">[flood]</span></p><p><span class="diczx1">勃然发怒决洪流，立拟沃杀九日沃。——</span><span class="diczx2">唐</span>· <span class="diczx2">卢仝</span><span class="smcs">《月蚀诗》</span></p><p><span class="cino">(10)</span> 浸泡，使没于水中 <span class="encs">[soak]</span></p><p><span class="diczx1">那个抓来就将水里沃。——<span class="smcs">《西游记》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>沃</strong> <span class="dicpy">wò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wò.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 肥美 <span class="encs">[fertile]</span></p><p><span class="diczx2">关中</span>为沃野，无凶年。——<span class="smcs">《史记·河渠书》</span></p><p><span class="diczx1">以盘石为沃田，以桀暴为良民。——<span class="smcs">《论衡·宣汉》</span></span></p><p><span class="diczx1">沃野千里。——<span class="smcs">《三国志·诸葛亮传》</span></span></p><p><span class="diczx1">膏之沃者。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《朱文公校昌黎先生集》</span></p><p><span class="diczx1">沃野万里。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(2)</span> 又如:沃地(肥沃的土地);沃田;沃淫(肥沃润泽的土地);沃润(肥沃润泽);沃饶(土地肥沃);沃畴(肥沃的田畴);沃瘠(土地的肥瘦);沃实(土地肥沃，物产丰盛);沃腴(肥沃);沃美(肥美)</p><p><span class="cino">(3)</span> 低 <span class="encs">[low]</span></p><p><span
