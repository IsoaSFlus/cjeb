                          <h2 class="h2_entry"><span class="orth">释</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>释</strong></p><p><strong>（<span class="diczx3">釋</span>）</strong></p><p><span class="dicpy">shì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shì.mp3"></a></span>  ㄕˋ</span></p><ol><li>解说，说明：解～。注～。～文。～义。</li><li>消除，消散：～疑。～怨。涣然冰～（像冰融化了一样，嫌隙和疑虑都完全消除）。</li><li>放开，放下：～放。保～。手不～卷。</li><li>佛教创始人释迦牟尼的简称，后泛指佛教：～氏。～教。～子（和尚）。～藏（<span class="dicpy">zàng </span>）（佛教经典的总汇，分经、律、论三藏，包括汉译佛经和中国的一些佛教著述）。～典。</li></ol>

                    <h2 class="h2_entry"><span class="orth">释</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>释</strong></p><p>釋 <span class="dicpy">shì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从采( <span class="dicpy">biàn</span>)，睪( <span class="dicpy">yì</span>)声。采，有辨别、分析义。本义:脱掉，解下)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[take off; unfasten]</span></p><p><span class="diczx1">释，解也。从采，取其分别物也。睪声。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">释获。——<span class="smcs">《仪礼·大射仪》</span></span></p><p><span class="diczx1">释左骖，以公命赠</span><span class="diczx2">孟明</span>。——<span class="smcs">《左传·僖公三十三年》</span></p><p><span class="diczx1">昭公出奔，民如释重负。——<span class="smcs">《谷梁传·昭公二十九年》</span></span></p><p><span class="diczx1">主人释服。——<span class="smcs">《仪礼》</span></span></p><p><span class="cino">(3)</span> 又如:释绂(解下印绶。指致仕罢官);释褐(除去平民的衣服，换上官服。即<span class="yhcs">“作官”</span>)</p><p><span class="cino">(4)</span> 解释<span class="encs">[explain]</span></p><p><span class="diczx1"><span class="encs">[</span><span class="diczx2">夫差</span>]</span>乃使行人 <span class="diczx2">奚斯</span>释言于 <span class="diczx2">齐</span>。——<span class="smcs">《国语·吴语》</span></p><p><span class="cino">(5)</span> 又如:释情消迹(消除误会;予以和解);释言(以言语自作解释);释例(说明所著书的凡例)</p><p><span class="cino">(6)</span> 释放;赦免 <span class="encs">[release;pardon]</span></p><p><span class="diczx2">怀王</span>竟听 <span class="diczx2">郑袖</span>，复释去 <span class="diczx2">张仪</span>。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="diczx1">豪民气尽，以额叩头，谢不敢。乃释之。——</span><span class="diczx2">明</span>· <span class="diczx2">高启</span><span class="smcs">《书博鸡者事》</span></p><p><span class="cino">(7)</span> 舍弃;抛弃 <span class="encs">[give up;abandon]</span></p><p><span class="diczx1">故释先王之法，而法其所以为法。——<span class="smcs">《吕氏春秋·察今》</span></span></p><p><span class="diczx1">乃请释子服何于</span><span class="diczx2">吴</span>。——<span class="smcs">《左传·哀公八年》</span></p><p><span class="diczx1">君其释申生也。——<span class="smcs">《国语·晋语》</span>。注:<span class="yhcs">“释，舍也。”</span></span></p><p><span class="cino">(8)</span> 又如:释法(舍弃法令而不施用)</p><p><span class="cino">(9)</span> 放下;放开 <span class="encs">[put down]</span></p><p><span class="diczx1">有卖油翁释担而立。——</span><span class="diczx2">宋</span>· <span class="diczx2">欧阳修</span><span class="smcs">《归田录》</span></p><p><span class="diczx1">庖丁释刀对曰:<span class="yhcs">“臣之所好者，道也。”</span>——<span class="smcs">《庄子·养生主》</span></span></p><p><span class="cino">(10)</span> 又如:力叮不释(用力叮住不放)</p><p><span class="cino">(11)</span> 消失 <span class="encs">[disappear]</span></p><p><span class="diczx1">虽欲爱君，惑不释也。——<span class="smcs">《国语·晋语》</span></span></p><p><span class="cino">(12)</span> 溶化;消融 <span class="encs">[melt]</span></p><p><span class="diczx1">涣兮其若冰之将释。——<span class="smcs">《老子》</span></span></p><p><span class="cino">(13)</span> 又如:如冰之将释(就像冰将消融那样)</p><p><span class="cino">(14)</span> 泡在水里或其他液体里 <span class="encs">[soak]</span></p><p><span class="diczx1">欲濡肉，则释而煎之以醢。——<span                         </div>
                        
                        <div class="content definitions knr">
						<div class="mob">【<a href="/zd/kx/?zm=酉集下" target="_blank">酉集下</a>】【<a href="/zd/kx/?zm=酉集下&amp;bs=釆" target="_blank">釆</a>】
                              釋·康熙筆画：<a href="/zd/kx/bh/?bh=20" target="_blank">20</a>　·部外筆画：13</div>
                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/91CB.svg">						  <p>【唐韻】賞職切【集韻】【韻會】施隻切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音適。【說文】解也。从釆，釆取其分別物也。【左傳·襄二十九年】春王正月，公在楚，釋不朝正于廟也。【註】釋，解也。【疏】解釋公所以不得親自朝正也。【吳語】乃使行人奚斯釋言於齊。【註】釋，解也。以言自解。</p><p>又【廣韻】捨也。【前漢·食貨志】今農事棄捐，而采銅者日蕃，釋其耒耨，冶鎔炊炭。【管子·霸形篇】釋實而攻虛，釋堅而攻膬，釋難而攻易。</p><p>又消也，散也。【前漢·景十三王傳】骨肉冰釋。【註】師古曰：冰釋，言消散也。【淮南子·俶眞訓】北方有不釋之冰。</p><p>又放也。【書·多方】開釋無辜，亦克用勸。【傳】開放無罪之人。【左傳·哀八年】請釋子服何於吳。</p><p>又【爾雅·釋詁】釋，服也。【疏】釋者，釋去恨怨而服也。</p><p>又【書·大禹謨】釋兹在兹。【傳】釋，廢也。</p><p>又【禮·王制】出征，執有罪反，釋奠于學，以訊馘告。【註】釋菜奠幣，禮先師也。</p><p>又【禮·禮器】禮釋回，增美質。【註】釋，猶去也。回，邪辟也。</p><p>又【儀禮·士虞禮】舉魚腊俎，俎釋三个。【註】釋，猶遺也。</p><p>又【書·伊訓】若虞機張，往省括于度則釋。【疏】釋弦發矢。</p><p>又潤也。【禮·內則】欲濡肉，則釋而煎之以醢。【疏】欲得濡肉，則以水潤釋而煎之以醢也。</p><p>又【詩·大雅】釋之叟叟。【傳】釋，淅米也。</p><p>又釋迦，佛號。今僧家皆稱釋氏。【支遁·詠人日詩】釋迦乗虛會。【梁昭明·東齋聽講】昔聞孔道貴，今覩釋花珍。</p><p>又姓。</p><p>又【韻會】或作澤。【詩·周頌】其耕澤澤。【註】言土解也。【周禮·冬官考工記】水有時以澤。</p><p>又【集韻】亦作繹。通作醳。【史記·魏世家】與其以秦醳衞，不如以魏醳衞。</p><p>又【字彙補】羊益切，音亦。悅也。【六書正譌】別作懌，非。【嵆康·琴賦】康樂者聞之，則欨愉懽釋。</p><p>又【韻補】叶施灼切，音爍。【楚辭·九章】凌陽侯之泛濫兮，忽翺翔之焉薄。心絓結而不解兮，思蹇產而不釋。</p><p>
