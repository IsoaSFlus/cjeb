                          <h2 class="h2_entry"><span class="orth">炎</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>炎</strong></p><p><span class="dicpy">yán  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yán.mp3"></a></span> ㄧㄢˊ</span></p><ol><li>热：～热。～凉（ａ．气候的热和冷；ｂ．喻人情势利，或亲热攀附，或冷漠疏远，反复无常）。趋～附势。～～（灼热；火炽盛；引申为兴盛，亦指威势显赫）。</li><li>身体的一部分发生红、肿、热、痛的现象：～症。</li><li>传说中的中国上古帝王中的一位，并成为中华民族千百年的象征：～帝。～黄。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> flame, blaze; hot</p><p><span class="z_ts2">德语</span> Entzündung  (S)</p><p><span class="z_ts2">法语</span> brûlant,enflammé,inflammation</p></div>

                    <h2 class="h2_entry"><span class="orth">炎</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>炎</strong> <span class="dicpy">yán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yán.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从二火。本义:火苗升腾)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[flame]</span></p><p><span class="diczx1">炎，火光上也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">火曰炎上。——<span class="smcs">《书·洪范》</span></span></p><p><span class="diczx1">秉畀炎火。——<span class="smcs">《诗·小雅·大田》</span></span></p><p><span class="diczx1">南有炎火千里。——<span class="smcs">《楚辞·大招》</span></span></p><p><span class="diczx1">顷之，烟炎张天(炎，通<span class="yhcs">“焰”</span>，火焰)。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(3)</span> 又如:炎上(火焰向上;火向上燃烧)</p><p><span class="cino">(4)</span> 焚烧 <span class="encs">[burn]</span></p><p><span class="diczx1">大炎昆冈，玉石俱焚。——<span class="smcs">《书·胤征》</span></span></p><p><span class="diczx1">今时复旱，如炎如焚。——<span class="smcs">《后汉书》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>炎</strong> <span class="dicpy">yán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yán.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 炎热，酷热 <span class="encs">[blazing;be burning hot]</span></p><p><span class="diczx1">赫赫炎炎。——<span class="smcs">《诗·大雅·云汉》</span>。传:<span class="yhcs">“热气也。”</span></span></p><p><span class="diczx1">南方曰炎天，又东北曰炎风。——<span class="smcs">《吕氏春秋·有始》</span></span></p><p><span class="diczx1">观炎气之相仍兮，窥烟液之所积。——<span class="smcs">《楚辞》</span></span></p><p><span class="diczx1">夏炎百木盛，阴郁增埋覆。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《南山》</span></p><p><span class="cino">(2)</span> 又如:炎暑(炎热酷暑);炎天(大热天);炎魃(旱鬼);炎威(夏天酷热的威势);炎月(暑月);炎辰(炎热的日子);炎兵(喻酷热);炎冷(犹冷热);炎序(炎夏季节)</p><p><span class="cino">(3)</span> 红色 <span class="encs">[red]</span>。如:炎卉(鲜艳的花草);炎波(红色的波浪);炎霞(红色的霞光)</p><p>◎ <strong>炎</strong> <span class="dicpy">yán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yán.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 炎症 <span class="encs">[inflammation]</span>。如:嗓子发炎;喉炎;扁桃腺炎</p><p><span class="cino">(2)</span> 指炎帝 <span class="encs">[Yan Diemperor]</span> 即传说中上古姜姓部族首领。相传少典娶于有嵪氏而生。原居姜水流域，后向东发展到中原地区。一说炎帝即神农氏。如:炎皇(炎帝神农氏);炎轩(炎帝神农氏与黄帝轩辕氏的并称)</p><p><span class="cino">(3)</span> 汉朝 <span class="encs">[Han dynasty]</span>。五行家谓汉以火德王，火曰炎上，故以<span class="yhcs">“炎”</span>指汉朝。亦泛指以火德王的朝代。如:炎正(汉朝);炎宋(赵宋自称以火德王，故称炎宋);炎运(以火德而兴的帝业之运);炎祚(指汉或宋的国统);炎纪(汉朝年代)</p><p><span class="cino">(4)</span>
