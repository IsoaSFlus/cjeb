                          <h2 class="h2_entry"><span class="orth">之</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>之</strong></p><p><span class="dicpy">zhī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> ㄓˉ</span></p><ol><li>助词，表示领有、连属关系：赤子～心。</li><li>助词，表示修饰关系：缓兵～计。不速～客。莫逆～交。</li><li>用在主谓结构之间，使成为句子成分：“大道～行也，天下为公”。</li><li>代词，代替人或事物：置～度外。等闲视～。</li><li>代词，这，那：“～二虫，又何知”。</li><li>虚用，无所指：久而久～。</li><li>往，到：“吾欲～南海”。</li></ol>

                    <h2 class="h2_entry"><span class="orth">之</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>之</strong> <span class="dicpy">zhī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。象艸过屮。枝茎益大。本义:出，生出，滋长)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[grow]</span></p><p><span class="diczx1">之，出也。象艸过屮，枝茎益大有所之，一者，地也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">如语焉而未之然。——<span class="smcs">《礼记》</span>。</span><span class="diczx2">俞樾</span>平议:<span class="yhcs">“此之字乃其本义。未之者，未出也。”</span></p><p><span class="cino">(3)</span> 往，朝某方向走，到…去 <span class="encs">[go to;leave]</span></p><p><span class="diczx1">之，适也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">自伯之东。——<span class="smcs">《诗·卫风·伯兮》</span></span></p><p><span class="diczx1">之</span><span class="diczx2">沛公</span>军。——<span class="smcs">《史记·项羽本纪》</span></p><p><span class="diczx1">之虚所卖之。(<span class="yhcs">“虚”</span>同<span class="yhcs">“墟”</span>)——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《童区寄传》</span></p><p><span class="diczx1">佯狂不知所之者。——</span><span class="diczx2">明</span>· <span class="diczx2">张溥</span><span class="smcs">《五人墓碑记》</span></p><p><span class="diczx1">吾欲之</span><span class="diczx2">南海</span>，何如?—— <span class="diczx2">清</span>· <span class="diczx2">彭端淑</span><span class="smcs">《为学一首示子侄》</span></p><p><span class="cino">(4)</span> 又如:之如(前往);之往(去;到);之官(上任;前往任所);之国(前往封地);之适(前往)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>之</strong> <span class="dicpy">zhī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> </p><p>〈代〉</p><p><span class="cino">(1)</span> 指示人或事物，相当于<span class="yhcs">“这个”</span><span class="yhcs">“那个”</span> <span class="encs">[this;that]</span></p><p><span class="diczx1">宣王说之。(之:指南郭处士来吹竽。)——<span class="smcs">《韩非子·内储说上》</span></span></p><p><span class="diczx1">虎因喜，计之曰。(之:这，指上文所说驴生了气只能踢的情况。)——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《三戒》</span></p><p><span class="diczx1">当分明记之。(之:指共读情景。)——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《祭妹文》</span></p><p><span class="diczx1">为之，则难者亦易矣。(之:代词，指天下事。)——</span><span class="diczx2">清</span>· <span class="diczx2">彭端淑</span><span class="smcs">《为学一首示子侄》</span></p><p><span class="cino">(2)</span> 又如:较之中学时代;因之，他将自己的整个心血都放了进去</p><p><span class="cino">(3)</span> 指代人或事物的名称，相当于他、她、它、他们 <span class="encs">[he;her;it;they;them]</span></p><p><span class="diczx1">郑商人</span><span class="diczx2">弦高</span>将市于 <span class="diczx2">周</span>，遇之。——<span class="smcs">《左传·僖公三十二年》</span></p><p><span class="cino">(4)</span> 又</p><p><span class="diczx2">蹇叔</span>哭之。</p><p><span class="diczx1">巫医乐师百工之人。(<span class="yhcs">“之”</span>是指示代词，指代<span class="yhcs">“巫医乐师百工”</span>。)——</span><span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/4E4B.svg">						  <p>〔古文〕㞢【唐韻】【正韻】止而切【集韻】【韻會】眞而切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音枝。【說文】出也，象艸過屮枝莖益大有所之。一者，地也。【玉篇】是也，適也，往也。【禮·檀弓】延陵季子曰：若䰟氣，則無不之也。</p><p>又於也。【禮·大學】之其所親愛而辟焉。【註】之，適也。【朱傳】猶於也。</p><p>又語助辭。【書·金縢】禮亦宜之。【禮·文王世子】秋冬亦如之。【正字通】凡之字，或句中，或語尾，或層出。如毛詩我之懷矣，共武之服，及女曰雞鳴章知子之來之，六句九之字，常華章左之左之，六句八之字，可以例推。</p><p>又此也。【詩·周南】之子于歸。【註】之子，是子也。</p><p>又變也。【易傳】辭也者，各指其所之。【孫奕示兒編】之字訓變，左傳，遇觀之否。言觀變爲否也。</p><p>又至也。【詩·鄘風】之死矢靡他。</p><p>又遺也。【揚子·法言】或問孔子，知其道之不用也。則載而惡乎之。曰：之後世君子。【註】言行道者貴乎及身，乃載以遺後世。</p><p>又姓。出【姓苑】。</p><p>又【郝敬讀書通】凡言之者，物有所指，事有所屬，地有所往，連屬之辭也。通作旃。【詩·唐風】舍旃舍旃。</p><p>又【魏風】上愼旃哉。<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>與之同。通作至。往彼曰之，到此曰至，音義互通。</p><p>又【韻補】叶職流切，音周。【楚辭·九章】呂望屠於朝歌兮，甯戚歌而飯牛。不逢堯舜與桓繆兮，世孰云而知之。叶上牛下求。　【周伯琦曰】古人因物制字。如之本芝草，乎本吁氣，焉本鳶，後人借爲助語，助語之用旣多，反爲所奪，又制字以別之，乃有芝字吁字鳶字。</p><p>
