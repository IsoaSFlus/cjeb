						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>伐</strong></p><p><span class="dicpy">fá  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fá.mp3"></a></span>  ㄈㄚˊ</span></p><ol><li>砍：～树。砍～。</li><li>征讨：讨～。～罪（征讨有罪的人）。口诛笔～。</li><li>自夸：～善（夸自己的好处）。～智。不矜不～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">伐</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>伐</strong> <span class="dicpy">fá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fá.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从人，从戈。甲骨文字形，像用戈砍人的头。本义:砍杀)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[kill]</span></p><p><span class="diczx1">伐，击也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">伐，杀也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">为酒醴，伐豚羔，舞长袖。——</span><span class="diczx2">宋</span>· <span class="diczx2">苏辙</span><span class="smcs">《太白山祈雨词》</span></p><p><span class="cino">(3)</span> 又如:伐祭(砍头祭天);伐杀(杀戮)</p><p><span class="cino">(4)</span> 砍伐 <span class="encs">[hack;fell;cut down]</span></p><p><span class="diczx1">伐竹取道。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《至小丘西小石潭记》</span></p><p><span class="diczx1">坎坎伐檀兮。——<span class="smcs">《诗·魏风·伐檀》</span></span></p><p><span class="diczx1">斯亦伐根以求木茂。——</span><span class="diczx2">唐</span>· <span class="diczx2">魏征</span><span class="smcs">《谏太宗十思疏》</span></p><p><span class="diczx1">伐薪烧炭南山中。 ——</span><span class="diczx2">白居易</span><span class="smcs">《卖炭翁》</span></p><p><span class="cino">(5)</span> 又如:伐取(砍伐取得);伐炭(伐木烧炭);伐薪(砍柴)</p><p><span class="cino">(6)</span> 讨伐，进攻 <span class="encs">[send armed forces to suppress;attack]</span></p><p><span class="diczx2">齐</span>师伐我。——<span class="smcs">《左传·庄公十年》</span></p><p><span class="diczx1">凡师有钟鼓曰伐，无曰侵。——<span class="smcs">《左传·庄公二十九年》</span></span></p><p><span class="diczx1">其后</span><span class="diczx2">秦</span>伐 <span class="diczx2">赵</span>。——<span class="smcs">《史记·廉颇蔺相如列传》</span></p><p><span class="diczx2">季氏</span>将伐 <span class="diczx2">颛臾</span>。——<span class="smcs">《论语·季氏》</span></p><p><span class="cino">(7)</span> 又如:口诛笔伐;伐叛(讨伐叛逆);伐罪吊民(讨伐有罪的君主，抚慰受难的百姓)</p><p><span class="cino">(8)</span> 自吹自擂，夸耀自己 <span class="encs">[brag about]</span></p><p><span class="diczx1">愿无伐善，无施劳。——<span class="smcs">《论语·公冶长》</span></span></p><p><span class="diczx1">每一令出，</span><span class="diczx2">平</span>伐其功。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="diczx1">不伐己功，不矜己能。——<span class="smcs">《史记·淮阴侯列传》</span></span></p><p><span class="cino">(9)</span> 又如:伐智(夸耀自己的才智);伐善(夸耀自己的长处);伐功矜能(吹嘘自己的功劳和才能);伐德(自夸其德)</p><p><span class="cino">(10)</span> 败坏，损伤 <span class="encs">[damage]</span></p><p><span class="diczx1">重被猛术。国本必伐。——</span><span class="diczx2">宋</span>· <span class="diczx2">秦观</span><span class="smcs">《治势下》</span></p><p><span class="cino">(11)</span> 又如:伐性(危害身心);伐命(残害性命);伐生(残害生命)</p><p><span class="cino">(12)</span> 打破、挫败 <span class="encs">[defeat;frustrate;baffle]</span></p><p><span class="diczx1">故上兵伐谋，其次伐交，再次伐兵。——<span class="smcs">《孙子·谋攻》</span></span></p><p><span class="cino">(13)</span> 敲击 <span class="encs">[knock]</span></p><p><span class="diczx1">征人伐鼓。——<span class="smcs">《诗·小雅·采芑》</span></span></p><p><span class="cino">(14)</span> 又如:伐鼓(敲鼓);伐梆(敲打梆子)</p><p><span class="cino">(15)</span> 引申为开凿 <span class="encs">[dig]</span></p><p><span class="diczx1">乃伐山石，刻之日月。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《汴州东西水门记》</span></p><p><span class="cino">(16)</span> 批评，责备 <span class="encs">[criticize;blame]</span></p><p><span class="diczx1">伐</span><span                           <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/4F10_136.svg"  /><span class="swbf">(伐)</span><span class="ywyy">擊也。</span><span class="swdz">詩勿翦勿伐傳、鉦人伐<img class='lazy k_img' data-original='//img.zdic.net/song/cn/21537.svg'><span class='k_txt'>𡔷</span>傳皆曰。伐、擊也。禮記郊特牲。二日伐<img class='lazy k_img' data-original='//img.zdic.net/song/cn/21537.svg'><span class='k_txt'>𡔷</span>何居。鄭曰。伐猶擊也。尙書。不愆于四伐五伐。鄭曰。一擊一刺曰伐。詩是伐是肆箋云。伐謂擊刺之。按此伐之本義也。引伸之乃爲征伐。周禮九伐注云。諸侯之於國。如樹木之有根。是以言伐云</span><span class="ywyy">从人持戈。</span><span class="swdz">戈爲句兵。亦曰<img class='lazy k_img' data-original='//img.zdic.net/song/cn/23AA0.svg'><span class='k_txt'>𣪠</span>兵。左傳擊之以戈是也。戍者、守也。故从人在戈下。入戈部。伐者、外擊也。故从人杖戈。入人部。房越切。十五部。</span><span class="ywyy">一曰敗也。</span><span class="swdz">此謂引伸之義。伐敗㬪韵。左傳。凡師有鐘<img class='lazy k_img' data-original='//img.zdic.net/song/cn/21537.svg'><span class='k_txt'>𡔷</span>曰伐。穀梁傳。斬樹木、壞宫室曰伐。攴部曰。敗者、毀也。公羊傳曰。春秋伐者爲客。伐者爲主。何云。伐人者爲客。讀伐、長言之。見伐者爲主。讀伐短言之。皆齊人語也。按今人讀房越切。此短言也。劉昌宗周禮大司馬、大行人、輈人皆房廢切。此長言也。劉係北音。周？、沈約韵書皆用南音。去入多強爲分別。而不合於古矣。伐人者有功。故左傳諸侯言時記功。大夫稱伐。史記明其等曰伐。積日曰閱。又引伸之自功曰伐。</span><span class="ywyy">亦斫也。</span><span class="swdz">大徐無此三字爲長。</span><span class="xh"></span></p>                          </div>
