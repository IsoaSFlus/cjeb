                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>烝</strong></p><p><span class="dicpy">zhēng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhēng.mp3"></a></span>  ㄓㄥˉ</span></p><ol><li>众多：～民。</li><li>古代特指冬天的祭祀：“冬祭白～”。</li><li>美。</li><li>进献。</li><li>同“<span class="diczx3">蒸</span>”，热气上升。</li><li>古代指与母辈淫乱。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> rise, steam; many, numerous</p><p><span class="z_ts2">德语</span> fortschreiten, vorrücken</p></div>

                    <h2 class="h2_entry"><span class="orth">烝</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>烝</strong> <span class="dicpy">zhēng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhēng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从火，丞声。本义:火气上行)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[evaporate]</span></p><p><span class="diczx1">烝，火气上升也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">阳气俱烝。——<span class="smcs">《国语·周语》</span></span></p><p><span class="diczx1">释之叟叟，烝之浮浮。——<span class="smcs">《诗·大雅·生民》</span></span></p><p><span class="cino">(3)</span> 又如:烝烝(热气升腾的样子);烝云(上腾的云气);烝濡(烝腾的湿气);烝润(湿热润发)</p><p><span class="cino">(4)</span> 用蒸汽加热。后作<span class="yhcs">“蒸”</span> <span class="encs">[steam]</span></p><p><span class="diczx1">烝之浮浮。——<span class="smcs">《诗·大雅·生民》</span></span></p><p><span class="diczx1">故枸木必将待櫽烝矫然后直。——<span class="smcs">《荀子·性恶》</span></span></p><p><span class="diczx2">易牙</span>烝其子首而进之。——<span class="smcs">《韩非子·难一》</span></p><p><span class="diczx2">桓南郡</span>每见人不快，辄嗔云:<span class="yhcs">“君得哀家梨，当复不烝食不?”</span>——<span class="smcs">《世说新语》</span></p><p><span class="cino">(5)</span> 又如:烝炊(蒸煮);烝壶(蒸熟的葫芦瓜);烝糕(以米粉或面粉发酵加糖蒸成的糕)</p><p><span class="cino">(6)</span> 一般指娶父亲的妻妾及兄长的妻妾 <span class="encs">[incest]</span></p><p><span class="diczx1">初，</span><span class="diczx2">惠公</span>之即位也少， <span class="diczx2">齐</span>人使 <span class="diczx2">昭伯</span>烝于 <span class="diczx2">宣姜</span>，不可，强之。——<span class="smcs">《左传·闵公二年》</span></p><p><span class="diczx1">见</span><span class="diczx2">武媚娘</span>而悦之，遂即东厢烝焉。—— <span class="diczx2">明</span>· <span class="diczx2">周清源</span><span class="smcs">《西湖二集》</span></p><p><span class="cino">(7)</span> 又如:烝淫(指与母辈通奸的淫行);烝报(乱伦之行);烝弑(奸淫母辈，残杀君父);烝乱(淫乱)</p><p><span class="cino">(8)</span> 用火烘烤 <span class="encs">[roast]</span></p><p><span class="diczx1">枸木必将待櫽栝烝矫燃后直。——<span class="smcs">《荀子·性恶》</span></span></p><p><span class="cino">(9)</span> 蒸发 <span class="encs">[evaporate]</span></p><p><span class="diczx1">月丽于上，山烝于下。——</span><span class="diczx2">王充</span><span class="smcs">《论衡》</span></p><p><span class="cino">(10)</span> 进;前进 <span class="encs">[advance]</span></p><p><span class="diczx1">烝我髦士。——<span class="smcs">《诗·小雅·甫田》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>烝</strong> <span class="dicpy">zhēng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhēng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 众多;众<span class="encs">[numerous;many]</span></p><p><span class="diczx1">有豕向蹄，烝涉波矣。——<span class="smcs">《诗·小雅·渐渐之石》</span></span></p><p><span class="diczx1">天生烝民。——<span class="smcs">《孟子·告子》</span></span></p><p><span class="cino">(2)</span> 又如:烝人(民众，百姓);烝民(民众，百姓);烝涉(成群涉水);烝庶(众人，民众);烝黎(百姓)</p><p><span class="cino">(3)</span> 美;美好 <span class="encs">[beautiful;fine]</span></p><p><span class="diczx2">文王</span>烝哉!——<span class="smcs">《诗·大雅·文王有声》</span></p>                    <div class="div copyright"> © 汉典 </div>
