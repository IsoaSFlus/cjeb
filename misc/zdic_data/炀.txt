                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>炀</strong></p><p><strong>（<span class="diczx3">煬</span>）</strong></p><p><span class="dicpy">yáng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yáng.mp3"></a></span> ㄧㄤˊ</span></p><ol><li>熔化金属。</li><li>火旺。</li><li>烘干，烤火。</li><li>古代谥法，去礼远众称“炀”。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> roast; scorch; melt; flame</p><p><span class="z_ts2">德语</span> duftete, gerochen</p><p><span class="z_ts2">法语</span> fondre,fondu</p></div>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>炀</strong></p><p>煬 <span class="dicpy">yáng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yáng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 熔炼金属 <span class="encs">[melt metals]</span></p><p><span class="diczx1">以一铁范…密布字印…持就火炀之。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">yàng</span></p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>炀</strong></p><p>煬 <span class="dicpy">yàng</span></p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从火，昜( <span class="dicpy">yáng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yáng.mp3"></a></span> )声。本义:烘烤)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[roast]</span></p><p><span class="diczx1">古者民不知衣服，夏多积薪，冬则炀之。——<span class="smcs">《庄子》</span></span></p><p><span class="cino">(3)</span> 向火(取暖);烤火 <span class="encs">[warm oneself]</span></p><p><span class="diczx1">炀，炙燥也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">冬则羊裘解札，短褐不掩形而炀灶口。——<span class="smcs">《淮南子》</span></span></p><p><span class="diczx1">若灶则不然，前之人炀，则后之人无从见也。——<span class="smcs">《战国策》</span></span></p><p><span class="cino">(4)</span> 炊 <span class="encs">[cook]</span>。如:炀者(灶下烧火的人);炀器(炉灶)</p><p><span class="cino">(5)</span> 遮蔽，蒙蔽，挡住 <span class="encs">[cover]</span>。如:炀没(湮没，消失);炀蔽(遮瞒;遮蔽)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>炀</strong></p><p>煬 <span class="dicpy">yàng</span></p><p>〈形〉</p><p><span class="cino">(1)</span> 炽热 <span class="encs">[hot]</span></p><p><span class="diczx1">观天火之炎炀兮，听大壑之波声。——</span><span class="diczx2">汉</span>· <span class="diczx2">东方朔</span><span class="smcs">《七谏》</span></p><p><span class="cino">(2)</span> 又如:炀火(烈火);炀旱(炎热干旱);炀炀(火盛的样子);炀和(融和;暖和)</p><p><span class="cino">(3)</span> 另见 <span class="dicpy">yáng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
