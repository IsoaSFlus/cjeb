                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>获</strong></p><p>獲 <span class="dicpy">huò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/huò.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从犬，蒦( <span class="dicpy">huó</span>)声。按甲骨文从隹从又，表示捕鸟在手。本义:猎得禽兽)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[capture;catch]</span></p><p><span class="diczx1">获，猎所获也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">田获三狐。——<span class="smcs">《易巽》</span></span></p><p><span class="diczx1">舍拔则获。——<span class="smcs">《诗·秦风·驷賳》</span></span></p><p><span class="diczx1">谋于野则获。——<span class="smcs">《左传·襄公三十一年》</span></span></p><p><span class="diczx1">今日畋乐乎?获多乎?——</span><span class="diczx2">司马相如</span><span class="smcs">《子虚赋》</span></p><p><span class="cino">(3)</span> 又如:获兽;获兔;获者(猎得禽兽者)</p><p><span class="cino">(4)</span> 俘获 <span class="encs">[capture]</span></p><p><span class="diczx1">遇犬获之。——<span class="smcs">《诗·小雅·巧言》</span></span></p><p><span class="diczx1">获者取左耳。——<span class="smcs">《周礼·大司马》</span></span></p><p><span class="diczx1">不获二毛。——<span class="smcs">《礼记·檀弓》</span></span></p><p><span class="diczx1">荻</span><span class="diczx2">百里孟明</span><span class="diczx2">视</span>。——<span class="smcs">《左传·僖公三十二年》</span></p><p><span class="cino">(5)</span> 又如:获丑(俘获敌众);获夷(能俘获夷狄者);获解(拘捕押送);拿获</p><p><span class="cino">(6)</span> 得到;取得 <span class="encs">[get]</span></p><p><span class="diczx1">获，得也。——<span class="smcs">《小尔雅·广言》</span></span></p><p><span class="diczx1">遭遇时变，以获爵位。——</span><span class="diczx2">杨恽</span><span class="smcs">《报孙会宗书》</span></p><p><span class="diczx1">初虽劳苦，卒获其庆。——<span class="smcs">《盐铁论·诛秦》</span></span></p><p><span class="diczx1">获莒挐。——<span class="smcs">《左传·僖公元年》</span></span></p><p><span class="diczx1">获宝玉大弓。——<span class="smcs">《左传·定公九年》</span></span></p><p><span class="diczx1">获者坐而获。——<span class="smcs">《仪礼·乡射礼》</span></span></p><p><span class="diczx1">凡得获货贿人民六畜者。——<span class="smcs">《周礼·朝士》</span></span></p><p><span class="diczx1">获有所闻。——</span><span class="diczx2">明</span>· <span class="diczx2">宋濂</span><span class="smcs">《送东阳马生序》</span></p><p><span class="diczx1">幸获名成。——</span><span class="diczx2">清</span>· <span class="diczx2">周容</span><span class="smcs">《芋老人传》</span></p><p><span class="diczx1">两人并获释。——<span class="smcs">《明史》</span></span></p><p><span class="diczx1">获矢无算。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="cino">(7)</span> 又如:获奖;获匹(获偶。得到配偶);获印(得到官印，意为得官);获成(得到和解，得到成功)</p><p><span class="cino">(8)</span> 得以，能够 <span class="encs">[can]</span>。如:获隽(得以考中);获隽公车(会试得中之意。隽:同<span class="yhcs">“俊”</span>，才智过人。汉以公家车递送应举的人，后以<span class="yhcs">“公车”</span>为入京应试主人的代称);获申(得以表达);获全(得以保全)</p><p><span class="cino">(9)</span> 遭受 <span class="encs">[subject
