                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>泛</strong></p><p><span class="dicpy">fàn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fàn.mp3"></a></span>  ㄈㄢˋ</span></p><ol><li>漂浮：～舟。</li><li>透出：脸上～出了红晕。</li><li>浮浅，不切实：浮～。空～。～～之交（友谊不深）。～～而谈。</li><li>一般地：～论。～指。～称。广～。～览。～读。</li><li>水向四处漫流：～溢。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> to drift, float; careless, reckless</p><p><span class="z_ts2">法语</span> flotter,se répandre,se dégager</p></div>

                    <h2 class="h2_entry"><span class="orth">泛</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>泛</strong> <span class="dicpy">fàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fàn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水，乏声。汎、泛实同一词。本义:漂浮)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[float]</span></p><p><span class="diczx1">汎，浮貌。从水，凡声。与泛略同。与氾迥别。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">汎彼柏舟。——<span class="smcs">《诗·邶风·柏舟》</span></span></p><p><span class="diczx1">汎舟于</span><span class="diczx2">河</span>。——<span class="smcs">《国语·晋语》</span>。注:<span class="yhcs">“浮也。”</span></p><p><span class="diczx1">一曰泛齐。——<span class="smcs">《周礼·酒正》</span></span></p><p><span class="diczx1">汎齐，浮蛾在上汎汎然也。——<span class="smcs">《释名·释饭食》</span></span></p><p><span class="diczx1">汎汎杨舟。——<span class="smcs">《诗·小雅·菁菁者莪》</span></span></p><p><span class="diczx1">乘天潢之汎汎兮。——<span class="smcs">《文选·张衡·思玄赋》</span></span></p><p><span class="diczx1">盖大苏泛</span><span class="diczx2">赤壁</span>云。—— <span class="diczx2">明</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="diczx1">泛</span><span class="diczx2">彭蠡口</span>，四望无际。—— <span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《过小孤山大孤山》</span></p><p><span class="cino">(3)</span> 又如:泛然(漂浮，浮动的样子);泛浮(漂浮);泛萍(比喻漂荡不定);泛蚁(浮在酒上的泡沫)</p><p><span class="cino">(4)</span> 漫溢，大水漫流 <span class="encs">[flood]</span></p><p><span class="diczx1">河水决</span><span class="diczx2">濮阳</span>，泛郡十六。——<span class="smcs">《汉书·武帝纪》</span></p><p><span class="cino">(5)</span> 又如:黄泛区;泛浸(泛滥淹没);泛涨(水涨溢);泛澜(漫溢横流)</p><p><span class="cino">(6)</span> 弥漫 <span class="encs">[fill the air]</span></p><p><span class="diczx1">凝烟泛城阙，凄风入轩房。——</span><span class="diczx2">南朝</span> <span class="diczx2">宋</span>· <span class="diczx2">刘铄</span><span class="smcs">《歌诗》</span></p><p><span class="cino">(7)</span> 浮现;露出 <span class="encs">[emerge]</span>。如:泛白(浮现白色)</p><p><span class="cino">(8)</span> 向上冒出 <span class="encs">[emit upwards]</span></p><p><span class="diczx1">坑里泛出半人深的水，油汪汪的，漂着汽油。——</span><span class="diczx2">杨朔</span><span class="smcs">《三千里江山》</span></p><p><span class="cino">(9)</span> 通<span class="yhcs">“覂”</span>( <span class="dicpy">fěng</span>)，倾覆 <span class="encs">[capsize;overturn;topple]</span></p><p><span class="diczx1">大命将泛，莫之振救。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《论积贮疏》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>泛</strong> <span class="dicpy">fàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fàn.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 假借为氾。广泛，普遍 <span class="encs">[extensive;general]</span></p><p><span class="diczx1">以</span><span class="diczx2">杜君</span>言泛汛之。—— <span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">情窦既开，必致于泛用无度。——</span><span class="diczx2">顾炎武</span><span class="smcs">《天下郡国利病书》</span></p><p><span class="cino">(2)</span>
