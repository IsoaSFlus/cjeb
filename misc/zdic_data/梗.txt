                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>梗</strong></p><p><span class="dicpy">gěng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gěng.mp3"></a></span>  ㄍㄥˇ</span></p><ol><li>植物的枝或茎：菜～子。</li><li>直，挺立：～着脖子。</li><li>正直，直爽：～直。</li><li>阻塞，妨碍：～塞（<span class="dicpy">sè</span>）。～阻。心肌～死。</li><li>大略：～概。</li><li>有刺的草木：～榆（刺榆）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">梗</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>梗</strong> <span class="dicpy">gěng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gěng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从木，更声。本义:一种树)</p><p><span class="cino">(2)</span> 刺榆(Hemiptelea davidii)的古名 <span class="encs">[David hemiptelea]</span></p><p><span class="diczx1">梗，山枌榆。荚可为芜荑也。——<span class="smcs">《说文》</span>。按，叶小于常榆，刺如柘刺，其木坚韧，中犊车材，即诗之山有枢。</span></p><p><span class="cino">(3)</span> 有刺的草木 <span class="encs">[spiny plant]</span></p><p><span class="diczx1">凡草木刺人，自关而东，或谓之梗。——<span class="smcs">《西京赋》</span></span></p><p><span class="diczx1">梗木为之靡拉。——<span class="smcs">《方言三》</span></span></p><p><span class="cino">(4)</span> 又如:梗林(多刺的林丛);梗莽(荆棘草莽);梗刺(草木茎上有刺针)</p><p><span class="cino">(5)</span> 草本植物的枝茎 <span class="encs">[stalk]</span></p><p><span class="diczx1">桃梗土偶。——<span class="smcs">《战国策·秦策》</span></span></p><p><span class="diczx1">今求柴胡桔梗于沮泽。——<span class="smcs">《战国策·齐策》</span></span></p><p><span class="cino">(6)</span> 又如:菠菜梗;梗子(植物的枝或茎);荷梗;芹菜梗</p><p><strong>词性变化</strong></p><p>◎ <strong>梗</strong> <span class="dicpy">gěng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gěng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 阻塞;阻碍 <span class="encs">[obstruct;hinder]</span></p><p><span class="diczx1">元恶未黜，天邑犹梗。——<span class="smcs">《梁书·高帝纪》</span></span></p><p><span class="diczx1">是时道路梗涩，人怀危惧，</span><span class="diczx2">承</span>每遇艰险，处之夷然。——<span class="smcs">《晋书·王承传》</span></p><p><span class="cino">(2)</span> 又如:梗涩(阻塞难通);梗化(阻塞教化;妨碍管教);梗滞(阻塞);梗拒(阻拦拒绝);梗沮(阻扰;干扰);梗碍(阻塞)</p><p><span class="cino">(3)</span> 假借为<span class="yhcs">“抗”</span>。抵御;抗拒 <span class="encs">[resist]</span></p><p><span class="cino">(4)</span> 又如:梗命(抗命);梗寇(作梗御寇);梗骜(亦指桀骜不驯的人)</p><p><span class="cino">(5)</span> 挺直 <span class="encs">[stiffen;straighten]</span>。如:梗着脖子</p><p><span class="diczx2">凤姐</span>听了，把头一梗。——<span class="smcs">《红楼梦》</span></p><p><span class="cino">(6)</span> 通<span class="yhcs">“鲠”</span>。食留咽中 <span class="encs">[choke with food]</span></p><p><span class="diczx1">谁生厉阶，至今为梗。——<span class="smcs">《诗·大雅·桑柔》</span></span></p><p>◎ <strong>梗</strong> <span class="dicpy">gěng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gěng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 耿直，刚正 <span class="encs">[upright]</span></p><p><span class="diczx1">子</span><span class="diczx2">文都</span>，性梗直，仕 <span class="diczx2">周</span>，为右侍上士。——<span class="smcs">《北史·汝阴王天赐传》</span>附 <span
