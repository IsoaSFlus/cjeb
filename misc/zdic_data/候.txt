						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>候</strong></p><p><span class="dicpy">hòu  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/hòu.mp3"></a></span>  ㄏㄡˋ</span></p><ol>    <li>等待：等～。～车室。～选人。守～。～补。～场。～审。～诊。</li>    <li>看望，问好：伺～。问～。</li>    <li>时节：时～。气～。～鸟。～虫。</li>    <li>事物在变化中的情状：火～儿。症～。</li>    <li>古代把五天称为“一候”，现气象学上仍沿用：～温（每五天的平均温度）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">候</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>候</strong> <span class="dicpy">hòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/hòu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。古文作<span class="yhcs">“矦”</span>。从人，侯声。本义:守望;侦察)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[reconnoitre]</span></p><p><span class="diczx1">矦，伺望也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">矦，覗也。——<span class="smcs">《广雅·释诂三》</span></span></p><p><span class="diczx1">市有候馆。——<span class="smcs">《周礼·遗人》</span>。注:<span class="yhcs">“楼可以观望者也。”</span></span></p><p><span class="cino">(3)</span> 又如:候馆(用以瞭望的高楼);候楼(用以瞭望的楼);侯遮(侦探敌情);候徼(在国境线上巡察警戒);候车(侦察敌情的战车)</p><p><span class="cino">(4)</span> 观测 <span class="encs">[observe]</span>。如:候风(观测风向);候景(测日影);候簿(天文观测记录簿)</p><p><span class="cino">(5)</span> 等候 <span class="encs">[wait for]</span></p><p><span class="diczx1">稚子候门。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《归去来兮辞》</span></p><p><span class="diczx1">即候苗成。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈》</span></p><p><span class="diczx1">候春发遣。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">移顺天府羁候。</span></p><p><span class="cino">(6)</span> 又如:候时(等候时机);候对(等候帝王召对);候结(等候审判;结案);候补道(等候实缺的候补道员)</p><p><span class="cino">(7)</span> 敬候 <span class="encs">[respectly wait]</span></p><p><span class="diczx1">彼矦人兮。——<span class="smcs">《诗·曹风·候人》</span>。传:<span class="yhcs">“道路迎送宾客者。”</span></span></p><p><span class="cino">(8)</span> 又如:候拜(迎候拜见);候人(周礼官名。掌管在路上迎送宾客等事);候驾(迎候帝王)</p><p><span class="cino">(9)</span> 看望;问候 <span class="encs">[visit;send one's respects to]</span></p><p><span class="diczx1">遣人候问。——<span class="smcs">《汉书·李广苏建传》</span></span></p><p><span class="diczx1">候太公太母起居。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《左忠毅公逸事》</span></p><p><span class="cino">(10)</span> 又如:候札(问候的信札);候函(问候的信函);候省(探视问候);候敬(问候时敬赠的礼物)</p><p><span class="cino">(11)</span> 预测;占验 <span class="encs">[forecast]</span></p><p><span class="diczx1">梦者六候。——<span class="smcs">《列子·周穆王》</span></span></p><p><span class="cino">(12)</span> 又如:候天(占验天象，以卜吉凶祸福);候物(占验物候);候星(占验星象);候气(占验节气的变化)</p><p><span class="cino">(13)</span> 服侍 <span class="encs">[attend]</span></p><p><span class="diczx1">性柔顺好礼，每帝视朝退，常具冠帔候接，佐御馔。——<span class="smcs">《宋史》</span></span></p><p><span class="cino">(14)</span> 诊察 <span class="encs">[examine]</span>。如:候脉</p><p><span class="cino">(15)</span> 〈方〉∶支付 <span class="encs">[pay]</span></p><p><span
