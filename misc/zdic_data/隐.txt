                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>隐</strong></p><p>隱 <span class="dicpy">yǐn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yǐn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 藏匿;隐蔽 <span class="encs">[hide;screen;shield;vover]</span></p><p><span class="diczx1">隐，蔽也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">隐，微也。——<span class="smcs">《尔雅》</span></span></p><p><span class="diczx1">隐五刃。——<span class="smcs">《国语·齐语》</span>。注:<span class="yhcs">“隐藏也。”</span></span></p><p><span class="diczx1">则事可以隐定。</span></p><p><span class="diczx1">隐忌雍蔽之人。——<span class="smcs">《荀子·致仕》</span>。注:<span class="yhcs">“隐，亦蔽也。”</span></span></p><p><span class="diczx2">桓公</span><span class="diczx2">管仲</span>虽善匿，弗能隐矣。——<span class="smcs">《吕氏春秋·重言》</span>。注:<span class="yhcs">“隐，蔽也。”</span></p><p><span class="diczx1">则盘纡隐深。——<span class="smcs">《文选·嵇康·琴赋》</span>。注:<span class="yhcs">“隐，幽深邃也。”</span></span></p><p><span class="diczx1">昔</span><span class="diczx2">帝鸿氏</span>有不才子，掩义隐贼，好行凶德。——<span class="smcs">《左传·文公十八年》</span></p><p><span class="diczx1">故隐居屠间。——<span class="smcs">《史记·魏公子列传》</span></span></p><p><span class="diczx1">可以隐形。——</span><span class="diczx2">三国</span><span class="diczx2">魏</span>· <span class="diczx2">邯郸淳</span><span class="smcs">《笑林》</span></p><p><span class="diczx1">日月隐耀。——</span><span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="diczx1">各隐卷底。——</span><span class="diczx2">明</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="cino">(2)</span> 又如:隐占身体(隐藏、掩护自己真实面目);隐遁(隐藏);隐谋(隐密计划);隐秀(隐藏智慧，不露锋芒或才华);隐候(在暗处等待);隐耀(隐藏光辉，隐藏身体。比喻才华不外露);隐身术(古代方士所隐蔽自己身形而使他人不可见的一种法术);隐冒(隐匿冒充);隐翳(掩蔽;遮蔽);隐掩(遮蔽掩护)</p><p><span class="cino">(3)</span> 隐瞒 <span class="encs">[conceal]</span></p><p><span class="diczx1">父为子隐，子为父隐。——<span class="smcs">《论语·子路》</span></span></p><p><span class="diczx1">上以无隐。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="cino">(4)</span> 又如:隐欺(隐瞒欺骗);隐情(隐瞒情况);隐拒(隐瞒抵赖);隐忍(忍耐，不露真情)</p><p><span class="cino">(5)</span> 隐测，审度 <span class="encs">[conjecture]</span></p><p><span class="diczx1">又隐实户口，料出无名万余人，以充军实。——<span class="smcs">《晋书·庾冰传》</span></span></p><p><span class="cino">(6)</span> 又如:隐心(审度);隐实(查证落实;核实)</p><p><span class="cino">(7)</span> 怜悯 <span class="encs">[pity to]</span></p><p><span class="diczx1">隐其无罪。——<span class="smcs">《孟子·梁惠王上》</span></span></p><p><span class="cino">(8)</span> 又如:隐恤(哀怜抚恤);隐恻(恻隐，同情怜惜);隐痛(内心深感苦痛);隐爱(恻隐疼爱);隐愍(怜悯);隐亲(怜悯抚恤)</p><p><span class="cino">(9)</span> 隐逸不出 <span class="encs">[hermit]</span></p><p><span class="diczx1">隐者也。——<span class="smcs">《论语·微子》</span></span></p><p><span class="diczx1">反国而隐焉。——</span><span class="diczx2">明</span>· <span class="diczx2">顾炎武</span><span class="smcs">《复庵记》</span></p><p><span class="cino">(10)</span> 又如:隐辟(隐退回避);隐静(避世隐居);隐学(隐居避世，读书自娱);隐吏(退隐的官吏);隐相(深居简出，不露面)</p><p><span class="cino">(11)</span> 塞，堵塞 <span class="encs">[block up;stop up]</span></p><p><span class="diczx1">士不隐塞，庶人不数罟。——<span class="smcs">《诗·小雅·鱼丽》</span></span></p><p><span class="cino">(12)</span> 又如:隐塞(堵塞)</p><p><span class="cino">(13)</span> 隐没;熄灭 <span class="encs">[vanish;go out;die out]</span></p><p><span class="diczx1">隐武事，行文道。——<span class="smcs">《国语·齐语》</span></span></p><p><span class="diczx1">隐武行文。——<span class="smcs">《春秋繁露·五行变救》</span></span></p><p><span class="diczx1">日斜光隐见，风还影合离。——</span><span class="diczx2">南朝</span>· <span class="diczx2">梁简文帝</span><span class="smcs">《咏栀子花》</span></p><p><span class="cino">(14)</span> 又如:隐形(隐没形体);隐灭(隐没消失);隐微(隐约细微);隐显(隐没与显现);隐息(休止);隐忽(时隐时现)</p><p><strong>词性变化</strong></p><p>◎ <strong>隐</strong></p><p>隱 <span class="dicpy">yǐn</span> <span class="ptr"><a class="audio_play_button class="diczx2">高祖</span>坐，言无所隐，时人甚惮焉。而隐厚笃实，瞻卹亲故。——<span class="smcs">《宋书·郑鲜之传》</span></p><p><span class="cino">(6)</span> 又如:隐冥(深沉含蓄);隐厚(稳重忠厚);隐隐桓桓(威武的样子);隐重(威重)</p><p><span class="cino">(7)</span> 宏大 <span class="encs">[great]</span>。如:隐虹(长虹)</p><p><span class="cino">(8)</span> 忧伤;疾苦 <span class="encs">[distressed;sad;sufferings]</span>。如:隐隐遑遑(忧愁不妥的样子)</p><p><span class="cino">(9)</span> 穷困 <span class="encs">[recluse]</span></p><p><span class="diczx1">贪夫殉身，圣亦保之;烈士殉名，隐亦好之。——</span><span class="diczx2">王闿运</span><span class="smcs">《邹汉勋传》</span></p><p><span class="cino">(4)</span> 又如:隐佚(隐逸者)</p><p><span class="cino">(5)</span> 隐语 <span class="encs">[enigmatic language]</span></p><p><span class="diczx2">齐威王</span>之时喜隐。——<span class="smcs">《史记·滑稽列传》</span></p><p><span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/96B1.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2450C.svg'><span class='k_txt'>𤔌</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/200CA.svg'><span class='k_txt'>𠃊</span>【唐韻】【正韻】於謹切【集韻】【韻會】倚謹切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音櫽。【爾雅·釋詁】隱，微也。【註】微謂逃藏也。【易·乾卦】龍德而隱者也。</p><p>又【禮·禮運】大道旣隱。【註】隱猶去也。</p><p>又【說文】蔽也。【玉篇】匿也。【論語】言及之而不言，謂之隱。【禮·檀弓】事親有隱而無犯。【魯語】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/206EC.svg'><span class='k_txt'>𠛬</span>五而已，無有隱者，隱乃諱也。</p><p>又【廣韻】私也。【論語】吾無隱乎爾。【疏】孔子敎人無所隱惜。</p><p>又【玉篇】不見也。【易·繫辭】巽稱而隱。【註】稱揚命令，而百姓不知其由。【史記·韓安國傳】壷遂之深中隱厚。</p><p>又【禮·曲禮】不以隱疾。【註】隱疾，衣中之疾也。</p><p>又【史記·秦始皇紀】隱宮徒<img class='lazy k_img' data-original='//img.zdic.net/song/cn/206EC.svg'><span class='k_txt'>𠛬</span>者，七十餘萬人。【註】宮<img class='lazy k_img' data-original='//img.zdic.net/song/cn/206EC.svg'><span class='k_txt'>𠛬</span>，一百日隱於隂室養之，故曰隱宮。</p><p>又【史記·滑稽傳】齊威王之時喜隱。【前漢·藝文志】隱書十八篇。【註】《劉向·別錄》云：隱書者，疑其言以相問，對者以慮思之，可以無不喩。</p><p>又【禮·玉藻】隱辟而後屨。【註】隱辟，俛逡巡而退著屨也。</p><p>又【爾雅·釋言】隱，占也。【註】隱度。【疏】占者，視兆以知吉凶，必先隱度。【禮·少儀】軍旅思險，隱情以虞。【註】隱，意也，思也。【後漢·安帝紀】隱親悉心，勿取浮華。【註】皆隱審盡心，勿取浮華不實者。</p><p>又【揚子·方言】隱，定也。</p><p>又【玉篇】安也。</p><p>又痛也。【詩·邶風】如有隱憂。【傳】痛也。【禮·檀弓】拜稽顙，哀戚之至隱也。稽顙，隱之甚也。【孟子】王若隱其無罪而就死地。【又】皆有怵惕惻隱之心。【前漢·韓安國傳】此仁人之所隱也。</p><p>又【左傳·昭二十五年】隱民皆取食焉。【註】隱約窮困。</p><p>又【定三年】君以弄馬之故，隱君身。【註】隱，憂約也。【荀子·儒效篇】隱隱兮其恐人之不當也。【註】隱隱，憂戚貌。</p><p>又【司馬相如·上林賦】湛湛隱隱。【註】隱隱，盛貌。</p><p>又【前漢·郊祀歌】休嘉砰隱溢四方。【註】砰隱，盛意。</p><p>又【左傳·襄二十三年】踰隱而待之。【註】隱，短牆也。</p><p>又【諡法】隱拂不成曰隱，不顯尸國曰隱，見美堅長曰隱。</p><p>又姓。【吳志】有廷尉左監隱蕃。</p><p>又【爾雅·釋草】蒡，隱葱。【註】似蘇，有毛，江東呼爲隱葱，藏以爲葅。</p><p>又【廣韻】【集韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>於靳切，音檼。【廣韻】隈隱之貌。</p><p>又【孟子】隱几而臥。【註】隱，倚也。於靳反。</p><p>又【集韻】築也。【前漢·賈山傳】厚築其外，隱以金椎。</p><p>又【集韻】於刃切，駰去聲。【禮·檀弓】旣葬而封，廣輪揜坎，其高可隱也。【註】隱，據也。封可手據，謂高四尺所。【釋文】隱，於刃反。　【集韻】俗作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/28F06.svg'><span class='k_txt'>𨼆</span>㥯隐。</p><p>
