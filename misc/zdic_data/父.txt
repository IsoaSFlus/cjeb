                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>父</strong> <span class="dicpy">fǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fǔ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 对有才德的男子的美称，多附缀于表字后面 <span class="encs">[courtesy name]</span></p><p><span class="diczx1">维师尚父，时维鹰扬。——<span class="smcs">《诗·大雅》</span></span></p><p><span class="diczx1">三月，公及</span><span class="diczx2">邾仪父</span>盟于 <span class="diczx2">眜</span>。——<span class="smcs">《春秋》</span></p><p><span class="diczx1">余弟安国平父，安上纯父。——</span><span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="cino">(2)</span> 对老年男子的尊称 <span class="encs">[old man]</span></p><p><span class="diczx1">纵江东父老怜而王我，我何面目见之?——<span class="smcs">《史记·项羽本纪》</span></span></p><p><span class="diczx1">父知之乎?(父:老先生)——<span class="smcs">《史记·张释之冯唐列传》</span></span></p><p><span class="cino">(3)</span> 又如:渔父;田父</p><p><span class="cino">(4)</span> 通<span class="yhcs">“甫”</span>。开始 <span class="encs">[beginning]</span></p><p><span class="diczx1">强梁者不得其死，吾将以为教父。——<span class="smcs">《老子·四十二章》</span></span></p><p><span class="cino">(5)</span> 又如:教父(教戒的开始)</p><p><span class="cino">(6)</span> 天。古人以为天地生万物，故称天为<span class="yhcs">“父”</span> <span class="encs">[heaven]</span></p><p><span class="diczx1">乾，天也，故称乎父;坤，地也，故称呼母。——<span class="smcs">《易·说卦》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>父</strong> <span class="dicpy">fǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fǔ.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“捕”</span>( <span class="dicpy">bǔ</span>)。捕捉，捉拿 <span class="encs">[catch;capture]</span></p><p><span class="diczx1">徒以而富(通福)之，父系而伏之。——<span class="smcs">《管子·侈靡》</span></span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">fù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fù.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>父</strong> <span class="dicpy">fù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (指事。甲骨文字形，象右手持棒之形。意思是:手里举着棍棒教子女守规矩的人是家长，即父亲。本义:父亲)</p><p><span class="cino">(2)</span> 同本义</p><p><span class="cino">(3)</span> 某人直系血统的上一代男性 <span class="encs">[father]</span></p><p><span
