                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>抵</strong> <span class="dicpy">dǐ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dǐ.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，氐( <span class="dicpy">dǐ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dǐ.mp3"></a></span> )声。本义:挤，推)</p><p><span class="cino">(2)</span> 同本义<span class="encs">[squeeze out]</span></p><p><span class="diczx1">抵，挤也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">抵，推也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">抵，距也。——<span class="smcs">《汉书·梁怀王揖传》</span>注</span></p><p><span class="cino">(3)</span> 又如:抵掎(排挤攻击);抵斥(排斥);抵排(排斥;抵触)</p><p><span class="cino">(4)</span> 抵赖，拒不承认事实 <span class="encs">[deny]</span></p><p><span class="diczx1">延年抵曰。——<span class="smcs">《汉书·田延年传》</span>。注:<span class="yhcs">“拒讳也。”</span></span></p><p><span class="cino">(5)</span> 又如:抵讳(抵赖;不认账);抵谰(抵赖，拒不承认)</p><p><span class="cino">(6)</span> 抵挡。抵抗，挡住压力 <span class="encs">[withstand;resist]</span>。如:抵配(拼;豁出去);抵冒(抗拒冒犯);抵拦(抗拒;抵挡)</p><p><span class="cino">(7)</span> 触犯 <span class="encs">[offend]</span>。如:抵牾(矛盾;抵触);抵死(冒死，触犯死罪，竭力，坚持);抵犯(触犯)</p><p><span class="cino">(8)</span> 抵偿 <span class="encs">[compensate]</span></p><p><span class="diczx1">伤人及盗抵罪。——<span class="smcs">《史记·高帝纪》</span></span></p><p><span class="cino">(9)</span> 又如:抵充(抵偿充当);抵补(抵充补足)</p><p><span class="cino">(10)</span> 诋毁 <span class="encs">[slander]</span>。如:抵诃(诋毁呵斥)</p><p><span class="cino">(11)</span> 抵达 <span class="encs">[arrive]</span></p><p><span class="diczx1">抵</span><span class="diczx2">九原</span>。——<span class="smcs">《史记·始皇纪》</span></p><p><span class="diczx1">抵营室。——<span class="smcs">《史记·天官书》</span></span></p><p><span class="diczx1">抵山谷中。——<span class="smcs">《汉书·李陵传》</span></span></p><p><span class="diczx1">乘虚直抵其城。——<span class="smcs">《资治通鉴·唐纪》</span></span></p><p><span class="cino">(12)</span> 又如:抵任(到任);抵官(到职);日内抵京</p><p><span class="cino">(13)</span> 相当 <span class="encs">[be equal to]</span></p><p><span class="diczx1">烽火连三月，家书抵万金。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《春望》</span></p><p><span class="cino">(14)</span> 又如:价抵连城</p><p><span class="cino">(15)</span> 碰 <span class="encs">[touch]</span>。如:抵足(脚对脚而睡;形容很亲近);抵掌(拍击着手掌，表示亲近)</p><p><span class="cino">(16)</span> 支撑 <span
