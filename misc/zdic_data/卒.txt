						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>卒</strong></p><p><span class="dicpy">zú  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span>  ㄗㄨˊ</span></p><ol><li>兵：士～。小～。兵～。</li><li>旧称差役：走～。</li><li>死亡：病～。生～年月。</li><li>完毕，终了：～岁（度过一年）。～业（毕业）。</li><li>终于：～胜敌军。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>卒</strong></p><p><span class="dicpy">cù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cù.mp3"></a></span>  ㄘㄨˋ</span></p><ol><p>　◎ 同“<span class="diczx3">猝</span>”。</p></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> soldier; servant; at last, finally</p><p><span class="z_ts2">德语</span> plötzlich, unerwartet, unvermittelt [ auch <a href="./猝">猝</a> cu4 ]  (Adj, Lit)​,Diener, Helfeshelfer, Lakai, Büttel  (S)​,Gewindeschneidbacke  (S, Tech)​,Soldat ( schwarzer Stein im chinesischen Schach )​  (S)​,Soldat  (S, Mil)​,etw. beenden, beendigen, etw. ein Ende machen  (V)​,sterben, verenden  (V)​,verenden, versterben  (V)​,endlich, schließlich, zum Schluß, zuletzt  (Adj)</p><p><span class="z_ts2">法语</span> soldat,finir,achever,mourir</p></div>

                    <h2 class="h2_entry"><span class="orth">卒</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>卒</strong> <span class="dicpy">cù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cù.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 同<span class="yhcs">“猝”</span>。仓促，急速 <span class="encs">[hastily]</span></p><p><span class="diczx1">行西逾陇卒。——<span class="smcs">《汉书·食货志》</span>。注:<span class="yhcs">“仓卒也。”</span></span></p><p><span class="diczx1">兴卒暴之作。——<span class="smcs">《汉书·成帝纪》</span></span></p><p><span class="diczx1">而卒惶急无以击轲，而乃以手共搏之。——<span class="smcs">《战国策·燕策》</span></span></p><p><span class="cino">(2)</span> 又如:卒拔(仓猝攻取);卒迫(仓促紧迫)</p><p><span class="cino">(3)</span> 突然 <span class="encs">[suddenly]</span></p><p><span class="diczx1">则亡以应卒。——<span class="smcs">《汉书·辛庆忌传》</span></span></p><p><span class="diczx1">卒然边境有急，数千百万之众，国胡以馈之?——</span><span class="diczx2">贾谊</span><span class="smcs">《论积贮疏》</span></p><p><span class="cino">(4)</span> 又如:卒倒(突然晕倒);卒暴(突然);卒然(忽然，突然)</p><p><span class="cino">(5)</span> 另见 <span class="dicpy">zú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span> </p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/卒卒">卒卒</a></span><span class="crefe"><a href="/hans/卒中">卒中</a></span></div>　<p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>卒</strong> <span class="dicpy">zú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span>
