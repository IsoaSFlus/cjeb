						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>遂</strong></p><p><span class="dicpy">suì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/suì.mp3"></a></span>  ㄙㄨㄟˋ</span></p><ol><li>顺，如意：～心。～愿。</li><li>成功，实现：未～。功成名～。</li><li>于是，就：服药后头痛～止。</li><li>通达：“何往而不～”。</li><li>进，荐：“不能退，不能～”。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>遂</strong> <span class="dicpy">suí</span></p><p><span class="cino">(1)</span> —如<span class="yhcs">“半身不遂”</span>（ <span class="dicpy">bànshēn-bùsuí</span>）(身体一侧发生瘫痪)</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">suì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/suì.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>遂</strong> <span class="dicpy">suì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/suì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>))</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[escape]</span></p><p><span class="diczx1">遂，亡也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> 行，往 <span class="encs">[go]</span></p><p><span class="diczx1">遂，往也。——<span class="smcs">《广雅·释诂一》</span></span></p><p><span class="diczx1">不能遂。——<span class="smcs">《易·大壮》</span></span></p><p><span class="diczx1">遂贤良。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="cino">(4)</span> 又如:遂行(通行;顺适地进行)</p><p><span class="cino">(5)</span> 称心如意;使得到满足 <span class="encs">[fulfill]</span></p><p><span class="diczx1">他年如遂隐，</span><span class="diczx2">五老</span>是知音。—— <span class="diczx2">唐</span>· <span class="diczx2">李中碧</span><span class="smcs">《庐山》</span></p><p><span class="diczx2">王吉</span>曰:长卿久宦游不遂，而来过我。——<span class="smcs">《史记·司马相如列传》</span></p><p><span class="cino">(6)</span> 又如:遂怀(遂愿);遂隐(满足隐遁的愿望);遂初(旧时辞去官职，实现隐退的本愿)</p><p><span class="cino">(7)</span> 顺利地完成;成功 <span class="encs">[succeed]</span></p><p><span class="diczx1">上无乏用，百事乃遂。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="diczx1">四者无一遂。——</span><span class="diczx2">汉</span>· <span class="diczx2">司马迁</span><span class="smcs">《报任安书》</span></p><p><span class="diczx1">令遂前功。——<span class="smcs">《后汉书·班超传》</span></span></p><p><span class="diczx1">介子推之从晋公子，既反国而隐焉，又范君之所有志而不遂者也。——</span><span class="diczx2">明</span>· <span class="diczx2">顾炎武</span><span class="smcs">《复庵记》</span></p><p><span class="cino">(8)</span> 又如:遂初(辞官归隐，达到原来的愿望);遂过(促成过失);遂志(实现志愿;满足愿望);遂事(成就事业;完成某事);遂功(成功。建立功业)</p><p><span class="cino">(9)</span> 推荐或举荐 <span class="encs">[recommend]</span>。如:遂贤(荐举贤能之士);遂能(荐举贤能之士);遂良(荐举贤良之士);遂材(荐举才能之士);遂举(进举，进用)</p><p><span class="cino">(10)</span> 通达 <span class="encs">[understand]</span></p><p><span class="diczx1">此其垦田，足以食其民;都邑遂路，足以处其民。——<span class="smcs">《商君书·箅地》</span></span></p><p><span class="cino">(11)</span> 又如:遂达(通达)</p><p><span class="cino">(12)</span> 表明 <span class="encs">[make known]</span>。如:遂威(展示威力)</p><p><span class="cino">(13)</span> 生长 <span class="encs">[grow]</span></p><p><span class="diczx1">禾稼春生，人必加功焉，故五谷得遂长。——<span class="smcs">《淮南子·修务》</span></span></p><p><span
