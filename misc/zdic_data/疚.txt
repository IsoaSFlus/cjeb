						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>疚</strong></p><p><span class="dicpy">jiù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiù.mp3"></a></span>  ㄐㄧㄡˋ</span></p><ol><li>长期生病。</li><li>忧苦，特指因自己过失而造成的心内痛苦：负～。内～。愧～。歉～。内省（<span class="dicpy">xǐng </span>）不～。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>疚</strong> <span class="dicpy">jiù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从疒( <span class="dicpy">chuáng</span>)，表示与疾病有关，久声。本义:久病)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[chronic illness]</span></p><p><span class="diczx1">无饥馑疾疚祸罪之殃。——<span class="smcs">《韩非子》</span></span></p><p><span class="cino">(3)</span> 又如:疚病(疾病);疚疾(疾病);疚头怪脑(丑头怪脸的样子)</p><p><span class="cino">(4)</span> 贫穷 <span class="encs">[poverty]</span></p><p><span class="diczx1">维昔之富不如时，维今之疚不如兹。——<span class="smcs">《诗·大雅·召旻》</span></span></p><p><span class="cino">(5)</span> 缺陷 <span class="encs">[defect]</span></p><p><span class="diczx1">行刑不疚。(不疚，无弊病。)——<span class="smcs">《国语·晋语》</span></span></p><p><span class="diczx1">令章靡疚，亦善之亚。——<span class="smcs">《文心雕龙》</span></span></p><p><span class="cino">(6)</span> 害;灾殃 <span class="encs">[calamity]</span></p><p><span class="diczx1">行刑不疚。(不疚，无弊病。)——<span class="smcs">《国语·晋语》</span></span></p><p><span class="diczx1">刚中正，履帝位而不疚，光明也。——<span class="smcs">《易·履》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>疚</strong> <span class="dicpy">jiù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiù.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 因有过失而感到内心惭愧痛苦 <span class="encs">[remorse]</span></p><p><span class="diczx1">使我心疚。——<span class="smcs">《诗·小雅·大东》</span></span></p><p><span class="cino">(2)</span> 又如:疚怀(内心不安。同疚心);疚惭(犹疚愧);疚悔(因负疚而后悔);疚负(犹过失);疚恶(犹错误)</p><p><span class="cino">(3)</span> 忧伤;忧虑 <span class="encs">[distressed;worried]</span>。如:疚怀(伤心;忧虑);疚茕(忧念);疚心疾首(形容忧心愁痛到极点)</p><p><span class="cino">(4)</span> 困惑 <span class="encs">[perplexed;puzzled]</span></p><p><span class="diczx1">设之以国家之患而不疚。——<span class="smcs">《国语》</span></span></p><p><span class="diczx1">行前定则不疚。道前定则不穷。——<span class="smcs">《孔子家语》</span></span></p><p>◎ <strong>疚</strong> <span class="dicpy">jiù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 败坏;伤害 <span class="encs">[in jure;damage]</span></p><p><span class="diczx1">西方之书有之曰:<span class="yhcs">“怀与安，实疚大事。”</span>——<span class="smcs">《国语》</span></span></p><p><span class="diczx1">今有负其子而汲者，队其子于井中，其母必从而道之。今岁凶民饥道饿，此疚重于队其子，其可无察邪?——<span class="smcs">《墨子》</span></span></p><p><span class="cino">(2)</span> 痛恨 <span class="encs">[hate]</span>。如:疚愆(犹愧恨)</p><p><span class="cino">(3)</span> 居丧 <span class="encs">[be in mourning]</span></p><p><span class="diczx1">自仲秋而在疚兮，逾履霜以践冰。——</span><span class="diczx2">潘岳</span><span
