                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>径</strong></p><p><strong>（<span class="diczx3">徑</span>）</strong></p><p><span class="dicpy">jìng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jìng.mp3"></a></span> ㄐㄧㄥˋ</span></p><ol><li>小路；亦指道路，方法：～道。山～。捷～。途～。大相～庭（相差太远）。</li><li>直，直捷了当：～直。～流。～情（任性）。～自。</li><li>数学上指连接圆心和圆周的直线：直～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">径</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>径</strong></p><p>徑 <span class="dicpy">jīng</span></p><p>〈动〉</p><p><span class="cino">(1)</span> 经过，行经 <span class="encs">[pass]</span></p><p><span class="diczx1">径，过也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">夜径泽中。——<span class="smcs">《汉书·高帝纪》</span>。按，行也。</span></p><p><span class="cino">(2)</span> 又如:径历(经过);径复(往返，迂回曲折)</p><p><strong>词性变化</strong></p><p>◎ <strong>径</strong></p><p>徑 <span class="dicpy">jīng</span></p><p>〈名〉</p><p><span class="cino">(1)</span> 南北为径 <span class="encs">[northsouthern]</span>。如:径轮(南北之间的长度)</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">jìng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jìng.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>径</strong></p><p>徑、逕 <span class="dicpy">jìng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jìng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从彳( <span class="dicpy">chì</span>)，表示与道路有关，巠声。本义:步行小路)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[footpath;path]</span></p><p><span class="diczx1">径，步道也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">径，小道也。——<span class="smcs">《字林》</span></span></p><p><span class="diczx1">艮为径路。——<span class="smcs">《易·说卦》</span></span></p><p><span class="diczx1">审端径术。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="diczx1">送丧不由径。——<span class="smcs">《礼记·曲礼》</span></span></p><p><span class="diczx1">行不由径。——<span class="smcs">《论语》</span></span></p><p><span class="diczx1">夫逃虚空者，藜藿柱乎鼪鼬之径。——<span class="smcs">《庄子·徐无鬼》</span></span></p><p><span class="diczx1">夫惟捷径以窘步。——<span class="smcs">《离骚》</span></span></p><p><span class="diczx1">三径就荒。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《归去来兮辞》</span></p><p><span class="diczx1">野径云俱黑。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《春夜喜雨》</span></p><p><span class="cino">(3)</span> 又如:径术(径路与沟洫);径路(小路);径逾(本意是逾越堤渠而走。引申为不走正道);径行(小路与大道);径界(小路);径畛(田间道路)</p><p><span class="cino">(4)</span> 泛指道路 <span class="encs">[road]</span></p><p><span class="diczx1">塞徯径。——<span class="smcs">《礼记·月令》</span>。注:<span class="yhcs">“徯径，禽兽之道也。”</span></span></p><p><span class="diczx1">皋兰被径兮路斯渐。——<span class="smcs">《楚辞·招魂》</span></span></p><p><span class="cino">(5)</span> 又如:径要(必经之要道;要领，精义)</p><p><span class="cino">(6)</span> 途径;门路 <span class="encs">[means;way]</span>。如:径路(比喻处世行事的途径);径途(途径;门径)</p><p><span class="cino">(7)</span> 直径 <span class="encs">[diameter]</span></p><p><span class="diczx1">员径八尺。——<span class="smcs">《后汉书·张衡传》</span></span></p><p><span class="diczx1">径寸之木。——</span><span class="diczx2">明</span>· <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/5F91.svg">						  <p>【集韻】古定切【韻會】【正韻】吉定切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音徑。【說文】步道也。【徐鍇曰】道不容車，故曰步道。【玉篇】小路也。【易·說卦傳】艮爲山，爲徑路。【疏】徯徑，細小狹路。【周禮·地官·遂人】夫閒有遂，遂上有徑。【註】徑容牛馬。【疏】徑不容車軌，而容牛馬及人之步徑。【禮·月令】審端徑術。【三輔決錄】蔣詡舍中三徑，唯羊仲求仲從之游。</p><p>又【廣雅】徑，袤也。【禮·曲禮】送喪不由徑。【註】徑，邪路也。</p><p>又疾也，直也。【史記·大宛傳】張騫曰：今使大夏，從羌中險，從蜀宜徑。【註】如淳曰：徑，疾也。【唐書·盧藏用傳】仕宦之捷徑。</p><p>又直波也。【爾雅·釋水】直波爲徑。【註】言徑涏。</p><p>又【集韻】徑，直也。【禮·檀弓】有直情而徑行者。</p><p>又猶行也。【左傳·僖二十五年】晉趙衰以壺飱從徑，餒而弗食。</p><p>又【集韻】堅靈切，音經。行過也。【史記·高祖紀】高祖被酒，夜徑澤中。</p><p>又【韻會】通作俓。【史記·司馬相如傳】俓峻赴險。</p><p>又與竟通。【史記·淳于髡傳】不過一斗，徑醉矣。</p>
