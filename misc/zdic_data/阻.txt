                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>阻</strong> <span class="dicpy">zǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zǔ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从阜( <span class="dicpy">jù</span>)声，且( <span class="dicpy">jǔ</span>)声。<span class="yhcs">“阜”</span>是土山，与高下险阻有关。本义:险要的地方)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[dangerous place]</span></p><p><span class="diczx1">阻，险也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">德行恒简以知阻。——<span class="smcs">《易·系辞》</span></span></p><p><span class="diczx1">黎民阻饥。——<span class="smcs">《书·舜典》</span></span></p><p><span class="diczx1">司险掌九州之图，以周知川泽之阻。——<span class="smcs">《周礼·司险》</span></span></p><p><span class="diczx1">左据</span><span class="diczx2">函谷</span>、二崤之阻，表以 <span class="diczx2">太华</span>、 <span class="diczx2">终南</span>之山。—— <span class="diczx2">汉</span>· <span class="diczx2">班固</span><span class="smcs">《西都赋》</span></p><p><span class="diczx1">罙入其阻。——<span class="smcs">《诗·商颂·殷武》</span></span></p><p><span class="cino">(3)</span> 又如:阻隘(险要之地);阻阨(据守险要)</p><p><span class="cino">(4)</span> 妨碍物;障隔 <span class="encs">[hindrance;resistance]</span></p><p><span class="diczx1">保其岩阻。——<span class="smcs">《三国志·诸葛亮传》</span></span></p><p><span class="cino">(5)</span> 又如;险阻(险要阻塞之地);阻障(障碍);阻寻(排除险阻寻访);阻积(险阻众多)</p><p><strong>词性变化</strong></p><p>◎ <strong>阻</strong> <span class="dicpy">zǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zǔ.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 地形崎岖多阻隔，道路难行 <span class="encs">[(of road) dangerous and difficult;not level]</span></p><p><span class="diczx1">溯洄从之，道阻且长。——<span class="smcs">《诗·秦风·蒹葭》</span></span></p><p><span class="diczx1">不知山林险阻、沮泽之形者，不能行军。——<span class="smcs">《孙子·军争》</span></span></p><p><span class="cino">(2)</span> 又如:阻修(阻隔而长远);阻长(道路艰险而遥远);阻深(险阻幽深)</p><p><span class="cino">(3)</span> 沮丧。也作<span class="yhcs">“沮”</span> <span class="encs">[dejected]</span></p><p><span class="diczx1">我之怀矣，自诒伊阻。——<span class="smcs">《诗·邶风·雄雉》</span></span></p><p><span class="diczx1">其人惭阻，即日去</span><span class="diczx2">苏</span>。——<span class="smcs">《清稗类钞》</span></p><p><span class="cino">(4)</span> 又如:意阻(意志消沉沮丧);阻丧(沮丧失色)</p><p>◎ <strong>阻</strong> <span class="dicpy">zǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zǔ.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 阻止;阻隔;阻拦 <span class="encs">[hinder;block;impede;resist]</span></p><p><span class="diczx1">夫仆俱阻。——<span class="smcs">《徐霞客游记·游黄山记》</span></span></p><p><span class="diczx1">阻风不行。——</span><span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《过小孤山大孤山》</span></p><p><span class="diczx1">幸亡阻也。——</span><span class="diczx2">明</span>· <span class="diczx2">宗臣</span><span
