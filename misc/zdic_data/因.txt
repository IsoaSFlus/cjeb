                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>因</strong></p><p><span class="dicpy">yīn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yīn.mp3"></a></span>  ㄧㄣˉ</span></p><ol>    <li>原故，原由，事物发生前已具备的条件：原～。～素。～果。病～。</li>    <li>理由：～为（<span class="dicpy">wèi</span>）。～而。</li>    <li>依，顺着，沿袭：～此。～之。～循（ａ．沿袭；ｂ．迟延拖拉）。～噎废食。陈陈相～。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> cause, reason; by; because (of)</p><p><span class="z_ts2">德语</span> Grund, Ursache  (S)​,folgen, fortführen  (V, Lit)​,aufgrund, nach, gemäß  (Konj)​,weil, wegen, infolge  (Konj)​,Yin  (Eig, Fam)</p><p><span class="z_ts2">法语</span> cause,raison,à cause de,selon</p></div>

                    <h2 class="h2_entry"><span class="orth">因</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>因</strong></p><p>囙 <span class="dicpy">yīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yīn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。从囗( <span class="dicpy">wéi</span>)大。大，人。甲骨文字形，象人在车席子上。本义:<span class="yhcs">“茵”</span>的本字，坐垫，车垫)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[mat;cushi on]</span></p><p><span class="diczx1">因，就也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">朱骏声</span>注:<span class="yhcs">“囗大俱非义。 <span class="diczx2">江氏永</span>曰:‘象茵褥之形，中象缝线文理。’按即茵之古文。 <span class="diczx2">江</span>说是也。”</span></p><p><span class="cino">(3)</span> 原因 <span class="encs">[cause]</span></p><p><span class="diczx1">贵贱虽复殊途，因果竟在何处?——<span class="smcs">《梁书·范缜传》</span></span></p><p><span class="diczx1">自谓得其势，无因有动摇。——</span><span class="diczx2">白居易</span><span class="smcs">《有木》</span>诗</p><p><span class="cino">(4)</span> 又如:因果报应(佛教指物有起因必有结果，善因得善果，恶因得恶果);前因后果</p><p><span class="cino">(5)</span> 机会 <span class="encs">[opportunity;chance]</span></p><p><span class="diczx1">于今无会因。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>因</strong> <span class="dicpy">yīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yīn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 依靠;凭借 <span class="encs">[depend on;rely on]</span></p><p><span class="diczx1">谁因谁极。——<span class="smcs">《诗·傭风·载驰》</span></span></p><p><span class="diczx1">论世之事，因为之备。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="diczx1">因人成事者。——<span class="smcs">《史记·平原君虞卿列传》</span></span></p><p><span class="diczx1">因厚币用事者臣</span><span class="diczx2">靳尚</span>。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="cino">(2)</span> 又如:因缘(凭借;依据)因赃假位(利用贿赂买得官职);因公假私(借公家的名义以谋取私利);因依(倚傍;依托);因附(依附);因国(可以依靠、利用的国家)</p><p><span class="cino">(3)</span> 沿袭，承袭 <span class="encs">[follow]</span></p><p><span class="diczx2">殷</span>因于 <span class="diczx2">夏</span>礼，所损益可知也。——<span class="smcs">《论语·为政》</span></p><p><span class="diczx1">因遗策。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《过秦论》</span></p><p><span class="diczx1">因</span><span class="diczx2">秦</span>宫室，据其府库。—— <span class="diczx2">张衡</span><span class="smcs">《东京赋》</span></p><p><span class="cino">(4)</span> 又如:因承(承袭，因袭);因习(相沿成习，沿袭)</p><p><span class="cino">(5)</span> 连接 <span class="encs">[connect]</span></p><p><span
