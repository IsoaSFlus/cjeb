                          <h2 class="h2_entry"><span class="orth">唐</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>唐</strong></p><p><span class="dicpy">táng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/táng.mp3"></a></span> ㄊㄤˊ</span></p><ol><li>夸大，虚夸：荒～。～大无验。</li><li>空，徒然：功不～捐（功夫不白费）。</li><li>传说中的中国朝代名，尧所建。</li><li>朝代名：～代。～诗。～僧。～人。～三彩。后～。</li><li>古代朝堂前或宗庙门内的大路。</li><li>古同“<span class="diczx3">螗</span>”，指蝉。</li><li>姓。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> <p>Tang dynasty; Chinese</p></p><p><span class="z_ts2">德语</span> Tang-Dynastie (618-907)​; daneben kleinere Dynastien desselben Namens  (S, Gesch)​,übertrieben  (Adj)​,umsonst  (Adj)​,Tang  (Eig, Fam)</p><p><span class="z_ts2">法语</span> pour rien,en vain,Tang (dynastie)​</p></div>

                    <h2 class="h2_entry"><span class="orth">唐</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>唐</strong> <span class="dicpy">táng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/táng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。小篆作字形。从口，庚声。本义:大话)</p><p><span class="cino">(2)</span> 〈方〉∶同本义 <span class="encs">[exaggerative;boastful;big words]</span></p><p><span class="diczx1">唐，大言也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">荒唐之言。——<span class="smcs">《庄子·天下》</span></span></p><p><span class="diczx1">唐之为言荡荡也。——</span><span class="diczx2">王充</span><span class="smcs">《论衡》</span></p><p><span class="cino">(3)</span> 广大 <span class="encs">[vast]</span></p><p><span class="diczx1">平原唐其坛曼兮。——</span><span class="diczx2">扬雄</span><span class="smcs">《甘泉赋》</span></p><p><span class="cino">(4)</span> 又如:唐唐(广大;浩荡);唐皇(气势宏伟盛大)</p><p><span class="cino">(5)</span> 在温室中生长的，人工培育的 <span class="encs">[hothouse]</span>。如:唐花;唐窖</p><p><span class="cino">(6)</span> 空;徒然 <span class="encs">[in vain;for nothing;to no avail]</span>。如:唐捐(虚耗;废弃;落空);唐劳(白白地劳动);唐丧(徒劳)</p><p><strong>词性变化</strong></p><p>◎ <strong>唐</strong> <span class="dicpy">táng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/táng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 朝代名。唐朝 <span class="encs">[Tang Dynasty]</span></p><p><span class="diczx2">唐</span>人尚未盛为之。—— <span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="diczx1">刻</span><span class="diczx2">唐</span>贤今人诗赋。—— <span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="diczx1">自</span><span class="diczx2">唐</span><span class="diczx2">显庆</span>以来。—— <span class="diczx2">清</span>· <span class="diczx2">姚鼐</span></p><p><span class="diczx1">登泰山记》</span></p><p><span class="diczx1">备于</span><span class="diczx2">唐</span>。—— <span class="diczx2">蔡元培</span><span class="smcs">《图画》</span></p><p><span class="cino">(2)</span> 公元 617年，隋太原留守李渊起兵，次年隋亡，李渊在关中称帝，国号唐，都长安。 907年为后梁朱温所灭。共历二十帝，二百九十年。如:唐巾(唐代皇帝所戴的一种便帽，后土人亦常戴此种帽子);唐体(唐诗的体式风格);唐环(唐玄宗的贵妃杨玉环);唐明皇(即唐玄宗);唐帽(唐巾。唐代帝王的一种便帽。后来士人多用);唐三藏(唐代玄奘法师的俗称);唐家(唐朝)</p><p><span class="cino">(3)</span> 传说中古帝尧政权的称号</p><p><span class="diczx1">唐虞之世。——</span><span class="diczx2">清</span>· <span class="diczx2">黄宗羲</span><span class="smcs">《原君》</span></p><p><span class="cino">(4)</span>                               唐·康熙筆画：<a href="/zd/kx/bh/?bh=10" target="_blank">10</a>　·部外筆画：7</div>
                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/5510.svg">						  <p>〔古文〕啺<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2327A.svg'><span class='k_txt'>𣉺</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/210EF.svg'><span class='k_txt'>𡃯</span>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>徒郎切，音堂。【說文】大言也。从口庚聲。【莊子·天下篇】荒唐之言。</p><p>又【史記·司馬相如·上林賦】瑉玉旁唐。【註】郭璞云：旁唐，盤薄。</p><p>又國名。【玉篇】堯稱唐者蕩蕩，道德至大之貌。【書·五子之歌】惟彼陶唐，有此冀方。【疏】韋昭云：陶，唐皆國名，猶湯稱殷商也。○按書傳皆言堯以唐侯升爲天子，不言封於陶唐。陶唐二字，或共爲地名，未必如昭言也。</p><p>又【詩·唐風疏】唐者，成王母弟叔虞所封其地，帝堯夏禹所都之墟，漢曰太原郡。</p><p>又漢縣名。【前漢·地理志】中山國唐縣。【註】故堯國也，唐水在西。【詩·唐風疏】皇甫謐曰：堯始封於唐，今中山唐縣是也。後徙晉陽，及爲天子，都平陽，於詩爲唐國，則唐國爲平陽也。</p><p>又魯地。【春秋·隱二年】公及戎盟于唐。【註】高平方輿縣北有武唐亭。</p><p>又【戰國策】左濟右天唐。【註】謂高唐。【前漢·地理志】平原郡高唐。【註】桑欽言，漯水所出也。</p><p>又【武帝紀】南巡狩至於盛唐。【註】韋昭曰：在南郡。</p><p>又【地理志】會稽郡錢唐。【註】武林水所出。</p><p>又【後漢·光武紀】進屠唐子鄉。【註】唐子鄉有唐子山，在今唐州湖陽縣西南。【廣韻】唐州，楚地。戰國時屬晉，後入於韓，秦屬南陽郡，後魏爲淮州，隋爲顯州，貞觀攺爲唐州，因唐城山爲名。</p><p>又姓。【廣韻】唐堯之後，子孫氏焉。【史記·秦本紀】孝文王立，尊唐八子爲唐太后。</p><p>又【屈原傳】楚有宋玉，唐勒，景差之徒。</p><p>又唐山，複姓。【前漢·禮樂志】高祖唐山夫人。</p><p>又【詩·<img class='lazy k_img' data-original='//img.zdic.net/song/cn/28EF0.svg'><span class='k_txt'>𨻰</span>風】中唐有甓。【傳】中，中庭也。唐，堂途也。【疏】爾雅釋宮云：廟中路謂之唐，堂途謂之<img class='lazy k_img' data-original='//img.zdic.net/song/cn/28EF0.svg'><span class='k_txt'>𨻰</span>。李巡曰：唐，廟中路名。孫炎引詩中唐有甓。堂途，堂下至門之徑也。然則唐之與<img class='lazy k_img' data-original='//img.zdic.net/song/cn/28EF0.svg'><span class='k_txt'>𨻰</span>，廟庭之異名耳，其實一也。故云：唐，堂途也。【周語】陂唐汚庳，以鍾其美。【註】唐，俗本作塘。【說文】無塘字。【莊子·徐無鬼】其求唐子也，而未始出域，有遺類矣。【註】唐子者，堂途給使令之人，猶周禮云門子耳。</p><p>又草名。【詩·鄘風】爰采唐矣。【傳】唐，蒙菜名。【爾雅·釋草】唐蒙，女蘿。女蘿，菟絲。</p><p>又弓名。【周禮·夏官·司弓矢】唐弓，大弓，以授學射者。</p><p>又叶徒紅切，音同。【歐陽修·楊諫議銘】震官太尉，四世以公。於陵正直，僕射於唐。</p><p>
