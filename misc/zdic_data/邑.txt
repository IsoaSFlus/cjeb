                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>邑</strong></p><p><span class="dicpy">yì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span>  ㄧˋ</span></p><ol><li>城市，都城：城～。都～。</li><li>旧指县：～人（同乡的人）。～庠（明清时称县学）。～宰。</li><li>古代诸侯分给大夫的封地：采～。</li><li>古同“<span class="diczx3">悒</span>”，愁闷不安。</li></ol>

                    <h2 class="h2_entry"><span class="orth">邑</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>邑</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。甲骨文字形。上为囗( <span class="dicpy">wéi</span>)，表疆域，下为跪着的人形，表人口。合起来表城邑。<span class="yhcs">“邑”</span>是汉字的一个部首，变体为右耳旁。从<span class="yhcs">“邑”</span>的字多和地名、邦郡有关。本义:国)</p><p><span class="cino">(2)</span> 古代称侯国为邑 <span class="encs">[state]</span></p><p><span class="diczx1">邑，国也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">段玉裁</span>注:<span class="yhcs">“<span class="smcs">《左传》</span>凡称人曰大国，凡自称曰敞邑。古国邑通称。”</span> <span class="diczx2">朱骏声</span>通训定声:<span class="yhcs">“<span class="smcs">《书》</span>‘西邑夏’、‘天邑商’、‘大邑周’，皆谓国。”</span></p><p><span class="diczx1">君惠徼福于敝邑之社稷。——<span class="smcs">《左传·僖公四年》</span></span></p><p><span class="cino">(3)</span> 又如:敝邑(对本国的谦称)</p><p><span class="cino">(4)</span> 国都，京城 <span class="encs">[capital]</span></p><p><span class="diczx1">邑外谓之郊。——<span class="smcs">《尔雅》</span>。</span><span class="diczx2">郭璞</span>注:邑，国都也</p><p><span class="diczx1">商邑翼翼。——<span class="smcs">《诗·商颂·殷武》</span></span></p><p><span class="cino">(5)</span> (翼翼:整齐的样子)</p><p><span class="diczx1">是以论其迁邑易京，则同规乎殷盘。——</span><span class="diczx2">汉</span>· <span class="diczx2">张衡</span><span class="smcs">《东京赋》</span></p><p><span class="cino">(6)</span> 又如:商邑(商都)。又指旧都</p><p><span class="diczx1">天子治居之城曰都，旧都曰邑也。——</span><span class="diczx2">唐</span>· <span class="diczx2">慧琳</span><span class="smcs">《一切经音义》</span></p><p><span class="cino">(7)</span> 又如:邑庙(城隍庙);邑野(都邑郊野)</p><p><span class="cino">(8)</span> 指古代无先君宗庙的都城 <span class="encs">[city]</span></p><p><span class="diczx1">凡邑，有宗庙先君之主曰都，无曰邑。——<span class="smcs">《左传》</span>。</span><span class="diczx2">孔颖达</span>疏:小邑有宗庙，则虽小曰都，无乃为邑，为尊宗庙，故小邑与大都同名</p><p><span class="cino">(9)</span> 又如:邑郛(城郭);邑门(城门;都邑之门)</p><p><span class="cino">(10)</span> 泛指一般城镇。大曰都，小曰邑 <span class="encs">[city]</span></p><p><span class="diczx1">小则获邑，大则获城——</span><span class="diczx2">宋</span>· <span class="diczx2">苏洵</span><span class="smcs">《六国论》</span></p><p><span class="diczx1">日扳</span><span class="diczx2">仲永</span>环谒于邑人。—— <span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《伤仲永》</span></p><p><span class="diczx1">邑小不足容。——</span><span class="diczx2">清</span>· <span class="diczx2">张廷玉</span><span class="smcs">《明史》</span></p><p><span class="diczx1">摄篆旁邑。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="diczx1">邑有</span><span class="diczx2">成名</span>。——<span class="smcs">《聊斋志异·促织》</span></p><p><span class="diczx1">俾入邑庠。</span></p><p><span class="cino">(11)</span> 又如:邑人(乡邑中的人);邑子(同乡，同邑人);邑屋(村舍);邑落(村落);邑党(乡党);邑闾(里巷的门);邑邻(邻里)</p><p><span class="cino">(12)</span> 旧时县的别称 <span class="encs">[county]</span></p><p><span class="diczx2">秦</span>有天下，裂都会而为之郡邑。—— <span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《封建论》</span></p><p><span class="diczx1">故今之邑民，最独畏旱而旱辄连年。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="diczx2"><span class="smcs">《上杜学士言开河书》</span></span></p><p><span
