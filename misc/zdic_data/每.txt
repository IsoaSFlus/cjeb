                          <h2 class="h2_entry"><span class="orth">每</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>每</strong></p><p><span class="dicpy">měi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/měi.mp3"></a></span>  ㄇㄟˇ</span></p><ol><li>指特定范围内的任何一个或一组：～人。～回。～组。</li><li>指特定范围内逐一出现的任何一次：～战必胜。～况愈下。</li><li>虽：“～有良朋，况也永叹”（虽有良朋好友，危难之时也只能为之长叹）。</li><li>古同“<span class="diczx3">们</span>”，中国宋元代口语。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> every, each</p><p><span class="z_ts2">德语</span> je, pro,oft, wiederholt,jeder (jede, jedes)​ (Pron)</p><p><span class="z_ts2">法语</span> chaque,chacun,tous</p></div>

                    <h2 class="h2_entry"><span class="orth">每</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>每</strong> <span class="dicpy">měi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/měi.mp3"></a></span> </p><p>〈形〉</p><p>(形声。小篆字形，从屮( <span class="dicpy">chè</span>)，母声。屮，草木初生。一说<span class="yhcs">“每”</span>为<span class="yhcs">“母”</span>的异体字。本义:草旺盛生长) 同本义 <span class="encs">[flourishly]</span></p><p><span class="diczx1">每，草盛上出也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">舆人诵曰:原田每每。——<span class="smcs">《左传·僖公二十八年》</span>。</span><span class="diczx2">杜</span>注:<span class="yhcs">“ <span class="diczx2">晋</span>军美盛，若原田之草每每然。”</span></p><p><strong>词性变化</strong></p><p>◎ <strong>每</strong> <span class="dicpy">měi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/měi.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 常常，经常 <span class="encs">[often;frequently]</span></p><p><span class="diczx1">妪每次谓余曰。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="diczx1">每大言曰。——</span><span class="diczx2">·</span><span class="diczx2">宗臣</span><span class="smcs">《宗子相先生集·报刘一丈书》</span></p><p><span class="cino">(2)</span> 又如:每常间(往日，平常);春秋佳日，每作郊游</p><p><span class="cino">(3)</span> 每一次 <span class="encs">[every time]</span></p><p><span class="diczx1">相如每朝。——<span class="smcs">《史记·廉颇蔺相如列传》</span></span></p><p><span class="diczx1">每与臣论。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p><span class="diczx1">每得降卒。——<span class="smcs">《资治通鉴·唐纪》</span></span></p><p><span class="diczx1">每冒风驰行。——</span><span class="diczx2">明</span>· <span class="diczx2">袁宏道</span><span class="smcs">《满井游记》</span></p><p><span class="cino">(4)</span> 又如:他每到学校，总要来看我;月刊每逢15出版</p><p>◎ <strong>每</strong> <span class="dicpy">měi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/měi.mp3"></a></span> </p><p>〈代〉</p><p><span class="cino">(1)</span> 各个 <span class="encs">[every;each]</span></p><p><span class="diczx1">每韵为一帖，木格贮之(意思是，把字按韵分类，分别放在木格里)。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="diczx1">每字为一印。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="diczx1">每一巨弹。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="cino">(2)</span> 又如:每字有二十余印;每天;每一分钱;每星期五</p><p>◎ <strong>每</strong> <span class="dicpy">měi</span> <span class="ptr"><a class="audio_play_button
