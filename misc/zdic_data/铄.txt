                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>铄</strong></p><p><strong>（<span class="diczx3">鑠</span>）</strong></p><p><span class="dicpy">shuò  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shuò.mp3"></a></span>  ㄕㄨㄛˋ</span></p><ol><li>熔化金属：众心成城，众口～金。</li><li>消毁，消损：“秦先得齐宋，则韩氏～”。</li><li>同“<span class="diczx3">烁</span>”。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> melt, smelt; shine</p><p><span class="z_ts2">德语</span> schmelzen, verschmelzen, einschmelzen  (V)</p><p><span class="z_ts2">法语</span> fondre</p></div>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>铄</strong></p><p>鑠 <span class="dicpy">shuò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shuò.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从金，乐声。本义:熔化)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[melt]</span></p><p><span class="diczx1">铄，销金也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">流金铄石些。——<span class="smcs">《楚辞·招魂》</span></span></p><p><span class="diczx1">夫铄金在炉，</span><span class="diczx2">庄蹻</span>不顾。——<span class="smcs">《盐铁论·诏圣》</span></p><p><span class="diczx1">大热铄石流金。——<span class="smcs">《淮南子·诠言训》</span></span></p><p><span class="diczx1">人无筋骨之强，爪牙之利，故割革而为甲，铄铁而为刃。——<span class="smcs">《淮南子·兵略》</span></span></p><p><span class="cino">(3)</span> 又如:铄金(销熔金属);铄石流金(使金石熔化。比喻酷热。同<span class="yhcs">“流金铄石”</span>);铄化(熔化);铄金点玉(熔化金属，玷污美玉)</p><p><span class="cino">(4)</span> 销毁 <span class="encs">[destroy]</span></p><p><span class="diczx1">彼人含其明，则天下铄矣。——<span class="smcs">《庄子·胠箧》</span></span></p><p><span class="cino">(5)</span> 又如:铄绝竽瑟(完全销毁乐器)</p><p><span class="cino">(6)</span> 削弱 <span class="encs">[weaken]</span></p><p><span class="diczx2">秦</span>先得 <span class="diczx2">齐</span><span class="diczx2">宋</span>，则 <span class="diczx2">韩氏</span>铄， <span class="diczx2">韩氏</span>铄则 <span class="diczx2">楚</span>孤而受兵也。——<span class="smcs">《战国策》</span></p><p><span class="cino">(7)</span> 渗入 <span class="encs">[permeate]</span></p><p><span class="diczx1">仁义礼智，非由外铄我也，我固有之也。——<span class="smcs">《孟子·告子上》</span></span></p><p><span class="cino">(8)</span> 毁谤 <span class="encs">[slander]</span>。如:铄金毁骨(比喻毁谤太多，使人无以自存)</p><p><span class="cino">(9)</span> 同<span class="yhcs">“烁”</span>。灼烁，光彩夺目 <span class="encs">[blaze]</span>。如:铄闪(闪烁);铄铄(光芒闪耀的样子)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>铄</strong></p><p>鑠 <span class="dicpy">shuò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shuò.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 明亮，光明 <span class="encs">[bright]</span></p><p><span class="diczx1">故其华表则镐镐铄铄，赫奕章灼若日月之丽天也。——</span><span class="diczx2">魏</span>· <span class="diczx2">何晏</span><span class="smcs">《景福殿赋》</span></p><p><span class="cino">(2)</span> 又如:铄亮(非常明亮)</p><p><span class="cino">(3)</span> 美好的样子 <span class="encs">[fine]</span>。如:铄懿渊积(德行美好，学问渊博)</p><p><span class="cino">(4)</span> 光辉美盛的样子 <span class="encs">[beautiful]</span>。如:铄颖(光辉美盛，颖秀突出)</p>                    <div class="div copyright"> © 汉典 </div>
