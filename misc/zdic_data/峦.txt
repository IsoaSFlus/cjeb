                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>峦</strong></p><p>巒 <span class="dicpy">luán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/luán.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从山，羉( <span class="dicpy">luán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/luán.mp3"></a></span> )声。本义:小而尖的山)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[small but pointed hill]</span></p><p><span class="diczx1">峦，山小而锐者。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">登峦而远望兮。——<span class="smcs">《楚辞·自悲》</span>。注:<span class="yhcs">“小山也。”</span></span></p><p><span class="diczx1">陟玉峦兮逍遥。——<span class="smcs">《楚辞·守志》</span>。注:<span class="yhcs">“山脊曰峦。玉峦，</span><span class="diczx2">昆仑山</span>也。”</span></p><p><span class="diczx1">丘峦崩摧。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span><span class="smcs">《梦游天姥吟留别》</span></p><p><span class="cino">(3)</span> 又如:山峦(连绵的山);峦壑(峰峦和山谷);峦峤(峭拔的山峰)</p><p><span class="cino">(4)</span> 泛指山 <span class="encs">[mountain]</span></p><p><span class="diczx1">谕以封峦。——<span class="smcs">《史记·司马相如传》</span>。集解:<span class="yhcs">“山也。”</span></span></p><p><span class="diczx1">簸丘跳峦。——<span class="smcs">《汉书·扬雄传》</span></span></p><p><span class="diczx1">陵峦超壑。——</span><span class="diczx2">张衡</span><span class="smcs">《西京赋》</span></p><p><span class="diczx1">桂殿兰宫，列冈峦之体势。——</span><span class="diczx2">王勃</span><span class="smcs">《滕王阁序》</span></p><p><span class="cino">(5)</span> 又如:峦丘(山丘);峦穴(山穴);峦谷(山谷)</p><p><span class="cino">(6)</span> 山脉，相连的群山 <span class="encs">[mountains in a range]</span></p><p><span class="diczx1">云中乱拍</span><span class="diczx2">禄山</span>舞，风过重峦下笑声。—— <span class="diczx2">唐</span>· <span class="diczx2">杜牧</span><span class="smcs">《过华清宫绝句三首》</span>之三</p><p><span class="cino">(7)</span> 山脊、山梁 <span class="encs">[mountain ridge]</span></p><p><span class="diczx1">城堡冈峦。——</span><span class="diczx2">清</span>· <span                           『說文解字注』<hr class="dichr">
                          <span class="jieshi">山小而銳。</span>釋山曰。巒山墮。說者謂爲一條。許於巒墮別爲二條。毛公釋嶞亦不云巒也。葢爾雅巒嶞二物。許云山小而銳、巒之解也。毛云山之隋隋小者、嶞之解也。後儒合爲一者、非是。劉淵林注蜀都賦曰。巒、山長而狹也。一曰山小而銳也。兼用爾雅、說文。而爾雅之讀與毛許異矣。<span class="jieshi">从山。䜌聲。</span>洛官切。十四部。                          </div>
