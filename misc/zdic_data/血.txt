                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>血</strong></p><p><span class="dicpy">xuè  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xuè.mp3"></a></span> ㄒㄩㄝˋ</span></p><ol><li>人或动物体内循环系统的不透明液体，大多为红色，主要成分为“血浆”、“血细胞”和“血小板”。味咸而腥：～型。～脂。～压。～糖。～迹。～汗。～泪。～洗。～书。～雨腥风。～海深仇。</li><li>人类因生育而自然形成的关系：～统。～缘。</li><li>喻刚强热烈：～性。～气方刚。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>血</strong></p><p><span class="dicpy">xiě  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiě.mp3"></a></span> ㄒㄧㄝˇ</span></p><ol><p>　◎ 义同“血”（xuè），用于口语。多单用，如“流了点儿血”。也用于口语常用词，如“鸡血”、“血块子”。</p></ol>

                    <h2 class="h2_entry"><span class="orth">血</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>血</strong> <span class="dicpy">xiě</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiě.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> <span class="encs">[口]</span>∶<span class="yhcs">“血”</span>( <span class="dicpy">xuè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xuè.mp3"></a></span> )的口语读音(多单用) <span class="encs">[blood]</span>。如:鸡血;血块子;血沥沥(血淋淋);血唬零喇(血糊淋粒。血淋淋的);血胡同(血湖同。于人丛中杀出的血路);血糊突(流血很多，像粥样);血渌渌(鲜血淋漓);血心(忠心;热忱);血的教训;流了一点血，不要紧</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">xuè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xuè.mp3"></a></span> </p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><hr class="dichr" /><span class="crefe"><a href="/hans/血豆腐">血豆腐</a></span><span class="crefe"><a href="/hans/血糊糊">血糊糊</a></span><span class="crefe"><a href="/hans/血淋淋">血淋淋</a></span><span class="crefe"><a href="/hans/血晕">血晕</a></span></div>　<p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>血</strong> <span class="dicpy">xuè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xuè.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (指事。小篆字形，从皿，<span class="yhcs">“一”</span>象血形。表示器皿中盛的是血。本义:牲血，供祭祀用)</p><p><span class="cino">(2)</span> 同本义。后泛指血液 <span class="encs">[blood]</span></p><p><span class="diczx1">血，祭所荐牲血也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">取其血膋。——<span class="smcs">《诗·信南山》</span>。笺:<span class="yhcs">“血以告杀也。”</span></span></p><p><span class="diczx1">以血祭祭社稷、五祀五岳。——<span class="smcs">《周礼·大宗伯》</span></span></p><p><span class="diczx1">血祭盛气也。——<span class="smcs">《礼记·郊特性》</span></span></p><p><span class="diczx1">叩其鼻以血社也。——<span class="smcs">《公羊传·僖公十九年》</span></span></p><p><span class="diczx1">原野厌人之肉，川谷流人之血。——</span><span class="diczx2">汉</span>· <span class="diczx2">扬雄</span><span class="smcs">《法言》</span></p><p><span class="cino">(3)</span> 又如:狗血喷头;贫血;呕心沥血;心血;血海的干系(关系非常重大的责任);血映征袍(喻战斗激烈);血肉蹀躞(死尸阻路，只能小步慢行)</p><p><span class="cino">(4)</span> 指妇女的月经 <span class="encs">[menses]</span></p><p><span
