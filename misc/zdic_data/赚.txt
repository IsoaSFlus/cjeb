						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>赚</strong></p><p><strong>（<span class="diczx3">賺</span>）</strong></p><p><span class="dicpy">zhuàn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàn.mp3"></a></span> ㄓㄨㄢˋ</span></p><ol><li>做买卖，与“<span class="diczx3">赔</span>”相对：～钱。</li><li>做买卖得的利：～头。</li><li>占便宜：这场电影看得真～，他给买票又买吃的。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>赚</strong></p><p>賺 <span class="dicpy">zhuàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从贝，兼声。从贝，与财富有关。本义:指买卖得盈利)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[make a profit;gain]</span></p><p><span class="diczx1">我也许过他二十两银子，只恨他没本事赚我的钱。——<span class="smcs">《警世通言》</span></span></p><p><span class="cino">(3)</span> 又如:赚手(获利);赚得(获得 <span class="encs">[利润]</span>);赚钱(获取利润);赚得不少;迅速赚一笔现金;每出售一桶可赚五元钱</p><p><span class="cino">(4)</span> 赢得;获得 <span class="encs">[gain;get]</span></p><p><span class="diczx2">西州</span>客，心边赚得，一味春偏。—— <span class="diczx2">元</span>· <span class="diczx2">刘敏中</span><span class="smcs">《凤凰台上忆吹箫》</span></p><p><span class="cino">(5)</span> 又如:赚杀(赢得，博得)</p><p><span class="cino">(6)</span> 〈方〉∶通过劳动挣<span class="encs">[钱]</span> <span class="encs">[earn]</span></p><p><span class="diczx1">年轻的时候，只帮人种地看牛，赚几个微细的工资。——</span><span class="diczx2">郁达夫</span><span class="smcs">《出奔》</span></p><p><span class="cino">(7)</span> 出错误 <span class="encs">[be wrong]</span></p><p><span class="diczx1">讶，赚矣!此辟谷药也。——</span><span class="diczx2">徐铉</span><span class="smcs">《稽神录拾遗》</span></p><p><span class="diczx1">然已只在康庄大道中，决不赚入蹊曲径矣。——</span><span class="diczx2">王阳明</span><span class="smcs">《传习录·答聂文蔚》</span></p><p><span class="cino">(8)</span> 耽误;错过 <span class="encs">[miss]</span></p><p><span class="diczx1">向日我一时见不到，赚了你终身。——<span class="smcs">《醒世恒言》</span></span></p><p><span class="cino">(9)</span> 诳骗;欺哄 <span class="encs">[cheat;deceive]</span></p><p><span class="diczx2">吕布</span>赚开城门，杀将进来了!——<span class="smcs">《全图绣像三国演义》</span></p><p><span class="diczx1">当时亲见元帅命中军去赚</span><span class="diczx2">刘豫</span>。——<span class="smcs">《说岳全传》</span></p><p><span class="cino">(10)</span> 又如:赚虎离山(比喻施用计谋骗对方离开其所凭借的有利位置);赚铜钱(骗钱);赚漏(骗取);赚辞(骗人的话);赚哄(欺骗);赚人(欺骗人)</p><p><span class="cino">(11)</span> 另见 <span class="dicpy">zuàn</span></p><p><strong>常用词组</strong></p><hr class="encs">[deceive;hoax;kid]</span></p><p><span class="diczx1">是我赚你们来，捉了请赏。——<span class="smcs">《水浒全传》</span></span></p><p><span class="cino">(2)</span> 又如:赚法(骗人的手法);赚杀(诳骗杀害);赚啜(哄骗);赚得(骗取)</p><p><span class="cino">(3)</span> 另见 <span class="dicpy">zhuàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
