                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>拥</strong></p><p>擁 <span class="dicpy">yōng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yōng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，雍声。本义:抱)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[hug]</span></p><p><span class="diczx1">闭门，聚众鸣鼓。众恐，拥柱而歌。——</span><span class="diczx2">汉</span>· <span class="diczx2">刘向</span><span class="smcs">《列女传》</span></p><p><span class="cino">(3)</span> 又如:拥树(拥抱小孩;环抱树木);拥矢(古代行投壶礼时，卑者抱矢而投);拥社(抱持社主);拥持(拥抱);拥膝(抱膝)</p><p><span class="cino">(4)</span> 簇拥 <span class="encs">[environ]</span></p><p><span class="diczx1">诸将所拥。——</span><span class="diczx2">清</span>· <span class="diczx2">侯方域</span><span class="smcs">《壮悔堂文集》</span></p><p><span class="diczx1">蜂拥而上。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="diczx1">乡民蚁拥。——<span class="smcs">《广东军务记》</span></span></p><p><span class="cino">(5)</span> 又如:前呼后拥;一群人拥着老教授走出来;拥攒(围拢);拥舍(厅堂边的小屋)</p><p><span class="cino">(6)</span> 环抱 <span class="encs">[surround]</span></p><p><span class="diczx1">回首孤城，依约青山拥。——</span><span class="diczx2">金</span>· <span class="diczx2">董解元</span><span class="smcs">《西厢记诸宫调》</span></p><p><span class="diczx1">旗翻光复照</span><span class="diczx2">神州</span>，虎踞龙蟠拥 <span class="diczx2">石头</span>。—— <span class="diczx2">柳亚子</span><span class="smcs">《题太平天国战史》</span></p><p><span class="cino">(7)</span> 又如:拥列(环抱耸列);拥炉(围炉取暖);拥带(屏障环绕)</p><p><span class="cino">(8)</span> 持;执持 <span class="encs">[hold]</span></p><p><span class="diczx2">哙</span>即带剑拥盾入军门。——<span class="smcs">《史记·项羽本纪》</span></p><p><span class="diczx1">拥至尹前。——</span><span class="diczx2">宋</span>· <span class="diczx2">胡仔</span><span class="smcs">《苕溪渔隐丛话》</span></p><p><span class="diczx1">拥火以入。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="diczx1">拥豪民马前。——</span><span class="diczx2">明</span>· <span class="diczx2">高启</span><span class="smcs">《书博鸡者事》</span></p><p><span class="diczx1">张黄盖拥纛。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="cino">(9)</span>
