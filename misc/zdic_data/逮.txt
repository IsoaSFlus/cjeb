                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>逮</strong> <span class="dicpy">dǎi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dǎi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> <span class="encs">[口]</span>∶捉，特指经过追赶或试图抓住 <span class="encs">[catch;capture]</span>。如:逮蚊子;逮特务;猫逮老鼠</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>逮</strong> <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，隶( <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> )声。本义:赶上;及;到)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[overtake;catch sb. up;catch up with]</span></p><p><span class="diczx1">逮，唐逮及也。——<span class="smcs">《说文》</span>。按，逮者，行相及也。古曰唐逮。</span></p><p><span class="diczx1">逮于袁娄而与之盟。——<span class="smcs">《公羊传·成公二年》</span></span></p><p><span class="diczx1">群后之逮在下。——<span class="smcs">《书·吕刑》</span></span></p><p><span class="diczx1">逮事父母。——<span class="smcs">《礼记·曲礼》</span></span></p><p><span class="diczx1">上不及</span><span class="diczx2">尧</span>、 <span class="diczx2">舜</span>，下不逮 <span class="diczx2">周</span>、 <span class="diczx2">孔</span>，亦一时之懿士。——<span class="smcs">《世说新语·排调》</span></p><p><span class="diczx1">纵轻体以迅赴，景追形而不逮。——</span><span class="diczx2">曹植</span><span class="smcs">《七启》</span></p><p><span class="diczx1">政逮于大夫四世矣。——<span class="smcs">《论语·季氏》</span></span></p><p><span class="diczx1">敌出不意，大惊扰，以枪上刺刀相搏击，而便捷猛鸷终弗逮。——</span><span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p><span class="cino">(3)</span> 又如:逮及(达到，到了某种程度);逮夜(到了晚上);逮至(及至;等到)</p><p><span class="cino">(4)</span> 逮捕，抓人(人在捉拿称<span class="yhcs">“逮”</span>，人逃亡而追拿称<span
