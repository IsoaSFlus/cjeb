						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>逾</strong></p><p><span class="dicpy">yú  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> ㄩˊ</span></p><ol><li>越过，超过：～期。～常（超过寻常）。～分（<span class="dicpy">fèn </span>）（过分）。～越。～恒（超过寻常）。</li><li>更加：～甚。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>逾</strong> <span class="dicpy">yú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，俞声。本义:越过，经过)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[cross;go through]</span></p><p><span class="cino">(3)</span> 又如:逾迈(走过;时光易逝);逾墙钻穴(指男女偷情;跳墙钻洞);逾墉(越过城墙);逾墙(越过矮墙);逾假不归(即假期已满而未归营);逾轨(越轨)</p><p><span class="cino">(4)</span> 超过;胜过 <span class="encs">[surpass;exceed]</span></p><p><span class="diczx1">无敢昏逾。——<span class="smcs">《书·顾命》</span></span></p><p><span class="diczx1">日月逾迈。——<span class="smcs">《书·秦誓》</span></span></p><p><span class="diczx1">物或逾秋不死，亦如人年多度百至于三百也。——</span><span class="diczx2">汉</span>· <span class="diczx2">王充</span><span class="smcs">《论衡·气寿》</span></p><p><span class="diczx1">越职逾法，以取名誉。——<span class="smcs">《汉书·宣帝纪》</span></span></p><p><span class="diczx1">去逾四十里，之虚所卖之。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《童区寄传》</span></p><p><span class="cino">(5)</span> 又如:逾千(超过一千);逾出(突出，超过一般);逾立(超过三十岁);逾世(超过当代人);逾冠(超过二十岁)</p><p><span class="cino">(6)</span> 指晋升职位 <span class="encs">[promote]</span></p><p><span class="diczx1">臣不逾级，则主位安。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《服疑》</span></p><p><span data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“愈”</span>。更加 <span class="encs">[even more]</span></p><p><span data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 远 <span class="encs">[far]</span></p><p><span class="diczx1">福逾刺凤。——<span
