                          <h2 class="h2_entry"><span class="orth">纬</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>纬</strong></p><p><strong>（<span class="diczx3">緯</span>）</strong></p><p><span class="dicpy">wěi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> ㄨㄟˇ</span></p><ol><li>织布时用梭穿织的横纱，编织物的横线，与“<span class="diczx3">经</span>”相对：～线。～纱。经～。</li><li>地理学上指在地面上的与赤道平行的线：～度。</li><li>东西的横路：～陌。</li><li>行星的古称：～星。</li><li>书名：古七～（指中国汉代以神学迷信附会儒家经义的书）。</li><li>编织：～萧（用蒿草编成帘子）。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>纬</strong></p><p>緯 <span class="dicpy">wěi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从糸( <span class="dicpy">mì</span>)，韦声。本义:织横纱)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[weave weft]</span></p><p><span class="diczx1">纬，织横丝也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">恃纬萧而食者。——<span class="smcs">《庄子·列御寇》</span>。释文:<span class="yhcs">“织也。”</span></span></p><p><span class="cino">(3)</span> 又如:纬萧(用蒿草编织窗帘子)</p><p><span class="cino">(4)</span> 缠捆，系束 <span class="encs">[tie]</span>。如:纬狗(拴狗)</p><p><span class="cino">(5)</span> 组织 <span class="encs">[organize]</span>。如:纬文(安排组织文章)</p><p><span class="cino">(6)</span> 治理 <span class="encs">[administer]</span>。如:纬世(治理天下。同经世、治世);纬武经文(指人文武兼备，有治世之才);纬地经天(治理国家)</p><p><strong>词性变化</strong></p><p>◎ <strong>纬</strong></p><p>緯 <span class="dicpy">wěi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 纬纱;织物的横丝 <span class="encs">[weft]</span>，它与经纱交叉，从一匹布的织边到织边。与<span class="yhcs">“经”</span>相对。如:纬经(犹经纬。纵横交错);纬编针织物</p><p><span class="cino">(2)</span> 东西向的道路或土地。与南北向的<span class="yhcs">“经”</span>相对 <span class="encs">[latitude]</span></p><p><span class="diczx1">凡地东西为纬。——<span class="smcs">《大戴礼记·易本命》</span></span></p><p><span class="diczx1">正督经纬。——<span class="smcs">《周髀算经》</span></span></p><p><span class="cino">(3)</span> 又如:纬途(古代都城内的东西大道)</p><p><span class="cino">(4)</span> 行星的古称 <span class="encs">[planet]</span></p><p><span class="diczx1">五纬相汁。——<span class="smcs">《西京赋》</span>。注:<span class="yhcs">“五星也。”</span></span></p><p><span class="cino">(5)</span> 又如:纬象(星象);纬星(行星的总称)</p><p><span class="cino">(6)</span> 琴徽，系琴弦的绳子 <span class="encs">[cord]</span></p><p><span class="diczx1">挟人筝而弹纬。——<span class="smcs">《楚辞·愍命》</span>。</span><span class="diczx2">王逸</span>注:<span class="yhcs">“纬，张弦也”</span>。</p><p><span class="cino">(7)</span> 地理学上所假想为地球上与赤道平行的南北分度线。在赤道以北的叫北纬，以南的叫南纬 <span class="encs">[latitude]</span></p><p><span class="diczx1">国中九经九纬。——<span class="smcs">《周礼·考工记·匠人》</span></span></p><p><span class="cino">(8)</span> <span class="yhcs">“纬书”</span>的简称。神学附会儒家经典的书 <span class="encs">[Wei book]</span>。如:纬谶(纬书与<span class="yhcs">“论语谶”</span>、<span class="yhcs">“河图”</span>、<span class="yhcs">“洛书”</span>等谶文的合称);纬候(纬书与<span class="smcs">《尚书中候》</span>的合称。亦为纬书的通称);纬术(纬书之学);纬图(纬书图谶之类的统称);纬说(纬书的学说)</p><p><span class="cino">(9)</span> 古筝上的弦 <span class="encs">[string]</span>。如:弹纬(弹弦)</p><p><strong>常用词组</strong></p><hr                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/7DEF.svg">						  <p>【廣韻】【集韻】【韻會】于貴切【正韻】于位切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音胃。【說文】織橫絲也。【釋名】緯，圍也。反覆圍繞，以爲經也。【左傳·昭二十四年】嫠不恤其緯。【註】織者常苦緯少。【莊子·列禦<img class='lazy k_img' data-original='//img.zdic.net/song/cn/21A25.svg'><span class='k_txt'>𡨥</span>】江上有家貧緯蕭而食者。【音義】緯，織也。</p><p>又【周禮·天官·冢宰體國經野疏】南北之道謂之經，東西之道謂之緯。</p><p>又【周禮·春官·大宗伯日月星辰註】星謂五緯。【疏】五緯，卽五星。言緯者，二十八宿隨天左轉爲經，五星右旋爲緯。</p><p>又圖緯。【司馬貞·三皇本紀】圖緯所載。</p><p>又【集韻】羽鬼切，音偉。【博雅】束也。【夏小正】農緯厥耒。</p><p>又【韻補】叶呼韋切。【劉向·九歎】慶忌囚于阱室兮，陳不占戰而赴圍。破伯牙之號鐘兮，挾人箏而彈緯。</p>
