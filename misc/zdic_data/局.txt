                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>局</strong> <span class="dicpy">jú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jú.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (会意。从口，从尺。<span class="yhcs">“尺”</span>示规矩法度。口易出错，故以尺相拘束。本义:局促)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[feel or show constraint]</span></p><p><span class="diczx1">局，促也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">不敢不局。——<span class="smcs">《诗·小雅·正月》</span></span></p><p><span class="diczx1">意局而辞野。——</span><span class="diczx2">潘民</span><span class="smcs">《乘舆箴》</span></p><p><span class="cino">(3)</span> 又如:局守(谨守);局定(限制，固定);局囿(局限，限制);局滞(局限停滞);局蹐(局限，局束);局脊(拘泥，拘束);局数(局束);局趣(拘束。同局促);局局蹐蹐(畏缩不安的样子);局促乡里(碌碌无为地呆在乡里)</p><p><span class="cino">(4)</span> 距离近 <span class="encs">[near;close]</span></p><p><span class="diczx1">涂路虽局，官守有限。——</span><span class="diczx2">曹丕</span><span class="smcs">《与吴质书》</span></p><p><span class="cino">(5)</span> 狭小，狭隘 <span class="encs">[narrow]</span>。如:局室(小室);局缩(狭窄);局迫(狭隘，狭窄);局戚(狭隘);局窄(狭窄，困窘)</p><p><strong>词性变化</strong></p><p>◎ <strong>局</strong> <span class="dicpy">jú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jú.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 弯曲 <span class="encs">[bend]</span></p><p><span class="diczx1">予发曲局。——<span class="smcs">《诗·小雅·采绿》</span></span></p><p><span class="diczx1">床不须局脚。——</span><span class="diczx2">殷芸</span><span class="smcs">《小说》</span></p><p><span class="cino">(2)</span> 又如:局局(俯身而大笑 的样子);局脚(装在器物底部的弯曲形高脚);局背(驼背)</p><p><span class="cino">(3)</span> 催逼;逼迫 <span class="encs">[force]</span></p><p><span class="diczx1">遂以局杀</span><span class="diczx2">湣公</span>于 <span class="diczx2">蒙泽</span>。——<span class="smcs">《史记》</span></p><p><span class="diczx1">屠户被众人局不过，只得连斟两碗酒喝了，壮一壮胆。——<span class="smcs">《儒林外史》</span></span></p><p>◎ <strong>局</strong> <span class="dicpy">jú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jú.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 部分 <span class="encs">[part]</span></p><p><span class="diczx1">左右有局。——<span class="smcs">《礼记·曲礼》</span>。注:<span class="yhcs">“部分也。”</span></span></p><p><span class="diczx1">局，部也。——<span class="smcs">《后汉书·袁绍传》</span>注</span></p><p><span class="diczx1">各司其局。——<span class="smcs">《礼记·曲礼》</span></span></p><p><span class="cino">(2)</span> 邻居 <span
