						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>足</strong></p><p><span class="dicpy">zú  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span>  ㄗㄨˊ</span></p><ol><li>脚：～下（对对方的敬称）。～迹。～球。～坛。失～。高～（敬辞，称别人的学生）。举～轻重。画蛇添～。</li><li>器物下部的支撑部分：鼎～。碗～。</li><li>充分，够量：～月。～见。～智多谋。</li><li>完全：～以。～色。</li><li>值得，够得上：不～为凭。微不～道。</li><li>增益：以昼～夜。</li></ol>

                    <h2 class="h2_entry"><span class="orth">足</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>足</strong> <span class="dicpy">zú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。甲骨文字形，上面的方口象膝，下面的<span class="yhcs">“止”</span>即脚，合起来指整个脚。本义:脚)</p><p><span class="cino">(2)</span> (同本义。秦汉以前，<span class="yhcs">“足”</span>和’趾”都表示<span class="yhcs">“脚”</span>;<span class="yhcs">“脚”</span>表示小腿。魏晋以后，三者都表示脚，但在书面语中，多用<span class="yhcs">“足”</span>) <span class="encs">[foot]</span></p><p><span class="diczx1">足，人之足也，在下，从止口。——<span class="smcs">《说文》</span>。按，膝下至跖之总名也。从止，即趾字，从口，象膝形。</span></p><p><span class="diczx1">震为足。——<span class="smcs">《易·说卦》</span></span></p><p><span class="diczx1">足恭而口圣。——<span class="smcs">《大戴礼记》</span></span></p><p><span class="diczx1">手足胼胝，面目黧。——<span class="smcs">《韩非子·外储说左上》</span></span></p><p><span class="diczx2">郑</span>人有欲买履者，先自度其足而置之其坐。——<span class="smcs">《韩非子》</span></p><p><span class="diczx1">假舆马者，非利足也，而致千里。——<span class="smcs">《荀子·劝学》</span></span></p><p><span class="diczx1">折臂断足。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="diczx2">东坡</span>现右足。—— <span class="diczx2">清</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="diczx2">鲁直</span>现左足。</p><p><span class="diczx1">细若文足。</span></p><p><span class="cino">(3)</span> 又如:足心(脚底的中心);足炉(用来暖脚的小火炉);足不出户(形容深居家门很少出门);足衣(袜子);足音跫然(形容久处寂寞，听到人的脚步声，便感到欢欣)</p><p><span class="cino">(4)</span> 支撑器物的脚 <span class="encs">[leg]</span></p><p><span class="diczx1">鼎折足，覆公餗。——<span class="smcs">《易·鼎》</span></span></p><p><span class="diczx1">如此则</span><span class="diczx2">荆</span>、 <span class="diczx2">吴</span>之势强，鼎足之形成矣。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(5)</span> 又如:鼎足</p><p><strong>词性变化</strong></p><p>◎ <strong>足</strong> <span class="dicpy">zú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zú.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 充实;完备;足够 <span class="encs">[enough;complete;ample]</span></p><p><span class="diczx1">国无九年之畜曰不足。——<span class="smcs">《谷梁传·庄公二十有八年》</span></span></p><p><span class="diczx1">杀所不足。——<span class="smcs">《墨子·公输》</span></span></p><p><span class="diczx1">取于食客门下足矣。——<span class="smcs">《史记·平原君虞卿列传》</span></span></p><p><span class="diczx1">兵甲已足。——</span><span class="diczx2">诸葛亮</span><span
