                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>斯</strong></p><p><span class="dicpy">sī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sī.mp3"></a></span>  ㄙˉ</span></p><ol><li>这，这个，这里：～人。～时。以至于～。</li><li>乃，就：有备～可以无患。</li><li>劈：“墓门有棘，斧以～之”。</li><li>古同“<span class="diczx3">厮</span>”，卑贱。</li><li>古同“<span class="diczx3">澌</span>”，尽。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">斯</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>斯</strong> <span class="dicpy">sī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sī.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从斤，其声。其所以盛木柿。斤，斧子。本义:劈;砍)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[split]</span></p><p><span class="diczx1">斯，析也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">墓门有棘，斧以斯之。——<span class="smcs">《诗·陈风·墓门》</span></span></p><p><span class="diczx1">有斯明享。——<span class="smcs">《书·酒诰》</span>。</span><span class="diczx2">郑</span>注:<span class="yhcs">“析也。”</span></p><p><span class="diczx1">厮(斯)舆之卒。——<span class="smcs">《汉书·严助传》</span>。注:<span class="yhcs">“析薪者也。”</span></span></p><p><span class="cino">(3)</span> 扯裂 <span class="encs">[tear]</span></p><p><span class="diczx1">斯，裂也。——<span class="smcs">《广雅》</span>。</span><span class="diczx2">王念孙</span>疏证:<span class="yhcs">“今俗语犹呼手裂为斯。”</span></p><p><span class="cino">(4)</span> 分开 <span class="encs">[divide]</span></p><p><span class="diczx1">斯而析之，精至于无伦，大至于不可围。——<span class="smcs">《庄子》</span></span></p><p><span class="cino">(5)</span> 距离;离开 <span class="encs">[leave]</span></p><p><span class="diczx2">华胥氏</span>之国…不知斯 <span class="diczx2">齐</span>国几千万里。——<span class="smcs">《列子·黄帝》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>斯</strong> <span class="dicpy">sī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sī.mp3"></a></span> </p><p>〈代〉</p><p><span class="cino">(1)</span> <span class="yhcs">“斯”</span>假借为<span class="yhcs">“此”</span>，这，这个 <span class="encs">[this]</span></p><p><span class="diczx1">斯，此也。——<span class="smcs">《尔雅·释诂》</span></span></p><p><span class="diczx1">斯言之玷。——<span class="smcs">《诗·大雅·抑》</span></span></p><p><span class="diczx1">登斯楼也，则有去国怀乡，忧馋畏讥，满目萧然，感极而悲者矣。——</span><span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="diczx1">而</span><span class="diczx2">毅宗</span>之语公主，亦曰:<span class="yhcs">“若何为生我家?”</span>痛哉斯言。—— <span class="diczx2">清</span>· <span class="diczx2">黄宗羲</span><span class="smcs">《原君》</span></p><p><span class="cino">(2)</span> 又如:斯世(此世，今世);斯人(此人);生于斯，长于斯</p><p>◎ <strong>斯</strong> <span class="dicpy">sī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sī.mp3"></a></span> </p><p><连></p><p>则。就 <span class="encs">[then]</span>——表示承接上文，得出结论</p><p><span class="diczx1">人之性，心有忧丧则悲，悲则哀，哀斯愤，愤斯怒，怒斯动，动则手足不静。——<span class="smcs">《淮南子》</span></span></p><p>◎ <strong>斯</strong> <span class="dicpy">sī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sī.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“厮”</span>。古时干粗杂活的奴隶或仆役。引申为卑贱 <span
