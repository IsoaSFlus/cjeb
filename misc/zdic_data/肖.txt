                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>肖</strong> <span class="dicpy">Xiāo</span></p><p>〈名〉</p><p>姓。<span class="yhcs">“萧”</span>字的俗写</p><p><strong>词性变化</strong></p><p>◎ <strong>肖</strong> <span class="dicpy">xiāo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiāo.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 假借为<span class="yhcs">“小”</span>。细微 <span class="encs">[small;tiny]</span></p><p><span class="diczx1">肖，小也。——<span class="smcs">《方言十二》</span></span></p><p><span class="diczx1">肖翘之物。——<span class="smcs">《庄子·ń箧》</span></span></p><p><span class="diczx1">达生之情者傀，达于知者肖。——<span class="smcs">《庄子》</span></span></p><p><span class="cino">(2)</span> 又如:肖翘(细小能飞的生物)</p><p>◎ <strong>肖</strong> <span class="dicpy">xiāo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiāo.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 假借为<span class="yhcs">“消”</span>。消灭;消失;衰退，衰微 <span class="encs">[perish;die out;exterminate;decline]</span></p><p><span class="diczx1">达于知者肖。——<span class="smcs">《庄子·列御寇》</span></span></p><p><span class="diczx2">申</span><span class="diczx2">吕</span>肖矣， <span class="diczx2">尚父</span>侧微。——<span class="smcs">《史记·太史公自序》</span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">xiào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiào.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>肖</strong> <span class="dicpy">xiào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiào.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从肉，小声。本义:相似，相像)</p><p><span class="cino">(2)</span> 同本义(侧重于本质特点上的相像) <span class="encs">[be like;be similar to]</span></p><p><span class="diczx1">肖，骨肉相似也。——<span class="smcs">《说文》</span>。字亦作俏</span></p><p><span class="diczx1">人俏天地之类。——<span class="smcs">《列子·杨朱》</span></span></p><p><span class="diczx1">肖形而蕃。——<span class="smcs">《淮南子·地形》</span></span></p><p><span class="diczx1">丹朱之不肖。——<span class="smcs">《孟子》</span></span></p><p><span class="diczx1">乃审厥象，俾以形旁求于天下，(</span><span class="diczx2">傅</span>)说 <span class="diczx2">筑</span><span class="diczx2">傅岩</span>之野，惟肖。——<span class="smcs">《书·说命上》</span></p><p><span class="diczx1">使之肖我。——</span><span class="diczx2">清</span>· <span class="diczx2">林觉民</span><span class="smcs">《与妻书》</span></p><p><span
