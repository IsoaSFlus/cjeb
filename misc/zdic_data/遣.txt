                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>遣</strong> <span class="dicpy">qiǎn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/qiǎn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)。本义:释放)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[release]</span></p><p><span class="diczx1">遣，纵也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">遣之日读诔。——<span class="smcs">《周礼·太史》</span>。注:<span class="yhcs">“谓祖庙之庭，大奠将行时也。”</span></span></p><p><span class="diczx1">大丧饰遣车。——<span class="smcs">《周礼·巾车》</span></span></p><p><span class="diczx1">书遣于策。——<span class="smcs">《仪礼·既夕礼》</span></span></p><p><span class="diczx1">遣车视牢具。——<span class="smcs">《礼记·杂记》</span>。<span class="yhcs">“送葬载牲体之车也。”</span></span></p><p><span class="diczx1">遣车一乘。——<span class="smcs">《礼记·檀弓》</span>。注:<span class="yhcs">“人臣赐车马者，乃得有遣车”</span>。按，此字古专于用凶礼。</span></p><p><span class="cino">(3)</span> 又如:遣俘(遣还俘虏);遣还(犹遣返)</p><p><span class="cino">(4)</span> 派。派去 <span class="encs">[send;dispatch]</span></p><p><span class="diczx2">赵王</span>于是遂遣 <span class="diczx2">相如</span>奉璧西入 <span class="diczx2">秦</span>。——<span class="smcs">《史记·廉颇蔺相如列传》</span></p><p><span class="diczx2">权</span>意不欲遣质。——<span class="smcs">《三国志·周瑜传》</span>注</p><p><span class="diczx1">乃遣</span><span class="diczx2">张良</span>往立 <span class="diczx2">信</span>为 <span class="diczx2">齐王</span>，征其兵击 <span class="diczx2">楚</span>。——<span class="smcs">《史记·淮阴侯列传》</span></p><p><span class="diczx1">愿勿遣。——</span><span class="diczx2">汉</span>· <span class="diczx2">刘向</span><span class="smcs">《列女传》</span></p><p><span class="diczx1">王终遣之。</span></p><p><span class="diczx1">太守即遣人随其往，寻向所志。——<span class="smcs">《桃花源记》</span></span></p><p><span class="diczx1">遣</span><span class="diczx2">李进诚</span>。——<span class="smcs">《资治通鉴·唐纪》</span></p><p><span class="cino">(5)</span> 又如:调遣(调派;差遣);差遣(分派到外面去工作);遣使(派遣使者);遣人去请医生;遣价(差遣仆人);遣官(派遣官员);遣员(差遣人员);遣军(派遣军队)</p><p><span class="cino">(6)</span> 贬谪;放逐 <span class="encs">[relegate;exile]</span></p><p><span class="diczx2">姜氏</span>与 <span class="diczx2">子犯</span>谋醉而遣之。——<span class="smcs">《左传·僖公二十三年》</span></p><p><span class="diczx1">遣归故郡。——<span class="smcs">《汉书·孔光传》</span></span></p><p><span class="diczx1">帝不悦而遣太子。——<span class="smcs">《潜夫论》</span></span></p><p><span class="cino">(7)</span> 又如:遣归(贬谪、释放或休弃而令归);遣谪(流放边远地区);遣刑(清代放逐犯人到极边远的地方去服苦役的刑罚);遣斥(斥逐);遣犯(指放逐在边地服刑的囚犯);遣流(发配，流放)</p><p><span class="cino">(8)</span> 发送;打发 <span class="encs">[send;dismiss]</span></p><p><span class="diczx1">笑而遣之。——</span><span class="diczx2">宋</span>· <span class="diczx2">欧阳修</span><span class="smcs">《归田录》</span></p><p><span class="diczx1">婿固不遣。——</span><span class="diczx2">明</span>· <span class="diczx2">崔铣</span><span class="smcs">《洹词·记王忠肃公翱三事》</span></p><p><span class="diczx1">厚资而遣之。——</span><span
