						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>仞</strong></p><p><span class="dicpy">rèn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rèn.mp3"></a></span> ㄖㄣˋ</span></p><ol><li>古代计量单位：一～（周尺八尺或七尺。周尺一尺约合二十三厘米）。山高万～。</li><li>测量深度。</li><li>古同“<span class="diczx3">韧</span>”，坚韧。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> ancient unit of measure (8 feet); 'fathom'</p><p><span class="z_ts2">德语</span> altes Längenmaß von 7 - 8 chin. Fuß</p><p><span class="z_ts2">法语</span> ren (mesure)​</p></div>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>仞</strong> <span class="dicpy">rèn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rèn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从人，刃声。本义:古代长度单位。周制八尺，汉制七尺)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[ren，an ancient measure of varying length]</span></p><p><span class="diczx1">仞，伸臂一寻八尺也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> 按，诸说不一</p><p><span class="diczx1">筑宫仞有三尺。——<span class="smcs">《礼记·祭义》</span></span></p><p><span class="diczx1">深八尺谓之洫，深二仞谓之浍。——<span class="smcs">《考工记·匠人》</span></span></p><p><span class="diczx2">太行</span>、 <span class="diczx2">王屋</span>二山，方七百里，高万仞。——<span class="smcs">《列子·汤问》</span></p><p><span class="diczx1">一片孤城万仞山。——</span><span class="diczx2">唐</span>· <span class="diczx2">王之涣</span><span class="smcs">《凉州词》</span></p><p><span class="diczx1">夫子之墙数仞。——<span class="smcs">《论语·子张》</span></span></p><p><span class="cino">(4)</span> 春秋时楚地名，在今湖北省十堰市东 <span class="encs">[Ren village]</span></p><p><span class="diczx2">子越</span>自 <span class="diczx2">石溪</span>， <span class="diczx2">子贝</span>自 <span class="diczx2">仞</span>以伐 <span class="diczx2">庸</span>。——<span class="smcs">《左传》</span></p><p><span class="cino">(5)</span> 用同<span class="yhcs">“刃”</span>。锋刃，借指刀剑之类的兵器 <span class="encs">[same as<span class="yhcs">“刃”</span>]</span></p><p><span class="diczx1">仞，通作刃。——<span class="smcs">《正字通》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>仞</strong> <span class="dicpy">rèn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rèn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 测量深度 <span class="encs">[fathom]</span></p><p><span class="diczx1">计丈数，揣高卑，度厚薄，仞沟洫。——<span class="smcs">《左传》</span></span></p><p><span class="cino">(2)</span> <span class="yhcs">“认”</span>的古字。</p><p><span class="cino">(3)</span> 承认 <span class="encs">[admit]</span></p><p><span class="diczx1">公理者，犹云众所同仞之界域。——</span><span class="diczx2">章炳麟</span><span class="smcs">《四惑论》</span></p><p><span class="cino">(4)</span> 通<span class="yhcs">“认”</span>。辨认 <span class="encs">[recognize]</span>。如:仞识论(认识论)</p><p><span class="cino">(5)</span> 认为，当作 <span class="encs">[regard as]</span></p><p><span class="diczx1">非其事者勿仞也…。仞人之事者败。——<span class="smcs">《淮南子·人间》</span></span></p><p><span class="diczx1">惟</span><span class="diczx2">通州</span><span class="diczx2">五代</span>所置，向无郡名，政和末，始赐名 <span class="diczx2">静海</span>，史家遂误仞改州为郡矣。—— <span class="diczx2">清</span>· <span class="diczx2">钱大昕</span><span class="smcs">《廿二史考异》</span></p><p><span class="cino">(6)</span> 通<span class="yhcs">“忍”</span> <span class="encs">[tolerate]</span></p><p><span class="diczx1">是故百姓冬不仞寒，夏不仞暑，作疾病死者，不可胜计也。——<span class="smcs">《墨子》</span></span></p><p>◎ <strong>仞</strong> <span class="dicpy">rèn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rèn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“韧”</span>。柔软而坚固 <span class="encs">[pliable but strong]</span></p><p><span class="diczx1">巩用黄牛之革。——<span class="smcs">《易·革》</span>。</span><span class="diczx2">三国</span><span class="diczx2">魏</span>· <span class="diczx2">王弼</span>注:<span class="yhcs">“牛之革坚仞不可变也。”</span></p><p><span class="cino">(2)</span> 通<span class="yhcs">“牣”</span>。满，充满 <span class="encs">[full]</span></p><p><span class="diczx1">虚宫馆而勿仞。——</span><span class="diczx2">司马相如</span><span class="smcs">《上林赋》</span></p><p><span class="diczx1">充仞其中者。——</span><span class="diczx2">司马相如</span><span class="smcs">《子虚赋》</span></p><p><span class="cino">(3)</span> 又如:仞积(堆满)</p>                    <div class="div copyright"> © 汉典 </div>

                          <h2 class="h2_entry"><span class="orth">仞</span><span class="span dictname"> 《康熙字典》</span></h2>
                                                  
                          
                        </div>
                        
                        <div class="content definitions knr">
						<div class="mob">【<a href="/zd/kx/?zm=子集中" target="_blank">子集中</a>】【<a href="/zd/kx/?zm=子集中&amp;bs=人" target="_blank">人</a>】
                              仞·康熙筆画：<a href="/zd/kx/bh/?bh=5" target="_blank">5</a>　·部外筆画：3</div>
                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/4EDE.svg">						  <p>【廣韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>而振切，音刃。【孔安國曰】八尺曰仞。【禮·祭義】築宮仞有三尺。【前漢·賈誼傳】鳳凰翔於千仞。</p><p>又度深曰仞。【左傳·昭三十二年】仞溝洫。</p><p>又與認通。【列子·天瑞篇】天地萬物不相離，仞而有之，皆惑也。【註】仞卽認。【淮南子·人閒訓】非其事者，勿仞也。仞人之事者敗。</p><p>又與牣通。滿也。【司馬相如·子虛賦】充仞其中。</p><p>又地名。【左傳·文十六年】楚子貝自仞以伐庸。【註】仞入庸道。　一作軔。【孟子】掘井九軔。　【正字通】古以周尺八尺爲仞，中人之身，長八尺，兩臂尋之，亦八尺，兩足步之，亦八尺。度高深以仞，度短長以尋，度地以步。《小爾雅》四尺曰仞，王肅从之。包咸鄭<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>謂七尺曰仞。《書》爲山九仞。《釋文》仞，七尺也。《應劭·漢書註》：五尺六寸曰仞。顏師古非之曰：八尺曰仞，取人申臂之一尋也。顏說與孔安國同，顏以八尺爲仞是也。《說文》以仞爲申臂一尋，非。一尋止六尺耳。或曰古尺短，周尺八尺，以今度之一寸，又減二分，應劭之說據漢尺也。《莊子·庚桑楚》步仞之丘註：七尺曰仞，亦非。</p>

                          <h2 class="h2_entry"><span class="orth">仞</span><span class="span dictname"> 《说文解字》</span></h2>
                                                  </div>
                        
                        <div class="content definitions snr">
						<div class="mob">【<a href="/zd/sw/?swju=卷八">卷八</a>】【<a href="/zd/sw/?swju=卷八&amp;swbs=人">人</a>】</div>
                                                    <div class="swnr">
                          『說文解字』<hr class="dichr">
                          <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/4EDE.svg">                          <p>伸臂一尋，八尺。从人刃聲。而震切</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/4EDE_111.svg"  /><span class="swbf">(仞)</span><span class="ywyy">伸臂一尋八尺。</span><span class="swdz">按此解疑非許之舊。恐後人改竄爲之。尺部下云。周制寸尺咫尋常仭諸度量。皆以人之體爲法。假令尋仭同物。許不當㒳舉之矣。諸家之說仭也。王肅、趙岐、王逸、曹操、李筌、顏師古、房玄齡、鮑彪諸人並曰八尺。而鄭周禮儀禮注、包咸論語注、高誘注呂氏春秋、王逸注大招招䰟、李謐明堂制度論、郭璞注司馬相如賦用司馬彪之說、陸德明莊子釋文則皆謂七尺。淮南子原道訓注八尺。而覽冥訓注則云七尺。百仭者七百尺。證以呂氏春秋注則原道注可疑。近歙程氏瑤田通藝錄有說曰。言七尺者是也。楊雄方言云。度廣曰尋。杜預左傳仭溝洫注。度深曰仭。二書皆言人伸㒳手以度物之名。而尋爲八尺、仭必七尺者、何也。同一伸手度物。而廣深用之。其勢自不得不異。人長八尺。伸㒳手亦八尺。用以度廣。其勢全伸而不屈。而用之以度深。則必上下其左右手而側其身焉。身側則胸與所度之物不能相摩。於是㒳手不能全伸而成弧之形。弧而求其弦以爲仭必不能八尺。故七尺曰仭。亦其勢然也。說文測下云。深所至也。玉篇云。度深曰測。測之爲言側也。余說與之合矣。玉裁謂。程說甚精。仭說可定矣。考工記。廣二尋、深二仭謂之澮。倘其度同八尺。何不皆曰二尋。如上文廣二尺、深二尺之例也。許書於尺下旣尋仭兼舉。尋者、八尺也。見寸部。則仭下必當云七尺。今本乃淺人所竄易耳。程氏又曰。小爾雅云四尺。應邵云五尺六寸。此其繆易見也。</span><span class="ywyy">从人。刄聲。</span><span class="swdz">而震切。十二部。或借爲牣滿字。孟子。掘井九軔。借軔爲之。</span><span class="xh"></span></p>                          </div>
