						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>稿</strong></p><p><span class="dicpy">gǎo  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǎo.mp3"></a></span>  ㄍㄠˇ</span></p><ol>    <li>谷类植物的茎秆：～秆。～荐（稻草编的垫子）。</li>    <li>文字、图画的草底，又喻事先考虑的计划：文～。～本。～件。讲～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">稿</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>稿</strong></p><p>稾 <span class="dicpy">gǎo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǎo.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从禾，高声。本义:禾秆)</p><p><span class="cino">(2)</span> 禾秆，谷类植物的茎杆。也作<span class="yhcs">“稾”</span> <span class="encs">[stalk of grain]</span></p><p><span class="diczx1">稾，秆也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">民惊走持稾。——<span class="smcs">《汉书·五行志》</span></span></p><p><span class="diczx1">又出稾税。——<span class="smcs">《书·禹贡传》</span></span></p><p><span class="diczx1">茭稾。——<span class="smcs">《汉书·赵安国传》</span></span></p><p><span class="diczx1">今又盛寒，马无稿草。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="diczx1">愿令民得入田，毋收稾稾为禽兽食。——<span class="smcs">《史记·萧相国世家》</span></span></p><p><span class="diczx1">(</span><span class="diczx2">皋涂</span>之山)有草焉，其状如稾茇，其叶如葵而赤黄，名曰无条，可以毒鼠。——<span class="smcs">《山海经·西山经》</span></p><p><span class="cino">(3)</span> 又如:稿本(香草名。一年生，根可入药);稿茇(香草名。即稾本);稿稾(指点燃后供照明用的草、麻);稿荐(用稻草编成的垫褥);稿苗(枯槁的禾苗);稿人(稻草人)</p><p><span class="cino">(4)</span> 诗文、图画等草底。亦指写成的文章、著作 <span class="encs">[draft]</span></p><p><span class="diczx2">怀王</span>使 <span class="diczx2">屈原</span>造为宪令， <span class="diczx2">屈原</span>属草稿未定， <span class="diczx2">上官大夫</span>见而欲夺之， <span class="diczx2">屈原</span>不与。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="cino">(5)</span> 又如:文稿(文章、公文的草稿);稿副(底稿);稿案(公文案牍;亦作<span class="yhcs">“文案师爷”</span>。为主官管理文秘事务的慕僚);稿书(草稿、初稿)</p><p><span class="cino">(6)</span> 箭杆 <span class="encs">[arrow’s pole]</span></p><p><span class="diczx1">特箭稾而茎立兮，独聆风于极危。——</span><span class="diczx2">汉</span>· <span class="diczx2">马融</span><span class="smcs">《长笛赋》</span></p><p><span class="cino">(7)</span> 又如:稿草(草稿。初写出的文稿);稿定(撰定文稿)</p><p><span class="diczx1">〈形〉</span></p><p><span class="cino">(8)</span> 通<span class="yhcs">“槁”</span>。干枯 <span class="encs">[withered stalk]</span></p><p><span class="diczx1">弃其本者，荣其稿矣。——<span class="smcs">《说苑·建本》</span></span></p><p><span class="cino">(9)</span> 又如:稿芼(枯干的草茎)</p><p><span class="cino">(10)</span> 指已刊行，但作者仍以为未周备，有待修订整理的书籍 <span class="encs">[manuscript]</span>。如:中国史稿;稿定(指著作已经完成定案，即可付印)</p><p><span class="cino">(11)</span> 样子;模样 <span class="encs">[appearance;look]</span></p><p><span class="diczx1">我看见哥儿的这个形容身段，言谈举动，怎么就和当儿国公爷一个稿子。——<span class="smcs">《红楼梦》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>稿</strong> <span class="dicpy">gǎo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǎo.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 叙说;论辈分。用同<span class="yhcs">“搞”</span> <span
