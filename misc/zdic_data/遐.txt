                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>遐</strong> <span class="dicpy">xiá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiá.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，叚( <span class="dicpy">xiá</span> <span class="encs">[far]</span></p><p><span class="diczx1">遐，远也。——<span class="smcs">《说文新附》</span></span></p><p><span class="diczx1">若陟遐，必自迩。——<span class="smcs">《书·太甲》</span></span></p><p><span class="diczx1">是以遐方疏俗，殊邻绝党之域，自上仁所不化，茂德所不绥，莫不蹻足抗首，请献厥珍。——</span><span class="diczx2">汉</span>· <span class="diczx2">扬雄</span><span class="smcs">《长杨赋》</span></p><p><span class="diczx1">矫首而遐观。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《归去来兮辞》</span></p><p><span class="diczx1">有艳淑女在闺房，室迩人遐在我傍。——<span class="smcs">《警世通言》</span></span></p><p><span class="diczx1">地遐怪物聚，寺古深殿存。——</span><span class="diczx2">宋</span>· <span class="diczx2">范祖禹</span><span class="smcs">《资州路东津寺》</span></p><p><span class="cino">(3)</span> 又如:遐僻(偏远);遐宣(远远宣扬);遐方(远方);遐布(传播到远处)</p><p><span class="cino">(4)</span> 长久，久远 <span class="encs">[long]</span></p><p><span class="diczx1">降尔遐福，维日不足。——<span class="smcs">《诗·小雅·天保》</span></span></p><p><span class="diczx1">入觐展遐恋，临轩慰来思。——</span><span class="diczx2">唐</span>· <span class="diczx2">德宗</span><span class="smcs">《送徐州张建封还镇》</span></p><p><span class="cino">(5)</span> 又如:遐福(久远之福);遐祉(永远长保的福祉)</p><p><span class="cino">(6)</span> 又长又大 <span class="encs">[long and big]</span></p><p><span class="diczx1">爰有遐狄，镣质轮菌，坐高门之侧堂，彰圣主之威神。——<span class="smcs">《文选·何晏·景福殿赋》</span></span></p><p><span class="cino">(7)</span> 又如:遐志(远大的志向);遐愁(深长的愁绪)</p><p><span class="cino">(8)</span> 疏远 <span class="encs">[distant]</span></p><p><span class="diczx1">毋金玉尔音，而有遐心。——<span class="smcs">《诗·小雅·白驹》</span>。</span><span class="diczx2">朱熹</span>集传:<span class="yhcs">“毋贵重尔之音声，而有远我之心。”</span></p><p><span class="cino">(9)</span> 又如:遐弃(疏远抛弃);遐遗(疏远遗弃)</p><p><strong>词性变化</strong></p><p>◎ <strong>遐</strong> <span class="dicpy">xiá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiá.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 前人;以往 <span class="encs">[former]</span></p><p><span class="diczx1">遵</span><span class="diczx2">许</span><span class="diczx2">郭</span>之遐轨。——<span class="smcs">《晋书》</span></p><p><span class="diczx1">仰天民之高衢兮，慕在昔之遐轨。——</span><span class="diczx2">汉</span>· <span class="diczx2">阮瑀</span><span class="smcs">《纪征赋》</span></p><p><span class="cino">(2)</span> 又如:遐轨(古人的遗迹;前人之法度);遐籍(古代的典籍)</p><p><span class="cino">(3)</span> 边远地方 <span class="encs">[distant land]</span>。如:遐荒(边远的荒芜地区);遐陬(边远角落);遐氓(边远地区的人民)</p><p>◎ <strong>遐</strong> <span class="dicpy">xiá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiá.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 远去 <span class="encs">[go far away]</span></p><p><span
