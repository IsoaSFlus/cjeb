						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>拿</strong></p><p><span class="dicpy">ná  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ná.mp3"></a></span>  ㄋㄚˊ</span></p><ol><li>用手取，握在手里：～笔。～枪。</li><li>掌握，把握：～主意。～权。～手。</li><li>挟（<span class="dicpy">xié</span>）制：～捏（故意刁难人，要挟人）。～大（自以为比别人强，看不起人，摆架子）。</li><li>侵蚀，侵害：让药水～白了。</li><li>逮捕，捉：～获。擒～。～问。</li><li>攻下，占领：一定要把敌人的碉堡～下来。</li><li>介绍，引出对象，相当于“<span class="diczx3">把</span>”：我～你当亲人看待。</li><li>介词，引出所凭借的工具、材料、方法等，相当于“<span class="diczx3">用</span>”：～笔来写。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>拿</strong></p><p>挐 <span class="dicpy">ná</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ná.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。手合起来，表示握持，字本作<span class="yhcs">“挐”</span>。从手。奴声。本义:握持)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[hold;take]</span></p><p><span class="diczx1">手里拿串拍板。——<span class="smcs">《水浒传》</span></span></p><p><span class="cino">(3)</span> 又如:从架上拿书;用钳子拿钉子;他拿起帽子和大衣走了;别拿我的东西</p><p><span class="cino">(4)</span> 捉拿 <span class="encs">[arrest;catch]</span></p><p><span class="diczx1">差人拿不将他来;差人都合他是一个人，谁肯拿他?——<span class="smcs">《醒世姻缘传》</span></span></p><p><span class="cino">(5)</span> 又如:缉拿(搜查捉拿);拿不将(拿不了);拿究(捉拿查究);拿捕(捉拿;逮捕);拿访(捉拿查访);拿送(逮捕解送);拿解(逮捕解送)</p><p><span class="cino">(6)</span> 矜持;摆 <span class="encs">[put on]</span>。如:拿一手(摆架子;要挟);拿三撇四(装模作样);拿身分(犹言摆架子);拿腔(拿乔。装腔作势);拿捻(故作姿态)</p><p><span class="cino">(7)</span> 刁难 <span class="encs">[make things difficult for sb.]</span></p><p><span class="diczx1">因怕</span><span class="diczx2">金桂</span>拿他，所以不敢透漏。——<span class="smcs">《红楼梦》</span></p><p><span class="cino">(8)</span> 如:拿把;拿捏</p><p><span class="cino">(9)</span> 把握 <span class="encs">[grasp;be sure of]</span>。如:拿准(十分有把握);拿稳(有把握;沉着)</p><p><span class="cino">(10)</span> 攻克;俘获 <span class="encs">[capture]</span>。如:拿下敌人的碉堡</p><p><span class="cino">(11)</span> 用手把一件东西从一处转移到另一处 <span class="encs">[bring]</span>。如:拿回家</p><p><span class="cino">(12)</span> 用…手段表现或运用权力 <span class="encs">[wield]</span>。如:拿钱垫人(以钱压人);拿权</p><p><span class="cino">(13)</span> 提起;举起;抬;吊 <span class="encs">[lift]</span>。如:他也不管人脸上过得去过不去，咳了一声，拿起脚就走了</p><p><strong>词性变化</strong></p><p>◎ <strong>拿</strong> <span class="dicpy">ná</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ná.mp3"></a></span> </p><p>〈介〉</p><p><span class="cino">(1)</span> 引进动作、行为所凭借的工具、材料或方法等。相当于<span class="yhcs">“用”</span> <span class="encs">[with]</span>。如:拿事实说话</p><p><span class="cino">(2)</span> 引进所处置的对象，相当于<span class="yhcs">“把”</span> <span class="encs">[equivalent to]</span>。如:别拿他当小孩</p><p><span class="cino">(3)</span> 和<span class="yhcs">“来说”</span>、<span class="yhcs">“来讲”</span>连用，举出要说明的事物或情况 <span class="encs">[for example]</span>。如:拿我们来说，缺勤是极少的 注：<span class="yhcs">“纷挐”</span>的<span class="yhcs">“挐”</span>不作<span class="yhcs">“拿”</span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a
