data-src-mp3="//img.zdic.net/audio/zd/py/pén.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从皿，分声。皿，泛指盘碗一类器具。本义:盛东西或洗涤用的器皿。通常为圆形，口大底小，比盘深)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[basin;pot;tub]</span></p><p><span class="diczx1">盆，盎也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">盆实二鬴，厚半寸，脣寸。——<span class="smcs">《考工记·陶人》</span></span></p><p><span class="diczx1">与其盆簝。——<span class="smcs">《周礼·牛人》</span>。</span><span class="diczx2">司农</span>注:<span class="yhcs">“所以盛血。”</span></p><p><span class="diczx1">盆器倾侧。——<span class="smcs">《虞初新志·秋声诗自序》</span></span></p><p><span class="cino">(3)</span> 又如:炭盆(烧木炭的火盆);脸盆;大铜盆(一种较浅的口大底小的盛器，多为圆形);盆吊(把囚犯倒吊处死的一种酷刑);盆冤(覆盆之冤，喻指蒙受莫白之冤);盆口精熟(精通、熟悉赌博的门道)</p><p><span class="cino">(4)</span> 古代量器。容量为古制十二斗八升 <span class="encs">[measurement's class="diczx1">夫奥者，老妇之祭也。盛于盆，尊于瓶。——<span class="smcs">《礼记·食货志》</span></span></p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>盆</strong> <span class="dicpy">pén</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pén.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 把物浸在水盆里 <span class="encs">[soak]</span></p><p><span class="diczx1">夫人缫，三盆手。——<span class="smcs">《礼记》</span></span></p><p><span class="cino">(2)</span> 水上涌四溢。也作<span class="yhcs">“湓”</span> <span class="encs">[brim over]</span></p><p><span class="diczx2">青</span>、 <span class="diczx2">冀</span>之域淫雨漏河， <span class="diczx2">徐</span>、 <span class="diczx2">岱</span>之滨海水盆溢。—— <span class="diczx2">汉</span>· <span class="diczx2">陈忠</span><span class="smcs">《因灾异上疏劾中侍伯荣》</span></p><p>◎ <strong>盆</strong> <span class="dicpy">pén</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pén.mp3"></a></span> </p><p>〈量〉</p><p><span class="cino">(1)</span> 古代计算量盆所盛数量的单位。古制十二斗八升为一盆</p><p><span class="diczx1">今是土之生五谷也，人善治之，则亩数盆，一岁而获之。——<span class="smcs">《荀子》</span></span></p><p><span class="cino">(2)</span> 计算一般容器盆所盛数量的单位</p><p><span class="diczx1">予购三百盆。——</span><span class="diczx2">清</span>· <span class="diczx2">龚自珍</span><span
