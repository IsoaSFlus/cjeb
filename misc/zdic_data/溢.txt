						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>溢</strong></p><p><span class="dicpy">yì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span>  ㄧˋ</span></p><ol><li>充满而流出来：～流。河水四～。洋～。充～。～于言表。</li><li>过分：～美。～恶。～誉。</li><li>超出：～出此数。</li><li>古同“<span class="diczx3">镒</span>”，古代重量单位。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>溢</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水，益声。本作<span class="yhcs">“益”</span>。象水从器皿中漫出。本义:水漫出来)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[overflow;brim over]</span></p><p><span class="diczx1">溢，器满也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">溢，盈也。——<span class="smcs">《尔雅》</span></span></p><p><span class="diczx1">溢，满也。——<span class="smcs">《广雅》</span>。按，过满为溢。</span></p><p><span class="diczx1">满而不溢。——<span class="smcs">《考经·诸侯》</span></span></p><p><span class="diczx1">东流不溢，孰知其故?——<span class="smcs">《楚辞》</span></span></p><p><span class="diczx1">惧满溢。——</span><span class="diczx2">唐</span>· <span class="diczx2">魏征</span><span class="smcs">《谏太宗十思疏》</span></p><p><span class="diczx1">溢于辞。——</span><span class="diczx2">清</span>· <span class="diczx2">黄宗羲</span><span class="smcs">《原君》</span></p><p><span class="cino">(3)</span> 又如:溢决(水满破堤);溢满(平满)</p><p><span class="cino">(4)</span> 水泛滥成灾，淹没 <span class="encs">[flood]</span></p><p><span class="diczx1">虽有凶旱水溢，民无菜色。——<span class="smcs">《礼记》</span></span></p><p><span class="cino">(5)</span> 又如:溢涌(洪水腾涌)</p><p><span class="cino">(6)</span> 满，充塞 <span class="encs">[be fill]</span>。如:溢气坌涌(才气横溢，喷涌而出);溢满(充满)</p><p><span class="cino">(7)</span> 增加;提高 <span class="encs">[increase]</span>。如:溢滋(增加);溢课(外加赋税)</p><p><strong>词性变化</strong></p><p>◎ <strong>溢</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 过度，过分 <span class="encs">[excessive]</span></p><p><span class="diczx1">夫两喜必多溢美之言，两怒必多溢恶之言。——<span class="smcs">《庄子》</span></span></p><p><span class="cino">(2)</span> 又如:溢羡(过分的利润);溢誉(过度的称誉);溢恶(过分指责);溢利(过分的利益);溢分(过分);溢美(过分赞美);溢望(过分地希望)</p><p><span class="cino">(3)</span> 过甚 <span class="encs">[exaggerated]</span>。如:溢尤(过甚);溢言(过甚的言辞);溢言虚美(过甚其辞、不符合实际地称美)</p><p><span class="cino">(4)</span> 非分 <span class="encs">[overstepping one's bounds]</span>。如:溢喜(分外的欣喜);溢志(非分的欲望);溢利(非分的盈利)</p><p><span class="cino">(5)</span> 过多;超过 <span class="encs">[surplus]</span>。如:溢羡(过多的盈利);溢销(超过额定销售量);溢额(超额)</p><p><span class="cino">(6)</span> 盛 <span class="encs">[vigorous]</span>。如:溢刚(旺盛强壮)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/溢出">溢出</a></span><span class="crefe"><a                     <div class="gycd"><div class="pz"><ruby><rbc><rb>溢</rb></rbc><rtc><rt>ㄧˋ</rt></rtc><rtc><rt>yì <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/gycd/zd/5913.ogg"></a></span> </rt></rtc></ruby></div><div class="gycd-item"><span class="gc_cx">动</span><ol><li><p class="def"><span class="gc_sy">液体漫出外流。</span><span class="gc_yy">《礼记·王制》：「虽有凶旱水<b>溢</b>，民无菜色。」</span><span class="gc_yy">《新唐书·卷九七·魏征传》：「是岁，大雨，谷、洛<b>溢</b>，毁宫寺十九，漂居人六百家。」</span></p></li><li><p class="def"><span class="gc_sy">泛指外流、流失。</span><span class="gc_yy">《二十年目睹之怪现状·第九四回》：「现在我们中国所用的全是墨西哥银圆，利权外<b>溢</b>，莫此为甚！」</span></p></li><li><p class="def"><span class="gc_sy">流露、充满。</span><span class="gc_lz">如：「热情洋<b>溢</b>」。</span><span class="gc_yy">《文选·陆机·文赋》：「文徽徽以溢目，音泠泠而 盈耳。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">副</span><ol><li><p class="def"><span class="gc_sy">过分、过度。</span><span class="gc_lz">如：「溢美」。</span><span class="gc_yy">汉·桓宽《盐铁论·错币》：「故人主积其食，守其用，制其有余，调其不足，禁<b>溢</b>羡，厄利涂，然后百姓可家给人足也。」</span><span class="gc_yy">南朝宋·鲍照〈征北世子诞育上疏〉：「不胜殊欢<b>溢</b>喜，谨奉表以闻。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">名</span><ol><li><p class="def"><span class="gc_sy">量词。古代计算重量的单位。二十两为一<b><b>溢</b></b>。</span><span class="gc_yy">《战国策·秦策一》：「白壁百 双，黄金万<b>溢</b>。」</span><span class="gc_yy">汉·高诱·注：「万<b>溢</b>，万金也。二十两为一<b>溢</b>也。」</span><span class="gc_yx">通「镒」。</span></p></li></ol></div></div>

                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/6EA2.svg">						  <p>【唐韻】夷質切【集韻】【韻會】【正韻】弋質切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音逸。【說文】器滿也。【爾雅·釋詁】溢，盈也。【孝經·諸侯章】滿而不溢。</p><p>又【爾雅·釋詁】溢，靜也。【疏】盈溢者宜靜。</p><p>又愼也。舍人曰：溢行之愼。</p><p>又洋溢也。【中庸】洋溢乎中國。</p><p>又匹溢，聲四散也。【王褒·洞簫賦】龢紛離其匹溢。</p><p>又【儀禮·喪服】朝一溢米。【註】二十四兩曰溢，爲米一升，二十四分升之一。</p><p>又【孔叢子·雜訓】兩手曰掬，一手曰溢。</p><p>又同鎰。【荀子·儒效篇】藏千溢之寶。</p><p>又通佾。【前漢·郊祀歌】千童羅舞成八溢。【註】溢，與佾同。列也。</p><p>又【集韻】食質切，音實。【儀禮】一溢米，劉昌宗讀。</p><p>又神至切，音示。【詩·周頌】假以溢我。徐邈讀。</p><p>又【韻補】叶於旣切，音意。【左思·魏都賦】沐浴福應，宅心醰粹，餘糧栖畝而勿收，頌聲載路而洋溢。</p><p>
