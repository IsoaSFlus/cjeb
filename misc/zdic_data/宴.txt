                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>宴</strong></p><p><span class="dicpy">yàn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yàn.mp3"></a></span>  ㄧㄢˋ</span></p><ol><li>以酒饭款待宾客：～客。～饮。～席。</li><li>聚会在一起吃酒饭：～会。～集。</li><li>酒席：设～。国～。盛（<span class="dicpy">shèng</span>）～。</li><li>乐（<span class="dicpy">lè</span>），安闲：～娱。～嬉。～乐（<span class="dicpy">yuè </span>）。～居。</li></ol>

                    <h2 class="h2_entry"><span class="orth">宴</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>宴</strong></p><p>讌、醼 <span class="dicpy">yàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yàn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从宀( <span class="dicpy">mián)，妟(yàn</span>)声。<span class="yhcs">“宀”</span>表示房屋，<span class="yhcs">“妟”</span>是<span class="yhcs">“安”</span>意思，也有表意作用。本义:请人吃饭喝酒，聚会在一起喝酒吃饭 )</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[entertain at a banquet]</span></p><p><span class="diczx1">宴，安也。——<span class="smcs">《说文》</span>。按，此字当为宴飨正字，亦作醼，作讌。</span></p><p><span class="diczx1">君子以饮食宴乐。——<span class="smcs">《易·需》</span>。</span><span class="diczx2">郑</span>注:<span class="yhcs">“宴，享宴也。”</span></p><p><span class="diczx1">吉甫宴喜。——<span class="smcs">《汉书·陈汤传》</span>引诗六月</span></p><p><span class="diczx1">宾客大宴。——<span class="smcs">《虞初新志·秋声诗自序》</span></span></p><p><span class="diczx2">太守</span>宴。—— <span class="diczx2">宋</span>· <span class="diczx2">欧阳修</span><span class="smcs">《醉翁亭记》</span></p><p><span class="diczx1">宴酣之乐。</span></p><p><span class="diczx1">添酒回灯重开宴。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="cino">(3)</span> 又如:宴犒(设宴犒赏);宴赏(设宴犒赏);宴饯(设宴招待);宴好(泛指宴饮聚会);宴衍(宴饮作乐);宴位(宴饮之处);宴笑(宴饮欢笑);宴娱(宴饮娱乐);宴饫;宴歌(宴饮歌唱);宴聚(宴饮聚会)</p><p><span class="cino">(4)</span> 进餐 <span class="encs">[eat;take]</span></p><p><span class="diczx1">逾庖而宴。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>宴</strong> <span class="dicpy">yàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yàn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (假借为<span class="yhcs">“安”</span>。安闲，安逸) <span class="encs">[easy;leisurely]</span></p><p><span class="diczx1">是与太子宴者也。——<span class="smcs">《汉书·贾谊传》</span>。注:<span class="yhcs">“谓安居。”</span></span></p><p><span class="diczx1">乃宴斯息。——</span><span class="diczx2">张衡</span><span class="smcs">《东京赋》</span>。注:<span class="yhcs">“安也。”</span></p><p><span class="diczx1">宴安鳩毒。——<span class="smcs">《左传》</span>。</span><span class="diczx2">孔颖达</span>疏:<span class="yhcs">“宴安自逸，若鳩毒之药。</p><p><span class="cino">(2)</span> 又如:宴坐(安坐;闲坐);宴私(谓公余闲居之时);宴玩(闲暇时赏玩);宴服(日常闲居时穿的衣服);宴居(闲居。一般指公余无事时);宴宴(安闲逸乐的样子);宴处(安居;闲居);宴语(闲谈);宴慰(闲居);宴堕(闲散不整的样子);宴默(安居静穆);宴逸(逸乐)</p><p><span class="cino">(3)</span> 安定，没有战事或动乱 <span
