                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>般</strong> <span class="dicpy">bān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bān.mp3"></a></span> </p><p>〈量〉</p><p><span class="cino">(1)</span> 种，类 <span class="encs">[kind;sort]</span></p><p><span class="diczx1">草木知春不久归，百般红紫斗芳菲。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《晚春》</span></p><p><span class="cino">(2)</span> 又如:这般东西;百般劝解;般般(种种;样样;件件)</p><p><strong>词性变化</strong></p><p>◎ <strong>般</strong> <span class="dicpy">bān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bān.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“搬”</span>。搬运 <span class="encs">[move;remove]</span></p><p><span class="diczx1">般，运也。——<span class="smcs">《玉篇》</span></span></p><p><span class="diczx1">自行般运出卖以办岁计。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(2)</span> 又如:般运(搬担。搬运);般剥(搬运，驳运)</p><p><span class="cino">(3)</span> 通<span class="yhcs">“班”</span>。布，颁布 <span class="encs">[issue;promulgate]</span></p><p><span class="diczx1">灵之来，神哉沛，先以雨，般裔裔。——<span class="smcs">《汉书·礼乐志》</span></span></p><p><span class="cino">(4)</span> 用同<span class="yhcs">“搬”</span> <span class="encs">[sow;play]</span>。如:般弄(搬弄。捉弄;摆布);般演(搬演。用戏剧或其他文艺形式表现某件事实或故事)</p><p>◎ <strong>般</strong> <span class="dicpy">bān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bān.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“斑”</span>。斑纹 <span class="encs">[mottled]</span></p><p><span class="diczx1">马黑脊而般臂蝼。——<span class="smcs">《周礼·内饔》</span></span></p><p><span class="diczx1">般般之兽。——<span class="smcs">《史记·司马相如传》</span>。索隐:<span class="yhcs">“般般，文采之貌也。”</span></span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">bō;pán</span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/般般件件">般般件件</a></span><span class="crefe"><a href="/hans/般配">般配</a></span></div><p><strong>其它字义</strong></p><hr class="dichr" /><p>◎ <strong>般</strong> <span class="dicpy">bō</span></p><p>另见 <span class="dicpy">bān;pán</span></p><p><strong>常用词组</strong><hr class="dichr" /></p><div class="cit type-xxjs"><span class="crefe"><a href="/hans/般若">般若</a></span></div><p><strong>其它字义</strong></p><hr class="dichr" /><p>◎ <strong>般</strong> <span class="dicpy">pán</span></p><p>〈名〉</p><p><span class="cino">(1)</span> 大船 <span class="encs">[big ship]</span></p><p><span class="diczx1">般，大船也。——<span class="smcs">《玉篇》</span></span></p><p><span class="cino">(2)</span> 水边的高地 <span class="encs">[highland in waterside]</span></p><p><span class="diczx1">般，水涯堆也。——<span class="smcs">《汉书音义》</span></span></p><p><span class="cino">(3)</span> 通<span class="yhcs">“鞶”</span>。囊 <span class="encs">[bag]</span></p><p><span class="diczx1">诸母般。——<span class="smcs">《谷梁传》</span>。</span><span class="diczx2">范宁</span>注:<span class="yhcs">“般，囊也。所以盛朝夕所须。”</span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>般</strong> <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/822C.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/26A17.svg'><span class='k_txt'>𦨗</span>【唐韻】北潘切【集韻】【韻會】【正韻】逋潘切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24CD7.svg'><span class='k_txt'>𤳗</span>。【說文】辟也。象舟之旋，从舟从殳。殳，所以旋也。</p><p>又【廣韻】般運。【集韻】移也。</p><p>又【集韻】數別之名。</p><p>又【廣韻】布還切【集韻】【韻會】【正韻】逋還切。<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>與班同。【爾雅·釋言】般，還也。【疏】般，還反也。【前漢·趙充國傳】明主般師罷兵。【註】鄧展曰：般，音班。班，還也。</p><p>又【賈誼傳】般紛紛其離此鄕兮。【註】般，音班。般，反也。</p><p>又分也，賜也。【揚子·太<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>經】建侯開國，渙爵般秩。</p><p>又布也。【前漢·郊祀歌】先以雨般裔裔。【註】般與班同。布也。</p><p>又人名。【前漢·古今人表】公輸般。【張衡·西京賦】命般爾之巧匠。【註】般，魯般。魯哀公時巧人。</p><p>又與班同。【禮·內則】馬黑脊而般臂漏。【註】般臂，前脛般般然也。【周禮·天官·內饔註】般臂，臂毛有文。</p><p>又獸名。【揚雄·羽獵賦】屨般首。【註】屨，謂以足蹈之也。</p><p>又【史記·司馬相如傳】般般之獸，樂我君囿。【註】謂騶虞也。</p><p>又【廣韻】薄官切【集韻】【韻會】【正韻】蒲官切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音盤。【玉篇】大船也。</p><p>又與盤同。盤庚。【周語】作般庚。</p><p>又【博雅】般桓，不進也。</p><p>又般還。【禮·投壷】主人般還曰辟。【疏】主人見賓之拜，乃般曲折還，謂賓曰：今辟而不敢受。亦作般旋。【抱朴子·廣譬卷】般旋之儀，見憎於祼踞之鄕。</p><p>又【爾雅·釋詁】般，樂也。【疏】般者，遊樂也。【揚子·太<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>經】大樂無閒，民神禽鳥之般。【註】般，樂也。【張衡·思<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>賦】惟般逸之無斁兮。</p><p>又【博雅】般，行也。【又】任也。</p><p>又與槃同。【莊子·田子方】公使人視之，則解衣般礴。【註】般，字又作槃。般礴，謂其坐也。</p><p>又【詩·周頌】篇名。</p><p>又地名。【前漢·地理志】濟南郡般陽縣。</p><p>又水名。【山海經】沂山，般水出焉，而東流注于河。</p><p>又與鞶同。【穀梁傳·桓三年】諸母般申之曰：謹愼從爾父母之言。【註】般，囊也。所以盛朝夕所須，以備舅姑之用。【疏】男子般革，婦人般絲，所以盛帨巾之屬，爲謹敬也。</p><p>又與磐同。【前漢·郊祀志】乾稱飛龍，鴻斬于般。【註】孟康曰：般，水涯堆也。師古曰：般，山石之安者。</p><p>又【集韻】【韻會】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>補滿切，音粄。漢縣名。今在齊州地。【前漢·地理志】平原般縣。【註】韋音逋坦反。師古曰：爾雅，九河一曰鉤般。郭璞云：水曲如鉤，流般桓也。然今其土俗如韋音。</p><p>又【集韻】一曰面平貌。</p><p>又【正字通】音缽。梵言般若，華言智慧。若，音惹。</p><p>又蒲先切，音駢。【張衡·西京賦】蚩尤秉鉞，奮鬣被般。禁禦不若，以知神姦。魑魅罔兩，莫能逢旃。</p>
