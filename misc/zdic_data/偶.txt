                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>偶</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从人，禺声。禺，通常读 <span class="dicpy">yù</span>，又音 <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> 。本义:仿人形制成的木偶)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[image;idol]</span></p><p><span class="diczx1">偶，相人也。——<span class="smcs">《说文》</span>。按，相人者，像人也。一名俑。</span></p><p><span class="diczx2">鲁</span>以偶人，而 <span class="diczx2">孔子</span>叹。——<span class="smcs">《淮南子·缪称》</span></p><p><span class="diczx1">见偶人与偶人相与语。——<span class="smcs">《史记·孟尝君列传》</span></span></p><p><span class="cino">(3)</span> 又如:头偶;偶戏(木偶戏)</p><p><span class="cino">(4)</span> 伙伴;同伴;与人共处 <span class="encs">[companion]</span></p><p><span class="diczx1">乃率其曹偶，亡之江中。——<span class="smcs">《史记·黥布列传》</span></span></p><p><span class="cino">(5)</span> 又如:偶行(结伴同行);偶攻(结伙攻击);偶耕(两人并耕);偶影(与影为偶，形容孤独);偶寝(同睡)</p><p><span class="cino">(6)</span> 对偶 <span class="encs">[antithesis]</span>。如:偶句(对偶的句子);偶对(指诗文的对偶);偶辞(对偶的词语);偶俪(骈体;对偶);偶属(连缀句子使成对偶);偶章(指偶句形式为主的诗章);偶倡(可以比并的歌词)</p><p><span class="cino">(7)</span> <span class="encs">[中医]</span>∶偶方(中药方剂名称，七方之一。指药味合于双数或两味药合成的方剂。亦指两方相合而成的复方) <span class="encs">[prescription with ingredients even in number]</span></p><p><strong>词性变化</strong></p><p>◎ <strong>偶</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 配合 <span class="encs">[coordinate;concert]</span></p><p><span class="diczx1">汝明勗偶王，在亶，乘兹大命。——<span class="smcs">《书·君奭》</span></span></p><p><span class="cino">(2)</span> 匹对;配偶 <span class="encs">[mate;spouse]</span></p><p><span class="diczx1">汝幸而偶我，又何不幸而生今日之中国。(偶，动词。与…为配偶，这里作<span class="yhcs">“嫁”</span>字解释。)——</span><span class="diczx2">清</span>· <span class="diczx2">林觉民</span><span class="smcs">《与妻书》</span></p><p><span class="cino">(3)</span> 又如:良偶;丧偶;偶成(配成婚姻);偶配(结成配偶);偶婚(即对偶婚)</p><p><span class="cino">(4)</span> 迎合;投合;适应 <span class="encs">[pander to]</span>。如:偶世(投合时世);偶俗(迎合世俗);偶时(适应时势);偶变投隙(投机取巧)</p><p><span class="cino">(5)</span> 假借为<span class="yhcs">“遇”</span>。遇合，得到赏识<span class="encs">[meet together;get along together]</span></p><p><span class="diczx2">然</span>士亦有遇合，贤者多如此二子，不得尽意，岂可胜道哉!——<span class="smcs">《史记·范雎蔡泽列传赞》</span></p><p>◎ <strong>偶</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈形〉</p><p>通<span class="yhcs">“耦”</span>。双数。与<span class="yhcs">“奇”</span>相对 <span class="encs">[even(number);in pairs]</span>。如:奇偶检查;偶年(逢双的年纪)</p><p>◎ <strong>偶</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
