                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>扑</strong></p><p><strong>（<span class="diczx3">撲</span>）</strong></p><p><span class="dicpy">pū  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pū.mp3"></a></span>  ㄆㄨˉ</span></p><ol><li>轻打，拍：～粉。～蝇。～打。</li><li>击拂的用具：粉～。</li><li>冲：～灭。～救。</li><li>相搏击：相～。～跌。</li><li>跌倒：～地。</li><li>伏：～在桌上看书。</li><li>杖，戒尺，亦为中国周代九刑之一。</li></ol>

                    <h2 class="h2_entry"><span class="orth">扑</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>扑</strong></p><p>撲 <span class="dicpy">pū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pū.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。左形，右声。本义:击，打)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[beat;strike]</span></p><p><span class="diczx1">扑，挨也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">扑，击也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">连杖曰扑。——<span class="smcs">《通俗文》</span></span></p><p><span class="diczx1">荫不祥之木为雷电所扑。——<span class="smcs">《淮南子·说林》</span></span></p><p><span class="diczx1">举筑扑</span><span class="diczx2">秦皇帝</span>，不中。——<span class="smcs">《史记·刺客列传》</span></p><p><span class="diczx1">吾即扑杀汝。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《左忠毅公逸事》</span></p><p><span class="cino">(3)</span> 又如:扑掌(拍掌。表示惊讶、惋惜);扑作教刑(用板子责打违反礼教的人);扑刑(指教师用戒尺、教鞭之类对学生所作的体罚);扑抶(杖击，鞭打);扑责(杖击责罚);扑枣(击落枣子)</p><p><span class="cino">(4)</span> 进攻 <span class="encs">[attack]</span></p><p><span class="diczx2">秦</span>破 <span class="diczx2">韩</span>、 <span class="diczx2">魏</span>，扑师 <span class="diczx2">武</span>。——<span class="smcs">《史记·周本纪》</span></p><p><span class="cino">(5)</span> 拂着;拂拭 <span class="encs">[whisk or wipe off]</span></p><p><span class="diczx1">旋见鸡伸颈摆扑。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(6)</span> 又如:扑复(扑击);扑手(扑击的招数);扑殄(扑灭);扑剪(扑灭铲除)</p><p><span class="cino">(7)</span> 倾倒。通<span class="yhcs">“仆”</span> <span class="encs">[fall]</span></p><p><span class="diczx2">秦</span>破 <span class="diczx2">韩</span><span class="diczx2">魏</span>，扑师武。——<span class="smcs">《史记·周本纪》</span></p><p><span class="diczx1">朽机懼倾扑。——</span><span class="diczx2">韩愈</span><span class="smcs">《纳凉联句》</span></p><p><span class="cino">(8)</span> 又如:扑翻身(跪下后两手伏地，扑倒身子);扑跌(相扑，摔交);扑破(摔破;摔坏);扑杀(摔死;击杀)</p><p><span class="cino">(9)</span> 直冲 <span class="encs">[rush at]</span>。如:孩子一下子扑到他妈的怀里去;老虎向山羊扑去</p><p><span class="cino">(10)</span> 赌博 <span class="encs">[gamble]</span>。如:扑卖(宋元时小商败推销用的一种博戏手法，以钱为赌具)</p><p><span class="cino">(11)</span> 尽全力 <span class="encs">[devote]</span>。如:一心扑在集体事业上</p><p><span class="cino">(12)</span> 〈方〉∶伏，趴 <span class="encs">[bend over]</span>。如:扑在桌上看地图</p><p><span class="cino">(13)</span> 喻指烟、气之属直冲(人的感官) <span class="encs">[assail]</span></p><p><span class="diczx1">剖之，有如烟扑口鼻。——</span><span
