<div class="enbox"><p><span class="z_ts2">英语</span> jade; green, blue</p><p><span class="z_ts2">法语</span> émeraude,vert bleuâtre,jade vert</p></div>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>碧</strong> <span class="dicpy">bì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从玉，从石，白声。本义:青绿色的玉石)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[green jade]</span></p><p><span class="diczx1">碧，石之青美者。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">高山，其下多青碧。——<span class="smcs">《山海经·西山经》</span></span></p><p><span class="cino">(3)</span> 又</p><p><span class="diczx1">章峨之山多瑶碧。</span></p><p><span class="diczx1">昆仑碧树瑶树在其北。——<span class="smcs">《淮南子·地形》</span></span></p><p><span class="diczx1">锡碧金银，众色炫耀。——<span class="smcs">《汉书·司马相如传》</span></span></p><p><span class="cino">(4)</span> 又如:碧卢(像玉的美石);碧箫(用碧玉制的箫);碧玉椽(用碧玉做的椽子)</p><p><span class="cino">(5)</span> 指代绿水 <span class="encs">[green water]</span></p><p><span class="diczx1">碧水东流至此回。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span><span class="smcs">《望天门山》</span></p><p><span class="diczx1">一碧万顷。——</span><span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="cino">(6)</span> 又如:碧波(清澄绿色的水波);碧泱泱(水流澄绿深广的样子);碧流(绿水);碧漪(清澈的水波。泛指绿水)</p><p><strong>词性变化</strong></p><p>◎ <strong>碧</strong> <span class="dicpy">bì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 青绿色 <span class="encs">[bluish green]</span></p><p><span class="diczx1">绿碧青丝绳。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="diczx1">碧峰巉然孤起。——</span><span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《过小孤山大孤山》</span></p><p><span class="diczx1">寒则苍，春则碧。——</span><span class="diczx2">王夫人</span><span class="smcs">《小云山记》</span></p><p><span class="cino">(2)</span> 又如:碧香(本指绿酒的馨，后为酒名);碧海(碧绿色的海);碧芳(绿叶);碧城(传说中仙人居住的地方)</p><p><span class="cino">(3)</span> 青白色，浅蓝色 <span class="encs">[clear blue]</span></p><p><span class="diczx1">春草碧色，春水渌波。——</span><span class="diczx2">江淹</span><span class="smcs">《别赋》</span></p><p><span class="diczx1">接天连叶无穷碧。——</span><span class="diczx2">宋</span>· <span class="diczx2">杨万里</span><span class="smcs">《晓出净慈寺送林子芳》</span></p><p><span class="cino">(4)</span>
