                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>唯</strong> <span class="dicpy">wéi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" class="diczx2">参</span>乎!吾道一以贯之。”</span> <span class="diczx2">曾子</span>曰:<span class="yhcs">“唯。”</span>——<span class="smcs">《论语·里仁》</span></p><p><span class="diczx2">秦王</span>跽而请曰:<span class="yhcs">“先生何以幸教寡人?”</span> <span class="diczx2">范睢</span>曰:<span class="yhcs">“唯，唯。”</span>——<span class="smcs">《史记·范睢蔡泽列传》</span></p><p><span class="cino">(2)</span> 又如:唯阿(唯、阿同为应对 语词。比喻差别不大);唯俞(应答辞);唯唯(恭逊的应辞);唯诺(应答;连 声应诺而不表异议)</p><p><strong>词性变化</strong></p><p>◎ <strong>唯</strong> <span class="dicpy">wéi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" class="smcs">《与陈伯之书》</span></p><p><span class="diczx1">唯余其一。——</span><span class="diczx2">吴均</span><span class="smcs">《与朱元思书》</span></p><p><span class="diczx1">唯巨石岿然。——<span class="smcs">《吕氏春秋·慎行论》</span></span></p><p><span class="cino">(2)</span> 又如:唯心(佛教语，谓一切诸法(指万事万物)，唯有内心。无心外之法。也称<span class="yhcs">“唯识”</span>);唯谨(唯有谨慎)</p><p><span class="cino">(3)</span> 以，因为 <span class="encs">[because]</span></p><p><span class="diczx1">唯无德也。——<span class="smcs">《国语·晋语》</span></span></p><p><span class="diczx1">冀之既病，则亦唯君故。——<span class="smcs">《左传》</span></span></p><p><span class="cino">(4)</span> 表示希望、祈使 <span class="encs">[wish]</span></p><p><span class="diczx1">唯大王与群臣熟计议之。——<span class="smcs">《史记》</span></span></p><p><span class="cino">(5)</span> 表示听任、任随 <span class="encs">[allow;let]</span>。常和<span class="yhcs">“所”</span>搭配</p><p><span class="diczx1">唯所纳之，无不如志。——<span class="smcs">《左传》</span></span></p><p>◎ <strong>唯</strong> <span class="dicpy">wéi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wéi.mp3"></a></span> </p><p><连></p><p>虽然，纵使 <span class="encs">[though]</span>——表示让步关系</p><p><span class="diczx1"><span class="encs">[</span><span class="diczx2">公孙</span>]</span> <span class="diczx2">弘</span>、<span class="encs">[ <span
