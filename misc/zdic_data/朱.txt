                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>朱</strong> <span class="dicpy">zhū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhū.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (指事。小篆字形，从木，一在其中指出这种木是红心的。 本义:赤心木)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[the tree with red core]</span></p><p><span class="diczx1">朱，赤心木，松柏属。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> 指朱色的物品 <span class="encs">[scarlet thing]</span></p><p><span class="diczx1">…朱(此指朱色缯帛)四百裹。——<span class="smcs">《穆天子传》</span></span></p><p><span class="cino">(4)</span> 姓</p><p><span class="diczx2">朱孝纯</span><span class="diczx2">子颍</span>。(朱孝纯，字子颍，山东历城人，乾隆进士，当时任泰安知府，善诗画。)—— <span class="diczx2">清</span>· <span class="diczx2">姚鼐</span><span class="smcs">《登泰山记》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>朱</strong> <span class="dicpy">zhū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhū.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 朱色，大红色 。古代称为正色 <span class="encs">[vermilion;bright-red]</span></p><p><span class="diczx1">网户朱缀。——<span class="smcs">《楚辞·招魂》</span></span></p><p><span class="diczx1">紫衣而朱冠。——<span class="smcs">《庄子·达生》</span></span></p><p><span class="diczx1">朱衣冠，执朱弓，挟朱矢。——<span class="smcs">《墨子·明鬼下》</span></span></p><p><span class="diczx1">朱门酒肉臭，路有冻死骨。——</span><span class="diczx2">杜甫</span><span class="smcs">《自京赴奉先县咏怀五百字》</span></p><p><span class="diczx2">禹</span>作为祭器，墨染其外，而朱画其内。——<span class="smcs">《韩非子·十过》</span></p><p><span class="diczx1">朱鳞火鬣。——</span><span class="diczx2">唐</span>· <span class="diczx2">李朝威</span><span class="smcs">《柳毅传》</span></p><p><span class="diczx1">朱缨宝饰之帽。——</span><span class="diczx2">明</span>· <span class="diczx2">宋濂</span><span class="smcs">《送东阳马生序》</span></p><p><span class="cino">(2)</span> 又如:朱印(朱色印记);朱封(用朱笔在封口作标记);朱押(用朱笔在封条等上面签字或画花押)</p><p>◎ <strong>朱</strong></p><p>硃 <span class="dicpy">zhū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhū.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> <span class="yhcs">“朱砂”</span>的简称<span
