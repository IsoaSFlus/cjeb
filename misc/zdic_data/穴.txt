						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>穴</strong></p><p><span class="dicpy">xué  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xué.mp3"></a></span>  ㄒㄩㄝˊ</span></p><ol><li>洞，窟窿：孔～。石～。～居。～隙。龙潭虎～。</li><li>人体可以进行针灸的部位，多为神经末梢密集或较粗的神经干经过的地方：～位。</li><li>旧艺人称在市场或广场上表演时所占的一块地方：掌～的（指地主或班主）。走～。～头。</li><li>姓。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> cave, den, hole; KangXi radical 116</p><p><span class="z_ts2">德语</span> Grab  (S)​,Höhle, Nest, Loch  (S)​,Radikal Nr. 116 = Loch, Höhle, Akupunkturpunkt  (S)​,Akupunkturpunkt  (S, Med)</p><p><span class="z_ts2">法语</span> grotte,caverne,point d'acupuncture</p></div>

                    <h2 class="h2_entry"><span class="orth">穴</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>穴</strong> <span class="dicpy">xué</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xué.mp3"></a></span> </p><p><span class="diczx1">〈名〉</span></p><p><span class="cino">(1)</span> (象形。小篆字形，上面是<span class="yhcs">“宀”</span>，表覆盖物;下面两边表示洞孔。本义:土窟窿，地洞)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[cave;hole]</span></p><p><span class="diczx1">穴，土室也。——<span class="smcs">《说文》</span>。按，象嵌空之形，非八字。</span></p><p><span class="diczx1">出自穴。——<span class="smcs">《易·需》</span>。注:<span class="yhcs">“阴之路也。”</span></span></p><p><span class="diczx1">有穴窈然，入之甚寒。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="diczx1">古之民未知为宫室时，就陵阜而居，穴而处。——<span class="smcs">《墨子·辞过》</span></span></p><p><span class="cino">(3)</span> 又如:洞穴(地洞或山洞);穴人(在洞穴中采钟乳石的人);穴流(洞穴中的水流)</p><p><span class="cino">(4)</span> 墓穴;埋棺材的坑 <span class="encs">[coffin pit;open grave]</span></p><p><span class="diczx1">临其穴。——<span class="smcs">《诗·秦风·黄鸟》</span></span></p><p><span class="diczx1">谷则异室，死则同穴。——<span class="smcs">《诗·王风·大车》</span></span></p><p><span class="cino">(5)</span> 又如:寿穴(生前造的墓穴)</p><p><span class="cino">(6)</span> 动物的窝 <span class="encs">[den]</span></p><p><span class="diczx1">狐狸驰赴穴，飞鸟翔故林。——</span><span class="diczx2">汉</span>· <span class="diczx2">王粲</span><span class="smcs">《七哀诗》</span></p><p><span class="cino">(7)</span> 又如:穴巢(鸟兽藏身的窝巢);虎穴;蚁穴</p><p><span class="cino">(8)</span> 指敌人或奸人盘据、藏匿的地方 <span class="encs">[lair]</span></p><p><span class="diczx1">上官欲察州里之豪，不能不假耳目，而奸人常为之穴。——</span><span class="diczx2">清</span>· <span class="diczx2">顾炎武</span><span class="smcs">《天下郡国利病书》</span></p><p><span class="cino">(9)</span> 又如:穴群(谓聚众盘据);穴巢(喻敌人或匪徒盘据、活动的场所)</p><p><span class="cino">(10)</span> 地道 <span class="encs">[tunnel]</span></p><p><span class="diczx1">审知穴之所在，凿穴迎之。——<span class="smcs">《墨子》</span></span></p><p><span class="cino">(11)</span> 又如:穴土(挖地道);穴师(挖地道的士兵);穴地(挖地道)</p><p><span class="cino">(12)</span> 水道 <span class="encs">[water course]</span></p><p><span class="diczx2">江</span><span class="diczx2">河</span>既导，万穴俱流。——<span class="smcs">《文选》</span></p><p><span class="cino">(13)</span> 中医指人体上可以针灸的部位，多为神经末稍密集或较粗的神经纤维经过的地方 <span class="encs">[acupuncture point;acupoint]</span>。如:穴道;穴位</p><p><strong>词性变化</strong></p><p>◎ <strong>穴</strong> <span class="dicpy">xué</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xué.mp3"></a></span> </p><p><span class="diczx1">〈动〉</span></p><p><span class="cino">(1)</span> 穴居;穴藏 <span class="encs">[live in caves]</span></p><p><span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/7A74.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/25922.svg'><span class='k_txt'>𥤢</span>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>胡決切，音坹。【說文】土室也。【易·繫辭】上古穴居而野處。【詩·大雅】陶復陶穴。【箋】未有寢廟，故覆穴而居。</p><p>又【玉篇】孔穴也。【孟子】鑽穴隙相窺。</p><p>又【廣韻】窟也。【易·需卦】出自穴。</p><p>又訓爲側。【爾雅·釋水】氿泉穴出。穴出，仄出也。</p><p>又官名。【周禮·秋官】穴氏掌攻蟄獸。【疏】凡獸皆藏穴中，故以穴爲官名，使取蟄獸。</p><p>又地名。【書·禹貢】鳥鼠同穴。【爾雅·釋地】岠齊州以南戴日爲丹穴。【左傳·文十一年】潘崇伐麇，至于鍚穴。【註】鍚穴，麇地。【水經注】中廬縣之西山謂之馬穴山。【左思·蜀都賦】嘉魚出于丙穴。【註】丙穴，在漢中沔陽縣北。</p><p>又【韻會】古穴切。【前漢·天文志】暈適背穴。【註】孟康曰：穴，或作鐍，其形如鐍。如淳曰：凡氣在日上爲冠爲戴，在旁直射爲珥，在旁如半環向日爲抱，向外爲背，有氣刺日爲鐍。鐍，抉傷也。</p><p>又叶胡桂切。【曹植·七啓】采英奇于側陋，宣皇明于巖穴。此甯子商歌之秋，而呂望所以投綸而逝。</p><p>又【集韻】戸橘切。【淮南子·原道訓】水居窟穴，人民有室。【孔融詩】言多令事敗，器漏苦不密。河潰蟻孔端，山壞由猿穴。</p><p>
