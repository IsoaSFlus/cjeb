						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>具</strong></p><p><span class="dicpy">jù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> ㄐㄩˋ</span></p><ol><li>器物：器～。</li><li>备有：～备。～有。别～只眼（形容具有独到的眼光和见解）。</li><li>备，办：～呈。～结。～名。～领。敬～菲酌。</li><li>古同“<span class="diczx3">俱</span>”，都，完全。</li><li>量词，用于某些器物和棺材、尸体：座钟一～。一～男尸。</li></ol>

                    <h2 class="h2_entry"><span class="orth">具</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>具</strong> <span class="dicpy">jù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。甲骨文字形，上面是<span class="yhcs">“鼎”</span>，下面是双手。表示双手捧着盛有食物的鼎器(餐具)。本义:准备饭食或酒席)</p><p><span class="cino">(2)</span> 同本义。泛指准备，备办 <span class="encs">[prepare]</span></p><p><span class="diczx1">具，供置也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">具，备也，办也。——<span class="smcs">《广韵》</span></span></p><p><span class="diczx1">以食具告。——<span class="smcs">《仪礼·士相见礼》</span>。注:<span class="yhcs">“犹办也。”</span></span></p><p><span class="diczx1">官备则具备。——<span class="smcs">《礼记·祭统》</span></span></p><p><span class="diczx1">寿为具召武帝显。——<span class="smcs">《汉书·何武传》</span></span></p><p><span class="diczx1">礼举仪具。——</span><span class="diczx2">张衡</span><span class="smcs">《东京赋》</span></p><p><span class="diczx1">具器械。——<span class="smcs">《孙子·谋攻》</span></span></p><p><span class="diczx1">故人具鸡黍。——</span><span class="diczx2">唐</span>· <span class="diczx2">孟浩然</span><span class="smcs">《过故人庄》</span></p><p><span class="diczx1">第二板已具。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="diczx1">别具本章。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(3)</span> 又如:具食(准备食物);具觞(备酒。觞:酒杯);具状(备办词状;写状子);具禀(备办上报的文件);具覆(备办回复的文件)</p><p><span class="cino">(4)</span> 写，题 <span class="encs">[write]</span></p><p><span class="diczx2">李</span>具状求在狱。—— <span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(5)</span> 又如:具草(拟稿，起草);具衔(题写官衔);知名不具;具结检验(与案件有关的人员出具保证，以免对尸体、现场等进行检验)</p><p><span class="cino">(6)</span> 具备，有 <span class="encs">[have;possess]</span></p><p><span class="diczx1">具乎其前。——</span><span class="diczx2">清</span>· <span class="diczx2">黄宗羲</span><span class="smcs">《柳敬亭传》</span></p><p><span class="diczx1">各具情态。——</span><span class="diczx2">明</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="diczx1">狱具矣。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(7)</span> 又如:具寿(和尚的通称。师父呼弟子或长老呼少年和尚时常用。具有尽其寿命及法身慧命的意思);具眼(具有鉴别事物的见识和能力);具足(具备)</p><p><span class="cino">(8)</span> 判决;定案 <span class="encs">[verdict]</span></p><p><span class="diczx1">有某姓兄弟，以把持公仓，法应立决，狱具矣。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>具</strong> <span class="dicpy">jù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 用具;器械
