						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>牌</strong></p><p><span class="dicpy">pái  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pái.mp3"></a></span>  ㄆㄞˊ</span></p><ol>    <li>用木板或其他材料做的标志：门～。路～。～子。～匾。～价。</li>    <li>凭证：金～。银～。腰～。</li>    <li>产品的商标：名～货。</li>    <li>古代兵士在战争时或现代警察在驱散示威者时用来遮护身体的东西：盾～。挡箭～。</li>    <li>娱乐或赌博用的东西：纸～。扑克～。麻将～。</li>    <li>神主、灵位或题着名字作为祭祀对象的木牌：～位。</li>    <li>词曲的调名：词～。曲～。～子曲。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>牌</strong> <span class="dicpy">pái</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pái.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从片，卑声。<span class="yhcs">“片”</span>指锯开的木头的一半，又泛指扁而薄的东西。本义:题榜，招牌)</p><p><span class="cino">(2)</span> 户籍板。犹今之户口簿 <span class="encs">[census plate]</span></p><p><span class="diczx1">簧牌，籍也。——<span class="smcs">《广雅》</span></span></p><p><span class="cino">(3)</span> 用木板、金属片或其他材料做成的标志 <span class="encs">[plate;sign;tablet]</span>。如:门牌;路牌;招牌;布告牌;名牌</p><p><span class="cino">(4)</span> 商标 <span class="encs">[brand;trademark]</span>。如:厂牌;名牌</p><p><span class="cino">(5)</span> 古时的凭证;符信 <span class="encs">[nameplate]</span>。如:牌金(金牌。古代官员的一种身份凭证);牌面(古代官吏、使节的一种身分凭证);牌印(令牌印信);牌令(道士用的法器);腰牌</p><p><span class="cino">(6)</span> 古代一种下行公文的名称 <span class="encs">[document]</span>。如:行牌;牌文(官府文书名。古时各部行道府以下官府所用的公文);牌使(传示帝王命令的使臣);牌司(古代收受讼状的府吏)</p><p><span class="cino">(7)</span> 娱乐或赌博用的卡片 <span class="encs">[card]</span>。如:扑克牌;骨牌;纸牌</p><p><span class="cino">(8)</span> 价格目录 <span class="encs">[list]</span>。如:牌价</p><p><span class="cino">(9)</span> 古时兵士打仗时用来遮护身体的<span class="yhcs">“盾”</span>的俗称 <span class="encs">[shield]</span>。如:藤牌;挡箭牌</p><p><span class="cino">(10)</span> 词曲的调门 <span class="encs">[pitch]</span>。如:词牌<span class="smcs">《菩萨蛮》</span>;曲牌<span class="smcs">《驻云飞》</span></p><p><span class="cino">(11)</span> 旌表纪念的建筑物 <span class="encs">[memorial]</span>。如:牌坊;牌楼</p><p><span class="cino">(12)</span> 样子，姿势 <span class="encs">[manner;air;shape]</span>。如:牌子儿(样儿);牌谱(姿势;样子)</p><p><span class="cino">(13)</span> <span class="yhcs">“牌军”</span>的略称 <span class="encs">[inferior officer]</span></p><p><span class="diczx1">一直奔回家来，又和</span><span class="diczx2">李牌</span>寻了一黄昏。——<span class="smcs">《水浒传》</span></p><p><span class="cino">(14)</span> 指<span class="yhcs">“万岁龙牌”</span> <span class="encs">[emperor's plate]</span></p><p><span class="diczx1">有天万寿，跟了抚台拜牌。——<span
