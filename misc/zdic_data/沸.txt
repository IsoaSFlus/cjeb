						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>沸</strong></p><p><span class="dicpy">fèi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fèi.mp3"></a></span>  ㄈㄟˋ</span></p><ol><li>开，滚，液体受热到一定温度时，内部发生气泡，表面翻滚，变成蒸气：～点。～水。～涌。～腾（亦喻事物蓬勃发展或情绪高涨）。～反盈天（形容人声喧闹，乱成一片）。人声鼎～。</li><li>波涌的样子：～郁（ａ．翻涌的样子；ｂ．愤懑不平的样子）。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>沸</strong> <span class="dicpy">fèi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fèi.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从水，弗声。本义:泉水大量涌出的样子)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[gushing out]</span></p><p><span class="diczx1">沸，滭沸滥泉也。——<span class="smcs">《说文》</span>。按，涌出之貌。</span></p><p><span class="diczx1">觱沸槛泉。——<span class="smcs">《诗·小雅·采菽》</span></span></p><p><span class="diczx1">百川沸騰。——<span class="smcs">《诗·小雅·十月之交》</span></span></p><p><span class="diczx1">愤泉秋沸。——</span><span class="diczx2">北周</span>· <span class="diczx2">庾信</span><span class="smcs">《哀江南赋》</span></p><p><span class="cino">(3)</span> 又如:沸沸(翻滚涌现的样子;行事迅速果断的样子);沸射(喷射);沸然(水腾涌的样子)</p><p><span class="cino">(4)</span> 水波翻涌的样子 <span class="encs">[surgent]</span></p><p><span class="diczx1">宫殿摆簸，云烟沸涌。——</span><span class="diczx2">唐</span>· <span class="diczx2">李朝威</span><span class="smcs">《柳毅传》</span></p><p><span class="cino">(5)</span> 又如:沸波(鸟名。即鱼鹰。因俯冲食水中鱼时扬起波浪);沸渭(水翻腾奔涌的样子);沸泻(水流翻滚奔腾)</p><p><span class="cino">(6)</span> 名声很响，影响很大 <span class="encs">[big-name]</span>。如:沸传(盛名远扬)</p><p><span class="cino">(7)</span> 杂乱，纷乱 <span class="encs">[disordered]</span>。如:沸乱(纷乱;烦乱);沸羹(比喻动荡混乱的局势)</p><p><strong>词性变化</strong></p><p>◎ <strong>沸</strong> <span class="dicpy">fèi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fèi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 把水烧开;沸腾 <span class="encs">[boil;bubble]</span></p><p><span class="diczx1">抽薪止沸，剪草除根。——</span><span class="diczx2">魏收</span><span class="smcs">《为侯景叛移梁朝文》</span></p><p><span class="cino">(2)</span> 又如:沸煎(沸滚，沸腾翻滚);沸溢(水翻滚漫出);沸动(翻动)</p><p><span class="cino">(3)</span> 声音喧闹或嘈杂 <span class="encs">[be noisy]</span></p><p><span class="diczx1">沸乎暴怒。——</span><span class="diczx2">司马相如</span><span class="smcs">《上林赋》</span></p><p><span class="diczx1">临河列阵，百余里中，钟鼓之声沸河动地。——<span class="smcs">《晋书·刘曜载记》</span></span></p><p><span class="cino">(4)</span> 又如:沸喧(沸激。像滚水一样喧闹);沸耳(声音喧腾);沸聒(喧腾，嘈杂);沸闹(喧闹);沸稠(繁密喧腾)</p><p>◎ <strong>沸</strong> <span class="dicpy">fèi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fèi.mp3"></a></span> </p><p>〈名〉</p><p>沸水 <span class="encs">[boiling
