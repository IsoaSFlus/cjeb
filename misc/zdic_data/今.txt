                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>今</strong> <span class="dicpy">jīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jīn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。从佶亼( <span class="dicpy">jí</span>)。本义:现在)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[the present]</span></p><p><span class="diczx1">今，是时也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">今，时辞也。——<span class="smcs">《苍颉篇》</span></span></p><p><span class="diczx1">迨其今兮。——<span class="smcs">《诗·召南·摽有梅》</span></span></p><p><span class="diczx1">于今三年。——<span class="smcs">《诗·豳风·东山》</span></span></p><p><span class="diczx1">吾今召君矣。——<span class="smcs">《史记·汲郑传》</span></span></p><p><span class="diczx1">今行而无信，则</span><span class="diczx2">秦</span>未可亲也。——<span class="smcs">《战国策·燕策》</span></p><p><span class="diczx1">今乃于</span><span class="diczx2">毛</span>先生而失之也。——<span class="smcs">《史记·平原君虞卿列传》</span></p><p><span class="diczx1">今其室十无一焉。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《捕蛇者说》</span></p><p><span class="diczx1">今之高爵显位。——</span><span class="diczx2">明</span>· <span class="diczx2">张溥</span><span class="smcs">《五人墓碑记》</span></p><p><span class="diczx1">今年四月。(指清朝光绪二十四年(1898年)，即戊戌年。四月，旧历。公历是六月。)——</span><span class="diczx2">清</span>· <span class="diczx2">梁启超</span><span class="smcs">《谭嗣同传》</span></p><p><span class="cino">(3)</span> 又如:今朝三明朝四(指一天天地推诿、拖延);今雨(新近结交的朋友);今愁古恨(今人之愁和古人之恨);今下(现时，眼下);今夕何夕(今夜是何夜?多用作赞叹语。谓此良辰)</p><p><span class="cino">(4)</span> 现代;当代。与<span class="yhcs">“古”</span>相对 <span class="encs">[modern times]</span></p><p><span class="diczx1">今之乐，犹古之乐也。——<span class="smcs">《孟子·梁惠王下》</span></span></p><p><span class="cino">(5)</span> 又如:今上(帝制时代，尊称当代在位的皇帝);今体(当代所通行的诗、文、书法等体裁。别于古体而言);今字(指当代所使用的文字。即:楷书;隶书)</p><p><span class="cino">(6)</span> 通<span class="yhcs">“金”</span> <span class="encs">[gold]</span>。如:今蝉蜕壳(金蝉脱壳。今，通<span class="yhcs">“金”</span>。比喻用计谋逃脱)</p><p><span class="cino">(7)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>今</strong> <span class="dicpy">jīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jīn.mp3"></a></span> </p><p>〈副〉</p><p>即将;立刻;马上 <span class="encs">[immediately;at once;right away]</span></p><p><span class="diczx1">夺</span><span class="diczx2">项王</span>天下者，必 <span class="diczx2">沛公</span>也，吾属今为之虏矣!——<span class="smcs">《史记·项羽本纪》</span></p><p>◎ <strong>今</strong> <span class="dicpy">jīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jīn.mp3"></a></span> </p><p>〈连〉</p><p>假使，如果 <span class="encs">[if]</span></p><p><span class="diczx1">今有人于此。——<span class="smcs">《墨子·公输》</span></span></p><p><span class="diczx1">今有构木钻燧于</span><span class="diczx2">夏后氏</span>之世者，必为 <span class="diczx2">鲧</span><span class="diczx2">禹</span>笑矣。(今，如果。)——<span class="smcs">《韩非子·五蠹》</span></p><p><span class="diczx1">今</span><span class="diczx2">王</span>鼓乐。——<span class="smcs">《孟子·梁惠王下》</span></p><p><span class="diczx1">今</span><span class="diczx2">括</span>一旦为将。—— <span class="diczx2">汉</span>· <span class="diczx2">刘向</span><span class="smcs">《列女传》</span></p><p><span class="diczx1">今若断斯织。</span></p><p>◎ <strong>今</strong> <span class="dicpy">jīn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jīn.mp3"></a></span> </p><p>〈代〉</p><p>这，此 <span class="encs">[this]</span>。如:今次(这次);今遭(这一回);今早(今朝;今天)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div                           <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/4ECA.svg">                          <p>是時也。从亼从乁。乁，古文及。居音切</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/4ECA_101.svg"  /><span class="swbf">(今)</span><span class="ywyy">是時也。</span><span class="swdz">今者對古之偁。古不一其時。今亦不一其時也。云是時者。如言目前。則目前爲今。目前已上皆古。如言趙宋。則趙宋爲今。趙宋已上爲古。如言魏晉。則魏晉爲今。魏晉已上爲古。班固作古今人表。漢人不與焉。而謂之古今人者、謂近乎漢者爲今人。遠乎漢者爲古人也。作古今人表者所以補漢書之所無。存漢已前之厓略也。亦謂三皇至漢以前迭爲古今人也。古今人用字不同、謂之古今字。張揖作古今字詁是也。自張揖已後。其爲古今字又不知幾更也。古今音之不同近世言之冣詳。自商周至近世、不知凡幾古今也。故今者、無定之䛐。約之以是時則兼賅矣。召南傳曰。今、急辭也。今急㬪韵。</span><span class="ywyy">从亼乀。</span><span class="swdz">會意。乀、逮也。乀亦聲。居音切。七部。</span><span class="ywyy">乀、古文及。</span><span class="swdz">見又部。</span><span class="xh"></span></p>                          </div>
