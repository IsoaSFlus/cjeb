                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>寇</strong> <span class="dicpy">kòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kòu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从宀( <span class="dicpy">mián</span>)，表示与室家房屋有关，从元(人)，从攴( <span class="dicpy">pū</span>)，表示持械击打。意思是手持器械的人，侵犯到房子里来打人。本义:入侵;侵犯)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[invade;plunder]</span></p><p><span class="diczx1">寇，暴也。与败贼 同义(朋侵)。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">寇贼奸宄。——<span class="smcs">《书·舜典》</span>。传:<span class="yhcs">“群行攻劫曰寇。”</span></span><span class="diczx2">郑</span>注:<span class="yhcs">“强取为寇。”</span></p><p><span class="diczx1">凡兵作于内为乱，于外为寇。——<span class="smcs">《左传·文公七年》</span></span></p><p><span class="diczx1">大兵不寇。——<span class="smcs">《吕氏春秋·贵公》</span></span></p><p><span class="diczx2">匈奴</span>背叛不臣，数为寇暴于边鄙。——<span class="smcs">《盐铁论·本议》</span></p><p><span class="diczx2">吐蕃</span>发十万兵寇 <span class="diczx2">西川</span>。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(3)</span> 又如:寇戎(来犯的敌军);寇虐(侵掠残害之行)寇钞(攻劫掠夺);寇剽(抢劫);寇钞(亦作<span class="yhcs">“寇抄”</span>。劫掠);寇攘(劫掠;侵扰);寇窃(抢劫;盗窃)</p><p><span class="cino">(4)</span> 砍伐 <span class="encs">[fell;cut;chop]</span></p><p><span class="diczx1">山木自寇也，膏火自煎也。——<span class="smcs">《庄子·人间世》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>寇</strong> <span class="dicpy">kòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kòu.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 入侵者 <span class="encs">[invader]</span></p><p><span class="diczx1">西山寇盗莫相侵。——</span><span class="diczx2">杜甫</span><span class="smcs">《登楼》</span></p><p><span class="cino">(2)</span> 敌人 <span class="encs">[enemy]</span></p><p><span class="diczx1">岂可复留此残寇，使长为国家之忧者。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(3)</span> 贼兵，敌军 <span class="encs">[foe]</span></p><p><span class="diczx1">在</span><span class="diczx2">宋</span>城上而待 <span class="diczx2">楚</span>寇矣。——<span class="smcs">《墨子·公输》</span></p><p><span class="diczx1">今寇众我寡。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(4)</span> 又如:寇乱(外寇与内乱。兵起于外为寇，起于内为乱)</p><p><span class="cino">(5)</span> 盗匪 <span class="encs">[bandit]</span></p><p><span class="diczx1">北与寇往来其间。——</span><span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《指南录后序》</span></p><p><span class="cino">(6)</span> 又如:寇逆(贼寇，叛逆);寇劫(指行劫的群盗);寇戎(匪患与战争);寇奸(寇贼奸宄)</p><p><span class="cino">(7)</span>
