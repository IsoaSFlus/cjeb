						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>政</strong></p><p><span class="dicpy">zhèng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span>  ㄓㄥˋ</span></p><ol><li>治理国家事务：～治。～府。～党。～权。～纲。～策。～令。～绩。～见。～客（为个人或某一集团利益从事政治活动的人）。～局。～变。参～。议～。</li><li>国家某一部门主管的业务：财～。邮～。民～。</li><li>家庭或集体生活中的事务：家～。</li><li>姓。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>政</strong> <span class="dicpy">zhēng</span></p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“征”</span>( <span class="dicpy">zhēng</span>)</p><p><span class="cino">(2)</span> 征伐 <span class="encs">[go on a punitive expedition]</span></p><p><span class="diczx1">先为之政。——<span class="smcs">《管子·势》</span></span></p><p><span class="diczx2">吴</span>政 <span class="diczx2">荆</span>。——<span class="smcs">《韩非子·内储下》</span></p><p><span class="diczx1">诸侯力政。——<span class="smcs">《大戴礼记·用兵》</span></span></p><p><span class="cino">(3)</span> 征税 <span class="encs">[levy]</span></p><p><span class="diczx1">宽政役。——<span class="smcs">《管子·小匡》</span></span></p><p><span class="diczx1">相地而衰政。——<span class="smcs">《荀子·王制》</span></span></p><p><span class="diczx1">庶民弛政。——<span class="smcs">《礼记·乐记》</span></span></p><p><span class="cino">(4)</span> 另见 <span class="dicpy">zhèng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>政</strong> <span class="dicpy">zhèng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意兼形声。从攴从正，正亦声。攴( <span class="dicpy">pū</span>)敲击。统治者靠皮鞭来推行其政治。<span class="yhcs">“正”</span>是光明正大。本义:匡正)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[correct]</span></p><p><span class="diczx1">政，正也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">政者，有所改更匡正。——<span class="smcs">《论语·有政》</span></span><span class="diczx2">马</span>注</p><p><span class="diczx1">必从上之政下。——<span class="smcs">《墨子·天志上》</span></span></p><p><span class="diczx1">俚句戏呈伯母大人粲政。——</span><span class="diczx2">清</span>· <span class="diczx2">秋瑾</span><span class="smcs">《致琴文书》</span></p><p><span class="cino">(3)</span> 又如:正己</p><p><span class="cino">(4)</span> 主持政事 <span class="encs">[govern]</span></p><p><span class="diczx1">盖善政者，视俗而施教，察失而立防，威德更兴，文武迭用。——<span class="smcs">《后汉书》</span></span></p><p><span class="cino">(5)</span> 又如:政本(为政的根本);政主(执政者);政官(掌管军政的官员);政术(施政的方法);政要(施政要领);政理(为政之道);政路(为政的途径)</p><p><strong>词性变化</strong></p><p>◎ <strong>政</strong> <span class="dicpy">zhèng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 政治;政事 <span class="encs">[politics;political affairs]</span></p><p><span class="diczx1">其政不获。——<span class="smcs">《诗·大雅·皇矣》</span>。释文:<span class="yhcs">“政，政教也。”</span></span></p><p><span class="diczx1">使帅其属而掌邦政。——<span class="smcs">《周礼·夏官》</span></span></p><p><span class="diczx1">荆国之为政。——<span class="smcs">《吕氏春秋·察今》</span></span></p><p><span class="diczx1">论厚薄为之政。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="diczx1">上下肃然，称为政理。——<span class="smcs">《后汉书·张衡传》</span></span></p><p><span class="diczx1">元政紊弛。——</span><span class="diczx2">明</span>· <span class="diczx2">高启</span><span class="smcs">《书博鸡者事》</span></p><p><span class="diczx1">言时政者。——</span><span class="diczx2">清</span>· <span class="diczx2">张廷玉</span><span
