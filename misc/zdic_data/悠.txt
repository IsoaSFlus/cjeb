                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>悠</strong></p><p><span class="dicpy">yōu  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yōu.mp3"></a></span>  ㄧㄡˉ</span></p><ol><li>久，远，长：～久。～远。～扬。</li><li>在空中摆动：～荡。晃～。转～。</li><li>稳住，控制：～着点劲。</li><li>闲适，闲散：～闲。～然。～忽（形容悠闲懒散）。～缓。～～（ａ．闲适，自由自在，如“白云～～”；ｂ．忧郁，如“～～我思”；ｃ．长久，遥远，如“～～长夜”；ｄ．众多；ｅ．荒谬，如“～～之谈”）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">悠</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>悠</strong> <span class="dicpy">yōu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yōu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从心，攸( <span class="dicpy">yōu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yōu.mp3"></a></span> )声。本义:忧思)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[miss;long for;think of]</span></p><p><span class="diczx1">悠，忧也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">悠，思也。——<span class="smcs">《尔雅》</span></span></p><p><span class="diczx1">悠悠我里。——<span class="smcs">《诗·小雅·十月之交》</span>。注:<span class="yhcs">“忧也。”</span></span></p><p><span class="diczx1">悠悠苍天兮。——<span class="smcs">《楚辞·初放》</span></span></p><p><span class="diczx1">悠哉悠哉，辗转反侧。——<span class="smcs">《诗·周南·关睢》</span></span></p><p><span class="diczx1">西北秋风至，</span><span class="diczx2">楚</span>客心悠哉。—— <span class="diczx2">南朝</span><span class="diczx2">梁</span>· <span class="diczx2">江淹</span><span class="smcs">《杂体诗三十首》</span></p><p><span class="cino">(3)</span> 又如:悠然(忧伤的样子);悠忧(忧伤)</p><p><span class="cino">(4)</span> 摇摆 <span class="encs">[swing]</span>。如:猴子从一个树枝悠到另一个树枝，穿过了密林;往上一悠，就举起来;悠搭(方言。摇晃，摆动)</p><p><span class="cino">(5)</span> 做事从容不迫 <span class="encs">[do things in a leisurely way]</span></p><p><span class="diczx1">听</span><span class="diczx2">安老爷</span>这等说，便道:<span class="yhcs">“喂!你悠着点儿，老头子!我一个出家人，不当家花拉的，你叫我那儿养小子去呀?”</span>——<span class="smcs">《儿女英雄传》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>悠</strong> <span class="dicpy">yōu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yōu.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 长;远 <span class="encs">[long;distant]</span></p><p><span class="diczx1">今吾道路悠远。——<span class="smcs">《国语·吴语》</span>。注:<span class="yhcs">“长也。”</span></span></p><p><span class="diczx1">悠远长怀。——<span class="smcs">《史记·司马相如传》</span></span></p><p><span class="diczx1">江山悠隔，朝宗无阶。——<span class="smcs">《晋书》</span></span></p><p><span class="diczx1">於乎悠哉，朕未有艾。——<span class="smcs">《诗·周颂·访落》</span></span></p><p><span class="diczx1">山海隔中州，相去悠且长。——</span><span class="diczx2">汉</span>· <span class="diczx2">苏武</span><span class="smcs">《诗四首》</span></p><p><span class="cino">(2)</span> 又如:悠永(久远);悠修(遥远);悠逖(远方);悠婉(悠扬婉转);悠短(长短);悠裔(辽远);悠漫(漫长);悠缅(久远;遥远);悠邈(久远);悠阔(辽阔);悠旷(遥远)</p><p><span class="cino">(3)</span> 飘扬的样子 <span class="encs">[fluttering]</span></p><p><span class="diczx1">建辰旒之太常，纷焱悠以容裔。——<span class="smcs">《文选·张衡·东京赋》</span></span></p><p><span class="cino">(4)</span>
