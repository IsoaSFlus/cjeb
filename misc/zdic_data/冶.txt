						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>冶</strong></p><p><span class="dicpy">yě  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yě.mp3"></a></span>  ㄧㄝˇ</span></p><ol><li>熔炼金属：～炼。～铸。～金。</li><li>好过分的装饰打扮（含贬义）：～容（a.打扮得很妖艳；b.妖艳的容貌）。妖～。</li><li>古同“<span class="diczx3">野</span>”：～游（原指春天或节日里男女出外游玩，后专指狎妓）。</li><li>姓。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>冶</strong> <span class="dicpy">yě</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yě.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从冫(古<span class="yhcs">“冰”</span>字)，台( <span class="dicpy">yí</span>)声。以冰融化解释金属销熔。 <span class="diczx2">段玉裁</span>:<span class="yhcs">“冰之融如铄金然，故炉铸亦曰冶。”</span>本义:销熔)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[smelt(metal)]</span></p><p><span class="diczx1">冶，销也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">冶，销铄也。——<span class="smcs">《三苍》</span></span></p><p><span class="diczx1">良冶之子。——<span class="smcs">《礼记·学记》</span></span></p><p><span class="diczx1">冶铸煮盐。——<span class="smcs">《史记·平淮书》</span></span></p><p><span class="cino">(3)</span> 又如:冶铸(冶炼铜铁，铸造器物);冶锻(冶炼);冶人(犹冶工);冶工(铸造金属器物的工人);冶士(古代从事冶炼铸造的刑徒)</p><p><span class="cino">(4)</span> 陶冶 <span class="encs">[cultivate;mould]</span></p><p><span class="diczx1">当是时，</span><span class="diczx2">文王</span>能陶冶天下之士。—— <span class="diczx2">王安石</span><span class="smcs">《上仁宗皇帝言事书》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>冶</strong> <span class="dicpy">yě</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yě.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 熔炼金属或铸造器物的场所 <span class="encs">[smeltery]</span>。如:冶监(掌管冶铸的官吏);冶家(以冶铸金属器物为生的人家)</p><p><span class="cino">(2)</span> 通<span class="yhcs">“野”</span>。田野，郊原 <span class="encs">[open country;the open;suburb]</span></p><p><span class="diczx1">冶游步春露。——古乐府<span class="smcs">《子夜四时歌》</span></span></p><p><span class="diczx1">是以时人谓之冶葛。——<span class="smcs">《北史·诸葛颖传》</span></span></p><p><span class="diczx1">不知身属冶游郎。——</span><span class="diczx2">李商隐</span><span class="smcs">《无题》</span></p><p><span class="cino">(3)</span> 姓</p><p>◎ <strong>冶</strong> <span class="dicpy">yě</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yě.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 艳丽 <span class="encs">[gorgeous]</span></p><p><span class="diczx1">美丽姚冶。——<span class="smcs">《荀子·非相》</span></span></p><p><span class="diczx1">纨袖渐冶，王颜掩姱。——</span><span class="diczx2">谢惠连</span><span class="smcs">《雪赋》</span></p><p><span class="cino">(2)</span> 又如:冶阳天(艳阳天);冶逸(容貌艳丽，神采飞扬);冶服(华丽的服装);冶丽(美丽异常;指美丽的女子);冶色(容颜美好)</p><p><span class="cino">(3)</span> 容态妖媚 <span class="encs">[seductively charming]</span></p><p><span class="diczx1">冶容诲淫。——<span class="smcs">《易·系辞上》</span></span></p><p><span class="cino">(4)</span> 又如:冶由(妩媚的笑态);冶步(步态妖冶妩媚)</p><p><span class="cino">(5)</span> 通<span class="yhcs">“野”</span> <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/51B6.svg">						  <p>【唐韻】羊者切【集韻】【韻會】【正韻】以者切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音野。【說文】消也。从仌，台聲。【三蒼】冶，銷也。遭熱卽流，遇冷卽合。與冰同志，故字从冰。【正韻】鎔也，銷也，鑄也。【韻會】爐鑄謂之冶。引尸子，蚩尤造九冶。</p><p>又鑄匠曰冶。【前漢·董仲舒傳】金之在鎔，惟冶者之所鑄。</p><p>又妖冶。【說文】女態。【易·繫辭】冶容誨淫。【正韻】裝飾也。</p><p>又姓。【廣韻】左傳，衞大夫冶厪。</p><p>又【玉篇】公冶，複姓。</p><p>又江南有梅根冶。【宋書·百官志】江南有梅根及冶塘二冶。【庾信·枯樹賦】南陵以梅根作冶。【孟浩然詩】火熾梅根冶。○按此皆晉時冶官所居。楊愼謂冶卽野字，非。</p><p>又【韻補】叶演女切，音與。【齊·武帝·估客樂詩】昔經樊鄧役，阻潮梅根冶。深懷悵往事，意滿辭不敘。</p><p>又叶鄔果切，音婐。容也。【謝惠連詩】酈生無文章，西施整妖冶。胡爲空耿介，悲哉君志瑣。</p>
