						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>拳</strong></p><p><span class="dicpy">quán  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/quán.mp3"></a></span>  ㄑㄩㄢˊ</span></p><ol><li>屈指卷（<span class="dicpy">juǎn </span>）握起来的手：～头（“<span class="diczx3">头</span>”读轻声）。</li><li>徒手的武术：～术。打～。太极～。～谱。～脚。</li><li>肢体弯曲：～曲。～起腿来。</li><li>量词，用于拳头打人的动作：打他几～。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>拳</strong> <span class="dicpy">quán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/quán.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。夲义:紧握的手。俗称拳头)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[fist]</span></p><p><span class="diczx1">拳，手也。——<span class="smcs">《说文》</span>。按，张之为掌，卷之为拳。”</span></p><p><span class="diczx2">鲁君</span>许诺，乃使吏鞟其拳。——<span class="smcs">《吕氏春秋》</span></p><p><span class="cino">(3)</span> 又如:挥拳;赤手空拳</p><p><span class="cino">(4)</span> 拳术;拳击 <span class="encs">[boxing]</span>。如:拳经(谈拳术的文字;拳术);打拳;太极拳;拳技(拳击的技法);拳法(拳击的技法)</p><p><span class="cino">(5)</span> 通<span class="yhcs">“弮”</span>。弓弩 <span class="encs">[bows]</span></p><p><span class="diczx1">士张空拳。——<span class="smcs">《汉书·李广传》</span></span></p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>拳</strong> <span class="dicpy">quán</span> <span class="yhcs">“蜷”</span>。屈曲;卷曲 <span class="encs">[curl]</span></p><p><span class="diczx1">其棱细则拳曲。——<span class="smcs">《庄子·人间世》</span></span></p><p><span class="diczx1">手不得拳，膝不得屈。——<span class="smcs">《颜氏家训·勉学》</span></span></p><p><span class="cino">(2)</span> 又如:拳拳弯弯(卷曲的样子);拳毛(卷曲的毛发);拳挛(拳曲;屈曲);拳毛(毛发卷曲);拳足(卷曲脚爪);拳发(头发卷曲)</p><p>◎ <strong>拳</strong> <span class="dicpy">quán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/quán.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 勇壮 <span class="encs">[brave]</span></p><p><span class="diczx1">无拳无勇。——<span class="smcs">《诗·巧言》</span>。传:<span class="yhcs">“力也。”</span></span></p><p><span class="diczx1">有拳勇股肱之力。——<span class="smcs">《国语·齐语》</span>。注:<span class="yhcs">“人勇为拳。”</span></span></p><p><span class="cino">(2)</span> 如:拳果(壮勇果断);拳勇(勇壮);拳捷(勇壮敏捷);拳猛(勇猛);拳腕(犹勇力);拳儇(勇壮矫捷)</p><p><span class="cino">(3)</span> 诚挚 <span class="encs">[honest]</span></p><p><span class="diczx1">拳拳之忠，终不能自列。——</span><span class="diczx2">司马迁</span><span class="smcs">《报任安书》</span></p><p><span class="diczx1">不胜拳拳。——<span class="smcs">《汉书·贡禹传》</span></span></p><p><span class="cino">(4)</span> 又如:拳拳好善(诚心诚意行善);拳挚(诚挚)</p><p>◎ <strong>拳</strong> <span class="dicpy">quán</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/quán.mp3"></a></span> </p><p>〈量〉</p><p><span class="cino">(1)</span> 表示动量，用于拿拳头打人的动作</p><p><span class="diczx1">你便在我脸上打一百拳也不妨!——<span class="smcs">《水浒传》</span></span></p><p><span class="cino">(2)</span> 表示物量，相当于<span class="yhcs">“桩”</span>、<span class="yhcs">“件”</span></p><p><span class="diczx1">我这一去，不得一拳儿好买卖不回来。——</span><span
