						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>偏</strong></p><p><span class="dicpy">piān  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/piān.mp3"></a></span>  ㄆㄧㄢˉ</span></p><ol><li>歪，不在中间：～斜。～离。～旁。～僻。～远。～锋（a.书法上指用毛笔时笔锋斜出的笔势；b.泛指做文章从侧面着手的方法）。不～不倚。</li><li>不全面，不正确：～爱。～见。～激。～狭。～执。～废。～颇。～听～信。</li><li>与愿望、预料或一般情况不相同：～～。～巧。</li><li>谦辞，向人称自己已用过茶饭等：我～过了，您请吃吧。</li></ol>

                    <h2 class="h2_entry"><span class="orth">偏</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>偏</strong> <span class="dicpy">piān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/piān.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从人，扁声。本义:不正，倾斜)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[leaning;slanting;inclined;tilted]</span></p><p><span class="diczx1">偏，颇也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">无偏无颇。——<span class="smcs">《书·洪范》</span></span></p><p><span class="diczx1">势有偏侧。——</span><span class="diczx2">南朝</span><span class="diczx2">宋</span>· <span class="diczx2">谢灵运</span><span class="smcs">《山居赋》</span></p><p><span class="cino">(3)</span> 又如:偏岸(斜岸);偏偏倒倒(东倒西歪。形容站不稳);偏侧(犹倾侧;偏斜);偏倒(倾斜)</p><p><span class="cino">(4)</span> 边远 <span class="encs">[remote]</span></p><p><span class="diczx1">心远地自偏。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《饮酒》</span></p><p><span class="cino">(5)</span> 又如:偏邦(远离中原地区的边远小国);偏伯(边远地方的长官);偏海(边远的滨海地区);偏泽(边远之地);偏州(僻远之州);偏陬(僻远之地)</p><p><span class="cino">(6)</span> 片面，侧重于某一方面或某一部分 <span class="encs">[one-sided;unilateral]</span></p><p><span class="cino">(7)</span> 又如:偏为(偏向;偏袒);偏任(片面信任);偏行(片面实行);偏定(片面论定);偏解(片面的见解);偏辞(片面之词);偏裨(偏将与裨将)</p><p><span class="cino">(8)</span> 独，特 <span class="encs">[particular]</span>。如:偏委(特殊照顾);偏惠(特殊的利益);偏嗜(特殊的爱好)</p><p><span class="cino">(9)</span> 偏私;不公正;不公平 <span class="encs">[selfish;partial;prejudiced]</span></p><p><span class="diczx1">王道荡荡，不偏不党;王道平平，不党不偏。——<span class="smcs">《墨子·兼爱下》</span></span></p><p><span class="diczx1">通则骄而偏。——<span class="smcs">《荀子·不苟》</span></span></p><p><span class="cino">(10)</span> 又如:偏佞(偏私奸巧);偏指(偏私的意愿);偏徇(偏私曲从);偏护(偏私的袒护);偏执(固执);偏介(孤高偏执);偏吝(偏执鄙吝);偏令(偏颇的政令);偏曲(偏颇隐晦);偏政(偏颇的政教);偏恣(偏颇任性);偏说(偏颇、片面的言论);偏质(偏颇之性);偏论(偏颇的议论)</p><p><span class="cino">(11)</span> 假借为<span class="yhcs">“遍”</span>。普遍 <span class="encs">[allover;everywhere]</span></p><p><span class="diczx1">则不可偏观也。——<span class="smcs">《墨子·小取》</span></span></p><p><span class="cino">(12)</span> 通<span class="yhcs">“扁”</span>。扁舟，小船 <span class="encs">[small boat;skiff]</span></p><p><span class="diczx1">乘偏舟于五湖。——<span class="smcs">《后汉书·隗嚣传》</span></span></p><p><span class="cino">(13)</span> 通<span class="yhcs">“翩”</span>。疾飞，飘扬 <span class="encs">[lightly;flying]</span></p><p><strong>词性变化</strong></p><p>◎ <strong>偏</strong> <span class="dicpy">piān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/piān.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 旁;一方 <span class="encs">[side]</span>。如:偏王(占据一地而称王);偏帅(一方守将);偏霸(偏据一方而称王);偏亡(缺少某一方面);偏至(在某一方面达到最高成就);偏长(一方面的特长);偏修(修道的旁门);偏室(正屋旁的房子);偏出(庶出;妾所生的子女)</p><p><span class="cino">(2)</span> 部属 <span class="encs">[subject]</span></p><p><span class="diczx1">举其偏，不为党。——<span class="smcs">《左传·襄公三年》</span></span></p><p><span class="cino">(3)</span> 配偶的一方 <span class="encs">[spouse]</span></p><p><span                   <div class="content definitions gnr">
                    <div class="gycd"><div class="pz"><ruby><rbc><rb>偏</rb></rbc><rtc><rt>ㄆㄧㄢ</rt></rtc><rtc><rt>piān <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/gycd/zd/0470.ogg"></a></span> </rt></rtc></ruby></div><div class="gycd-item"><span class="gc_cx">形</span><ol><li><p class="def"><span class="gc_sy">倾斜、不中正。</span><span class="gc_lz">如：「偏心」、「偏见」、「不偏不倚」。</span><span class="gc_yy">《书经·洪范》：「无偏无陂，遵王之义。」</span><span class="gc_yy">唐·白居易〈长恨歌〉：「云髻半<b>偏</b>新睡觉，花冠不整下堂来。」</span></p></li><li><p class="def"><span class="gc_sy">不周全的、片面的。</span><span class="gc_yy">《礼记·乐记》：「乐极则忧，礼粗则<b>偏</b>矣。」</span></p></li><li><p class="def"><span class="gc_sy">不是中心地带、荒僻的。</span><span class="gc_lz">如：「偏远」、「偏僻」。</span><span class="gc_yy">晋·陶渊明〈饮酒诗〉二○首之五：「问君何能尔？心远地自<b>偏</b>。」</span></p></li><li><p class="def"><span class="gc_sy">深、多。</span><span class="gc_yy">唐·元稹〈酬窦校书二十韵〉：「音徽千里断，魂梦两情<b>偏</b>。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">副</span><ol><li><p class="def"><span class="gc_sy">倾于某一面的、不公平的。</span><span class="gc_yy">《汉书·卷九四·匈奴传上》：「朕闻天不颇复，地不<b>偏</b> 载。」</span></p></li><li><p class="def"><span class="gc_sy">表示故意跟外来要求相反的。</span><span class="gc_lz">如：「他要我去，我<b>偏</b>不去。」</span></p></li><li><p class="def"><span class="gc_sy">表示出乎意料之外或与意愿相反的。</span><span class="gc_lz">如：「屋漏<b>偏</b>遭连夜雨。」</span><span class="gc_yy">唐·无名氏〈伊川歌·第三〉：「可怜闺里月，<b>偏</b>照汉家营。」</span></p></li><li><p class="def"><span class="gc_sy">恰巧、正好。</span><span class="gc_yy">唐·皇甫冉〈曾东游以诗寄之〉诗：「正是扬帆时，<b>偏</b>逢江上客。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">动</span><ol><li><p class="def"><span class="gc_sy">表示「吃过饭」的客套话。</span><span class="gc_yy">《红楼梦·第一四回》：「凤姐才吃饭，见他们来了，便笑道：『好长腿子，快上来罢！』宝玉道：『我们<b>偏</b>了。』」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">名</span><ol><li><p class="def"><span class="gc_sy">姓。如汉代有<b><b>偏</b></b>吕。</span></p></li></ol></div></div>
