						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>劣</strong></p><p><span class="dicpy">liè  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/liè.mp3"></a></span>  ㄌㄧㄝˋ</span></p><ol><li>恶，坏：恶～。～迹。</li><li>低下，弱下：～势。～等。低～。优～。</li><li>小于一定标准的：～弧（小于半圆的弧）。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> bad, inferior; slightly</p><p><span class="z_ts2">德语</span> schlecht, minderwertig  (Adj)</p><p><span class="z_ts2">法语</span> mauvais,inférieur</p></div>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>劣</strong> <span class="dicpy">liè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/liè.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (会意。从力，从少。这是合并两个字的意思而构成的会意字。本义:弱;小)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[weak;small]</span></p><p><span class="diczx1">劣，弱也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">寿命长短，骨体强劣，各有人焉。——</span><span class="diczx2">曹植</span><span class="smcs">《辩道论》</span></p><p><span class="cino">(3)</span> 又如:劣马(瘦弱的马;又指烈性马);劣弱(衰弱;懦弱)</p><p><span class="cino">(4)</span> 少，不足 <span class="encs">[few]</span></p><p><span class="diczx1">夫德劣故用兵，犯法故施刑。——<span class="smcs">《论衡》</span></span></p><p><span class="cino">(5)</span> 不好;低劣，与<span class="yhcs">“优”</span>相对 <span class="encs">[bad; evil]</span></p><p><span class="diczx1">优劣异名。——<span class="smcs">《论衡·气寿》</span></span></p><p><span class="diczx1">必能使行阵和睦，优劣得所。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p><span class="diczx1">羸牛劣马。——<span class="smcs">《齐民要术·养牛马》</span></span></p><p><span class="cino">(6)</span> 又如:劣性(恶劣的品性);劣缺(不好，恶劣);劣衿(品行恶劣的生员)</p><p><strong>词性变化</strong></p><p>◎ <strong>劣</strong> <span class="dicpy">liè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/liè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 认为不好 <span class="encs">[have a low opinion of; think lowly of]</span> 以为不及 <span class="encs">[inferior]</span></p><p><span class="diczx1">成以其小，劣之。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(2)</span> 劣下(低能;低下);劣弟(在平辈或朋友间表示自谦的称呼);劣丈(在世交中，长辈对晚辈表示自谦的称呼)</p><p><span class="cino">(3)</span> 顽皮;乖巧 <span class="encs">[naughty]</span>。如:劣相(乖僻);劣角(乖劣，顽皮);劣缺(乖戾，狠毒，顽劣)</p><p><span class="cino">(4)</span> 强;美 <span class="encs">[strong;fine]</span></p><p><span class="diczx1">论着雄心力劣牙爪，今日也合消，也合消封妻荫子，禄重官高。——</span><span class="diczx2">元</span>· <span class="diczx2">尚仲贤</span><span class="smcs">《三夺槊》</span></p><p><span class="cino">(5)</span> 暴烈，不驯服 <span class="encs">[viollent]</span>。如:劣性(暴烈性格);劣彆(倔强;固执)</p><p>◎ <strong>劣</strong> <span class="dicpy">liè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/liè.mp3"></a></span> </p><p>〈副〉</p><p>仅，只 <span class="encs">[only]</span></p><p><span class="diczx1">学</span><span class="diczx2">谢朓</span>劣得<span class="yhcs">“黄鸟度青枝”</span>。——<span class="smcs">《梁书·钟嵘传》</span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/劣等">劣等</a></span><span class="crefe"><a href="/hans/劣迹">劣迹</a></span><span class="crefe"><a href="/hans/劣蹶">劣蹶</a></span><span class="crefe"><a href="/hans/劣马">劣马</a></span><span class="crefe"><a href="/hans/劣绅">劣绅</a></span><span class="crefe"><a href="/hans/劣势">劣势</a></span><span class="crefe"><a href="/hans/劣质">劣质</a></span><span class="crefe"><a href="/hans/劣种">劣种</a></span></div>                    <div class="div copyright"> © 汉典 </div>
