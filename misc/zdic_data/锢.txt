                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>锢</strong></p><p><strong>（<span class="diczx3">錮</span>）</strong></p><p><span class="dicpy">gù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gù.mp3"></a></span>  ㄍㄨˋ</span></p><ol><li>把金属熔化以浇灌堵塞空隙：～露（用熔化的金属堵塞金属物品的漏洞，如“～～锅”。亦作“锢漏”）。</li><li>禁闭：禁～。～蔽。</li><li>古同“<span class="diczx3">痼</span>”，痼疾。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> run metal into cracks; confine</p><p><span class="z_ts2">德语</span> aufhalten, aufhören ,zurückhalten</p><p><span class="z_ts2">法语</span> obstruer,solide</p></div>

                    <h2 class="h2_entry"><span class="orth">锢</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>锢</strong></p><p>錮 <span class="dicpy">gù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从金，固声。本义:用熔化的金属堵塞空隙)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[fill up cracks with molten metal]</span></p><p><span class="diczx1">锢，铸塞也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">徐锴</span>曰:<span class="yhcs">“铸铜铁以塞隙也。”</span></p><p><span class="diczx1">釭锏键钻冶锢鐈。——<span class="smcs">《急就篇》</span></span></p><p><span class="diczx1">冶铜锢其内。——<span class="smcs">《汉书·贾山传》</span></span></p><p><span class="cino">(3)</span> 又如:锢露(亦作锢路。用熔化的金属堵塞金属物品的漏洞);锢南山(用铜铁熔液堵塞的南山。比喻胸中郁结不消的块垒)</p><p><span class="cino">(4)</span> 禁闭;禁锢 <span class="encs">[imprison]</span></p><p><span class="diczx1">(</span><span class="diczx2">巫臣</span>)遂奔 <span class="diczx2">晋</span>，…… <span class="diczx2">子反</span>请以重币锢之。——<span class="smcs">《左传·成公二年》</span></p><p><span class="diczx1">可是在中国，那时是确无写处的，禁锢得比罐头还严密。——<span class="smcs">《为了忘却的记念》</span></span></p><p><span class="diczx1">锢诏狱，昼夜搒讯。——<span class="smcs">《明史·海瑞传》</span></span></p><p><span class="cino">(5)</span> 又如:锢党(遭禁锢的朋党成员);锢钥(禁锢锁闭)</p><p><span class="cino">(6)</span> 束缚;闭塞 <span class="encs">[restrain;unenlightened]</span>。如:锢身(以枷锁加于人身);锢送(加以枷锁押送)</p><p><span class="cino">(7)</span> 垄断，包揽 <span class="encs">[monopoly]</span>。如:锢房(专房。指集宠爱于一身)</p><p><span class="cino">(8)</span> 禁止参加政治活动或出任官职 <span class="encs">[debar from holding office]</span>。如:锢籍(贬为庶人，居留原籍)</p><p><span class="cino">(9)</span> 封闭;关闭 <span class="encs">[block;close]</span>。如:锢闭(关闭;禁闭)</p><p><strong>词性变化</strong></p><p>◎ <strong>锢</strong></p><p>錮 <span class="dicpy">gù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gù.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 疾病经久不治的。通<span class="yhcs">“痼”</span> <span class="encs">[chronic]</span></p><p><span class="diczx1">失今不治，必为锢疾。——<span class="smcs">《汉书·贾谊传》</span>。</span><span class="diczx2">师古</span>曰:<span class="yhcs">“锢疾，坚久之疾。”</span></p><p><span class="cino">(2)</span> 又如:锢病(经久难治的疾病)</p><p><span class="cino">(3)</span> 通<span class="yhcs">“固”</span>。坚固 <span class="encs">[solid;firm]</span>。如:锢定(牢固确定)</p><p><span class="cino">(4)</span> 通<span class="yhcs">“固”</span>。顽固 <span class="encs">[stubborn]</span>。如:锢陋(难于更改的陋规);锢见(成见)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a
