                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>滥</strong></p><p><strong>（<span class="diczx3">濫</span>）</strong></p><p><span class="dicpy">làn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làn.mp3"></a></span> ㄌㄢˋ</span></p><ol><li>流水漫溢：泛～。</li><li>不加选择，不加节制：～用职权。宁缺勿～。～伐。</li><li>浮泛不合实际：陈词～调。～竽充数（喻没有真正的才干，而混在行家里面充数，或以次充好。有时亦表示自谦）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">滥</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>滥</strong></p><p>濫 <span class="dicpy">làn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水，监声。本义:浮起，浮现)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[float; gush; emerge]</span></p><p><span class="diczx1">滥，泛也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx2">江</span>出于 <span class="diczx2">岷山</span>，其始出也，其源可以滥觞。——<span class="smcs">《荀子·子道》</span></p><p><span class="diczx1">滥泉正出。——<span class="smcs">《尔雅·释水》</span></span></p><p><span class="diczx1">滥水之潘为渊。——<span class="smcs">《列子·黄帝》</span></span></p><p><span class="cino">(3)</span> 大水漫出，泛滥 <span class="encs">[overflow;flood]</span></p><p><span class="diczx1">大水泛滥郡国十五有余。——<span class="smcs">《汉书·谷永传》</span></span></p><p><span class="diczx1">其水阳焊不耗，阴霖不滥。——<span class="smcs">《水经注·湿水》</span></span></p><p><span class="diczx1">当</span><span class="diczx2">尧</span>之时，水逆行，泛滥于中国。——<span class="smcs">《孟子·滕文公》</span></p><p><span class="cino">(4)</span> 又如:泛滥(江河湖泊的水溢出);滥泉(从地下向上涌出的泉水);滥汜(泛滥，浮沉)</p><p><span class="cino">(5)</span> 浸渍，泡在水里 <span class="encs">[soak]</span></p><p><span class="diczx1">滥车之水。——<span class="smcs">《管子·地图》</span></span></p><p><span class="cino">(6)</span> 蔓延;波及 <span class="encs">[spread]</span>。如:滥炎(蔓延的火);祸滥及平人</p><p><span class="diczx1">因怒而滥刑。——</span><span class="diczx2">唐</span>· <span class="diczx2">魏征</span><span class="smcs">《谏太宗十思疏》</span></p><p><span class="cino">(7)</span> 又如:滥漫(任意，随便);滥杀(胡乱杀戳无罪的人);滥浊(过度混乱);滥用职权;滥冗(过多);滥刑(过量的刑罚;任意判罪或施刑)</p><p><span class="cino">(8)</span> 通<span class="yhcs">“欿”</span>。贪欲;卑污 <span class="encs">[greedy;depraved]</span></p><p><span class="diczx2">虞公</span>滥于宝于马。——<span class="smcs">《吕氏春秋》</span></p><p><span class="cino">(9)</span> 又如:滥官(贪官);滥墨(指官吏贪财好贿者);滥溢(胡作非为)</p><p><span class="cino">(10)</span> 通<span class="yhcs">“敛”</span>。收聚 <span class="encs">[gather]</span></p><p><span class="diczx1">至刑不滥。——<span class="smcs">《淮南子·汜论》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>滥</strong></p><p>濫 <span class="dicpy">làn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 过度;超过限度;漫无准则 <span class="encs">[excessive;wanton]</span></p><p><span class="diczx1">生穑省用，不滥其度。——<span class="smcs">《周书·程典》</span></span></p><p><span class="diczx1">小人究斯滥矣。——<span class="smcs">《论语·卫灵公》</span></span></p><p><span class="cino">(2)</span> 虚妄不实 <span class="encs">[unreal]</span></p><p><span class="diczx1">不僭不滥。——<span
