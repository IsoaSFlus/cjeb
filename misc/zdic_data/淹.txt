                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>淹</strong></p><p><span class="dicpy">yān  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span>  ㄧㄢˉ</span></p><ol><li>浸没（<span class="dicpy">mò</span>）：水～。～没（<span class="dicpy">mò</span>）。～灌。</li><li>皮肤被汗液浸渍：胳肢窝被汗～得又痛又痒。</li><li>广：～博。～通。～贯（渊博而贯通）。～雅（渊博高雅）。</li><li>滞，久留：～留。久～。～滞。～月（滞留一月）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">淹</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>淹</strong> <span class="dicpy">Yān</span></p><p>〈名〉</p><p>(形声。从水，奄声。本义:古水名，即今金沙江的一段) 同本义。在四川省南部。即今金沙江自发源地至四川省攀枝花市的一段 <span class="encs">[Yan River]</span></p><p><span class="diczx1">淹水，一曰复水也，出今四川宁远府徼外，合金沙江入</span><span class="diczx2">江</span>。——<span class="smcs">《字林》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>淹</strong> <span class="dicpy">yān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 浸泡 <span class="encs">[soak]</span></p><p><span class="diczx1">淹芳芷于腐井兮。——</span><span class="diczx2">刘向</span><span class="smcs">《九叹·怨思》</span></p><p><span class="diczx1">淹之以乐好。——<span class="smcs">《礼记·儒行》</span>。注:<span class="yhcs">“谓浸渍也。”</span></span></p><p><span class="cino">(2)</span> 又如:淹荠燎菜(形容肮脏、破旧);淹渍(浸泡，淹浸)</p><p><span class="cino">(3)</span> 淹没;沉没 <span class="encs">[flood;submerge]</span></p><p><span class="diczx1">宅中水淹。——<span class="smcs">《北史·皇甫和传》</span></span></p><p><span class="cino">(4)</span> 又如:淹沉(沉溺)</p><p><span class="cino">(5)</span> 逗留;挽留 <span class="encs">[stay for a long time]</span></p><p><span class="diczx1">淹回水而凝滞。——<span class="smcs">《楚辞·屈原·涉江》</span></span></p><p><span class="diczx1">为从者之淹。——<span class="smcs">《左传·僖公三十三年》</span></span></p><p><span class="diczx1">淹久于敝邑。</span></p><p><span class="cino">(6)</span> 又如:淹息(逗留休息);淹回(徘徊;逗留);淹泊(停留;滞留)</p><p><span class="cino">(7)</span> 拖延 <span class="encs">[put off]</span>。如:淹缠(拖延;滞留)</p><p><span class="cino">(8)</span> 离开人世 <span class="encs">[die]</span>。如:淹忽(去世，死亡);淹逝(去逝)</p><p><span class="cino">(9)</span> 通<span class="yhcs">“腌”</span>。以盐渍食物 <span class="encs">[salt]</span></p><p><span class="diczx1">煎鱼切肝，羊淹鸡寒。——<span class="smcs">《盐铁论·散不足》</span></span></p><p>◎ <strong>淹</strong> <span class="dicpy">yān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 广博;深入 <span class="encs">[wide]</span></p><p><span class="diczx2">平子</span>淹通(通达)，故虑周而藻密。——<span class="smcs">《文心雕龙·体性》</span></p><p><span class="diczx1">淹究经术。——</span><span class="diczx2">欧阳修</span><span class="smcs">《新唐书》</span></p><p><span class="cino">(2)</span> 又如:淹贯精微(渊博贯通、精深微妙);淹泓(渊深广大);淹明(渊博通达);淹洽(渊博;深通);淹决(深通)</p><p><span class="cino">(3)</span> 满 <span class="encs">[full]</span>。如:淹月(满月);淹心(心满意足)</p><p><span class="cino">(4)</span> 时间长 <span class="encs">[long]</span></p><p><span class="diczx1">淹，久也。——<span class="smcs">《尔雅》</span></span></p><p><span class="diczx1">无令舆师淹于居地。——<span class="smcs">《左传·成公二年》</span></span></p><p><span class="diczx1">日月忽其不淹兮。——<span class="smcs">《离骚》</span></span></p><p><span
