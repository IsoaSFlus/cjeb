						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>洞</strong></p><p><span class="dicpy">dòng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dòng.mp3"></a></span>  ㄉㄨㄥˋ</span></p><ol><li>窟窿，深穴，孔：～穴。山～。～箫。空～。漏～。～府。～天。</li><li>打洞，打成洞（穿透）：一狼～其中。</li><li>透彻地，清楚地：～悉。～穿。</li><li>说数字时用来代替零。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>洞</strong> <span class="dicpy">dòng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dòng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从水，同声。本义:水流急)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[rapid]</span></p><p><span class="diczx1">洞，疾流也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> 幽深 <span class="encs">[deep and serene]</span>。如:洞虚(深幽)</p><p><span class="cino">(4)</span> 广阔 <span class="encs">[broad]</span>。如:洞庭(广阔的庭院;又指湖名)</p><p><span class="cino">(5)</span> 深 <span class="encs">[deep]</span>。如:洞壑(深谷);洞豁(深广);洞井(深洞)</p><p><span class="cino">(6)</span> 清澈 <span class="encs">[clear]</span>。如:洞观(清澈见底);洞朗(清彻明亮);洞光(透明通亮)</p><p><span class="cino">(7)</span> 透彻 <span class="encs">[thoroughly]</span>。如:洞溢(透彻，详赡);洞视(透视，看得清楚);洞彻(洞澈。清澈;了解透彻)</p><p><strong>词性变化</strong></p><p>◎ <strong>洞</strong> <span class="dicpy">dòng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dòng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 窟窿;孔穴 <span class="encs">[hole]</span></p><p><span class="diczx1">心气内洞。——<span class="smcs">《素问·四气调神大论》</span>。注:<span class="yhcs">“洞，谓中空也。”</span></span></p><p><span class="diczx1">港洞坑谷。——</span><span class="diczx2">马融</span><span class="smcs">《长笛赋》</span></p><p><span class="diczx1">距洞百余步。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="cino">(2)</span> 又如:洞隙(孔缝);洞肛(肛门);袖子磨破一个洞</p><p><span class="cino">(3)</span> 拱道，位于拱下的道路 <span class="encs">[archway]</span>。如:城门洞</p><p><span class="cino">(4)</span> 口令中用以代替数目中的<span class="yhcs">“零”</span> <span class="encs">[zero]</span></p><p>◎ <strong>洞</strong> <span class="dicpy">dòng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dòng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 贯穿;通达 <span class="encs">[communicate]</span></p><p><span class="diczx1">转视积薪后，一狼洞其中。——<span class="smcs">《聊斋志异·狼三则》</span></span></p><p><span class="diczx1">骈其指，力斫之，可断牛项;横搠之，可洞牛腹。——<span class="smcs">《聊斋志异·铁布衫法》</span></span></p><p><span class="cino">(2)</span> 又如:洞出(穿洞而出);洞壁(在墙上打洞);洞连(连接相通);洞户(室与室相通的门户);洞越(瑟底两孔相通)</p><p><span class="cino">(3)</span> 通晓;知悉 <span class="encs">[thoroughly understand]</span></p><p><span class="diczx1">为人鸢肩豺目，洞精矘眄口吟舌言，裁能书计。——<span class="smcs">《后汉书·梁冀传》</span></span></p><p><span class="cino">(4)</span> 又如:洞习(通晓熟悉);洞达(通晓明白);洞烛(明察);洞冥(洞察昏暗之处)</p><p><span class="cino">(5)</span> 敞开 <span class="encs">[open]</span>。如:洞敞(敞开);洞辟(大开)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit
