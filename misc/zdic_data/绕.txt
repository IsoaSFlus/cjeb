                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>绕</strong></p><p><strong>（<span class="diczx3">繞</span>）</strong></p><p><span class="dicpy">rào  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rào.mp3"></a></span> ㄖㄠˋ</span></p><ol><li>缠：～线。缠～。</li><li>纠缠，弄迷糊：～嘴。～口令。</li><li>走弯曲迂回的路：～远。～道。～越。</li><li>围着转：～场一周。围～。环～。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>绕</strong></p><p>繞、遶 <span class="dicpy">rào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rào.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从糸( <span class="dicpy">mì</span>)，尧声。本义:缠束)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[wind;coil]</span></p><p><span class="diczx1">绕，缠也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">绕黄山而款牛首。——</span><span class="diczx2">张衡</span><span class="smcs">《西京赋》</span>。注:<span class="yhcs">“裹也。”</span></p><p><span class="cino">(3)</span> 又如:把丝绕在卷筒上;绕搭(缠绕);绕缭(缭绕，盘旋围绕);绕手(犹棘手);绕网(陷入罗网);绕结(回旋缠结)</p><p><span class="cino">(4)</span> 环绕 <span class="encs">[surround;encircle;revolve around]</span></p><p><span class="diczx1">白水绕东城。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span><span class="smcs">《送友人》</span></p><p><span class="diczx1">黄芦苦竹绕宅生。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="cino">(5)</span> 又如:绕梁声(形容歌声优美动听，长久萦回耳畔);绕袭(迂回突袭敌人)</p><p><span class="cino">(6)</span> 迂回绕过 <span class="encs">[bypass]</span></p><p><span class="diczx1">一边是陡山，一边是深峪，更无别处好绕。——<span class="smcs">《老残游记》</span></span></p><p><span class="cino">(7)</span> 又如:绕弯(波折);绕过暗礁;绕袭(绕道袭击)</p><p><span class="cino">(8)</span> 逃避…力量，规避 <span class="encs">[get around]</span>。如:你必须明确回答，想绕是绕不过去的</p><p><span class="cino">(9)</span> 使模糊 <span class="encs">[confuse]</span>。如:你的话把他绕住了</p><p><span class="cino">(10)</span> 纠缠(如事情或问题) <span class="encs">[be in a trangle]</span></p><p><span class="diczx1">我看昨天晚上你是有点绕住了;遇上急事，谁也保不住迷头。——</span><span class="diczx2">老舍</span><span class="smcs">《骆驼祥子》</span></p><p><span class="cino">(11)</span> 弯曲 <span class="encs">[bend]</span></p><p><span class="diczx1">眉连娟以增绕兮，目流睇而横波。——<span class="smcs">《文选·傅毅·舞赋》</span></span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span
