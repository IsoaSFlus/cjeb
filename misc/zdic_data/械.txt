data-src-mp3="//img.zdic.net/audio/zd/py/xiè.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从木，戒声。本义:脚镣手铐一类的刑具)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[fetters，shackles，etc.]</span></p><p><span class="diczx1">械，桎梏也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">拘罪人曰桁械。谓穿木加足曰械，大械曰桁。——<span class="smcs">《通俗文》</span></span></p><p><span class="diczx1">在手曰械。——<span class="smcs">《后汉书·吴传》</span>注</span></p><p><span class="diczx1">械系</span><span class="diczx2">敬</span><span class="diczx2">广武</span>。——<span class="smcs">《汉书·娄敬传》</span></p><p><span class="diczx1">三缢加别械。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">求脱械。</span></p><p><span class="diczx1">地上刑械。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《左忠毅公逸事》</span></p><p><span class="cino">(3)</span> 又如:械送(加刑具押送);械索(泛指刑具。用刑具拷掠);械节(枷械等刑具的结合处);械解(戴上刑具押送);械颈(加械于颈)</p><p><span class="cino">(4)</span> 器械，用具 <span class="encs">[appliance;instrument;tool]</span></p><p><span class="diczx1">有械于此，一日浸百畦。——<span class="smcs">《庄子·天地》</span></span></p><p><span class="diczx1">云梯之械。——<span class="smcs">《墨子公输》</span></span></p><p><span class="diczx1">攻械尽。</span></p><p><span class="diczx1">以牒为械。</span></p><p><span class="cino">(5)</span> 又如:械用(器械用具);械器(器械;器具);械具(机械用具)</p><p><span class="cino">(6)</span> 武器 <span class="encs">[weapon]</span>。如:械战(犹械斗);枪械</p><p><strong>词性变化</strong></p><p>◎ <strong>械</strong> <span class="dicpy">xiè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xiè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 拘系;枷住;拘束 <span class="encs">[restrain]</span></p><p><span class="diczx1">苟入狱，不问罪之有无，必械手足。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">半与城门充校尉，谁将</span><span class="diczx2">斜谷</span>械 <span class="diczx2">阳陵</span>。—— <span class="diczx2">龚自珍</span><span class="smcs">《己亥杂诗》</span></p><p><span class="cino">(2)</span> 又如:械致(拘系押送);械槛(拘系于囚车)</p><p><span class="cino">(3)</span> 治 <span class="encs">[manage]</span></p><p><span class="diczx1">桥揉斤械。——<span class="smcs">《文选·马融·长笛赋》</span></span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/械斗">械斗</a></span><span class="crefe"><a
