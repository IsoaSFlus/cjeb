                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>冀</strong> <span class="dicpy">Jì</span></p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从北，异( <span class="dicpy">yì</span>)声。本义:冀州。古九州之一)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[Ji prefecture]</span></p><p><span class="diczx1">冀，北方州也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">两河间曰冀州。——<span class="smcs">《尔雅·释地》</span></span></p><p><span class="diczx1">河内曰冀州。——<span class="smcs">《周礼·职方氏》</span>。</span><span class="diczx2">朱骏声</span>曰:<span class="yhcs">“地亘今盛京 <span class="diczx2">直隶</span><span class="diczx2">山西</span>、 <span class="diczx2">河南</span>各府州， <span class="diczx2">淮南</span>地形，正中 <span class="diczx2">冀州</span>，曰中土。”</span></p><p><span class="diczx1">惟彼</span><span class="diczx2">陶唐</span>，有此 <span class="diczx2">冀</span>方。——<span class="smcs">《书·五子之歌》</span>。 <span class="diczx2">孔</span>传:<span class="yhcs">“ <span class="diczx2">陶唐</span>帝 <span class="diczx2">尧</span>氏，都 <span class="diczx2">冀州</span>，统天下四方。”</span></p><p><span class="cino">(3)</span> 古代国名<span class="encs">[Ji state]</span>。在今山西省河津县东北</p><p><span class="cino">(4)</span> 河北省的简称 <span class="encs">[Hopei province]</span></p><p><strong>词性变化</strong></p><p>◎ <strong>冀</strong> <span class="dicpy">jì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 假借为<span class="yhcs">“觊”</span>。希望，期望 <span class="encs">[hope]</span></p><p><span class="diczx1">吾冀而朝夕修我。——<span class="smcs">《国语·鲁语》</span></span></p><p><span class="diczx1">冀至珠庭焉。——<span class="smcs">《史记·孝武纪》</span></span></p><p><span class="diczx1">释其耒而守株，冀复得兔。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="diczx1">冀三横唯余其一。——<span class="smcs">《世说新语·自新》</span></span></p><p><span class="diczx1">冀</span><span class="diczx2">君实</span>或见恕。—— <span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《答司马谏议书》</span></p><p><span class="diczx1">冀缓急或可救。——</span><span class="diczx2">清</span>· <span class="diczx2">梁启超</span><span class="smcs">《谭嗣同传》</span></p><p><span class="cino">(2)</span> 又如:冀其成功;希冀;冀图、冀愿(希图;希望;企望)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/冀求">冀求</a></span><span
