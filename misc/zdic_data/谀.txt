                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>谀</strong></p><p>諛 <span class="dicpy">yú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从言，臾声。本义:奉承，谄媚) 同本义 <span class="encs">[flatter]</span></p><p><span class="diczx1">谀，谄也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">不择是非而言谓之谀。——<span class="smcs">《庄子·渔夫》</span></span></p><p><span class="diczx1">以不善和人者谓之谀。——<span class="smcs">《荀子·脩身》</span></span></p><p><span class="diczx1">从命病君谓之谀。——<span class="smcs">《说苑·臣术》</span></span></p><p><span class="diczx1">官盛则近谀。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《师说》</span></p><p><span class="diczx1">诬谀之徒。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="diczx1">谄谀之臣。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="cino">(2)</span> 又如:谀墓之金(为人作墓志所得的报酬);谀颂(阿谀奉承);谀悦(谄媚;奉承);谀墓(指替人作墓志阿谀死者);谀佞(惯于用花言巧语阿谀谄媚的人);谀言(谄媚的言语)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>谀</strong></p><p>諛 <span class="dicpy">yú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 谄媚的话 <span class="encs">[flattery]</span></p><p><span class="diczx1">唯囿是恢，唯谀是信。——<span class="smcs">《汉书·韦贤传》</span></span></p><p><span class="cino">(2)</span> 又如:谀史(有溢美之辞的史传)</p><p>◎ <strong>谀</strong></p><p>諛 <span class="dicpy">yú</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yú.mp3"></a></span> </p><p>〈形〉</p><p>和悦柔顺 <span class="encs">[be kindly and genial]</span>。如:谀然(愉悦和顺的样子)</p>                    <div class="div copyright"> © 汉典 </div>
