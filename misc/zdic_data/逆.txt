                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>逆</strong></p><p><span class="dicpy">nì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/nì.mp3"></a></span>  ㄋㄧˋ</span></p><ol><li>方向相反，与“<span class="diczx3">顺</span>”相对：～流。～行。～风。～转（<span class="dicpy">zhuǎn </span>）（局势恶化）。莫～之交。</li><li>抵触，不顺从：忤～。忠言～耳。</li><li>背叛，背叛者或背叛者的：叛～。～产。</li><li>迎接：～旅（旅店）。</li><li>预先：～料（预料）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">逆</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>逆</strong> <span class="dicpy">nì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/nì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，屰( <span class="dicpy">nì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/nì.mp3"></a></span> )声。本义:迎，迎接，迎着)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[meet]</span></p><p><span class="diczx1">逆，迎也。关东曰逆，关西曰迎。——<span class="smcs">《说文》</span></span></p><p><span class="diczx2">宣公</span>如 <span class="diczx2">齐</span>逆女。——<span class="smcs">《左传·成公十四年》</span></p><p><span class="diczx1">上卿逆于境。——<span class="smcs">《国语·周语上》</span></span></p><p><span class="diczx2">楚庄王</span>伐 <span class="diczx2">郑</span>， <span class="diczx2">郑伯</span>肉袒牵羊以逆。—— <span class="diczx2">宋</span>· <span class="diczx2">苏轼</span><span class="smcs">《留侯论》</span></p><p><span class="diczx2">晋侯</span>逆夫人 <span class="diczx2">嬴氏</span>以归。——<span class="smcs">《左传·僖公二十三年》</span></p><p><span class="diczx1">尝过</span><span class="diczx2">青城山</span>，遇雪，舍于逆旅。—— <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·人事一》</span></p><p><span class="diczx2">陶子</span>将辞逆旅之馆，永归于本宅。—— <span class="diczx2">陶潜</span><span class="smcs">《自祭文》</span></p><p><span class="cino">(3)</span> 又如:逆战(迎战);逆厘(迎福纳吉);逆劳(迎劳);逆暑(迎夏。迎接暑气到来的祭礼);逆视(迎视);逆送(迎送);逆女(迎接女儿归宁)</p><p><span class="cino">(4)</span> 迎击;迎战 <span class="encs">[beat head-on]</span></p><p><span class="diczx1">并力逆</span><span class="diczx2">操</span>。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(5)</span> 又如:逆涛(顶着浪涛);逆击(迎击);逆格(迎击);逆拒(迎击，抵御)</p><p><span class="cino">(6)</span> 倒;反;向相反方向活动 <span class="encs">[reverse;inverse;converse]</span></p><p><span class="diczx1">水逆流百余里。——</span><span class="diczx2">郦道元</span><span class="smcs">《水经注·江水》</span></p><p><span class="cino">(7)</span> 又如:逆电流;逆鳞(倒生的鳞片);逆向(反方向);逆溢(水倒流泛滥);逆推(由下向上、由后向前推测)</p><p><span class="cino">(8)</span> 抵触;不顺;违背 <span class="encs">[defy;disobey;go against]</span></p><p><span class="diczx1">逆节伤化，不道。——<span class="smcs">《汉书·赵尹韩张两王传》</span></span></p><p><span class="diczx1">本末舛逆，首尾衡决——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《治安策》</span></p><p><span class="diczx1">且以一璧之故，逆强</span><span class="diczx2">秦</span>之欢，不可。——<span class="smcs">《史记·廉颇蔺相如列传》</span></p><p><span class="diczx1">忠言逆耳。——</span><span class="diczx2">明</span>· <span
