                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>哨</strong></p><p><span class="dicpy">shào  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shào.mp3"></a></span>  ㄕㄠˋ</span></p><ol>    <li>巡逻，警戒防守的岗位：～兵。～卡（<span class="dicpy">qiǎ ㄑㄧㄚˇ</span>）。～所。岗～。放～。</li>    <li>古代军队的编制单位，历代标准不一。</li>    <li>一种小笛：～子。</li>    <li>用口吹出的高尖音：口～儿。</li>    <li>鸟叫。</li></ol>

                    <h2 class="h2_entry"><span class="orth">哨</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>哨</strong> <span class="dicpy">shào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shào.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从口，肖声。本义:口小不能容的，一说口歪不正的)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[of mouth wry]</span></p><p><span class="diczx1">哨，不容也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">哨，口不容也。——<span class="smcs">《韵会》</span>引<span class="smcs">《说文》</span></span></p><p><span class="diczx1">某有枉矢哨壶。——礼记·投壶》</span></p><p><span class="cino">(3)</span> 又如:哨壶(口不正的壶)</p><p><span class="cino">(4)</span> 细狭尖锐 <span class="encs">[slender]</span></p><p><span class="diczx1">哨，小也。——<span class="smcs">《后汉书·马融传》</span>注</span></p><p><span class="diczx1">大匈哨后。——</span><span class="diczx2">马融</span><span class="smcs">《广成颂》</span></p><p><span class="cino">(5)</span> 又如:哨棒(行路防身的长木棍);哨腿(长腿。指高个子)</p><p><strong>词性变化</strong></p><p>◎ <strong>哨</strong> <span class="dicpy">shào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shào.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 巡逻 <span class="encs">[patrol]</span></p><p><span class="diczx1">质明，避哨竹林中。——<span class="smcs">《 指南录 后序》</span></span></p><p><span class="cino">(2)</span> 又如:哨口(巡逻了望的关卡、岗哨);哨船(在江中巡逻警戒的小兵船);哨骑(巡逻骑的马)</p><p><span class="cino">(3)</span> 侦察 <span class="encs">[reconnaitre]</span></p><p><span class="diczx1">男亲哨见，当道并无寨栅，军皆屯于山上。——<span class="smcs">《三国演义》</span></span></p><p><span class="cino">(4)</span> 又如:哨路(探路);哨见(探见);哨军(侦察巡逻的士兵)</p><p><span class="cino">(5)</span> 设岗放哨 <span class="encs">[sentry]</span>。如:哨线(警戒线)</p><p><span class="cino">(6)</span> 撮口或用哨子发出响声 <span class="encs">[whistle]</span></p><p><span class="diczx2">张顺</span>略哨一声，只见江上渔船撑扰来到岸边。—— <span class="diczx2">施耐庵</span><span class="smcs">《水浒传》</span></p><p><span class="cino">(7)</span> 又如:哨聚(召集，聚集)</p><p><span class="cino">(8)</span> 吹;喷 <span class="encs">[blow]</span></p><p><span class="diczx1">阴风飒飒，是神兵口内哨来烟;黑雾漫漫，是鬼祟暗中喷出气。——<span class="smcs">《西游记》</span></span></p><p><span class="cino">(9)</span> 叫;鸟叫 <span class="encs">[chirp]</span></p><p><span class="diczx2">行者</span>又嚎， <span class="diczx2">沙弥</span>又哨，你须不夺人之好。—— <span class="diczx2">元</span>· <span class="diczx2">王实甫</span><span class="smcs">《西厢记》</span></p><p><span class="cino">(10)</span> 逗弄;哄弄 <span class="encs">[tease]</span>。如:哨哄(逗弄)</p><p>◎ <strong>哨</strong> <span class="dicpy">shào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shào.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 岗哨，站岗放哨的地方 <span class="encs">[lookout post]</span>。如:哨卡;哨位;哨所;哨岗;哨官(管理一哨的武官);哨楼</p><p><span class="cino">(2)</span> 一队人马。古代军事术语。泛称战阵的两翼或军队的一支、一队 <span class="encs">[detachment]</span></p><p><span class="diczx2">宋江</span>传令 <span
