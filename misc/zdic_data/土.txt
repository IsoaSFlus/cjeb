                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>土</strong></p><p><span class="dicpy">tǔ  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǔ.mp3"></a></span> ㄊㄨˇ</span></p><ol><li>地面上的泥沙混合物：～壤。黄～。</li><li>疆域：国～。领～。</li><li>本地的，地方性的：故～。</li><li>民间生产的（区别于“<span class="diczx3">洋</span>”）：～方（民间流传的药方，亦称“偏方”）。</li><li>不合潮流：～气。</li><li>未熬制的鸦片：烟～。</li><li>中国古代乐器八音之一。</li><li>中国少数民族，主要分布于青海省：～族。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">土</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>土</strong> <span class="dicpy">tǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǔ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (象形。甲骨文字形，上象土块，下象地面。金文中空廓变填实，小篆又变为线条。本义:泥土，土壤)</p><p><span class="cino">(2)</span> 同本义。沉积于地面上的泥沙混合物。潮湿时称泥土，地质学上称土壤 <span class="encs">[soil;earth]</span></p><p><span class="diczx1">土，地之吐生物者也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">百谷草木丽乎土。——<span class="smcs">《易·彖传》</span></span></p><p><span class="diczx2">禹</span>敷土。——<span class="smcs">《书·禹贡》</span></p><p><span class="diczx1">辨十有二土之名物。——<span class="smcs">《周礼·大司徒》</span></span></p><p><span class="diczx1">君之视臣为土芥，则臣视君如寇仇。——<span class="smcs">《孟子·离娄》</span></span></p><p><span class="diczx1">焉置土石。——<span class="smcs">《列子·汤问》</span></span></p><p><span class="diczx1">如土石何。</span></p><p><span class="diczx1">不若燔土。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈·活板》</span></p><p><span class="diczx1">土膏微润。——</span><span class="diczx2">明</span>· <span class="diczx2">袁宏道</span><span class="smcs">《满井游记》</span></p><p><span class="cino">(3)</span> 又如:土粉(粉刷墙壁用的白垩土);土力(土壤肥沃的程度);土化(施肥使土壤肥沃);土坷拉(土块);土山(泥土堆积成的山);土城(土筑之城);土堡(用土石筑成的碉堡);土堰(土筑成的拦水坝);土阶茅屋(谓居住俭朴)。又指炼金术者的四大元素之一和五行之一。如:水、火、木、金、土</p><p><span class="cino">(4)</span> 土地 <span class="encs">[land;ground]</span></p><p><span class="diczx1">甘食其土之有。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《捕蛇者说》</span></p><p><span class="cino">(5)</span> 又如:土工(从事挖土埋人工作的人);土宇(土地住宅);土田(土地田畴);土功(治水土的工程);土作(土木工程);土薄(土地硗薄);守土有责;土公(土地神)</p><p><span class="cino">(6)</span> 尘土 <span class="encs">[dirt]</span>。如:土雨(飞扬的尘土);他鞋上都是土</p><p><span class="cino">(7)</span> 领土，国土 <span class="encs">[territory;domain]</span></p><p><span class="diczx1">天府之土。——<span class="smcs">《三国志·诸葛亮传》</span></span></p><p><span class="diczx1">北土禾平。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="diczx1">能守其土。——</span><span class="diczx2">宋</span>· <span class="diczx2">苏洵</span><span class="smcs">《六国论》</span></p><p><span class="diczx1">守土吏或降。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="cino">(8)</span> 又如:土疆(领土);土境(领土，国土)</p><p><span class="cino">(9)</span> 乡土，居处。某人的出生地、出身地、居住地或国籍地 <span class="encs">[country]</span>。如:故土;土棍(地方上的无赖、地痞);土团(乡里的自卫组织);土豪(乡里仗势欺人的人);土风(乡土歌谣);土思(思念故乡);土宇(乡土和屋宅)</p><p><span class="cino">(10)</span> 坟墓 <span class="encs">[mound]</span>。如:土眼(坟);土中人(坟中人);土中宅(坟墓);土骨堆(坟)</p><p><span class="cino">(11)</span> 烟土，粗制的鸦片 <span class="encs">[opium]</span>。如:土栈(贩运、囤积鸦片烟的商行);运土</p><p><span class="cino">(12)</span>                           <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/571F_979.svg"  /><span class="swbf">(土)</span><span class="ywyy">地之吐生萬物者也。</span><span class="swdz">吐土㬪韵。釋名曰。土、吐也。吐萬物也。</span><span class="ywyy"><img class='lazy k_img' data-original='//img.zdic.net/song/cn/2011E.svg'><span class='k_txt'>𠄞</span>象地之上、地之中。</span><span class="swdz">上各本作下。誤。今依韵會正。地之上、謂平土面者也。土二橫當齊長。士字則上十下一。上橫直之長相等。而下橫可隨意。今俗以下長爲土字。下短爲士字。絕無理。</span><span class="ywyy">丨、物出形也。</span><span class="swdz">此所謂引而上行讀若？也。合二字象形爲會意。它魯切。廣韵引文字指歸曰。無㸃。按文字指歸葢以無㸃者它魯切。有㸃者徒古切。田地主也。釋氏書國土必讀如杜是也。五部。</span><span class="ywyy">凡土之屬皆从土。</span><span class="xh"></span></p>                          </div>
