						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>枝</strong></p><p><span class="dicpy">zhī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span>  ㄓˉ</span></p><ol>    <li>由植物主干上分出来的茎条：树～。～干（<span class="dicpy">gàn </span>）。竹～。节外生～。～柯。～节（ａ．由一件事生发的其他问题；ｂ．细碎的，不重要的）。</li>    <li>量词，指杆形的：一～铅笔。</li>    <li>古同“<span class="diczx3">支</span>”，支持，分支。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>枝</strong></p><p><span class="dicpy">qí  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/qí.mp3"></a></span>  ㄑㄧˊ</span></p><ol>    <p>　◎ 古同“<span class="diczx3">歧</span>”，岔。</p></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>枝</strong> <span class="dicpy">zhī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从木，支声。本义:主干上分出的茎条)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[branch]</span></p><p><span class="diczx1">枝，木别生条也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">草荄之枝。——<span class="smcs">《素问·移精变气论》</span></span></p><p><span class="diczx1">千朵万朵压枝低。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《江畔独步寻花》</span></p><p><span class="diczx1">不蔓不枝。——</span><span class="diczx2">宋</span>· <span class="diczx2">周敦颐</span><span class="smcs">《爱莲说》</span></p><p><span class="diczx1">明月别枝惊鹊。——</span><span class="diczx2">宋</span>· <span class="diczx2">辛弃疾</span><span class="smcs">《西江月》</span></p><p><span class="diczx1">正用一枝鸣。——</span><span class="diczx2">宋</span>· <span class="diczx2">洪迈</span><span class="smcs">《容斋续笔》</span></p><p><span class="cino">(3)</span> 又如:枝任(树叶多的树枝);枝末(树枝的梢头);枝主(枝条和主干);枝杪(树木枝条的梢头);枝柯(枝条);枝干(树枝和树干)</p><p><span class="cino">(4)</span> 古代指嫡长子以外的宗族子孙 <span class="encs">[collateral branch]</span></p><p><span class="diczx1"><span class="smcs">《诗》</span>云:<span class="yhcs">“本枝百世。”</span>——<span class="smcs">《左传·庄公六年》</span></span></p><p><span class="cino">(5)</span> 又如:枝庶(嫡长子以外的支系);枝嗣(旁支的子孙);枝萼(枝上的花萼。喻宗族旁支)</p><p><span class="cino">(6)</span> 通<span class="yhcs">“肢”</span>。人的手足与鸟兽翼足的总称<span class="encs">[limb]</span></p><p><span class="diczx1">四枝坚固。——<span class="smcs">《管子·内业》</span></span></p><p><span class="diczx1">若生四枝。——<span class="smcs">《荀子·儒效》</span></span></p><p><span class="diczx1">人之有形体四枝。——<span class="smcs">《吕氏春秋·圜道》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>枝</strong> <span class="dicpy">zhī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> </p><p>〈量〉</p><p><span
