                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>惟</strong></p><p><span class="dicpy">wéi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wéi.mp3"></a></span>  ㄨㄟˊ</span></p><ol><li>单，只：～独。～一。～有。～恐。～命是从。～我独尊。</li><li>但是：雨虽止，～路仍甚泥泞。</li><li>文言助词，常用于句首：～妙～肖。～二月既望。</li><li>想，思考：思～（亦作“思维”）。～度（<span class="dicpy">dù</span>）（思量，揣度）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">惟</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>惟</strong> <span class="dicpy">wéi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wéi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从心，隹( <span class="dicpy">zhuī</span>)声。本义:思考;思念)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[think]</span></p><p><span class="diczx1">惟，凡思也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">臣窃惟事势。——</span><span class="diczx2">贾谊</span><span class="smcs">《治安策》</span></p><p><span class="diczx1">吾惟之，坚子固不足遣。——<span class="smcs">《汉书·张良传》</span></span></p><p><span class="cino">(3)</span> 又如:惟思(思虑);惟度(思考忖度);惟念(思念;考虑)</p><p><span class="cino">(4)</span> 为，是 <span class="encs">[be]</span></p><p><span class="diczx1">万邦黎献，共惟帝臣。——<span class="smcs">《书·益稷》</span></span></p><p><span class="diczx1">不变惟何。——<span class="encs">[英]</span></span><span class="diczx2">赫胥黎</span>著、 <span class="diczx2">严复</span>译<span class="smcs">《天演论》</span></p><p><span class="cino">(5)</span> 有 <span class="encs">[have]</span></p><p><span class="diczx1">相亦惟终。——<span class="smcs">《礼记》</span></span></p><p><span class="cino">(6)</span> 听从，随从 <span class="encs">[obey]</span></p><p><span class="diczx1">大国苟不以为然也，则亦惟命耳。——</span><span class="diczx2">刘献廷</span><span class="smcs">《广阳杂记》</span></p><p><span class="cino">(7)</span> 又如:惟惟(听从顺服的样子)</p><p><span class="cino">(8)</span> 愿，希望 <span class="encs">[hope]</span></p><p><span class="diczx1">若草庵见毁一节，成功之后，万惟留心。——<span class="smcs">《英烈传》</span></span></p><p><span class="cino">(9)</span> 又如:惟所欲为(想干什么就干什么，任意而为)</p><p><span class="cino">(10)</span> 在于 <span class="encs">[lie in]</span></p><p><span class="diczx1">德惟善政，政在养民。——<span class="smcs">《书·大禹谟》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>惟</strong> <span class="dicpy">wéi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wéi.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 用来限定范围，相当于<span class="yhcs">“只有”</span>、<span class="yhcs">“只是”</span> <span class="encs">[only]</span></p><p><span class="diczx1">惟明克允。——<span class="smcs">《书·舜典》</span></span></p><p><span class="diczx1">惟士为能。——<span class="smcs">《孟子·梁惠王上》</span></span></p><p><span class="diczx1">惟闻女叹息。——<span class="smcs">《乐府诗集·木兰诗》</span></span></p><p><span class="diczx1">惟吾德馨。——</span><span class="diczx2">唐</span>· <span class="diczx2">刘禹锡</span><span class="smcs">《陋室铭》</span></p><p><span class="diczx1">惟有乳下孙。——</span><span class="diczx2">杜甫</span><span class="smcs">《石壕吏》</span></p><p><span class="diczx1">惟手熟尔。——</span><span class="diczx2">宋</span>· <span class="diczx2">欧阳修</span><span class="smcs">《归田录》</span></p><p><span class="diczx1">惟，独也。常语也。或作唯、维。——</span><span class="diczx2">清</span>· <span class="diczx2">王引之</span><span class="smcs">《经传释词》</span></p><p><span class="cino">(2)</span> 又如:惟一惟精(精诚专事，毫无杂念);惟义是从(只要有道理就坚决服从);惟口起羞(谓言语不慎，招致羞辱);惟利是求(一心为利，别的什么都不顾);惟精惟一(精纯专一)</p><p><span class="cino">(3)</span> 表示动作、行为的重复，相当于<span class="yhcs">“又”</span> <span class="encs">[again]</span>。如:惟新(更新)</p><p><span class="cino">(4)</span> 相当于<span class="yhcs">“犹”</span>、<span class="yhcs">“还”</span> <span class="encs">[still]</span></p><p><span
