						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>浪</strong></p><p><span class="dicpy">làng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làng.mp3"></a></span>  ㄌㄤˋ</span></p><ol><li>大波：波～。海～。巨～。风～。～涛。～潮（亦喻大规模的社会运动）。风平～静。</li><li>像波浪起伏的：麦～。声～。</li><li>没有约束，放纵：放～。流～。～费。～迹。～人。孟～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">浪</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>浪</strong> <span class="dicpy">làng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 波浪，水因风吹、石激而形成的涌动 <span class="encs">[wave]</span></p><p><span class="diczx1">风起浪涌。——<span class="smcs">《世说新语·雅量》</span></span></p><p><span class="diczx1">长风破浪会有时，直挂云帆济沧海。——</span><span class="diczx2">李白</span><span class="smcs">《行路难》</span></p><p><span class="diczx1">温泉毖涌而自浪。——</span><span class="diczx2">左思</span><span class="smcs">《魏都赋》</span></p><p><span class="cino">(2)</span> 又如:风浪(水面上的风和波浪);激浪(汹涌急剧的波浪);浪波(浪头。波浪);浪淘淘(波浪翻滚的样子)</p><p><span class="cino">(3)</span> 像波浪起伏的样子 <span class="encs">[sth.resembling wave]</span>。如:热浪;林浪;麦浪</p><p><strong>词性变化</strong></p><p>◎ <strong>浪</strong> <span class="dicpy">làng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 逛，无事到处遛 <span class="encs">[wave;wander]</span>。如:他没事到处浪;浪走(无目的胡乱走);浪跄(走路不稳);浪迹萍踪(四方漫游，行踪无定)</p><p><span class="cino">(2)</span> 勾引，劝…发生性关系 <span class="encs">[seduce]</span>。如:浪汉(挑逗男人)</p><p><span class="cino">(3)</span> 〈方〉∶敞开，亮出来 <span class="encs">[open]</span></p><p><span class="diczx1">接着是家家都<span class="yhcs">“浪山头”</span>了。——</span><span class="diczx2">茅盾</span><span class="smcs">《春蚕》</span></p><p>◎ <strong>浪</strong> <span class="dicpy">làng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 放荡;放纵 <span class="encs">[unrestrained;dissolute]</span></p><p><span class="diczx1">谑浪笑敖。——<span class="smcs">《诗·邶风·终风》</span></span></p><p><span class="cino">(2)</span> 又如:浪弟子(称行为放荡、不知检束的青年人);浪事(不好的事;令人生厌的事);浪荡鬼(对放荡不检的人的鄙称)</p><p><span class="cino">(3)</span> 言词不干净;随口说脏话 <span class="encs">[dirty]</span></p><p><span class="diczx1">小岭这个令，浪得很!好好的说一个飞觞解秽罢。——</span><span class="diczx2">清</span>· <span class="diczx2">魏子安</span><span class="smcs">《花月痕》</span></p><p><span class="cino">(4)</span> 空 <span class="encs">[empty]</span>。如:浪广(犹空泛);浪荡(空阔);浪荡灯(悬空挂着的灯，吊灯)</p><p><span class="cino">(5)</span> 无用 <span class="encs">[useless]</span></p><p><span class="diczx1">你这个不知死活的东西!这府里希罕你的那扔不了的浪东西!——<span class="smcs">《红楼梦》</span></span></p><p><span class="cino">(6)</span> 破烂 <span class="encs">[tattered;ragged;worn-out]</span></p><p><span class="diczx1">这府里希罕你的那朽不了的浪东西?——<span class="smcs">《红楼梦》</span></span></p><p><span class="cino">(7)</span> 轻易;随便 <span class="encs">[freely]</span>。如:浪舌(随意乱说);浪骂(乱骂);浪侃(胡侃。胡说，瞎扯);浪使(乱花费);浪语(胡说，乱说)</p><p>◎ <strong>浪</strong> <span class="dicpy">làng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/làng.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 徒然;白白地 <span class="encs">[in vain]</span>。如:浪死(徒然死去;白白送死);浪泊(徒然停留);浪度(虚度)</p><p><span
