                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>仕</strong></p><p><span class="dicpy">shì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shì.mp3"></a></span>  ㄕˋ</span></p><ol><li>做官：出～。～宦。学而优则～。～途。～女（a.宫女，贵族妇女；b.以社会上层妇女为题材的中国画。均亦作“士女”）。</li><li>审察：“弗问弗～”。</li><li>古同“<span class="diczx3">事</span>”，事业。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> official; serve government</p><p><span class="z_ts2">德语</span> Beamter werden,Leibwächter (roter Stein im chinesischen Schach)</p><p><span class="z_ts2">法语</span> exercer une fonction publique</p></div>

                    <h2 class="h2_entry"><span class="orth">仕</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>仕</strong> <span class="dicpy">shì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从人，从士。本义:做官)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[be an official;fill an office]</span></p><p><span class="diczx1">学而优则仕。——<span class="smcs">《论语·子张》</span></span></p><p><span class="diczx1">常经典籍为业，未遑仕进之事。——<span class="smcs">《后汉书·崔骃传》</span></span></p><p><span class="diczx2">申子</span>请仕其从兄官。——<span class="smcs">《韩非子·外储说左上》</span></p><p><span class="diczx1">不仕无义。——<span class="smcs">《论语·微子》</span></span></p><p><span class="diczx1">使天下仕者皆欲立于王之朝。——<span class="smcs">《孟子·梁惠王上》</span></span></p><p><span class="diczx1">时墨者</span><span class="diczx2">东郭先生</span>将北适中山以干仕。—— <span class="diczx2">马中锡</span><span class="smcs">《中山狼传》</span></p><p><span class="diczx1">乙先得举，登仕路，闻甲落魄，笑不顾，交以绝。——</span><span class="diczx2">清</span>· <span class="diczx2">周容</span><span class="smcs">《芋老人传》</span></p><p><span class="cino">(3)</span> 又如:仕官(就任官职);仕途经济(通过<span class="yhcs">“学而优则仕”</span>的道路去做官);仕止(出仕或隐退);仕路(升官之路)</p><p><span class="cino">(4)</span> 通<span class="yhcs">“事”</span>。从事;做事 <span class="encs">[engage]</span></p><p><span class="diczx1">滔滔</span><span class="diczx2">江</span>、 <span class="diczx2">汉</span>，南国之纪;尽瘁以仕，宁莫我有。——<span class="smcs">《诗·小雅·四月》</span></p><p><span class="cino">(5)</span> 学习政事，见习试用 <span class="encs">[study]</span></p><p><span class="diczx1">仕，学也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">段玉裁</span>注:<span class="yhcs">“古义宦训仕，仕训学。”</span>。 <span class="diczx2">朱</span>曰:<span class="yhcs">“犹今言试用也。”</span></p><p><span class="cino">(6)</span> 审察。检查 <span class="encs">[examine]</span></p><p><span class="diczx1">弗躬弗亲，庶民弗信;弗问弗仕，勿罔君子。——<span class="smcs">《诗·小雅·节南山》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>仕</strong> <span class="dicpy">shì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 职位 <span class="encs">[post]</span></p><p><span class="diczx1">或著显绩而婴时戮，或有大才而无贵仕，皆扬清风于上烈，垂令闻而不已。——</span><span class="diczx2">晋</span>· <span class="diczx2">潘岳</span><span class="smcs">《西征赋》</span></p><p><span class="cino">(2)</span> 通<span class="yhcs">“士”</span>。古代四民之一。指以道艺、武勇谋求仕进的人 <span class="encs">[intellect]</span></p><p><span class="diczx1">丰水有芑，</span><span class="diczx2">武王</span>岂不仕。——<span class="smcs">《诗·大雅·文王有声》</span></p><p><span class="diczx1">此非能仕之所耻也。——<span class="smcs">《韩非子·说难》</span></span></p><p><span class="diczx1">有仕于此，而子悦之。——<span class="smcs">《孟子·公孙丑下》</span></span></p><p><span class="cino">(3)</span> 又如:仕子(泛指文人、学子);仕林(士大夫阶层)</p><p><span class="cino">(4)</span> 官宦，官员 <span class="encs">[official]</span></p><p><span class="diczx1">仕女往住夜游。——</span><span class="diczx2">孟元老</span><span class="smcs">《东京梦华录》</span></p><p><span class="cino">(5)</span> 又如:仕版(官员的花名册);仕户(官宦人家);仕子(有官职的人);仕籍(旧指记载官吏名籍的簿册。亦借指仕途，官场);仕门(仕宦之家);仕流(一般的官属);仕家(官宦之家);仕禄(官职，职务)</p><p><span class="cino">(6)</span> 通<span class="yhcs">“事”</span>。事情 <span
