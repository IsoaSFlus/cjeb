                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>跟</strong></p><p><span class="dicpy">gēn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gēn.mp3"></a></span>  ㄍㄣˉ</span></p><ol><li>脚的后部，踵：脚后～。</li><li>鞋袜的后部：高～鞋。袜后～儿。</li><li>随在后面，紧接着：～随。～踪。</li><li>赶，及：～不上班。</li><li>和，同：我～他在一起工作。</li><li>对，向：我已经～他说了。</li></ol>

                    <h2 class="h2_entry"><span class="orth">跟</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>跟</strong> <span class="dicpy">gēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gēn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从足，艮( <span class="dicpy">gèn</span>)声。本义:足后为跟)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[heel]</span></p><p><span class="diczx1">跟，足踵也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">足后曰跟，在下方着地，一体任之，象木根也。——<span class="smcs">《释名·释形体》</span></span></p><p><span class="diczx1">阽焦原而跟趾。——</span><span class="diczx2">张衡</span><span class="smcs">《思玄赋》</span></p><p><span class="diczx1">诘曲犹能辨跟肘。——</span><span class="diczx2">苏轼</span><span class="smcs">《凤翔八观》</span></p><p><span class="cino">(3)</span> 又如:跟蹠(脚后跟);跟踵(脚后跟);跟胫(脚跟与小腿);跟趾(脚后跟)</p><p><span class="cino">(4)</span> 器官的基底部分。亦指物体的底部或后部 <span class="encs">[base]</span></p><p><span class="diczx1">每册之跟，自书精楷以表之。——<span class="smcs">《清稗类钞·鉴赏类》</span></span></p><p><span class="cino">(5)</span> 又如:崖跟(山崖底部);跟底(底部)</p><p><strong>词性变化</strong></p><p>◎ <strong>跟</strong> <span class="dicpy">gēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gēn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 追随于后 <span class="encs">[follow]</span></p><p><span class="diczx1">有元地脚保识人前去跟寻。——</span><span class="diczx2">吴自牧</span><span class="smcs">《梦粱录》</span></p><p><span class="cino">(2)</span> 又如:跟我来;跟他走;小偷一直跟着他;跟屁股(紧跟不舍)</p><p><span class="cino">(3)</span> <span class="encs">[仆人]</span> 侍奉主人 <span class="encs">[wait upon]</span>。如:跟前人(被收房的丫头);跟底(跟前，身边);跟马(旧时称跟随在官员车轿后面骑马的随从);跟房(旧时随嫁的婢女)</p><p><span class="cino">(4)</span> 穿着(鞋)，趿 <span class="encs">[put on]</span></p><p><span class="diczx1">驾长檐车，跟高齿屐。——</span><span class="diczx2">北齐</span>· <span class="diczx2">颜之推</span><span class="smcs">《颜氏家训》</span></p><p><span class="cino">(5)</span> 比;及;抵 <span class="encs">[compare with]</span>。如:不跟(不及)</p><p><span class="cino">(6)</span> 旧时指女子嫁人 <span class="encs">[marry]</span>。如:跟人</p><p>◎ <strong>跟</strong> <span class="dicpy">gēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gēn.mp3"></a></span> </p><p>〈介〉</p><p><span class="cino">(1)</span> 和，同 <span class="encs">[with]</span></p><p><span class="diczx1">甚么王八蛋的客，有胆子的快来跟三爷碰碰。——<span class="smcs">《老残游记》</span></span></p><p><span class="cino">(2)</span> 只和指人的名词组合，介绍出与主体共同完成某一动作的对象。如:跟老乡们一起劳动;有事要跟群众商量</p><p><span class="cino">(3)</span> 表示与某事物有无联系。如:他跟这事没关系;高山的气压跟平地上不一样</p><p><span class="cino">(4)</span> 向，对——表示与动作有关的对方，只和指人的名词组合 <span class="encs">[to]</span>。如:他要跟你谈一谈;跟大爷碰碰(同大爷较量较量);这事还没有跟领导汇报</p><p><span class="cino">(5)</span> 表示处所，相当于<span class="yhcs">“从”</span> <span
