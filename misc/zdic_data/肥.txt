                          <h2 class="h2_entry"><span class="orth">肥</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>肥</strong></p><p><span class="dicpy">féi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/féi.mp3"></a></span>  ㄈㄟˊ</span></p><ol><li>含脂肪多的，与“<span class="diczx3">瘦</span>”相对：～肉。～胖。～缺（指收入多的官职）。～硕（a.大而肥胖；b.果实大而饱满）。～马轻裘。</li><li>土质含养分多的：～沃。～美（a.肥沃；b.肥壮、丰美）。</li><li>能增加田地养分的东西（如粪、豆饼、化学配合剂等）：～料。～力。</li><li>使田地增加养分：～田。</li><li>指衣服鞋袜等宽大，与“<span class="diczx3">瘦</span>”相对：～大。～瘦儿。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>肥</strong> <span class="dicpy">féi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/féi.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (会意。从肉，本义:脂肪多)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[corpulent;fat]</span></p><p><span class="diczx1">肥，多肉也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">坚土之人肥。——<span class="smcs">《大戴礼记·易本命》</span></span></p><p><span class="diczx1">瞻肥瘠。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="diczx1">吾牲牷肥。——<span class="smcs">《左传·桓公六年》</span>。</span><span class="diczx2">服</span>注:<span class="yhcs">“牛羊曰肥。”</span></p><p><span class="diczx1">庖有肥肉，厩有肥马。——<span class="smcs">《孟子·梁惠王上》</span></span></p><p><span class="cino">(3)</span> 又如:肥猪;肥肉(喻指有钱的人);肥腻(含油脂多的食物;油脂多);肥汉(肥胖的男子)</p><p><span class="cino">(4)</span> 引申为肥沃 <span class="encs">[fertile]</span></p><p><span class="diczx1">不爱珍器重肥饶之地。——</span><span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《过秦论》</span></p><p><span class="cino">(5)</span> 又如:这块地肥极了;肥衍(土地肥沃);肥饶(肥沃富饶)</p><p><span class="cino">(6)</span> 茁壮，粗大 <span class="encs">[sturdy;stout]</span></p><p><span class="diczx1">桃花流水鳜鱼肥。——</span><span class="diczx2">唐</span>· <span class="diczx2">张志和</span><span class="smcs">《渔歌子》</span></p><p><span class="cino">(7)</span> 又如:肥茂(肥壮茂盛);肥茁(肥壮);肥润(肥壮润泽);肥泽(肌肉丰润)</p><p><span class="cino">(8)</span> 富裕 <span class="encs">[rich]</span>。如:肥强(富饶强盛)</p><p><span class="cino">(9)</span> 衣服又宽又大 <span class="encs">[loose]</span>。如:伙穿一条裤子都嫌肥;肥大</p><p><span class="cino">(10)</span> 赚钱又不费力 <span class="encs">[lucrative]</span>。如:肥秩(收入丰厚的官位)</p><p><strong>词性变化</strong></p><p>◎ <strong>肥</strong> <span class="dicpy">féi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/féi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 使土地肥沃 <span class="encs">[fertilize]</span></p><p><span class="diczx1">掩地表亩，剌草殖谷，多粪肥田，是农夫众庶之事也。——<span class="smcs">《荀子·富国》</span></span></p><p><span class="cino">(2)</span> 使发财 <span class="encs">[enrich]</span>。如:肥己</p><p><span class="cino">(3)</span> 通<span class="yhcs">“非”</span>( <span class="dicpy">fēi</span>)。责难 <span class="encs">[censure;blame]</span></p><p><span class="diczx1">目所偏视，</span><span class="diczx2">晋国</span>爵之;口所偏肥， <span class="diczx2">晋国</span>黜之。——<span class="smcs">《列子·黄帝》</span></p><p>◎ <strong>肥</strong> <span class="dicpy">féi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/féi.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 肥料 <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/80A5.svg">						  <p>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>符非切，音腓。【說文】多肉也。从肉从卪，會意。【徐曰】肉不可過多，故从卪寓戒。【禮·禮運】安之以樂，而不達于順，猶食而弗肥也。</p><p>又【博雅】肥，盛也。</p><p>又【廣韻】肥腯。【蔡邕·獨斷】凡祭宗廟禮，牲之別名，豚曰腯肥。【詩·小雅】旣有肥牡，以速諸舅。【左傳·桓六年】博碩肥腯。</p><p>又田有肥瘠。【書·禹貢厥田惟中中傳】田之高下肥瘠。九州之中爲第五。</p><p>又馬亦稱肥。【前漢·食貨志】冠蓋相望，乗堅策肥。</p><p>又饒裕也。【易·遯卦】上九肥遯。【疏】肥，饒裕也。</p><p>又幡名。【吳語】建肥胡。【註】肥胡，幡名。</p><p>又鳥名。【山海經】英山有鳥焉，其名曰肥遺。</p><p>又蛇名。【山海經】太華之山有蛇焉，名曰肥<img class='lazy k_img' data-original='//img.zdic.net/song/cn/27525.svg'><span class='k_txt'>𧔥</span>。【註】肥，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/27525.svg'><span class='k_txt'>𧔥</span>，皆毒蟲也。</p><p>又水之初出同流者名肥。【爾雅·釋水】歸異出同流，肥。【疏】謂小水支分歸入大海則異，其泉源初出，則同流者名肥。</p><p>又水名。【詩·衞風】我思肥泉。【疏】肥泉，是衞水也。</p><p>又地名。【前漢·地理志】合肥。【註】屬九江郡。應劭曰：夏水出父城東南，至此與淮合，故曰合肥。</p><p>又縣名。【史記·高祖功臣年表】肥如侯蔡寅。【註】肥如，縣名，屬遼西。</p><p>又國名。【左傳·昭十二年】秋八月壬午滅肥。【註】肥，白狄也。</p><p>又姓。【史記·趙世家】先問先王貴臣肥義。</p><p>又人名。季康子名肥。見【左傳·哀三年】。</p><p>又【集韻】補美切，音秕。薄也。【列子·黃帝篇】口所偏肥，晉國黜之。【註】薄也。</p><p>又水名。肥者，通作淝。【詩·衞風】我思肥泉。【釋文】肥，或作淝。</p>
