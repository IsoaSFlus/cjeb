                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>殆</strong></p><p><span class="dicpy">dài  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span>  ㄉㄞˋ</span></p><ol>    <li>危：危～。危乎～哉。知足不辱，知止不～（懂得满足不贪心就不会受辱，懂得适可而止就不会遭到危险）。</li>    <li>大概，几乎：伤亡～尽。</li>    <li>古同“<span class="diczx3">怠</span>”，懈怠。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>殆</strong> <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从歹( <span class="dicpy">è</span>)，台声。本义:危险)</p><p><span class="cino">(2)</span> 同本义<span class="encs">[danger]</span></p><p><span class="diczx1">殆，危也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">民今方殆，视民梦梦——<span class="smcs">《诗·小雅·正月》</span></span></p><p><span class="diczx1">亦曰殆哉。——<span class="smcs">《书·秦誓》</span>。传:<span class="yhcs">“危殆。”</span></span></p><p><span class="diczx2">晋</span>有三不殆。——<span class="smcs">《左传·昭公四年》</span>。注:<span class="yhcs">“危也。”</span></p><p><span class="diczx1">兵殆于垂沙。——<span class="smcs">《荀子·议兵》</span>。注:<span class="yhcs">“谓危亡也。”</span></span></p><p><span class="diczx1">知己知彼，百战不殆。——<span class="smcs">《孙子·谋攻》</span></span></p><p><span class="diczx1">三守不完，则国危身殆。——<span class="smcs">《韩非子·三守》</span></span></p><p><span class="diczx1">备危恐殆，急置太子，祸乃无从起。——<span class="smcs">《韩非子·扬权》</span></span></p><p><span class="diczx1">此二者，殆物也。——<span class="smcs">《韩非子·忠孝》</span></span></p><p><span class="cino">(3)</span> 又如:殆危(危险)</p><p><span class="cino">(4)</span> 困乏;疲惫 <span class="encs">[tired]</span></p><p><span class="diczx1">以有涯隨无涯，殆已。——<span class="smcs">《庄子·养生主》</span>。</span><span class="diczx2">向</span>注:<span class="yhcs">“疲困之谓也。”</span></p><p><span class="diczx1">日将暮，步履颇殆，休于路侧。——<span class="smcs">《聊斋志异》</span></span></p><p><span class="cino">(5)</span> 假借为<span class="yhcs">“怠”</span>。懒惰 <span class="encs">[lazy;idle]</span></p><p><span class="diczx1">周行而不殆。——<span class="smcs">《老子·二十五章》</span></span></p><p><span class="diczx1">受命不殆。——<span class="smcs">《诗·商颂·玄鸟》</span></span></p><p><span class="diczx1">滋敝邑休殆。——<span class="smcs">《左传·昭公五年》</span></span></p><p><span class="diczx1">学而不思则罔，思而不学则殆。——<span class="smcs">《论语·为政下》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>殆</strong> <span class="dicpy">dài</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dài.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 表推测，相当于<span class="yhcs">“大概”</span>、<span class="yhcs">“几乎”</span> <span class="encs">[nearly]</span></p><p><span class="diczx1">殆于不可。——<span class="smcs">《孟子》</span></span></p><p><span class="diczx2">沛公</span>殆天授。——<span class="smcs">《史记·留侯世家》</span></p><p><span class="diczx1">殆有神护者。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="diczx1">游历殆遍。——</span><span class="diczx2">宋</span>· <span class="diczx2">沈括</span><span class="smcs">《梦溪笔谈》</span></p><p><span class="diczx2">扬州</span>城下，进退不由，殆例送死。—— <span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《指南录·后序》</span></p><p><span class="cino">(2)</span> 又如:敌人伤亡殆尽;殆其(大概);殆庶(庶几，近似)</p><p><span class="cino">(3)</span> 表示范围，相当于<span class="yhcs">“仅仅”</span>、<span class="yhcs">“只”</span> <span class="encs">[only]</span></p><p><span class="diczx1">此殆空言，非至计也。——<span class="smcs">《汉书·赵充国传》</span>。</span><span class="diczx2">颜师古</span>注:<span class="yhcs">“殆，仅也。”</span></p><p><span class="cino">(4)</span> 表示肯定，相当于<span
