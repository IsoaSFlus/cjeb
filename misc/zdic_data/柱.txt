                          <h2 class="h2_entry"><span class="orth">柱</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>柱</strong></p><p><span class="dicpy">zhù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhù.mp3"></a></span>  ㄓㄨˋ</span></p><ol><li>建筑物中直立的起支撑作用的构件：～子。～石。顶梁～。</li><li>形状像柱子的东西，或作用重要如柱子的：水～。～臣（对国家有重要作用的臣子）。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> pillar, post; support; lean on</p><p><span class="z_ts2">德语</span> Betonpfeiler  (S)​,Säule  (S)​,Zhu  (Eig, Fam)</p><p><span class="z_ts2">法语</span> pilier,poteau</p></div>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>柱</strong> <span class="dicpy">zhǔ</span></p><p>〈动〉</p><p><span class="cino">(1)</span> 支撑。通<span class="yhcs">“拄”</span> <span class="encs">[support]</span></p><p><span class="diczx1">且鳌足可以柱天，体必长大，不容于天地，</span><span class="diczx2">女娲</span>虽圣，何能杀之?——<span class="smcs">《论衡·谈天》</span></p><p><span class="cino">(2)</span> 又如:柱天(撑天;支天);柱地(立柱支地)</p><p><span class="cino">(3)</span> 挡住;阻塞 <span class="encs">[stop]</span></p><p><span class="diczx1">内人共举机以柱门，兵斫门，不能破。——<span class="smcs">《三国志》</span></span></p><p><span class="cino">(4)</span> 拄 <span class="encs">[lean on]</span></p><p><span class="diczx1">手把定筇竹，柱杖行田野，出村房。——<span class="smcs">《刘知远诸宫调》</span></span></p><p><span class="cino">(5)</span> 又如:柱杖(手杖)</p><p><span class="cino">(6)</span> 另见 <span class="dicpy">zhù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhù.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>柱</strong> <span class="dicpy">zhù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从木，主声。本义:屋柱)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[post;pillar;column]</span></p><p><span class="diczx1">柱，楹也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">剪屏柱楣。——<span class="smcs">《仪礼·丧服》</span>。注:<span class="yhcs">“所谓梁闇。”</span></span></p><p><span class="diczx1">东至于底柱。——<span class="smcs">《书·禹贡》</span></span></p><p><span class="diczx1">散木也，…以为柱则蠹。——<span class="smcs">《庄子·人间世》</span></span></p><p><span class="diczx2">荆轲</span>废，乃引其匕首以掷 <span class="diczx2">秦王</span>，不中，中柱。——<span class="smcs">《战国策·燕策》</span></p><p><span class="diczx1">柱以白璧。——</span><span class="diczx2">唐</span>· <span class="diczx2">李朝威</span><span class="smcs">《柳毅传》</span></p><p><span class="cino">(3)</span> 又如:柱磉石(支撑柱子的基石);铜柱;楣柱;柱桷(屋柱和椽子);柱梁(柱和梁);柱栋(屋柱与栋梁);柱极(殿柱和正梁);柱楣(茅庐的立柱和前檐;柱和梁)</p><p><span class="cino">(4)</span> 像柱子的东西 <span class="encs">[sth.shaped like a column]</span></p><p><span class="diczx1">王以名使</span><span class="diczx2">括</span>(赵括)，若胶柱鼓瑟耳。——<span class="smcs">《史记·廉颇蔺相如列传》</span></p><p><span class="cino">(5)</span> 又如:水柱;冰柱;花柱</p><p><span class="cino">(6)</span> 另见 <span                     <div class="gycd"><div class="pz"><ruby><rbc><rb>柱</rb></rbc><rtc><rt>ㄓㄨˋ</rt></rtc><rtc><rt>zhù <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/gycd/zd/4529.ogg"></a></span> </rt></rtc></ruby></div><div class="gycd-item"><span class="gc_cx">名</span><ol><li><p class="def"><span class="gc_sy">屋中直立以支持梁的粗木。</span><span class="gc_yy">《史记·卷八六·刺客传·荆轲传》：「秦王方环<b>柱</b>走，卒惶急。」</span><span class="gc_yx">也称为「柱子」。</span></p></li><li><p class="def"><span class="gc_sy">泛称细长像<b><b>柱</b></b>形的东西。</span><span class="gc_lz">如：「水<b>柱</b>」、「花<b>柱</b>」。</span></p></li><li><p class="def"><span class="gc_sy">琴瑟上系弦的木条。</span><span class="gc_yy">唐·李商隐〈锦瑟〉诗：「锦瑟无端五十弦，一弦一<b>柱</b>思华年。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">动</span><ol><li><p class="def"><span class="gc_sy">支持。</span><span class="gc_yy">汉·王充《论衡·谈天》：「且鼇足可以<b>柱</b>天，体必长大，不容于天地，女娲虽圣，何能杀之？」</span></p></li><li><p class="def"><span class="gc_sy">讥刺。</span><span class="gc_yy">《汉书·卷六七·朱云传》：「既论难，连<b>柱</b>五鹿君。」</span><span class="gc_yx">通「拄」。</span></p></li></ol></div></div>
