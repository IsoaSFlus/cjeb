                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>迷</strong></p><p><span class="dicpy">mí  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mí.mp3"></a></span>  ㄇㄧˊ</span></p><ol><li>分辨不清，失去了辨别、判断的能力：～信。～糊。～津。～惘。～蒙（ａ．昏暗看不清；ｂ．神志模糊不清；ｅ．使迷惑，受蒙蔽）。执～不悟。</li><li>醉心于某种事物，发生特殊的爱好：～恋。入～。</li><li>沉醉于某种事物的人：棋～。革新～。</li><li>使人陶醉：景色～人。</li></ol>

                    <h2 class="h2_entry"><span class="orth">迷</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>迷</strong> <span class="dicpy">mí</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mí.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，米声。本义:迷路;分辨不清)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[be lost;be confused]</span></p><p><span class="diczx1">迷，惑也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">烈风雷雨弗迷。——<span class="smcs">《书·舜典》</span></span></p><p><span class="diczx1">凡夫失其所欲之路，而妄行之，则为迷。——<span class="smcs">《韩非子·解老》</span></span></p><p><span class="diczx1">及行迷之未远。——<span class="smcs">《离骚》</span></span></p><p><span class="diczx1">迷不知宠之门。——<span class="smcs">《楚辞·惜诵》</span></span></p><p><span class="diczx1">迷其所在。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="diczx1">入溆浦余儃佪兮，迷不知吾所如。——<span class="smcs">《楚辞·九章·涉江》</span></span></p><p><span class="diczx1">高低冥迷，不知西东。——</span><span class="diczx2">杜牧</span><span class="smcs">《阿房宫赋》</span></p><p><span class="diczx1">遂迷，不复得路。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《桃花源记》</span></p><p><span class="diczx1">其若迷。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《朱文公校昌黎先生集》</span></p><p><span class="diczx1">无迷其途。</span></p><p><span class="diczx1">道中迷雾。——</span><span class="diczx2">清</span>· <span class="diczx2">姚鼐</span><span class="smcs">《登泰山记》</span></p><p><span class="cino">(3)</span> 又如:迷离徜彷(模糊不明);迷罔;迷岸(佛家语。指迷惑而不悟的生死流转世界)</p><p><span class="cino">(4)</span> 迷惑，使…辨不清 <span class="encs">[delude;confuse]</span></p><p><span class="diczx1">乱花渐欲迷人眼，浅草才能没马蹄。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《钱塘湖春行》</span></p><p><span class="cino">(5)</span> 又如:迷魂(迷惑人的事理，使人身不由己);迷网(指迷惑人的事物像网罗一样，使人陷落而无以自拔);迷夺时明(迷惑、剥夺当今明哲之士的舆论);迷眩缠陷(以声色迷惑，为罗网陷阱);迷途</p><p><span class="cino">(6)</span> 迷恋 <span class="encs">[be addicted to;be infatuated with]</span></p><p><span class="diczx1">迷花倚石忽已暝(迷恋着花，依倚着石，不觉天色已经晚了。暝，天黑、夜晚)。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span><span class="smcs">《梦游天姥吟留别》</span></p><p><span class="cino">(7)</span> 昏迷 <span class="encs">[fall into a stupor;be in a coma]</span></p><p><span class="diczx2">扁鹊</span>遂饮二人毒酒，迷死三日。——<span class="smcs">《列子·汤问》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>迷</strong> <span class="dicpy">mí</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mí.mp3"></a></span> </p><p>〈名〉</p><p>狂热爱好者，通常是作为观众而非直接参加者 <span
