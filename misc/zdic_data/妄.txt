                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>妄</strong></p><p><span class="dicpy">wàng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wàng.mp3"></a></span>  ㄨㄤˋ</span></p><ol><li>胡乱，荒诞不合理：轻举～动。胆大～为。～自尊大。～图。狂～。～想。</li><li>非分的，不实的：姑～听之。～自菲薄。～作主张。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> absurd, foolish, reckless; false</p><p><span class="z_ts2">德语</span> sich anmaßen  (V)​,absurd  (Adj)​,albern  (Adj)​,unbesonnen  (Adj)​,widersinnig  (Adj)</p><p><span class="z_ts2">法语</span> faux,insensé,téméraire</p></div>

                    <h2 class="h2_entry"><span class="orth">妄</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>妄</strong> <span class="dicpy">wàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wàng.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> (形声。从女，亡声。本义:胡乱)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[at random]</span></p><p><span class="diczx1">妄，乱也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">不通于轻重谓之妄言。——<span class="smcs">《管子·山至数》</span></span></p><p><span class="diczx1">以人自观谓之度，反度为妄。——<span class="smcs">《贾子道术》</span></span></p><p><span class="diczx1">施妄者，乱之始也。——<span class="smcs">《春秋繁露》</span></span></p><p><span class="diczx1">妄自菲薄。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p><span class="diczx1">多求妄用。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="diczx1">妄传伯夷。——</span><span class="diczx2">清</span>· <span class="diczx2">黄宗羲</span><span class="smcs">《原君》</span></p><p><span class="diczx1">毋妄发。——</span><span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p><span class="cino">(3)</span> 又如:妄口(胡说);妄系(无故抓人入罪);妄扳平人(胡乱牵连无辜之人);妄施(随便地施恩);妄折(无原则地屈从);妄行(随便行动);妄言(乱说)</p><p><strong>词性变化</strong></p><p>◎ <strong>妄</strong> <span class="dicpy">wàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wàng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 虚妄，极不真实 <span class="encs">[unreal]</span></p><p><span class="diczx1">妄，虚妄。——<span class="smcs">《广韵》</span></span></p><p><span class="diczx1">此言妄也。——<span class="smcs">《论衡·问孔》</span></span></p><p><span class="diczx1">故得妄誉。——<span class="smcs">《大戴礼记·文王官人》</span></span></p><p><span class="diczx1">妄见之也。——</span><span class="diczx2">汉</span>· <span class="diczx2">王充</span><span class="smcs">《论衡·订鬼篇》</span></p><p><span class="cino">(2)</span> 又如:妄誉(不实的赞美);妄诞(虚妄不实);妄执(虚妄的执念)</p><p><span class="cino">(3)</span> 不法，胡作非为 <span class="encs">[illegally]</span></p><p><span class="diczx1">彼好专利而妄。——<span class="smcs">《左传·哀公二十五年》</span></span></p><p><span class="diczx1">若再妄求，恐非后福。——<span class="smcs">《梼杌闲评——明珠缘》</span></span></p><p><span class="cino">(4)</span> 又如:妄诛(不当责罚而责罚;滥杀);妄杀(乱杀);妄作(无知而任意胡为);妄进(非分图进)</p><p><span class="cino">(5)</span> 狂乱，毫无约束 <span class="encs">[rash;unrestrained]</span></p><p><span class="diczx1">性悖妄。——</span><span class="diczx2">清</span>· <span class="diczx2">纪昀</span><span class="smcs">《阅微草堂笔记》</span></p><p><span class="cino">(6)</span> 又如:妄男子(狂妄无知的男子)</p><p><span class="cino">(7)</span> 荒谬，荒诞 <span
