                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>雄</strong></p><p><span class="dicpy">xióng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xióng.mp3"></a></span>  ㄒㄩㄥˊ</span></p><ol><li>阳性的，与“<span class="diczx3">雌</span>”相对：～性。～鸡。～狮。～蕊。～蜂。</li><li>强有力的：～壮。～健。～伟。～厚。～浑。～劲。～奇。～踞。～视。～姿。～心。～关。～图。～辩。～才大略。</li><li>强有力的人或国家：～杰（ａ．才能出众的人；ｂ．才能出众）。～俊。英～。枭～。奸～。称～。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>雄</strong> <span class="dicpy">xióng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xióng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从隹( <span class="dicpy">zhuī</span>)，厷( <span class="dicpy">gōng</span>)声。从隹，与鸟有关。本义:公鸟)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[male bird]</span></p><p><span class="diczx1">雄，鸟父也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">谁知鸟之雌雄。——<span class="smcs">《诗·小雅·正月》</span></span></p><p><span class="diczx1">雄雉于飞。——<span class="smcs">《诗·邶风·雄雉》</span></span></p><p><span class="diczx1">但见悲鸟号古木，雄飞雌从绕林间。——</span><span class="diczx2">李白</span><span class="smcs">《蜀道难》</span></p><p><span class="cino">(3)</span> 又如:雄父(公鸡)</p><p><span class="cino">(4)</span> 泛指动植物能产生精细胞的 <span class="encs">[male]</span></p><p><span class="diczx1">雄狐绥绥。——<span class="smcs">《诗·齐风·南山》</span></span></p><p><span class="diczx1">走亦曰雄。——<span class="smcs">《周礼·庖人》</span>注</span></p><p><span class="diczx1">双兔傍地走，安能辨我是雄雌。——<span class="smcs">《木兰诗》</span></span></p><p><span class="diczx1">雄兔脚扑朔。——<span class="smcs">《乐府诗集·木兰诗》</span></span></p><p><span class="diczx1">剑有雄雌。——</span><span class="diczx2">晋</span>· <span class="diczx2">干宝</span><span class="smcs">《搜神记》</span></p><p><span class="diczx1">一雄一雌</span></p><p><span class="cino">(5)</span> 又如:雄鸟;雄蕊;雄花;雄孤;雄猿;雄竹</p><p><span class="cino">(6)</span> 指男子 <span class="encs">[man]</span></p><p><span class="diczx2">淳</span>死之后，新妇遗腹得雄，便以为家门之幸。—— <span class="diczx2">明</span>· <span class="diczx2">夏完淳</span><span class="smcs">《狱中上母书》</span></p><p><span class="cino">(7)</span> 指为首者，居前列者 <span class="encs">[chieftain;head;chief]</span></p><p><span class="diczx1">有夫出征，而丧其雄。——<span class="smcs">《左传·襄公十年》</span></span></p><p><span class="cino">(8)</span> 杰出的人物 <span class="encs">[heroes]</span></p><p><span class="diczx2">刘备</span>，天下枭雄。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(9)</span> 又如:雄长(称雄，称霸);雄豪</p><p><span class="cino">(10)</span> 强有力的国家 <span class="encs">[powerful state]</span></p><p><span class="diczx1">今数雄已灭。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="diczx1">绝世之雄。——</span><span class="diczx2">清</span>· <span class="diczx2">梁启超</span><span class="diczx2">汉王</span>挑战决雌雄。——<span class="smcs">《史记》</span></p><p><span class="cino">(13)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>雄</strong> <span class="dicpy">xióng</span> <span
