						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>伤</strong></p><p><strong>（<span class="diczx3">傷</span>）</strong></p><p><span class="dicpy">shāng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shāng.mp3"></a></span> ㄕㄤˉ</span></p><ol><li>受损坏的地方：工～。～痕。</li><li>损害：～了筋骨。～脑筋。～神。劳民～财。</li><li>因某种致病因素而得病：～风。～寒。</li><li>因过度而感到厌烦：～食。</li><li>妨碍：无～大体。</li><li>悲哀：悲～。哀～。神～。感～。～悼。～逝（悲伤地怀念去世的人）。</li><li>得罪：～众。开口～人。</li></ol>

                    <h2 class="h2_entry"><span class="orth">伤</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>伤</strong></p><p>傷 <span class="dicpy">shāng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shāng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从人，塲( <span class="dicpy">shāng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shāng.mp3"></a></span> )省声。本义:皮肉破损处;创伤)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[wound;injury;trauma]</span></p><p><span class="diczx1">伤，创也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">命理瞻伤。——<span class="smcs">《礼记·月令》</span>。注:<span class="yhcs">“创之浅者曰伤。”</span></span></p><p><span class="diczx1">以杙抉其伤。——<span class="smcs">《左传·襄公十七年》</span></span></p><p><span class="diczx1">无面伤。——<span class="smcs">《左传·哀公元年》</span></span></p><p><span class="diczx1">匠石运斤成风，听而斵之，尽垩而鼻不伤。——<span class="smcs">《庄子·徐无鬼》</span></span></p><p><span class="diczx1">咶其叶，则口烂而受伤。——<span class="smcs">《庄子·人间世》</span></span></p><p><span class="diczx1">则援剑戟而逐之，不避死伤。——<span class="smcs">《荀子·正论》</span></span></p><p><span class="diczx1">其伤于缚者，即幸留，病数月乃瘳。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">城中死伤日积，巷哭声相闻。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="cino">(3)</span> 又如:枪伤;两虎相斗，必有一伤;负伤;轻伤;重伤;遍体鳞伤;死伤;刀伤;烧伤，灼伤;烫伤;冻伤</p><p><span class="cino">(4)</span> 丧事;丧祭 <span class="encs">[funeral arrangements;mourning]</span></p><p><span class="diczx1">是故明君饰食饮吊伤之礼。——<span class="smcs">《管子·君臣下》</span></span></p><p><span class="cino">(5)</span> 又如:伤辞 (古时为相识者死亡而作的哀辞)</p><p><span class="cino">(6)</span> 损失 <span class="encs">[loss]</span></p><p><span class="diczx1">天下得其地，则其利少;攻其国，则其伤大。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="cino">(7)</span> 受伤的人 <span class="encs">[the wounded]</span></p><p><span class="diczx1">虏救死扶伤不给 。——</span><span class="diczx2">司马迁</span><span class="smcs">《报任安书》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>伤</strong></p><p>傷 <span class="dicpy">shāng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shāng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 损伤;伤害 <span class="encs">[injure;hurt]</span></p><p><span class="diczx2">川壅</span>而溃，伤人必多。——<span class="smcs">《国语·周语》</span></p><p><span class="diczx1">杀人者死，伤人及盗抵罪。——<span
