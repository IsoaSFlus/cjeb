						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>奢</strong></p><p><span class="dicpy">shē  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shē.mp3"></a></span> ㄕㄜˉ</span></p><ol>    <li>用钱没有节制，过分享受：～侈。～靡。穷～极欲。</li>    <li>过分的：～盼。～求。～望。</li>    <li>夸张：～言。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>奢</strong> <span class="dicpy">shá</span></p><p>〈名〉</p><p><span class="cino">(1)</span> 姓</p><p><span class="diczx1">明代有奢崇明</span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">shē</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shē.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>奢</strong> <span class="dicpy">shē</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shē.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从大，者声。本义: 奢侈，不节俭)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[luxurious;extravagant]</span></p><p><span class="diczx1">奢，张也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">徐灏</span>曰:<span class="yhcs">“奢者侈靡放纵之义。故曰‘张’，言其张大也。”</span></p><p><span class="diczx1">奢言淫乐而显侈靡。——</span><span class="diczx2">司马相如</span><span class="smcs">《子虚赋》</span></p><p><span class="diczx1">礼，与其奢也，宁俭。——<span class="smcs">《论语·八佾》</span></span></p><p><span class="diczx1">常以俭得之，以奢失之。——<span class="smcs">《韩非子·十过》</span></span></p><p><span class="diczx1">反俭曰奢，言夸大于人也。——<span class="smcs">《太平御览》</span>引<span class="smcs">《说文》</span></span></p><p><span class="diczx2">秦</span>爱纷奢，人亦念其家。—— <span class="diczx2">唐</span>· <span class="diczx2">杜牧</span><span class="smcs">《阿房宫赋》</span></p><p><span class="diczx1">时国王骄奢，不遵典宪。——<span class="smcs">《后汉书·张衡传》</span></span></p><p><span class="cino">(3)</span> 又如:奢尚(奢侈的风尚);奢欲(奢侈的欲望);奢绮(奢侈华丽);奢薄(风俗奢侈，人情，浇薄);奢阔(奢侈阔绰);奢愿(过分的愿望);奢溢(奢侈过制);奢僭(奢侈逾礼);奢佚(骄奢放纵)</p><p><span class="cino">(4)</span> 过分 <span class="encs">[excessive]</span></p><p><span class="diczx1">欲起一台，以为奢费而不作。——<span class="smcs">《潜夫论·俘侈》</span></span></p><p><span class="diczx1">摘鲜焙芳旋封裹，至精至好且不奢。——</span><span class="diczx2">唐</span>· <span class="diczx2">卢仝</span><span class="smcs">《走笔谢孟谏议寄新茶》</span></p><p><span class="cino">(5)</span> 出色;美好 <span class="encs">[fine]</span></p><p><span class="diczx1">玉面添娇舞态奢。——</span><span class="diczx2">唐</span>· <span class="diczx2">刘禹锡</span><span class="smcs">《和乐天柘枝》</span></p><p><span class="cino">(6)</span> 又如:奢遮煞(好到顶点;充其量)</p><p><strong>词性变化</strong></p><p>◎ <strong>奢</strong> <span class="dicpy">shē</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shē.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 夸张，夸大 <span class="encs">[exaggerate;magnify]</span></p><p><span class="diczx2">光</span>夫人 <span class="diczx2">显</span>改 <span class="diczx2">光</span>生的时所造茔制，而更奢大之。——<span class="smcs">《汉书·宣帝纪》</span></p><p><span class="diczx1">故夫夸目者尚奢，惬心者贵当。——</span><span class="diczx2">陆机</span><span class="smcs">《文赋》</span></p><p><span class="cino">(2)</span> 胜过 <span class="encs">[surpass]</span></p><p><span class="diczx1">彼肆人之男女，丽美奢乎</span><span class="diczx2">许</span>史。——<span class="smcs">《文选·张衡·西京赋》</span></p><p><span class="cino">(3)</span> 另见 <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/5962.svg">						  <p>【廣韻】式車切【集韻】【韻會】詩車切【正韻】詩遮切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音賖。【說文】張也。【司馬相如·子虛賦】盛推雲夢以爲高奢。</p><p>又侈也。【毛詩序】蜉蝣之詩，刺奢也。【杜牧之·阿房宮賦】秦愛紛奢。</p><p>又美人名。【荀子·賦篇】閭娵子奢，莫之媒也。</p><p>又世謂媼壻曰阿奢。【通鑑】竇懷貞再娶韋后乳嫗爲妻，奏請，輒自署皇后阿奢不慙。</p><p>又西竺譽人曰蘭奢。【朱子·語錄】王導爲相，只周旋人過一生，嘗坐客二十許人，逐一稱讚，獨不及西僧，徐謂僧曰蘭奢。</p><p>又姓。奢比，黃帝七輔之一。【路史】奢比辨乎東，以爲土師。【三才圖會】作奢北。【國名記】有奢北國。</p><p>又【陸機·感丘賦】或披褐以敦儉兮，或侯服以崇奢。或延祚于黃耉兮，或喪志于札瘥。○按奢瘥俱六麻韻，亦在歌韻。《正字通》沿《字彙》之誤，將奢轉入歌韻，謬。</p><p>又神名。【山海經】奢比之尸在其北。</p><p>又或作奓。詳奓字註。</p><p>又食遮切，音蛇。苗姓。明萬曆閒有奢崇明，奢進。</p><p>
