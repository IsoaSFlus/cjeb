                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>恩</strong> <span class="dicpy">ēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ēn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从心，因声。本义:恩惠)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[美favor;benevolence]</span></p><p><span class="diczx1">恩，惠也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">恩者，仁也。——<span class="smcs">《礼记·丧服四制》</span></span></p><p><span class="diczx1">如此，则上下之恩结矣。——<span class="smcs">《韩非子·用人》</span></span></p><p><span class="diczx1">夫人有恩而背之，不祥莫大焉。——</span><span class="diczx2">明</span>· <span class="diczx2">马中锡</span><span class="smcs">《中山狼传》</span></p><p><span class="cino">(3)</span> 又</p><p><span class="diczx1">先生之恩，生死而骨肉也。</span></p><p><span class="cino">(4)</span> 又如:忘恩;报恩;恩润(恩惠德泽);恩义(恩情与道义);恩分(恩惠情分);恩惠教化。古代多指官方对人民而言;恩田(佛家田。三福之一。指父母)</p><p><span class="cino">(5)</span> 情谊 <span class="encs">[friendship]</span></p><p><span class="diczx1">大丈夫处世，遇知己之主，外托君臣之义，内结骨肉之恩，言必行，计必从。——<span class="smcs">《三国演义》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>恩</strong> <span class="dicpy">ēn</span> <span class="ptr"><a between husband and wife;benevolence]</span></p><p><span class="diczx1">明主知之，故不养恩爱之心而增威严之势。——<span class="smcs">《韩非子·六反》</span></span></p><p><span class="cino">(2)</span> 又如:恩客(妓女对特别亲爱的客人的称呼);恩意(恩爱的心意);恩狎(宠爱亲热);恩慈(宠爱慈惠)</p><p>◎ <strong>恩</strong> <span class="dicpy">ēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ēn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 恩赐，加恩 <span class="encs">[bestow (favors，charity，etc.)]</span></p><p><span class="diczx2">傅太后</span>恩 <span class="diczx2">赵太后</span>。——<span class="smcs">《汉书·外戚传》</span></p><p><span class="diczx1">彼不我恩(加恩，优待)也。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《童区寄传》</span></p><p><span class="cino">(2)</span> 又如:恩赍(恩赐的财物);恩台(老百姓对地方官的尊称);恩相(对长官的尊称);恩家(奴仆尊称主人);恩主(对施恩于己者的敬称)</p><p><span class="cino">(3)</span> 感恩;感谢 <span class="encs">[feel grateful;be thankful]</span>。如:恩抚(谓抚育之恩);恩环(衔环以报恩);恩女(父女);恩养钱(亦称<span class="yhcs">“恩养礼钱”</span>。出卖儿女时买方所付钱财的婉称，也指对曾收养自己儿女的人的报酬)</p><p><span class="cino">(4)</span> 爱 <span
