						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>构</strong></p><p><strong>（<span class="diczx3">構</span>）</strong></p><p><span class="dicpy">gòu  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gòu.mp3"></a></span> ㄍㄡˋ</span></p><ol><li>结成，组合，造：～造。～思。～图。虚～。～筑。～想。</li><li>作品：佳～。</li><li>落叶乔木，叶卵形，花淡绿色。木材可制器具，皮为造纸原料。</li></ol>

                    <h2 class="h2_entry"><span class="orth">构</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>构</strong></p><p>構 <span class="dicpy">gòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gòu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。古字为<span class="yhcs">“冓。从木，冓( <span class="dicpy">góu</span>)声。字本作“冓”</span>，金文象屋架两面对构形。本义:架木造屋)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[construct;build]</span></p><p><span class="diczx1">冓，交积材也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">构，盖也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> 字亦作<span class="yhcs">“搆”</span></p><p><span class="diczx1">大搆驾，兴宫室。——<span class="smcs">《淮南子·本经》</span></span></p><p><span class="diczx1">筑土构木。——<span class="smcs">《淮南子·泛论》</span></span></p><p><span class="diczx1">观其结构。——<span class="smcs">《鲁灵光殿赋》</span></span></p><p><span class="diczx1">有圣人作，构木为巢以避群害，而民悦之，使王天下，号之曰有巢氏。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="diczx1">骊山北构而西折，直走咸阳。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜牧</span><span class="smcs">《阿房宫赋》</span></p><p><span class="cino">(4)</span> 又如:构木(架木造物);构天(构造天地);构巢(构木为巢);构厦(营造大厦);构第(营造宅第)</p><p><span class="cino">(5)</span> 构成，造成 <span class="encs">[compose]</span></p><p><span class="diczx1">邪秽在身，怨之所构。——<span class="smcs">《荀子·劝学》</span></span></p><p><span class="diczx2">楚</span>尝与 <span class="diczx2">秦</span>构难，战于 <span class="diczx2">汉中</span>。——<span class="smcs">《史记·张仪列传》</span></p><p><span class="cino">(6)</span> 又如:构难(结成怨仇);构怨(结怨);构愁(结怨，结仇)</p><p><span class="cino">(7)</span> 招致，引起 <span class="encs">[create;cause]</span></p><p><span class="diczx1">相(视)彼泉水，载清载浊。我日构祸，曷云能穀?——<span class="smcs">《诗·小雅·四月》</span></span></p><p><span class="diczx1">吾闻</span><span class="diczx2">秦</span><span class="diczx2">楚</span>构兵。——<span class="smcs">《孟子·告子》</span></p><p><span class="cino">(8)</span> 又如:构乱(叛乱);构战(交战);构兵(交兵;交战);构祸(造成祸乱)</p><p><span class="cino">(9)</span> 诬陷，陷害 <span class="encs">[frame up]</span></p><p><span class="diczx2">宣姜</span>与公子 <span class="diczx2">朔</span>构 <span class="diczx2">急子</span>。——<span class="smcs">《左传·桓公十六年》</span></p><p><span class="cino">(10)</span> 又如:构害(谋害，陷害);构抉(搜寻捏造);构死(诬陷而死)</p><p><span class="cino">(11)</span> 挑拨离间 <span class="encs">[sow dissension]</span></p><p><span class="diczx1">彼实构吾二君，寡君若得而食之，不厌。——<span class="smcs">《左传·僖公三十二年》</span></span></p><p><span class="cino">(12)</span> 又如:构结(勾结串通);构扇(挑衅扇动);构间(挑拨离间);构合(从中挑拨，使彼此结成嫌隙)</p><p><span class="cino">(13)</span> 连结，交合 <span
