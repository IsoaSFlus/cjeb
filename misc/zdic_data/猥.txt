                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>猥</strong></p><p><span class="dicpy">wěi  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span>  ㄨㄟˇ</span></p><ol><li>众，多。</li><li>琐碎烦杂：～滥。～杂。</li><li>苟且：“然窃恨足下不深惟其终始，而～随俗之毁誉也”。</li><li>鄙陋，下流：卑～。～亵。～劣。</li><li>谦辞，犹言辱：“先帝不以臣卑鄙，～自枉屈，三顾臣于草庐之中”。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>猥</strong> <span class="dicpy">wěi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> </p><p>〈名〉</p><p>(形声。从犬，畏声。本义:狗叫声) 同本义 <span class="encs">[bark]</span></p><p><span class="diczx1">猥，犬吠声。——<span class="smcs">《说文》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>猥</strong> <span class="dicpy">wěi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 许多，众多 <span class="encs">[numerous]</span></p><p><span class="diczx1">以人猥计其野。——<span class="smcs">《管子·八观》</span>。注:<span class="yhcs">“众也。”</span></span></p><p><span class="diczx1">山水猥至。——<span class="smcs">《长笛赋》</span>。注:<span class="yhcs">“多也。”</span></span></p><p><span class="cino">(2)</span> 又如:猥多(众多;繁多);猥盛(众多);猥众(众多);猥滥(多而滥);猥集(多而集中);猥闶(高大的建筑物重叠。形容其多)</p><p><span class="cino">(3)</span> 庞杂 <span class="encs">[multifarious]</span>。如:猥细(繁琐);猥役(杂役);猥复(繁复，重复);猥杂(繁杂;杂乱);猥酿(杂乱;鄙陋)</p><p><span class="cino">(4)</span> 鄙陋 <span class="encs">[vulgar]</span>。如:猥衰(非常狼狈);猥人(鄙俗的人);猥士(鄙贱之士);猥下(鄙陋;低下);猥俗(俚俗);猥细(鄙陋卑下)</p><p><span class="cino">(5)</span> 猥亵，淫秽 <span class="encs">[obscene]</span>。如:猥嫚(下流侮慢);猥茸(鄙陋猥琐)</p><p><span class="cino">(6)</span> 卑鄙，可耻，不体面 <span class="encs">[base]</span>。如:猥缩(畏缩，委靡不大方);猥情(懒惰);猥倡(下等的妓女);淫猥(淫秽);猥贱(卑贱;下贱);猥陋(卑鄙浅陋);猥琐龌龊(卑鄙庸俗，使人生厌)</p><p><span class="cino">(7)</span> 大;粗大 <span class="encs">[big]</span>。如:猥大(粗大;壮大)</p><p><span class="cino">(8)</span> 浅薄 <span class="encs">[superficial]</span>。如:猥局(浅陋狭隘)</p><p><span class="cino">(9)</span> 劣质;下等 <span class="encs">[inferior]</span>。如:猥弱(平庸，低下);猥凡(低劣，质差);猥酒(劣质酒);猥官(低级杂吏);猥拙(拙劣)</p><p><span class="cino">(10)</span> 害怕 <span class="encs">[be afraid of;be scared]</span></p><p><span class="diczx1">待要自己赶来擒捉，一来也被打得着实有些狼狈，二来也被这个母大虫打得猥了。他虽前世是个狐精转化，那狐狸毕竟也还怕那老虎。——<span class="smcs">《醒世姻缘传》</span></span></p><p><span class="cino">(11)</span> 谦词，等于说<span class="yhcs">“辱”</span>，指降低身分，用于他人对自己的行动 <span class="encs">[I]</span></p><p><span class="diczx1">猥自枉屈。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p>◎ <strong>猥</strong> <span class="dicpy">wěi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wěi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 假借为<span class="yhcs">“委”</span>。委曲;积聚 <span class="encs">[accumulate]</span></p><p><span class="diczx1">兼受其猥。——<span class="smcs">《汉书·五行志》</span></span></p><p><span class="diczx1">今猥被以大罪，恐其遂畔。——<span class="smcs">《汉书·王莽传》</span></span></p><p><span class="diczx1">科别其条，勿猥勿并。——<span class="smcs">《汉书·董仲舒传》</span></span></p><p><span class="cino">(2)</span>
