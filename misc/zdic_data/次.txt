						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>次</strong></p><p><span class="dicpy">cì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cì.mp3"></a></span>  ㄘˋ</span></p><ol><li>第二：～日。～子。～等。～要。</li><li>质量、品质较差的：～品。～货。</li><li>等第，顺序：～第。～序。名～。</li><li>化学上指酸根或化合物中少含两个氧原子的：～氯酸。</li><li>中间：胸～。</li><li>量词，回：～数（<span class="dicpy">shù</span>）。初～。三番五～。</li><li>旅行所居止之处所：旅～。舟～。～所。</li><li>姓。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>次</strong> <span class="dicpy">cì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从欠，二声。本义:临时驻扎和住宿)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[stop]</span></p><p><span class="diczx1">次，舍也。——<span class="smcs">《礼记·檀弓上》</span>注</span></p><p><span class="diczx2">楚</span>师伐 <span class="diczx2">郑</span>， <span class="diczx2">次</span>于 <span class="diczx2">鱼陵</span>。——<span class="smcs">《左传·襄公十八年》</span></p><p><span class="diczx2">泰伯</span>素服郊次。——<span class="smcs">《左传·僖公三十三年》</span></p><p><span class="diczx2">王</span>次于 <span class="diczx2">河朔</span>。——<span class="smcs">《书·泰誓中》</span></p><p><span class="cino">(3)</span> 指行军在一处停留三宿以上 <span class="encs">[stay for more than three days]</span></p><p><span class="diczx1">初一交战，</span><span class="diczx2">操</span>军不利，引次江北。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(4)</span> 又如:次止(驻止;止宿)</p><p><strong>词性变化</strong></p><p>◎ <strong>次</strong> <span class="dicpy">cì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 第二，居其次 <span class="encs">[the next in order;the second]</span></p><p><span class="diczx1">凡用兵之法，全国为上，破国次之。——<span class="smcs">《孙子·谋攻》</span></span></p><p><span class="cino">(2)</span> 又如:次从(然后;接着);次后(以后;续后);次早(第二天早上)</p><p><span class="cino">(3)</span> 副，贰 <span class="encs">[second;vice]</span>。如:次车(副车);次介(副使);次相(副丞相)</p><p><span class="cino">(4)</span> 属于质量差的 <span class="encs">[subquality]</span>。如:次品</p><p>◎ <strong>次</strong> <span class="dicpy">cì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 顺序，等第 <span class="encs">[sequence;order]</span></p><p><span class="diczx2">陈胜</span>、 <span class="diczx2">吴广</span>皆次当行，为屯长。——<span class="smcs">《史记·陈涉世家》</span></p><p><span 						<div class="mob">【<a href="/zd/kx/?zm=辰集下" target="_blank">辰集下</a>】【<a href="/zd/kx/?zm=辰集下&amp;bs=欠" target="_blank">欠</a>】

                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/6B21.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/26B8F.svg'><span class='k_txt'>𦮏</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/2055E.svg'><span class='k_txt'>𠕞</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/20923.svg'><span class='k_txt'>𠤣</span>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>七四切，音佽。【說文】不前不精也。【徐曰】不前是次於上也，不精是其次也。【周禮·冬官考工記】畫繢之事，靑與白相次也，赤與黑相次也。【左傳·襄二十四年】太上有立德，其次有立功，其次有立言。</p><p>又師止曰次。【左傳·莊三年】凡師一宿爲舍，再宿爲信，過信爲次。【書·泰誓】戊午，王次于河朔。</p><p>又位次。【周禮·春官】大史祭之日，執書以次位常。【疏】謂執行祭祀之書，各居所掌位次也。【左傳·襄二十三年】敬共朝夕，恪居官次。</p><p>又次舍。【周禮·天官】宮正，以時比宮中之官府、次舍之衆寡。【鄭註】次諸吏直宿，若今之部署諸廬者，舍其所居寺。【又】宮伯，授八次八舍之職事。【註】鄭司農云：庶子衞王宮，在內爲次，在外爲舍。</p><p>又凡舍皆曰次。【左傳·襄二十六年】師陳焚次。【杜註】次，舍也。焚舍，示必死。</p><p>又安行旅之處爲旅次。【易·旅二爻】旅卽次。</p><p>又處也。【魯語】五<img class='lazy k_img' data-original='//img.zdic.net/song/cn/206EC.svg'><span class='k_txt'>𠛬</span>三次。【註】次，處也。三次，謂朝、野、市。</p><p>又張幄於所止之處亦曰次。【周禮·天官】掌次，朝日祀五帝，則張大次小次。【鄭註】次謂幄也。大幄，初往所止居也。小幄，謂接祭退俟之處。</p><p>又【儀禮·士冠禮】賔就次。【鄭註】次，門外更衣處。必帷幕簟席爲之。</p><p>又市亭也。【周禮·地官】司市，于思次以令市，而聽大治大訟。涖于介次，而聽小治小訟。【註】思次，若今市亭也。介次，市亭之屬。別小者也。鄭司農云：次，市中<img class='lazy k_img' data-original='//img.zdic.net/song/cn/202EB.svg'><span class='k_txt'>𠋫</span>樓。</p><p>又星之躔舍爲次。【禮·月令】日窮于次。【註】次，舍也。正義曰：謂去年季冬，日次於<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>枵，從此以來，每引移次他辰，至此月窮盡，還次<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>枵，故云日窮於次。</p><p>又天有十二次，地有十二辰。次之與辰，上下相值。如星紀在丑，斗牛之次。<img class='lazy k_img' data-original='//img.zdic.net/song/cn/248E5.svg'><span class='k_txt'>𤣥</span>枵在子，虛危之次。</p><p>又胷中曰胷次。【莊子·田子方】喜怒哀樂，不入於胷次。【註】次，中也。</p><p>又席閒曰席次。【孔稚圭·北山移文】眉軒席次。</p><p>又至也。【史記·酷吏杜周傳】內深次骨。【註】李奇曰：其用罪，深刻至骨。</p><p>又造次，猶言草次。急遽貌。【論語】造次必於是。【前漢·河閒獻王傳】造次必於儒者。</p><p>又編髮爲首飾之名。【儀禮·士昏禮】女次純衣纁袡。【註】次，首飾，今時髲也。【疏】《周禮》追師掌后之首服副編次。言次第髮長短爲之，如髲鬄也。別作䯸。</p><p>又雞次，楚典名。【戰國策】蒙穀獻雞次之典而百官治。</p><p>又水名，在高平。【水經注】若水與石門水合，水有五原，東水導源高平縣西八十里，西北流，次水注之。</p><p>又諸次，山名。亦水名。【山海經】有大次山，小次山。</p><p>又諸次之山，諸次之水出焉，東流注于河，是山多木無草。</p><p>又居次，匈奴女號，若漢公主。【前漢·常惠傳】獲單于父行及嫂居次。【匈奴傳】王昭君長女爲須卜居次，小女爲當于居次。</p><p>又姓。【呂氏春秋】荆有勇士次非，亦作佽。</p><p>又【集韻】資四切，音恣。楡次，地名。【廣輿記】楡次，縣屬太原府。</p><p>又【集韻】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>津私切，音咨。次且，欲前不前也。【易·夬卦】其行次且。《易經考異》作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/28025.svg'><span class='k_txt'>𨀥</span>跙，王、鄭、馬皆作趦趄。</p><p>又湑次，漢縣名，在武威郡。【孟康曰】次音咨。</p><p>又【集韻】才資切，音慈。具次，山名，通作茨。</p><p>又【楚辭·九歎】今余邦之橫陷兮，宗鬼神之無次。閔先嗣之中絕兮，心惶惑而息悲。次亦叶慈。</p><p>
