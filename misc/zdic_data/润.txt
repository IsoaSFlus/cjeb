						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>润</strong></p><p><strong>（<span class="diczx3">潤</span>）</strong></p><p><span class="dicpy">rùn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rùn.mp3"></a></span> ㄖㄨㄣˋ</span></p><ol><li>不干枯，湿燥适中：湿～。～泽。滋～。</li><li>加油或水使不干枯：～肠。～滑。浸～。</li><li>细腻光滑：光～。滑～。珠圆玉～。</li><li>使有光泽，修饰：～饰。～色。</li><li>利益：利～。分～。</li><li>以财物酬人：～笔。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>润</strong></p><p>潤 <span class="dicpy">rùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rùn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水， 闰( <span class="dicpy">rùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rùn.mp3"></a></span> )声。本义:雨水下流，滋润万物)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[moisten]</span></p><p><span class="diczx1">润，水曰润下。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">润，渍也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">润之以风雨。——<span class="smcs">《易·系辞》</span></span></p><p><span class="diczx1">雨润万物。——<span class="smcs">《论衡·雷虚》</span></span></p><p><span class="diczx1">温润而泽。——<span class="smcs">《礼记·聘义》</span></span></p><p><span class="diczx1">随风潜入夜，润物细无声。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《春夜喜雨》</span></p><p><span class="cino">(3)</span> 又如:润一润喉咙;润下(雨水下以滋润万物);润雨(受雨水滋润);润改(滋润涵养);润滋(滋润)</p><p><span class="cino">(4)</span> 修饰，使有光彩 <span class="encs">[embellish]</span></p><p><span class="diczx1">润，饰也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">必待明圣润色。——<span class="smcs">《汉书·终军传》</span></span></p><p><span class="cino">(5)</span> 又如:润屋(装饰房屋);润正(润色纠正);润笔之绢(为人作诗文书画所得的报酬);润改(润色修改)</p><p><span class="cino">(6)</span> 使得到好处，扶助 <span class="encs">[assist]</span></p><p><span class="diczx1">泽加百姓，功润诸侯。——<span class="smcs">《汉书》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>润</strong></p><p>潤 <span class="dicpy">rùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rùn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 潮湿 <span class="encs">[moist]</span></p><p><span class="diczx1">衣润费垆烟。——</span><span class="diczx2">宋</span>· <span class="diczx2">周邦彦</span><span class="smcs">《满庭芳·风老莺雏》</span></p><p><span class="cino">(2)</span> 又如:润肺爽口(痛快;酣畅);润浸(浸湿;浸透);润溽(湿润);润滋滋(润湿的样子)</p><p><span class="cino">(3)</span> 细腻光滑 <span class="encs">[smooth]</span></p><p><span class="diczx1">嵌岩窦穴，怪奇万状，色泽莹润，亦与它石迥异。——</span><span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《过小孤山大孤山》</span></p><p><span class="cino">(4)</span> 又如:润气(光泽);润腻(光滑细腻;润泽细腻)</p><p>◎ <strong>润</strong></p><p>潤 <span class="dicpy">rùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/rùn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 雨 <span
