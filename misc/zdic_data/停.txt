                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>停</strong> <span class="dicpy">tíng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tíng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从人，亭声。本义:停止，止息)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[stop;cease;halt]</span></p><p><span class="diczx1">停，定也，定于所在也。——<span class="smcs">《释名·释言语》</span></span></p><p><span class="diczx1">潢汙行潦之水。——<span class="smcs">《左传·隐公三年》</span>。注:<span class="yhcs">“潢汙，停水。”</span></span></p><p><span class="diczx1">停杯投箸不能食。——</span><span class="diczx2">唐</span>· <span class="diczx2">李白</span><span class="smcs">《行路难》</span></p><p><span class="diczx1">琵琶声停欲语迟。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="diczx1">余亦停弗上。——<span class="smcs">《徐霞客游记·游黄山记》</span></span></p><p><span class="diczx1">时方冬停遣。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(3)</span> 又如:停床(指亡故);停市(停止交易);停刑(停止执行刑罚);停步(停止前进)</p><p><span class="cino">(4)</span> 停留，暂时居住 <span class="encs">[stay]</span></p><p><span class="diczx1">停数日。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《桃花源记》</span></p><p><span class="diczx1">停于大室之隅。——</span><span class="diczx2">唐</span>· <span class="diczx2">李朝威</span><span class="smcs">《柳毅传》</span></p><p><span class="cino">(5)</span> 又如:停留长智(耽搁久了，自会想出主意来);停伫(停留伫立);停逗(停留);停眠(停留住宿);停待(等待;停留);停缓(停留缓行);停蓄(停留蓄积);停积(停留蓄积)</p><p><span class="cino">(6)</span> 停放;停泊 <span class="encs">[be parked;berth]</span></p><p><span class="diczx1">停车坐爱枫林晚，霜叶红于二月花。——</span><span class="diczx2">杜牧</span><span class="smcs">《山行》</span></p><p><span class="diczx1">停船暂借问，或恐是同乡。——</span><span class="diczx2">崔颢</span><span class="smcs">《长干行》</span></p><p><span class="cino">(7)</span> 又如:停丧挂孝(停棺在家等待吊丧，悬挂孝幛，戴孝尽礼);停柩(停灵。停放灵柩或在埋葬前暂时停放)</p><p><span class="cino">(8)</span> 蓄积 <span class="encs">[stagnate]</span>。</p><p><span class="cino">(9)</span> 如:停蓄(停积);停食(积食);停浸(聚积);停涵(停蓄沉浸);停塌(屯积);停潦(积水)</p><p><span class="cino">(10)</span> 抚育 <span class="encs">[nourish]</span>。</p><p><span class="cino">(11)</span> 如:停育;停毒(化育;养育)</p><p><span class="cino">(12)</span> 耸立 <span class="encs">[tower]</span>。</p><p><span class="cino">(13)</span> 如:停停(耸立貌;高貌)</p><p><span class="cino">(14)</span> 停滞 <span class="encs">[stagnant]</span>。</p><p><span class="cino">(15)</span> 如:停阴(集结不散的阴云);停云(停止不动的云);停云落月(在信中表示对友人的思慕)</p><p><strong>词性变化</strong></p><p>◎ <strong>停</strong> <span class="dicpy">tíng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tíng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 成数，总数分成几份，其中一份叫一停 <span class="encs">[portion;part(of a total)]</span>。</p><p><span
