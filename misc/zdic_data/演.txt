                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>演</strong></p><p><span class="dicpy">yǎn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yǎn.mp3"></a></span>  ㄧㄢˇ</span></p><ol><li>根据事理推广发挥：推～。讲～（亦作“演讲”）。～义（以史书及传说的材料为基础，增添一些细节，用章回体写成的小说）。～绎（一种推理方法，由一般原理推出关于特殊情况下的结论）。</li><li>依照程式练习：～练。～示。～算。～武。～习。～兵场。</li><li>不断变化：～变。～化。</li><li>把技艺当众表现出来：表～。～出。～播。～奏。主～。义～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">演</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>演</strong> <span class="dicpy">yǎn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yǎn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从水，寅声。本义:水长流)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[flow]</span></p><p><span class="diczx1">演，长流也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">南国风光当世少，</span><span class="diczx2">西陵</span>演浪过 <span class="diczx2">江</span>难。—— <span class="diczx2">李白</span><span class="smcs">《代封人寄翁参枢先輩》</span></p><p><span class="diczx1">子未闻</span><span class="diczx2">杨都</span>之巨伟也，左沧海，右 <span class="diczx2">岷山</span>，龟鸟津其落， <span class="diczx2">江</span><span class="diczx2">汉</span>演其源。——<span class="smcs">《艺文类聚》</span></p><p><span class="cino">(3)</span> 水流经地下 <span class="encs">[underflow]</span></p><p><span class="diczx1">演以</span><span class="diczx2">潜沫</span>，浸以 <span class="diczx2">绵雒</span>。——<span class="smcs">《文选·左思·蜀都赋》</span>。 <span class="diczx2">刘逵</span>注:<span class="yhcs">“水潜行曰演。此二水伏流，故曰‘演以潜沫’。”</span></p><p><span class="cino">(4)</span> 延展 <span class="encs">[txtend;probng]</span></p><p><span class="diczx1">演，延也，言蔓延而广也。——<span class="smcs">《释名·释言语》</span></span></p><p><span class="diczx1">留侯演成。——</span><span class="diczx2">班固</span><span class="smcs">《西都赋》</span></p><p><span class="diczx1">有所摇演。——</span><span class="diczx2">马融</span><span class="smcs">《长笛赋》</span></p><p><span class="cino">(5)</span> 又如:演延(绵延，广远);演迤(绵延不绝貌);演溢(蔓延满溢)</p><p><span class="cino">(6)</span> 湿润 <span class="encs">[moisten]</span></p><p><span class="diczx1">夫水土演而民用也。——<span class="smcs">《国语·周语》</span></span></p><p><span class="cino">(7)</span> 演变 <span class="encs">[evolve]</span>。如:演展(演变发展);演递(演变发展)</p><p><span class="cino">(8)</span> 通<span class="yhcs">“衍”</span>。推演 <span class="encs">[deduce]</span></p><p><span class="diczx1">于是敷演旧章，奏而改焉。——<span class="smcs">《三国志·高堂隆传》</span></span></p><p><span class="diczx1">敷演皇极，流化万里。——<span class="smcs">《三国志·胡综传》</span></span></p><p><span class="cino">(9)</span> 又如:演禽星(占卜的一种，以星禽推知人事吉凶);演证(推演证明);演绎法(即演绎推理)</p><p><span class="cino">(10)</span> 表演 <span class="encs">[perform]</span>。如:演梨园(演戏);演述(表演叙述)</p><p><span class="cino">(11)</span> 练习 <span class="encs">[drill;practise]</span>。如:演兵场(练兵的场所);演武场;演武厅(练武的场所或厅堂)</p><p><span class="cino">(12)</span> 传布 <span class="encs">[disseminate]</span></p><p><span class="diczx1">主公，若演真经，须寻佛地。——<span class="smcs">《西游记》</span></span></p><p><span class="cino">(13)</span> 又如:演迤(流布;流传);演法(谓传布教义的方式);演教(传布教化或指传布佛教教义);演迪(谓继承推广)</p><p><span class="cino">(14)</span> 阐述，讲解 <span
