                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>衷</strong> <span class="dicpy">zhōng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhōng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从衣。中声。本义:贴身的内衣)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[underwear]</span></p><p><span class="diczx1">衷，里亵衣。——<span class="smcs">《说文》</span>。</span><span class="diczx2">段玉裁</span>注:<span class="yhcs">“亵衣有在外者，衷则在内者也。”</span></p><p><span class="cino">(3)</span> 又如:衷衣(里衣，内衣);衷服(贴身内衣)</p><p><span class="cino">(4)</span> 通<span class="yhcs">“中”</span>。内心 <span class="encs">[heart]</span></p><p><span class="diczx1">汝不察吾衷。——</span><span class="diczx2">清</span>· <span class="diczx2">林觉民</span><span class="smcs">《与妻书》</span></p><p><span class="cino">(5)</span> 又如:衷怀歉仄(心里感到内疚抱歉);衷言(心里话);衷素(内心真情)</p><p><span class="cino">(6)</span> 中心，中央 <span class="encs">[centre]</span></p><p><span class="diczx1">佩，衷之旗也。——<span class="smcs">《左传·闵公二年》</span></span></p><p><span class="diczx1">发命之不衷。——<span class="smcs">《左传·昭公十六年》</span></span></p><p><span class="diczx1">服之不衷。——<span class="smcs">《左传·僖公二十四年》</span></span></p><p><span class="cino">(7)</span> 又如:衷正(中正)</p><p><span class="cino">(8)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>衷</strong> <span class="dicpy">zhōng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhōng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 贴身穿着;穿在里面 <span class="encs">[wear inside]</span></p><p><span class="diczx1">或衷其襦。——<span class="smcs">《谷梁传》</span></span></p><p><span class="diczx2">楚</span>人衷甲。——<span class="smcs">《左传·襄公二十七年》</span></p><p><span class="diczx1">今人服公裳必衷以背子。——</span><span class="diczx2">宋</span>· <span class="diczx2">程大昌</span><span class="smcs">《演繁露·背子中禅》</span></p><p><span class="cino">(2)</span> 又如:衷襦(衬在里面的短衣);衷甲(在衣服里面穿铠甲)</p><p><span class="cino">(3)</span> 引申为包围 <span class="encs">[surround;encircle]</span></p><p><span class="diczx1">衷戎师，前后击之，尽殪。——<span class="smcs">《左传》</span></span></p><p>◎ <strong>衷</strong> <span class="dicpy">zhōng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhōng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 正;正派 <span class="encs">[right;upright]</span></p><p><span class="diczx2">楚</span>辟我衷。——<span class="smcs">《左传·昭公六年》</span>。注:<span class="yhcs">“正也。”</span></p><p><span class="diczx1">发命之不衷。——<span class="smcs">《左传·昭公十六年》</span>。注:<span class="yhcs">“当也。”</span></span></p><p><span class="diczx1">史馆取为衷据。——</span><span class="diczx2">清</span>· <span class="diczx2">章学诚</span><span class="smcs">《邵与桐别传》</span></p><p><span class="cino">(2)</span> 又如:衷据(正确的根据)</p><p><span class="cino">(3)</span> 善;福 <span class="encs">[good;happy]</span></p><p><span class="diczx1">今天降衷于</span><span class="diczx2">吴</span>。——<span class="smcs">《国语·吴语》</span></p><p><span class="diczx1">和衷哉。——<span class="smcs">《书·皋谟》</span>。传:<span class="yhcs">“善也。”</span></span></p><p><span class="cino">(4)</span> 忠诚 <span class="encs">[loyal faithful]</span></p><p><span class="diczx1">欲对衷言不从。——<span class="smcs">《荀子·成相》</span>。注:<span class="yhcs">“诚也。”</span></span></p><p><span class="diczx1">不从命乃衷。——<span class="smcs">《荀子·子道》</span></span></p><p><span class="cino">(5)</span> 又如:衷愚(愚忠。尽忠而不明事理。多用作臣子上言于帝王的自谦之词);衷赤(赤诚之心)</p><p><span class="cino">(6)</span> 另见 <span
