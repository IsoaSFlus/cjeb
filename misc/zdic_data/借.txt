                    <h2 class="h2_entry"><span class="orth">借</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>借</strong> <span class="dicpy">jiè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从人，昔声。本义:借，借进，借出)</p><p><span class="cino">(2)</span> 借进 <span class="encs">[borrow]</span></p><p><span class="diczx1">借，假也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">朱骏声</span>曰:<span class="yhcs">“即藉字之转注。古只作藉。”</span></p><p><span class="diczx1">借，假借也。——<span class="smcs">《广韵》</span></span></p><p><span class="diczx1">借无不给。——<span class="smcs">《晋书·阮裕传》</span></span></p><p><span class="diczx1">借旁近与之。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《伤仲永》</span></p><p><span class="diczx2">允修</span>借书。—— <span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《黄生借书说》</span></p><p><span class="diczx1">书非借不能读。</span></p><p><span class="diczx1">借者之用心。</span></p><p><span class="diczx1">其借书亦类予。</span></p><p><span class="cino">(3)</span> 又如:求借(请求别人借给);挪借(暂时借用别人的钱);借当(借人东西去当铺当钱);借当头(借将要拿去典当的东西);借典(借，借用)</p><p><span class="cino">(4)</span> 借出 <span class="encs">[lend]</span></p><p><span class="diczx1">有马者，借人乘之。——<span class="smcs">《论语·卫灵公》</span></span></p><p><span class="diczx1">权势不可以借人。——<span class="smcs">《韩非子·内储说下》</span></span></p><p><span class="cino">(5)</span> 又如:他借书给我们;借给他几块钱;借一肩儿(挑担者让别人替自己挑一会，自己可以休息一下);借借(借出);借赁(租借)</p><p><span class="cino">(6)</span> 帮助 <span class="encs">[help]</span></p><p><span class="diczx1">水天空阔，恨东风，不借世间英物。——</span><span class="diczx2">文天祥</span><span class="smcs">《念奴娇》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>借</strong></p><p>藉 <span class="dicpy">jiè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jiè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 假托;借口 <span class="encs">[use as a pretext]</span></p><p><span class="diczx1">若苟有以藉口。——<span class="smcs">《左传·成公二年》</span></span></p><p><span class="cino">(2)</span> 又如:借言(借口，假托的理由);借情(假托某种情况或缘故)</p><p><span class="cino">(3)</span> 凭借;依靠 <span class="encs">[rely on]</span></p><p><span class="diczx1">借子杀之。——<span class="smcs">《墨子·公输》</span></span></p><p><span class="diczx1">其可以借以美名者，以外权重之。——<span class="smcs">《韩非子·孤愤》</span></span></p><p><span class="diczx1">借天降大雨。——<span class="smcs">《广东军务记》</span></span></p><p><span class="diczx1">借寨墙为蔽。——</span><span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p><span class="diczx1">借征入境。——<span class="encs">[英]</span></span><span class="diczx2">赫胥黎</span>著、 <span class="diczx2">严复</span>译<span class="smcs">《天演论》</span></p><p><span class="cino">(4)</span> 又如:借倩(托请);借水行舟(借用别人的力量或条观条件办事);借步檐躲雨(喻指借他人力量庇护自己);借势儿(乘机)</p><p>◎ <strong>借</strong></p><p>藉 <span class="dicpy">jiè</span> <span
