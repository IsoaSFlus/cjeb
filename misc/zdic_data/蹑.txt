                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>蹑</strong></p><p><strong>（<span class="diczx3">躡</span>）</strong></p><p><span class="dicpy">niè  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/niè.mp3"></a></span>  ㄋㄧㄝˋ</span></p><ol><li>踩，踏：“张良、陈平～汉王足”。～足。</li><li>追踪，跟随，轻步行走的样子：追～。～踪。～手～脚。</li></ol>

                    <h2 class="h2_entry"><span class="orth">蹑</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>蹑</strong></p><p>躡 <span class="dicpy">niè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/niè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从足，聂声。本义:踩踏，有意识地踩踏)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[tread]</span></p><p><span class="diczx1">蹑，蹈也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx2">康子</span>履 <span class="diczx2">魏桓子</span>，蹑其踵。——<span class="smcs">《战国策·秦策》</span></p><p><span class="diczx2">张良</span>、 <span class="diczx2">陈平</span>蹑 <span class="diczx2">汉王</span>足。——<span class="smcs">《史记·淮阴侯列传》</span></p><p><span class="diczx1">蹑屐登崖。——</span><span class="diczx2">明</span>· <span class="diczx2">刘基</span><span class="smcs">《诚意伯刘文成公文集》</span></p><p><span class="diczx1">农夫蹑丝履。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="diczx1">足下蹑丝履。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="cino">(3)</span> 又</p><p><span class="diczx1">蹑履相逢迎。</span></p><p><span class="cino">(4)</span> 又如:蹑冻(踏着未融化的冰雪行路);蹑球(踩球);蹑蹬(踩踏石级);蹑级(顺着台阶逐级上下)</p><p><span class="cino">(5)</span> 登 <span class="encs">[climb]</span></p><p><span class="diczx1">世胄蹑高，英俊沉下僚。——</span><span class="diczx2">左思</span><span class="smcs">《咏史》</span></p><p><span class="cino">(6)</span> 又如:飞空蹑壁;蹑级(逐级提拔);蹑空(旧谓得道成仙之人可腾空而行或停留空中);蹑虚(凌空);蹑云(高耸入云;腾云)</p><p><span class="cino">(7)</span> 穿 <span class="encs">[put on]</span></p><p><span class="diczx1">农夫蹑丝履。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="cino">(8)</span> 又如:蹑屐(拖着木屐;穿着木屐);蹑蹈(穿鞋袜)</p><p><span class="cino">(9)</span> 放轻 <span class="encs">[脚步]</span> 悄悄地走。通<span class="yhcs">“捻”</span> <span class="encs">[lighten(one’s step)]</span>。如:他蹑脚儿走了;蹑跀(小步;轻步);蹑脚(轻步行走的样子)</p><p><span class="cino">(10)</span> 追随;追赶 <span class="encs">[follow]</span></p><p><span class="diczx1">蹑迹披求。(跟随蛤蟆的踪迹，披开丛草寻求。蹑，追踪，披，拨开。)——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(11)</span> 又如:蹑景(追赶日影);蹑追(跟踪追击);蹑迹(追踪;跟踪);蹑风(追逐疾风);蹑捕(追捕);蹑逐(追逐);蹑寇(追击敌寇);蹑击(追击)</p><p><span class="cino">(12)</span> 踮起脚尖走路 <span class="encs">[walk on tiptoe]</span>。如:蹑跟(高提脚跟而以脚尖轻步行走)</p><p><span class="cino">(13)</span> 到;前往 <span class="encs">[go to]</span>。如:蹑蹀(往来小步的样子);蹑踱(来回寻找);蹑屣(远行;跋涉;奔波)</p><p><span class="cino">(14)</span> 超越;胜过 <span
