                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>绰</strong></p><p>綽 <span class="dicpy">chāo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/chāo.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 抓取 <span class="encs">[grab]</span></p><p><span class="diczx1">绰枪上马。——<span class="smcs">《水浒传》</span></span></p><p><span class="cino">(2)</span> 又如:绰起一根棍子</p><p><span class="cino">(3)</span> 举起 <span class="encs">[lif up]</span>。如:绰起(立起，竖起)</p><p><span class="cino">(4)</span> 顺;应 <span class="encs">[comply with;conform to]</span>。如:绰经(趁势;顺势)</p><p><span class="cino">(5)</span> 另见 <span class="dicpy">chuò;chuo</span></p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>绰</strong></p><p>綽 <span class="dicpy">chuò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/chuò.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从糸( <span class="dicpy">mì</span>)，卓声。本义:宽大，舒缓)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[ample;spacious;enough]</span></p><p><span class="diczx1">宽兮绰兮。——<span class="smcs">《诗·卫风·淇奥》</span></span></p><p><span class="cino">(3)</span> 又如:绰步(大步);绰板婆(大脚婆娘);绰绰(宽舒裕如)</p><p><span class="cino">(4)</span> 姿态柔美 <span class="encs">[gentle]</span></p><p><span class="diczx1">柔情绰态，媚于语言。——</span><span class="diczx2">曹植</span><span class="smcs">《洛神赋》</span></p><p><span class="cino">(5)</span> 又如:绰态(柔媚多姿);绰俏(漂亮;俊俏)</p><p><span class="cino">(6)</span> 隐隐约约 <span class="encs">[faintly]</span>。如:绰见(瞧见，看见);绰注(琴声自下而上为绰，自上而下为注)</p><p><span class="cino">(7)</span> 端正 <span class="encs">[upright]</span></p><p><span class="diczx1">曾经绰立侍丹墀，绽蕊宫花拂面权。——</span><span class="diczx2">唐</span>· <span class="diczx2">元稹</span><span class="smcs">《酬孝甫见赠诗》</span></p><p><span class="cino">(8)</span> 又如:绰立</p><p><strong>词性变化</strong></p><p>◎ <strong>绰</strong></p><p>綽 <span class="dicpy">chuò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
