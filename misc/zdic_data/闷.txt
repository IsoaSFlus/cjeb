						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>闷</strong></p><p><strong>（<span class="diczx3">悶</span>）</strong></p><p><span class="dicpy">mèn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mèn.mp3"></a></span> ㄇㄣˋ</span></p><ol><li>心烦，不舒畅：愁～。沉～。郁～。～懑。～～不乐。</li><li>密闭，不透气：～子车。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>闷</strong></p><p><strong>（<span class="diczx3">悶</span>）</strong></p><p><span class="dicpy">mēn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mēn.mp3"></a></span> ㄇㄣˉ</span></p><ol><li>因空气不流通而引起的感觉：～气。～热。</li><li>密闭，使不透气：茶刚沏上，～会儿再喝。</li><li>不吭声，不声张：他只是～头苦干。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>闷</strong></p><p>悶 <span class="dicpy">mēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mēn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 气闷，因气不通畅而引起的不快之感 <span class="encs">[stuffy;stifling;air is close]</span></p><p><span class="diczx1">闭则热而闷。——<span class="smcs">《素问·风论》</span></span></p><p><span class="cino">(2)</span> 又如:闷吐(心闷想吐);闷绝(晕倒。一说窒息而死);闷痒(因身体不舒爽而发痒);闷窒(窒息);闷躁(气闷烦躁)</p><p><span class="cino">(3)</span> 沉默不出声或声音不响的样子 <span class="encs">[(of a sound)muffled]</span></p><p><span class="diczx1">闷然而后应。——<span class="smcs">《庄子·德充符》</span></span></p><p><span class="cino">(4)</span> 又如:闷默(静默不出声);闷闷(闷声不响)</p><p><strong>词性变化</strong></p><p>◎ <strong>闷</strong></p><p>悶 <span class="dicpy">mēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mēn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 密闭，使不透气 <span class="encs">[cover tightly]</span>。如:茶刚泡上，闷一会儿再喝</p><p><span class="cino">(2)</span> 呆在家里不出门 <span class="encs">[shut oneself or sb. indoors]</span>。如:别老闷在家里</p><p><span class="cino">(3)</span> 过饱 <span class="encs">[oversaturate]</span>。如:肉吃多了，闷住了</p><p><span class="cino">(4)</span> 昏闷;失去知觉 <span class="encs">[depressed]</span>。如:闷香(一种麻醉毒品，强盗常用它熏人后强奸或盗物)</p><p><span class="cino">(5)</span> 另见 <span class="dicpy">mèn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/mèn.mp3"></a></span> </p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/闷雷">闷雷</a></span><span class="crefe"><a href="/hans/闷气">闷气</a></span><span class="crefe"><a href="/hans/闷热">闷热</a></span><span class="crefe"><a href="/hans/闷声">闷声</a></span><span class="crefe"><a href="/hans/闷头儿">闷头儿</a></span></div><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>闷</strong></p><p>悶 <span class="dicpy">mèn</span>
