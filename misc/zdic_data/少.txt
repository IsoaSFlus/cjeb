						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>少</strong></p><p><span class="dicpy">shǎo  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǎo.mp3"></a></span> ㄕㄠˇ</span></p><ol>    <li>数量小的，与“<span class="diczx3">多</span>”相对：多～。～量。～许。</li>    <li>缺，不够：缺～。减～。</li>    <li>不经常：～有。～见。</li>    <li>短时间：～等。～候。～顷。</li>    <li>丢，遗失：屋里～了东西。</li>    <li>轻视：“且夫我尝闻～仲尼之闻而轻伯夷之义者，如吾弗信，今我睹子之难穷也”。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>少</strong></p><p><span class="dicpy">shào  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shào.mp3"></a></span> ㄕㄠˋ</span></p><ol>    <li>年纪轻或年轻人：～年。～女。～壮（年轻力壮）。</li>    <li>古代辅佐长官的副职：～傅。～保。</li>    <li>次级的：～校。～将（<span class="dicpy">jiàng </span>）。</li>    <li>姓。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>少</strong> <span class="dicpy">shǎo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǎo.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (会意。小篆从小，丿( <span class="dicpy">piě</span>)声。本义:不多)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[few;little;less]</span></p><p><span class="diczx1">少，不多也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">少，微也。——<span class="smcs">《太玄·玄衡》</span></span></p><p><span class="diczx1">夫少者，多之所贵也。——<span class="smcs">《易·略例》</span></span></p><p><span class="diczx1">宾少进。——<span class="smcs">《仪礼·乡射礼》</span></span></p><p><span class="diczx1">险以远，则至者少。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="diczx1">人民少而财有余。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="cino">(3)</span> 又如:少一时(少时，过一会儿);少可(少，至少;稍好，稍愈);少冗(较忙);少甚么(少甚末，少是末，少甚。有的是，不少，不希罕);少才(小聪明);少年(不几年);少好(少而精);少言寡语(谓平时说话不多);少说(说少一点);少气(气不足)</p><p><span class="cino">(4)</span> 薄弱 <span class="encs">[weak]</span></p><p><span class="diczx1">力少而不畏强。——<span class="smcs">《韩非子》</span></span></p><p><span class="cino">(5)</span> 通<span class="yhcs">“小”</span> <span class="encs">[small]</span></p><p><span class="diczx1">仲春之月，安萌芽，养幼少，存诸孤。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="diczx2">武王</span>崩， <span class="diczx2">成王</span>幼少。——<span class="smcs">《淮南子·汜论》</span></p><p><span class="diczx1">朝廷幼少，恃我而已。——<span class="smcs">《后汉书·王允传》</span></span></p><p><span class="diczx1">尿不能卒出，痛引少腹。——<span class="smcs">《诸病源候论·石淋候》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>少</strong> <span class="dicpy">shǎo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǎo.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 短缺，距原来的要求还差 <span class="encs">[lack;be deficient;be short of]</span></p><p><span class="diczx1">理无专在，而学无止境也，然则问可少耶?——</span><span class="diczx2">清</span>· <span class="diczx2">刘开</span><span class="smcs">《问说》</span></p><p><span class="diczx1">今少一人，愿君即以</span><span
