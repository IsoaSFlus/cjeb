                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>阕</strong></p><p><strong>（<span class="diczx3">闋</span>）</strong></p><p><span class="dicpy">què  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/què.mp3"></a></span>  ㄑㄩㄝˋ</span></p><ol><li>停止，终了：乐～。服～（古代三年之丧满）。</li><li>量词，歌曲或词，一首为一阕；一首词的一段亦称一阕，前一段称“上阕”，后一段称“下阕”。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> close, shut; watch tower</p><p><span class="z_ts2">法语</span> section d'une chanson,fermé</p></div>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>阕</strong></p><p>闋 <span class="dicpy">què</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/què.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从门，癸( <span class="dicpy">guǐ</span>)声。本义:祭事结束而闭门)</p><p><span class="cino">(2)</span> 同本义。引申为止息;终了 <span class="encs">[cease;end]</span></p><p><span class="diczx1">阕，事已闭门也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">瞻彼阕者，虚室生白，吉祥止止。——<span class="smcs">《庄子·人间世》</span></span></p><p><span class="diczx1">繁肴既阕，亦有寒羞。——</span><span class="diczx2">张协</span><span class="smcs">《七命》</span></p><p><span class="cino">(3)</span> 尽;无剩余 <span class="encs">[exhaust]</span></p><p><span class="diczx1">俾民心阕。——<span class="smcs">《诗·小雅·节南山》</span></span></p><p><span class="diczx1">物物卬市，日阙亡储。——<span class="smcs">《汉书》</span>。</span><span class="diczx2">颜师古</span>注:<span class="yhcs">“阕，尽也。日阕，言当日即尽，不蓄积也。”</span></p><p><span class="cino">(4)</span> 旧指服丧期满 <span class="encs">[(mourning term) end]</span></p><p><span class="diczx1">于是三年礼阕，乃相与刊勒金石。——<span class="smcs">《山阳太守祝睦后碑》</span></span></p><p><span class="cino">(5)</span> 乐终<span class="encs">[stop]</span></p><p><span class="diczx1">阕，曲终为阕。——<span class="smcs">《古今韵会举要》</span></span></p><p><span class="diczx1">主人答拜，乐阕。——<span class="smcs">《仪礼》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>阕</strong></p><p>闋 <span class="dicpy">què</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/què.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 乐曲 <span class="encs">[word]</span></p><p><span class="diczx1">歌数阕。——<span class="smcs">《史记·留侯世家》</span></span></p><p><span class="diczx1">曲终阕尽，余弦更兴。——</span><span class="diczx2">汉</span>· <span class="diczx2">马融</span><span class="smcs">《长笛赋》</span></p><p><span class="diczx1">歌曲以阕为称。——</span><span class="diczx2">宋</span>· <span class="diczx2">吴曾</span><span class="smcs">《能改斋漫录》</span></p><p><span class="cino">(2)</span> 乐一遍为一阕 <span class="encs">[turn]</span></p><p><span class="diczx1">昔</span><span class="diczx2">葛天氏</span>之乐，三人操牛尾投足以歌八阕。——<span class="smcs">《吕氏春秋》</span></p><p><span class="cino">(3)</span> 空隙 <span class="encs">[gap]</span></p><p><span class="diczx1">瞻彼阕者，虚室生白。——<span class="smcs">《庄子》</span></span></p><p>◎ <strong>阕</strong></p><p>闋 <span class="dicpy">què</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/què.mp3"></a></span>                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/95CB.svg">						  <p>【唐韻】【集韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>傾雪切，音缺。【說文】事已閉門也。【玉篇】止也。【博雅】闋，訖也。</p><p>又【玉篇】息也。【詩·小雅】君子如居，俾民心闋。【傳】息也。【後漢·楊璇傳】剽人盜邑者，不闋時月。【註】息也。</p><p>又【廣韻】盡也。【前漢·王莽傳】物物卬市，日闋亡儲。【註】盡也，言當日卽盡。【張景陽·七命】繁肴旣闋，亦有寒羞。</p><p>又【玉篇】終也。【增韻】樂終也。【禮·文王世子】有司告以樂闋。【註】闋，終也。【周禮·夏官·大司馬】鼓戒三闋。【前漢·張良傳】歌數闋。</p><p>又空也。【莊子·人閒世】瞻彼闋者，虛室生白。【註】闋，空也。室比喻心，心能空虛，則純白獨生也。</p><p>又【正韻】門開也。</p><p>又【集韻】暌桂切，音䙆。止也。【禮·郊特牲】卒爵而樂闋。【註】闋，止也。</p><p>又【廣韻】【集韻】【韻會】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>苦穴切，音湀。義同。</p><p>又【集韻】古穴切，音玦。馬名。【爾雅·釋畜】回毛在背，闋廣。【疏】旋毛在背者名闋廣。</p><p>又【韻補】叶音餽。【曹植·酒賦】敘嘉賔之歡會兮，惟耽樂之旣闋。日晻暗於桑楡兮，命僕夫而皆逝。</p>
