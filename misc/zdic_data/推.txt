						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>推</strong></p><p><span class="dicpy">tuī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tuī.mp3"></a></span>  ㄊㄨㄟˉ</span></p><ol><li>手抵物体向外或向前用力使物移动：～车。～搡。～拿。～倒。～敲。顺水～舟。</li><li>使用工具向前移动进行工作：～头。～土。</li><li>使事情开展：～广。～行（<span class="dicpy">xíng </span>）。～动。～销。</li><li>进一步想，由已知之点想到其它：～及。～究。～导。～论。～测。～事。</li><li>往后挪动（时间上）：～延。～移。</li><li>辞让，脱卸：～让。～却。～委。～辞。</li><li>举荐，指出某人优点：～许。～重（<span class="dicpy">zhòng </span>）。～举。～崇。</li><li>让出，献出：～恩。～心置腹。</li></ol>

                    <h2 class="h2_entry"><span class="orth">推</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>推</strong> <span class="dicpy">tuī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tuī.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，隹( <span class="dicpy">zhuī</span>)声。本义:手向外用力使物体移动或向前移动)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[push]</span></p><p><span class="diczx1">推，排也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">推，軵也，前也。——<span class="smcs">《仓颉篇》</span></span></p><p><span class="diczx1">若己推而内之沟中。——<span class="smcs">《孟子·万章上》</span></span></p><p><span class="diczx1">苟有险，余必下推车。——<span class="smcs">《左传·成公二年》</span></span></p><p><span class="diczx1">推堕附马河中。——</span><span class="diczx2">明</span>· <span class="diczx2">宋濂</span><span class="smcs">《送东阳马生序》</span></p><p><span class="diczx1">始欲着推字。——</span><span class="diczx2">宋</span>· <span class="diczx2">胡仔</span><span class="smcs">《苕溪渔隐丛话》</span></p><p><span class="diczx1">作推敲之势。</span></p><p><span class="cino">(3)</span> 又如:推毂(推动车轮轴，使车前进);推抢(推拥抢夺，争抢);推沟(推下沟);推行(推动物体向前);推引(推动;推拉);推天抢地(形容乱推乱搡);推转(推出处死)</p><p><span class="cino">(4)</span> 使工具向前移动进行工作 <span class="encs">[push forward]</span>。如:推磨;推豆腐;推米面;推头;推草坪;推子</p><p><span class="cino">(5)</span> 推移，变化、移动和发展。按顺序更换 <span class="encs">[shove]</span></p><p><span class="diczx1">与世推移。——<span class="smcs">《史记·屈原贾生列传》</span></span></p><p><span class="diczx1">推赤心于天下。——</span><span class="diczx2">南朝</span><span class="diczx2">梁</span>· <span class="diczx2">丘迟</span><span class="smcs">《与陈伯之书》</span></p><p><span class="cino">(6)</span> 又如:推革(推移变易);推次(相推以次第);推换(推移变换);推遇(随着遭遇而推移变迁);推涉(推移;更易);推变(推移变化);推荡(推移);推嬗(推移演变)</p><p><span class="cino">(7)</span> 寻求;探索 <span class="encs">[seek]</span></p><p><span class="diczx1">旁推交通。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="cino">(8)</span> 又如:推求;推索(推求寻索);推迹(根据迹象推求);推极(推求穷究);推详(推究详察);推纠(推究纠察);推原(从本原上推究);推案(考查推求);推讨(推究探讨)</p><p><span class="cino">(9)</span> 推行，推广施行 <span class="encs">[force]</span></p><p><span class="diczx1">推其所为。——<span class="smcs">《孟子·梁惠王上》</span></span></p><p><span class="cino">(10)</span> 又如:推销</p><p><span class="cino">(11)</span> 荐举;推选 <span class="encs">[recommend]</span>。如:推士(推举贤才);推功(推举功勋;按照功劳);推借(推许荐引);推致(推荐延请);推补(荐举补缺官员);推擢(推荐提拔)</p><p><span class="cino">(12)</span> 推测;推断 <span class="encs">[infer]</span></p><p><span
