                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>傅</strong> <span class="dicpy">fù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从人，尃( <span class="dicpy">fū</span>)声。本义:辅佐)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[assist]</span></p><p><span class="diczx1">傅，相也。——<span class="smcs">《说文》</span>。按，与俌略同。</span></p><p><span class="diczx1">傅，傅之德义。——<span class="smcs">《汉书·贾谊传》</span>。</span><span class="diczx2">师古</span>曰:<span class="yhcs">“傅，辅也。”</span></p><p><span class="diczx2">郑伯</span>傅王。——<span class="smcs">《左传·僖公二十八年》</span></p><p><span class="cino">(3)</span> 又如:傅父(古代辅育王室子女的年长男性保育人员);傅御(古代辅佐君王或诸侯办事的官员);傅弼(辅弼;辅佐)</p><p><span class="cino">(4)</span> 辅导 <span class="encs">[give guidance in study or training]</span></p><p><span class="diczx1">有</span><span class="diczx2">楚</span>大夫于此，欲其子之 <span class="diczx2">齐</span>语也，则使 <span class="diczx2">齐</span>人傅诸?使 <span class="diczx2">楚</span>人傅诸?——<span class="smcs">《孟子》</span></p><p><span class="diczx1">今日傅教帝。——<span class="smcs">《汉书·陈平传》</span></span></p><p><span class="cino">(5)</span> 又如:傅姆(古时辅导、保育贵族子女的年老妇人);傅相(古时辅导国君、诸侯王之官);傅导(辅导)</p><p><span class="cino">(6)</span> 靠近，迫近 <span class="encs">[close to]</span></p><p><span class="diczx1">傅，近也。——<span class="smcs">《小尔雅》</span></span></p><p><span class="diczx1">重欲傅人。——<span class="smcs">《考工记·庐人》</span></span></p><p><span class="diczx1">亦傅于天。——<span class="smcs">《诗·小雅·菀柳》</span></span></p><p><span class="cino">(7)</span> 又如:傅郭(靠近外城);傅近(靠近)</p><p><span class="cino">(8)</span> 依附，依凭 <span class="encs">[depend on]</span></p><p><span class="diczx1">皮之不存，毛将安傅?——<span class="smcs">《左传·僖公十四年》</span></span></p><p><span class="cino">(9)</span> 又如:傅堞(凭借城堞);傅丽(依靠)</p><p><span class="cino">(10)</span> 通<span class="yhcs">“附”</span>。附着 <span class="encs">[stick to]</span></p><p><span class="diczx1">辩士见一端，或以意傅着星历，虚造</span><span class="diczx2">匈奴</span>、 <span class="diczx2">乌孙</span>、 <span class="diczx2">西羌</span>之难。——<span class="smcs">《汉书·息夫躬傅》</span></p><p><span class="diczx1">毋为虎傅翼，将飞入邑，择人而食之。——<span class="smcs">《韩非子》</span></span></p><p><span class="cino">(11)</span> 又如:傅人(依附于他人);傅于(附着)</p><p><span class="cino">(12)</span> 涂 <span class="encs">[apply]</span>。如:傅粉(敷粉);傅彩(在绘画中加上色彩);傅色(着色);傅致(涂上);傅饰(涂饰)</p><p><span class="cino">(13)</span> 附会;强加 <span class="encs">[be attached with]</span>。如:傅重(附会罪状);傅时(趋附时势);傅益(因附会而增加);傅理(附会于理);傅构(辅会虚构)</p><p><span class="cino">(14)</span> 捎带，随带 <span class="encs">[bring]</span></p><p><span class="diczx1">边人驩激，执弓傅矢，唯恐不得进。——<span class="smcs">《宋史》</span></span></p><p><span class="cino">(15)</span> 通<span class="yhcs">“赙”</span>。以财物助丧事 <span class="encs">[give property for funeral arrangements]</span></p><p><span class="diczx1">若国札丧则令傅补之。——<span class="smcs">《周礼·秋官下·小行人》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>傅</strong> <span class="dicpy">fù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/fù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 教师，师傅，古时特指帝王的相或帝王、诸侯之子的老师 <span class="encs">[teacher]</span>。如:太傅;少傅</p><p><span class="cino">(2)</span> 姓</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a
