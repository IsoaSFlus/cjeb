						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>滑</strong></p><p><span class="dicpy">huá  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/huá.mp3"></a></span>  ㄏㄨㄚˊ</span></p><ol><li>光溜，不粗涩：～溜。光～。～润。～利。</li><li>在光溜的物体表面上溜动：～冰。～雪。～行。～翔。～梯。～坡。</li><li>狡诈，不诚实：～头。狡～。～头～脑。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">滑</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>滑</strong> <span class="dicpy">huá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/huá.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从水，骨声。本义:滑溜，光滑)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[smooth;slippery]</span></p><p><span class="diczx1">间关莺语花底滑。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="diczx1">道中迷雾冰滑。——</span><span class="diczx2">清</span>· <span class="diczx2">姚鼐</span><span class="smcs">《登泰山记》</span></p><p><span class="cino">(3)</span> 又如:滑擦(滑擦擦。形容很光滑);滑出律(形容光滑);滑踏(滑溜;不便行走);滑刺溜(形容非常光滑)</p><p><span class="cino">(4)</span> 流利;婉转 <span class="encs">[fluent]</span>。如:滑辣(形容水响的声音);滑熟(熟练;惯熟);滑怏(形容关系融洽或没隔阂);滑串流口(谓人言词流利便捷)</p><p><span class="cino">(5)</span> 狡诈;油滑。亦指狡猾的人 <span class="encs">[cunning;crafty]</span></p><p><span class="diczx1">吏谨将之无铍(</span><span class="dicpy">pī</span>通披)滑。——<span class="smcs">《荀子·成相》</span></p><p><span class="diczx1">为人上，操下如束湿薪。滑贼任威。——<span class="smcs">《史记·酷吏列传·宁成》</span></span></p><p><span class="cino">(6)</span> 又如:滑贼(狡猾狠戾);滑民(狡猾之民);滑人(滑头的人);滑鬼(狡诈或油滑的人)</p><p><span class="cino">(7)</span> 浮华不实 <span class="encs">[showy]</span></p><p><span class="diczx1">但有时也会使革命精神转成浮滑。——</span><span class="diczx2">鲁迅</span><span class="smcs">《集外集拾遗补编》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>滑</strong> <span class="dicpy">huá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/huá.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 滑动;滑过 <span class="encs">[slip;slide]</span></p><p><span class="diczx1">间关莺语花底滑。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="diczx1">道中迷雾冰滑。——</span><span class="diczx2">清</span>· <span class="diczx2">姚鼐</span><span class="smcs">《登泰山记》</span></p><p><span class="cino">(2)</span> 又如:滑擦(打滑);滑垒(滑步上垒);滑落;滑了一跤;滑下;滑倒;在错误的道路上越滑越远;滑行;滑雪</p><p>◎ <strong>滑</strong> <span class="dicpy">huá</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/huá.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 古时指使菜肴柔滑的作料 <span class="encs">[smooth agent]</span>。亦指使菜肴柔滑。如:滑甘(古时用以给菜肴调味的佐料)</p><p><span class="cino">(2)</span> 中药。滑石 <span class="encs">[talc]</span></p><p><span class="diczx1">以滑养窍。——<span class="smcs">《周礼》</span></span></p><p><span class="cino">(3)</span> 中医脉象之一，指脉搏跳动流畅 <span class="encs">[smooth pulse]</span></p><p><span class="diczx1">老先生这脉息，右寸略见弦滑。——<span
