                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>激</strong></p><p><span class="dicpy">jī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span>  ㄐㄧˉ</span></p><ol><li>水受阻遏，震荡而涌或飞溅：冲～。～荡。～浊扬清（冲击污水，让清水上来，喻打击坏人坏事，奖励好人好事）。</li><li>冷水突然浇淋或冲、泡食物：～酸菜。</li><li>鼓动，使人的感情冲动：刺～。～励。</li><li>感情冲动：感～。～动。～昂慷慨。</li><li>急剧的，强烈的：～烈。～进。</li><li>过分直率：～厉。～切。</li><li>〔～光〕某些物质的原子中的粒子受光或电刺激，使低能级的原子变成高能级原子，而辐射出相位、频率、方向等完全相同的光，称“激光”。亦称“莱塞”、“镭射”。</li><li>〔～光器〕产生激光的装置。</li><li>〔～素〕内分泌腺分泌的物质。亦称“荷尔蒙”。</li><li>鲜明：唇如～丹。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>激</strong> <span class="dicpy">jī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。本义:水势受阻遏后腾涌或飞溅)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[surge]</span></p><p><span class="diczx1">激，水碍袤疾波也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">激而行之，可使在山。——<span class="smcs">《孟子》</span></span></p><p><span class="diczx1">激水之疾，至于漂石者，势也。——<span class="smcs">《孙子·势》</span></span></p><p><span class="diczx1">泉水激石。——</span><span class="diczx2">吴均</span><span class="smcs">《与朱元思书》</span></p><p><span class="cino">(3)</span> 又如:激激(急流声;水势湍急的样子);激浪(怒涛);激射(喷射，冲击)</p><p><span class="cino">(4)</span> 鼓动人心，使有所感发 <span class="encs">[excite]</span></p><p><span class="diczx1">激而发之。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="diczx1">激于义而死。——</span><span class="diczx2">明</span>· <span class="diczx2">张溥</span><span class="smcs">《五人墓碑记》</span></p><p><span class="diczx1">激众愤。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="cino">(5)</span> 又如:激烦(麻烦);激作(振奋激励);激拔(激励提拔);激奖(激励劝勉);激聒(嘟囔);激激聒聒(嘟嘟囔囔)</p><p><span class="cino">(6)</span> 抑制 <span class="encs">[restrain]</span>。如:激贪(抑制贪婪);激贪厉俗(抑制贪婪之风，劝勉良好的世俗)</p><p><span class="cino">(7)</span> 冷水突然刺激身体使得病 <span class="encs">[fall ill from getting wet]</span>。如:他被雨水激着了</p><p><span class="cino">(8)</span> 快速变冷 <span class="encs">[chill]</span>。如:把西瓜放在冷水里激一激</p><p><strong>词性变化</strong></p><p>◎ <strong>激</strong> <span class="dicpy">jī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 迅疾;猛烈 <span class="encs">[rapid;acute;fierce]</span></p><p><span class="diczx1">比如顺风而呼，声非加疾，其势激也。——<span class="smcs">《史记·游侠列传》</span></span></p><p><span class="cino">(2)</span> 又如:激水(湍急的水流);激矢(急飞的箭);激合(迅速集合);激迫(急迫);激辞(激烈的言辞)</p><p><span class="cino">(3)</span> 高亢;激昂 <span class="encs">[resoundirg;reverating;aroused]</span></p><p><span class="diczx1">振声激扬。——<span class="smcs">《后汉书·张衡传》</span></span></p><p><span class="cino">(4)</span> 又如:激卬(振奋昂扬);激楚(声音高亢凄凉;又为古代歌曲名);激抗(高亢);激壮(激昂雄壮);激宕(激越跌荡)</p><p><strong>常用词组</strong></p><hr
