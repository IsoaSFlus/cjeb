                    <h2 class="h2_entry"><span class="orth">穰</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>穰</strong> <span class="dicpy">ráng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ráng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从禾，襄( <span class="dicpy">xiāng</span>)声。本义:禾茎中白色柔软的部分。也指瓜果的肉)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[pulp]</span>?</p><p><span class="cino">(3)</span> 丰收 <span class="encs">[bumper harvest]</span></p><p><span class="diczx1">世之有饥穰，天之行也，</span><span class="diczx2">禹</span><span class="diczx2">汤</span>被之矣。—— <span class="diczx2">汉</span>· <span class="diczx2">贾谊</span><span class="smcs">《论积贮疏》</span></p><p><span class="diczx1">穰岁之秋。——<span class="smcs">《韩非子·五蠹》</span></span></p><p><span class="cino">(4)</span> 又如:穰川(获得丰收的平原地方);穰岁(丰年)</p><p><span class="cino">(5)</span> 泛指黍稷稻麦等植物的杆茎 <span class="encs">[stalk]</span></p><p><span class="diczx1">烧黍穰。——</span><span class="diczx2">贾思勰</span><span class="smcs">《齐民要术·种谷》</span></p><p><span class="cino">(6)</span> 又如:穰子(刍料;喂牲口的草)</p><p><span class="cino">(7)</span> 饱满的稻粒 <span class="encs">[fully grown paddy]</span></p><p><span class="diczx1">凡稻最佳者九穰一秕，…则六穰四秕者容有之。——<span class="smcs">《天工开物》</span></span></p><p><span class="cino">(8)</span> 填，充实 <span class="encs">[full]</span></p><p><span class="diczx1">心是虚底物，性是里面穰肚馅草。性之理包在心内，到发时却是性底出来。——<span class="smcs">《朱子语类》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>穰</strong> <span class="dicpy">ráng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ráng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“禳”</span>。祭名，去邪除恶之祭 <span class="encs">[offer a sacrifice to exorcise evils]</span></p><p><span class="diczx1">见道旁有穰田者。——<span class="smcs">《史记·滑稽淳于髡传》</span></span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">rǎng</span></p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>穰</strong> <span class="dicpy">rǎng</span></p><p>〈形〉</p><p><span class="cino">(1)</span> 盛多;众多 <span class="encs">[abundant]</span></p><p><span class="diczx2">高唐州</span>城池虽小，人稠物穰，军广粮多。——<span class="smcs">《水浒全传》</span></p><p><span class="cino">(2)</span> 又如:穰浩(众多);穰穰(众多)</p><p><span class="cino">(3)</span> 通<span class="yhcs">“攘”</span>。烦乱;纷乱 <span class="encs">[confused]</span>。如:穰穰(纷乱貌)</p><p><span class="cino">(4)</span> 繁荣 <span class="encs">[prosperous]</span></p><p><span class="diczx2">长安</span>中浩穰，于三辅尤为剧。——<span class="smcs">《汉书》</span></p><p><span class="cino">(5)</span> 兴旺;旺盛 <span class="encs">[flourishing]</span></p><p><span class="diczx2">汉</span>世之论，自 <span class="diczx2">贾谊</span>已繁穰，其次渐与辞赋同流。—— <span class="diczx2">章炳麟</span><span class="smcs">《国故论衡·论式》</span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>穰</strong> <span class="dicpy">rǎng</span></p><p>［数］</p><p><span class="cino">(1)</span> 中国物理学会曾作为10 18 的词头的用字 <span class="encs">[exa-]</span>——国际制用<span class="yhcs">“艾可萨”</span>，简称<span class="yhcs">“艾”</span>，符号E</p><p><span class="cino">(2)</span> 万万秭 <span class="encs">[one hundred million times of 秭]</span></p><p><span class="diczx1">凡大数之法，万万曰亿，…万万秭曰穰，万万穰曰沟。——<span class="smcs">《孙子算经》</span></span></p><p>◎ <strong>穰</strong> <span class="dicpy">rǎng</span></p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“攘”</span>。排除;排斥 <span class="encs">[get rid of;repel]</span>。如:穰灾(攘灾;祛灾)</p><p><span
