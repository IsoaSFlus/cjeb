						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>幢</strong></p><p><span class="dicpy">chuáng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/chuáng.mp3"></a></span>  ㄔㄨㄤˊ</span></p><ol><li>古代原指支撑帐幕、伞盖、旌旗的木竿，后借指帐幕、伞盖、旌旗。</li><li>〔～～〕形容影子摇晃，如“人影～～”。</li><li>刻着佛号或经咒的石柱：经～。石～。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>幢</strong></p><p><span class="dicpy">zhuàng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàng.mp3"></a></span>  ㄓㄨㄤˋ</span></p><ol><p>　◎ 量词，指房屋：一～大楼。</p></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>幢</strong> <span class="dicpy">chuáng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/chuáng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从巾，童声。本义:古时作为仪仗用的一种旗帜)</p><p><span class="cino">(2)</span> 垂筒形、饰有羽毛、锦绣的旗帜。古代常在军事指挥、仪仗行列、舞蹈表演中使用 <span class="encs">[pennant or streamer used in ancient China]</span>。如:幢幡(佛前所立的旌旗。后泛指一般旌旗);幢队(行军时，举旗帜作先导的部队);幢牙(军营前饰有羽毛的大旗)</p><p><span class="cino">(3)</span> 刻着佛号或经咒的石柱子。书写佛号或经咒于帛上者称经幢，刻于石上者称石幢 <span class="encs">[stone pillar]</span>。如:幢相(佛幢)</p><p><span class="cino">(4)</span> 古代军队编制单位。百人为幢 <span class="encs">[hundred soldier]</span>。如:幢主(部队首长)</p><p><span class="cino">(5)</span> 量词。俗称房屋或楼房一所为一幢</p><p><span class="cino">(6)</span> 另见 <span class="dicpy">zhuàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàng.mp3"></a></span> </p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>幢</strong> <span class="dicpy">zhuàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàng.mp3"></a></span> </p><p>〈名〉</p><p>张挂在舟、车上的帷幔 <span class="encs">[a curtain (for a carriage)to screen from sun]</span></p><p><span class="diczx1">抚鸿幢，御矰缴，方舟并驾，俛仰极乐。——<span class="smcs">《后汉书》</span></span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>幢</strong> <span class="dicpy">zhuàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 愚昧无知 <span class="encs">[ignorant]</span></p><p><span class="diczx1">是故君先立于仁，则大夫忠而士信，民敦，工璞，商悫，女憧，妇空空。七者，教之志也。——<span class="smcs">《大戴礼记》</span></span></p><p><span class="cino">(2)</span> 又如:幢愚(愚昧);憧憧(憨愚无知的样子)</p><p>◎ <strong>幢</strong> <span class="dicpy">zhuàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"
