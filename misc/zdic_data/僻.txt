						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>僻</strong></p><p><span class="dicpy">pì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pì.mp3"></a></span>  ㄆㄧˋ</span></p><ol>    <li>偏，距离中心地区远的：偏～。～静。～陋。～野。～远。穷乡～壤。</li>    <li>不常见的：冷～。生～。</li>    <li>性情古怪，不合群：孤～。怪～。乖～。～戾。～性。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>僻</strong> <span class="dicpy">pì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pì.mp3"></a></span> </p><p>〈动〉</p><p>(形声。从人，辟声。本义:退避;回避) 同本义 <span class="encs">[avoid]</span></p><p><span class="diczx1">僻，辟也。…<span class="smcs">《诗》</span>曰:<span class="yhcs">“宛如左僻。”</span>——<span class="smcs">《说文》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>僻</strong> <span class="dicpy">pì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 偏僻;偏远;很少有人去的 <span class="encs">[remote;out-of-the-way]</span></p><p><span class="diczx1">虽僻远其何伤。——<span class="smcs">《楚辞·涉江》</span></span></p><p><span class="diczx1">而荆僻也。——<span class="smcs">《吕氏春秋·慎行》</span></span></p><p><span class="diczx2">浔阳</span>地僻无音乐。—— <span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p><p><span class="cino">(2)</span> 又如:僻路(偏僻的小路);僻壤(偏僻的地方);僻小(偏僻狭小);僻穴(偏僻的洞穴);僻陋(偏僻简陋);僻巷(偏僻小巷);僻净(偏僻清静)</p><p><span class="cino">(3)</span> 邪僻 <span class="encs">[evil]</span></p><p><span class="diczx1">民之多僻。——<span class="smcs">《诗·大雅·板》</span></span></p><p><span class="cino">(4)</span> 又如:僻王(邪僻不正的国君);僻邪(邪恶);僻事(邪僻之事);僻淫(邪避淫佚);僻乱(邪僻反常，不合正道);僻诡(邪避诡诈);僻错(邪僻乖张)</p><p><span class="cino">(5)</span> 偏执 <span class="encs">[stubborn]</span>。如:僻违(乖张，违背常理);僻介(偏执耿直);僻固(偏执，固执);僻陋(性情偏执，见识浅陋);僻鄙(偏执鄙陋);僻愚(僻陋暗昧);僻数(僻陋的技巧);僻滞(偏执拘泥)</p><p><span class="cino">(6)</span> 冷僻 <span class="encs">[rare;uncommon]</span>。如:僻书(冷僻的书籍，极罕见的书籍);僻涩(冷僻晦涩);僻缓(冷僻而非必需);僻隐(冷僻幽隐而不易见);僻事(不是平常习见的典故);僻字</p><p><span class="cino">(7)</span> 怪僻 <span class="encs">[eccentric]</span>。如:僻语(怪僻的语言);僻放(怪僻放恣); 僻戾(怪僻暴戾); 僻恣(怪僻而纵恣); 僻谬(乖僻荒谬， 违背正理); 僻违(乖僻不合)</p><p><span class="cino">(8)</span> 歪;斜 <span class="encs">[askew;inclined]</span>。如:僻然(倾侧的样子);僻侧(倾斜)</p><p>◎ <strong>僻</strong> <span class="dicpy">pì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/pì.mp3"></a></span> </p><p>〈名〉</p><p>通<span class="yhcs">“嬖”</span>。受宠爱的人 <span class="encs">[favorite]</span></p><p><span class="diczx1">便僻好色。——<span class="smcs">《韩非子·八奸》</span></span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/僻典">僻典</a></span><span class="crefe"><a href="/hans/僻见">僻见</a></span><span class="crefe"><a href="/hans/僻径">僻径</a></span><span class="crefe"><a href="/hans/僻静">僻静</a></span><span class="crefe"><a href="/hans/僻陋">僻陋</a></span><span class="crefe"><a href="/hans/僻路">僻路</a></span><span class="crefe"><a href="/hans/僻壤">僻壤</a></span><span class="crefe"><a href="/hans/僻性">僻性</a></span><span class="crefe"><a href="/hans/僻野">僻野</a></span><span class="crefe"><a href="/hans/僻远">僻远</a></span></div>                    <div class="div copyright"> © 汉典 </div>

                          『說文解字』<hr class="dichr">
                          <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/50FB.svg">                          <p>避也。从人辟聲。《詩》曰：“宛如左僻。”一曰从㫄牽也。普擊切</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/50FB_327.svg"  /><span class="swbf">(僻)</span><span class="ywyy">辟也。</span><span class="swdz">辟大徐本作避。非是。辟者、法也。引伸爲辟人之辟。辟人而人避之亦曰辟。若周禮閽人。凡外內命婦出入。則爲之辟。孟子。行辟人可也。曲禮。若主人拜。則客還辟辟拜。郊特牲。有由辟焉。包咸論語注。躩盤辟皃也。投壷。主人盤旋曰辟。賓盤旋曰辟。大射儀。賓辟。注曰。辟、逡遁不敢當盛。他書辟人、辟邪、辟寒、辟塵之類。語意大略相似。自屛之者言。則閽人、離婁篇、郊特牲是也。自退者言。則曲禮、投壷、論語注所云是也。舟部般、辟也。卽旋辟、盤辟之謂。辟之言邊也。屛於一邊也。僻之本義如是。廣韵曰。誤也。邪僻也。此引伸之義。今義行而古義廢矣。詩曰。民之多僻。</span><span class="ywyy">从人。辟聲。</span><span class="swdz">普擊切。十六部。</span><span class="ywyy">詩曰。宛如左僻。</span><span class="swdz">魏風葛屨文。此引詩證僻之本義也。今詩作辟者。俗人不解。易字而音避。</span><span class="ywyy">一曰從旁牽也。</span><span class="swdz">此義稍別異。</span><span class="xh"></span></p>                          </div>
