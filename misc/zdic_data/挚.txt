                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>挚</strong> <span class="dicpy">zhì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。从手，从执。本义: 古代初次求见人时所送的礼物，见面礼)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[present]</span></p><p><span class="diczx1">挚，握持也。——<span class="smcs">《说文》</span>。字亦作贽。</span></p><p><span class="diczx1">以禽作六挚。孤执皮帛，卿执羔，大夫执雁，士执雉，庶人执鹅，工商执鸡。——<span class="smcs">《周礼·大宗伯》</span></span></p><p><span class="diczx1">奠挚见于君。——<span class="smcs">《仪礼·士冠礼》</span></span></p><p><span class="diczx1">执挚以相见。——<span class="smcs">《礼记·郊特牲》</span></span></p><p><span class="diczx1">先人尝受其挚，阅不终篇，而屡叹</span><span class="diczx2">葛氏</span>之多子也。—— <span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《葛兵祖墓志铭》</span></p><p><span class="cino">(3)</span> 又如:挚币(古时用为祭祀或馈赠礼物的束帛)</p><p><span class="cino">(4)</span> 古国名 <span class="encs">[Zhi state]</span>。在今河南省汝南县东南</p><p><span class="diczx2">挚</span><span class="diczx2">仲氏</span><span class="diczx2">任</span>，自彼 <span class="diczx2">殷商</span>，来嫁于 <span class="diczx2">周</span>。——<span class="smcs">《诗·大雅·大明》</span></p><p><span class="cino">(5)</span> 通<span class="yhcs">“贽”</span>。见面礼<span class="encs">[gift cpresented to a senior at one's first visit]</span></p><p><span class="diczx1">奠挚见于君。——<span class="smcs">《仪礼·士冠礼》</span></span></p><p><span class="diczx1">以禽作六挚。——<span class="smcs">《周礼·春官·大宗伯》</span></span></p><p><span class="diczx1">执挚以相见。——<span class="smcs">《礼记·郊特牲》</span></span></p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>挚</strong> <span class="dicpy">zhì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 逮;捕;捉 <span class="encs">[arrest;seize]</span></p><p><span class="diczx1">挚执妻子。——<span class="smcs">《吕氏春秋》</span></span></p><p><span class="cino">(2)</span> 又如:挚执(拘捕)</p><p><span class="cino">(3)</span> 击;搏击 <span class="encs">[strike]</span></p><p><span class="diczx1">行冬令，则风寒不时，鹰隼蚤挚，四鄙入保。——<span class="smcs">《淮南子》</span></span></p><p><span class="cino">(4)</span> 攫取 <span class="encs">[pinch]</span></p><p><span class="diczx1">百卉具零，刚虫搏挚。——<span class="smcs">《文选·张衡·西京赋》</span></span></p><p><span class="cino">(5)</span> 至;到 <span class="encs">[arrive at]</span></p><p><span class="diczx1">天曷不降威，大命不挚。——<span class="smcs">《书·西伯戡黎》</span></span></p><p><span class="diczx1">挚而有别。——<span class="smcs">《诗·关雎》</span>传</span></p><p>◎ <strong>挚</strong> <span class="dicpy">zhì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhì.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 诚恳，恳切 <span class="encs">[ernest;sincere]</span>。如:挚伴(亲密诚挚的伙伴);挚切(恳切);挚热(恳切热烈);挚醇(诚挚淳朴)</p><p><span class="cino">(2)</span> 通<span class="yhcs">“鸷”</span>。凶猛<span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/646F.svg">						  <p>【唐韻】【集韻】【韻會】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>脂利切，音至。【說文】握持也。一曰至也。【爾雅·釋詁】臻也。【註】摯，執也，所以表至也。【書·西伯戡黎】大命不摯。【註】言受大命者，何不至也。【詩·周南·關雎傳】雎鳩摯而有別。【箋】摯之言至也，謂情意至然而有別。</p><p>又執物以爲相見之禮也。與贄質通。【禮·曲禮】凡摯，天子鬯，諸侯圭，卿羔，大夫鴈，士雉，庶人之摯匹。【周禮·春官·大宗伯】以禽作六摯。【註】所執以自致也。</p><p>又進也。【戰國策】近習之人，其摯諂也固矣。</p><p>又極也。【周禮·冬官考工記函人】凡甲鍛不摯，則不堅。【註】摯之言致。【疏】謂熟之至極。【前漢·韓安國傳贊】臨其摯而顛墜。</p><p>又傷折也。【禮·月令】孟春行冬令，則水潦爲敗，雪霜大摯。</p><p>又與鷙通。【禮·曲禮】前有摯獸，則載貔貅。【疏】摯獸，虎狼之屬。</p><p>又【儒行】鷙蟲攫搏。【疏】蟲是鳥獸通名，但獸摯從執下手，鳥鷙從執下鳥。</p><p>又國名。【詩·大雅】摯仲氏任。【周語】昔摯疇之國也，由大任。【註】摯疇二國，任姓，奚仲仲虺之後，太任之家也。</p><p>又人名。【蔡邕·釋誨】伊摯有負鼎之衒。【註】摯，伊尹名也。</p><p>又【正韻】支義切，音寘。義同。</p><p>又【集韻】【韻會】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>陟利切，音致。【廣雅】解也。</p><p>又與輊通。低也。【周禮·冬官考工記·輈人】大車之轅摯，其登又難。【註】摯，輖也。【又】大車平地旣節軒摯之任。或作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2838C.svg'><span class='k_txt'>𨎌</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/2694E.svg'><span class='k_txt'>𦥎</span>。</p><p>又作輖。</p><p>又職日切，音質。姓也。【前漢·貨殖傳】京師富人，茂陵摯綱。【後漢·馬融傳】京兆摯恂。【晉書·摯虞傳】京兆，長安人。　从執持之執，本作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/22D37.svg'><span class='k_txt'>𢴷</span>。</p>
