						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>耦</strong></p><p><span class="dicpy">ǒu  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span>  ㄡˇ</span></p><ol><li>两个人在一起耕地。</li><li>〔～合〕物理学上指两个或两个以上的体系或两种运动形式之间通过各种相互作用而彼此影响以至联合起来的现象。</li><li>同“<span class="diczx3">偶</span>”。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>耦</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从耒，禺( <span class="dicpy">yù</span>或 <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> )声。耒( <span class="dicpy">lěi</span>)，翻土工具。本义:二人并肩耕地)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[plough side by side]</span></p><p><span class="diczx1">耦，耒广五寸为伐，二伐为耦。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">十千维耦。——<span class="smcs">《诗·周颂·噫嘻》</span></span></p><p><span class="diczx1">譬如农夫作耦。——<span class="smcs">《国语·吴语》</span></span></p><p><span class="diczx1">禹见耕者耦立而式。——<span class="smcs">《荀子·大略》</span></span></p><p><span class="diczx2">长沮</span>、 <span class="diczx2">桀溺</span>耦而耕。——<span class="smcs">《论语·微子》</span></p><p><span class="cino">(3)</span> 又如:耦耕(两人并力一起耕种。为古代犁田的一种方式);耦犁(一种耕作法)。引申为二人一组。如:耦俱无猜(双方都无猜疑);耦居(谓两人同处);耦俱(指相处融洽)</p><p><span class="cino">(4)</span> 匹敌;相对 <span class="encs">[match]</span>。如:耦国(对等之国);耦象(指箭耙);耦语(相对私语)</p><p><span class="cino">(5)</span> 遇，逢 <span class="encs">[meet with]</span></p><p><span class="diczx1">燕成双，鸾对影，耦新知。——</span><span class="diczx2">宋</span>· <span class="diczx2">孙光宪</span><span class="smcs">《酒泉子》</span></p><p><span class="cino">(6)</span> 合，符合 <span class="encs">[coordinate]</span>。如:耦世(适应世俗)</p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>耦</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 配偶 <span class="encs">[spouse]</span>。如:自择良耦;正为嘉耦</p><p><span class="cino">(2)</span> 偶像 <span class="encs">[image;idol]</span>。如:木耦;土耦。耦人(土木等制成的人俑)</p><p><span class="cino">(3)</span> 同<span class="yhcs">“偶”</span>。双数，成对 <span class="encs">[even]</span></p><p><span class="diczx1">阳卦奇，阴卦耦。——<span class="smcs">《易系辞下》</span></span></p><p><span class="cino">(4)</span> 又如:耦语(两个相对私语。同偶语);耦俱无猜(两者均无疑恨);耦立(犹对偶);耦丽，耦俪(对耦的文辞)</p><p><span class="cino">(5)</span> 躯体 <span class="encs">[body]</span></p><p><span class="diczx2">南郭</span><span class="diczx2">子綦</span>隐机而坐，仰天而嘘，嗒焉似丧其耦。——<span class="smcs">《庄子》</span></p><p>◎ <strong>耦</strong> <span class="dicpy">ǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/ǒu.mp3"></a></span> </p><p>〈量〉</p><p>双，两个一组 <span class="encs">[two;pair]</span></p><p><span class="diczx1">射者三耦。——<span class="smcs">《左传·襄公二十九年》</span></span></p><p><span class="diczx1">耦俱无猜。——<span class="smcs">《左传·僖公九年》</span></span></p><p><span class="diczx1">嗒焉似丧其耦。——<span                     <div class="gycd"><div class="pz"><ruby><rbc><rb>耦</rb></rbc><rtc><rt>ㄡˇ</rt></rtc><rtc><rt>ǒu <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/gycd/zd/5801.ogg"></a></span> </rt></rtc></ruby></div><div class="gycd-item"><span class="gc_cx">名</span><ol><li><p class="def"><span class="gc_sy">古代的一种农具。</span><span class="gc_yy">《说文解字·耒部》：「<b>耦</b>，耕广五寸为伐，二伐为<b>耦</b>。」</span></p></li><li><p class="def"><span class="gc_sy">一对、二个。</span><span class="gc_yy">《左传·襄公二十九年》：「公享之，展庄叔执币，射者三<b>耦</b>。」</span><span class="gc_yy">晋·杜预·注：「二人为<b>耦</b>。」</span><span class="gc_yy">《三国志·卷四七·吴书·吴主权传》：「今孤父子亲自受田，车中八牛以为四<b>耦</b>。」</span></p></li><li><p class="def"><span class="gc_sy">配偶。</span><span class="gc_yy">《左传·桓公六年》：「人各有<b>耦</b>，齐大非吾<b>耦</b>也。」</span><span class="gc_yy">《聊斋志异·卷十·炙戬》：「请赐离婚书，听君自择良<b>耦</b>。」</span></p></li><li><p class="def"><span class="gc_sy">双数。</span><span class="gc_yy">《易经·系辞下》：「阳卦奇，阴卦<b>耦</b>。」</span><span class="gc_yy">唐·李德裕〈文章论〉：「意尽而止，成篇不拘于只<b>耦</b>。」</span></p></li><li><p class="def"><span class="gc_sy">姓。如汉代有<b><b>耦</b></b>嘉。</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">形</span><ol><li><p class="def"><span class="gc_sy">淤积的。</span><span class="gc_yy">《吕氏春秋·孝行览·必己》：「牛缺居上地，大儒也，下之邯郸，遇盗于<b>耦</b>沙之中。」</span></p></li></ol></div><div class="gycd-item"><span class="gc_cx">动</span><ol><li><p class="def"><span class="gc_sy">两人并耕。</span><span class="gc_yy">《论语·微子》：「长沮桀溺<b>耦</b>而耕，孔子过之。」</span><span class="gc_yy">《荀子·大略》：「禹见耕者<b>耦</b>，立而式。」</span><span class="gc_yy">唐·杨倞·注：「两人并耕为<b>耦</b>。」</span></p></li></ol></div></div>

                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/8026.svg">						  <p>【唐韻】五口切【集韻】【韻會】【正韻】語口切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音偶。【說文】耒廣五寸爲伐，二伐爲耦。【詩·周頌】亦服爾耕，十千維耦。【箋】耜廣五寸，二耜爲耦。一川之閒萬夫，故有萬耦。【周禮·冬官考工記·匠人】二耜爲耦。一耦之伐，廣尺，深尺，謂之<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24C15.svg'><span class='k_txt'>𤰕</span>。【疏】兩人耕爲耦。</p><p>又凡二人爲耦。【左傳·襄二十九年】射者三耦。【註】二人爲耦。【周禮·天官·掌次】射則張耦次。【註】耦。俱升射者。【疏】天子大射六耦，在西郊。賔射亦六耦，在朝。燕射三耦，在寢。</p><p>又【射人】王以六耦，諸侯以四耦，孤卿大夫以三耦。【莊子·齊物論】嗒焉，似喪其耦。</p><p>又匹也，配也。【左傳·桓六年】人各有耦，齊大，非吾耦也。【宣三年】石癸曰：吾聞姬姞耦，其子孫必蕃。【註】姞姓宜爲姬配耦。</p><p>又【釋名】遇也。【前漢·高帝紀】耦語者棄市。【註】耦，對也。</p><p>又【玉篇】不畸也。【易·繫辭】陽卦奇，隂卦耦。【註】陽卦二隂，故奇爲之君。隂卦二陽。故耦爲之主。</p><p>又通也。【淮南子·要略】所以應待萬方，覽耦百變也。【註】耦，通也。</p><p>又地名。【列子·說符篇】牛缺者，上地之大儒也。下之邯鄲，遇盜於耦沙之中。</p><p>又人名。【左傳·文十五年】宋華耦來盟，其官皆從之。【註】耦，華督曾孫也。</p><p>又姓。【五音集韻】漢有侍中耦嘉。</p><p>又【韻補】叶語綺切，音擬。【詩·周頌】駿發爾私，終三十里。亦服爾耕，十千維耦。</p><p>又叶五舉切，音語。【楚辭·九思】言逝邁兮北徂，叫我友兮配耦。日隂曀兮未光，闃<img class='lazy k_img' data-original='//img.zdic.net/song/cn/231BA.svg'><span class='k_txt'>𣆺</span>窕兮靡睹。</p><p>
