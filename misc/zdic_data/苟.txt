                    <h2 class="h2_entry"><span class="orth">苟</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>苟</strong> <span class="dicpy">gǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǒu.mp3"></a></span> </p><p>〈名〉</p><p>(形声。从艸，句( <span class="dicpy">gōu</span>)声。本义:草名。又:菜名) 同本义 <span class="encs">[grass name]</span></p><p><span class="diczx1">苟，苟草也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">苟贞夫。——<span class="smcs">《急就篇》</span>。注<span class="yhcs">“苟，草名也。”</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>苟</strong> <span class="dicpy">gǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǒu.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 随便，轻率 <span class="encs">[careless;negligent;indifferent]</span></p><p><span class="diczx1">生亦我所欲，所欲有甚于生者，故不为苟得也。——<span class="smcs">《孟子·告子上》</span></span></p><p><span class="cino">(2)</span> 又如:一丝不苟;不苟言笑;苟言(言谈轻率随便)</p><p><span class="cino">(3)</span> 贪求 <span class="encs">[greedy]</span>。如:苟利(贪求不正当的利益);苟冒(贪求);苟欲(贪求;不正当的欲望);苟贪(贪求)</p><p><span class="cino">(4)</span> 卑下 <span class="encs">[humble]</span>。如:苟贱(卑鄙下贱);苟辞(诌媚之言)</p><p>◎ <strong>苟</strong> <span class="dicpy">gǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǒu.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 姑且;暂且 <span class="encs">[tentatively;for the moment]</span></p><p><span class="diczx1">苟者，偷合之称。所以行为廉隅，不存德义，谓之苟且。——<span class="smcs">《匡谬正俗》</span>。按，累呼曰苟且，单呼曰苟。</span></p><p><span class="diczx1">且苟而赋。——<span class="smcs">《国语·鲁语》</span></span></p><p><span class="diczx1">临财毋苟得。——<span class="smcs">《礼记·曲礼上》</span></span></p><p><span class="diczx1">苟全性命于乱世，不求闻达于诸侯。——</span><span class="diczx2">诸葛亮</span><span class="smcs">《出师表》</span></p><p><span class="diczx1">虽董之以严刑，振之以威怒，终苟免而不怀仁。——</span><span class="diczx2">唐</span>· <span class="diczx2">魏征</span><span class="smcs">《谏太宗十思疏》</span></p><p><span class="cino">(2)</span> 又如:苟全性命(暂时保全性命);苟简(苟且简略);苟止(苟且行事;勉强停止);苟且行止(不正当的行为);苟容(苟且依附。指被权贵者所收容接纳);苟且因循(得过且过，不求进取);苟免(苟且免于损害)</p><p>◎ <strong>苟</strong> <span class="dicpy">gǒu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gǒu.mp3"></a></span> </p><p><连></p><p><span class="cino">(1)</span> 若;如果;假使 <span class="encs">[if]</span></p><p><span class="diczx1">苟富贵，无相忘。——<span class="smcs">《史记·陈涉世家》</span></span></p><p><span class="diczx1">苟粟多而财有余，何为而不成?——</span><span class="diczx2">汉</span>·
