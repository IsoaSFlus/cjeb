                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>晏</strong> <span class="dicpy">yàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yàn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从日，安声。本义:晴朗)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[sunny]</span></p><p><span class="diczx1">晏，天清也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">晏，明也。——<span class="smcs">《小尔雅》</span></span></p><p><span class="diczx1">于是天清日晏。——</span><span class="diczx2">杨雄</span><span class="smcs">《羽猎赋》</span>。注:<span class="yhcs">“无云之处也。”</span></p><p><span class="diczx1">鶤日知晏阴蜡知雨。——<span class="smcs">《淮南子·缪称》</span>。注:<span class="yhcs">“无云也。”</span></span></p><p><span class="diczx1">日出清济为晏。——<span class="smcs">《汉书·郊祀志上集》</span>注</span></p><p><span class="cino">(3)</span> 又如:晏日(晴天);晏阴(晴阴);晏温(天气晴暖)</p><p><span class="cino">(4)</span> 通<span class="yhcs">“安”</span>( <span class="dicpy">ān</span>)。平静;安逸 <span class="encs">[peaceful]</span></p><p><span class="diczx1">以定晏阴之所成。——<span class="smcs">《礼记·月令》</span>。注:<span class="yhcs">“安也。阴称，晏。”</span></span></p><p><span class="diczx1">而海内晏如。——<span class="smcs">《汉书·诸侯王表》</span>。注:<span class="yhcs">“安然也。”</span></span></p><p><span class="diczx1">言笑晏晏。——<span class="smcs">《诗·卫风·氓》</span></span></p><p><span class="diczx1">河清海晏，大德宽仁。——<span class="smcs">《西游记》</span></span></p><p><span class="cino">(5)</span> 又如:晏安(平静;安逸);晏如(安坐;闲坐);晏居(闲居;安居);晏室(闲居之室;私室)</p><p><span class="cino">(6)</span> 通<span class="yhcs">“旰”</span>( <span class="dicpy">gàn</span>)。迟 <span class="encs">[late]</span></p><p><span class="diczx1">冉子退潮，子曰:<span class="yhcs">“何晏也?”</span>。——<span class="smcs">《论语·子路》</span>。</span><span class="diczx2">皇</span>疏:<span class="yhcs">“晏，晚也。”</span></p><p><span class="diczx1">二子待君，曰晏。——<span class="smcs">《吕氏春秋·慎小》</span>。高诱注:<span class="yhcs">“晏，暮也。”</span></span></p><p><span class="diczx1">及年岁之未晏兮。——<span class="smcs">《楚辞·离骚》</span></span></p><p><span class="diczx1">你们今天怎么回来得这样晏?——</span><span class="diczx2">巴金</span><span class="smcs">《家》</span></p><p><span class="cino">(7)</span> 又如:晏眠(迟睡);晏然(迟迟;迟缓的样子);晏灯(长明灯;终夜不熄的灯)</p><p><span class="cino">(8)</span> 通<span class="yhcs">“安”</span>( <span class="dicpy">ān</span>)。温和;和柔 <span class="encs">[mild]</span></p><p><span class="diczx1">言笑晏晏，信誓旦旦。——<span class="smcs">《诗·卫风·氓》</span>。传:<span class="yhcs">“晏晏，和柔也。”</span></span></p><p><span class="diczx1">粲也晏也。——</span><span class="diczx2">扬雄</span><span class="smcs">《法言·孝至》</span>。 <span class="diczx2">李轨</span>注:<span class="yhcs">“晏，和柔。”</span></p><p><span class="diczx1">冻登赤天，晏入玄泉。——</span><span class="diczx2">汉</span>· <span class="diczx2">扬雄</span><span class="smcs">《太玄》</span></p><p><span class="cino">(9)</span> 又如:晏阴(柔和之阴，微阴)</p><p><span class="cino">(10)</span> 鲜艳;华美 <span class="encs">[bright]</span></p><p><span class="diczx1">羔裘晏兮，三英粲兮。——<span class="smcs">《诗·郑风·羔裘》</span></span></p><p><span class="cino">(11)</span> 又如:晏灿(明朗)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span
