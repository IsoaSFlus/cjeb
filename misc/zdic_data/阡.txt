                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>阡</strong> <span class="dicpy">qiān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/qiān.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从阜，千声。<span class="yhcs">“阜”</span>土山。与土有关。本义:田间南北向的小路)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[a footpath between fields，leading north and south]</span></p><p><span class="diczx1">阡，路东西为陌，南北为阡。——<span class="smcs">《说文新附》</span></span></p><p><span class="diczx1">阡，阡陌。——<span class="smcs">《广韵》</span></span></p><p><span class="diczx1">为田开阡陌封疆，而赋税平。——<span class="smcs">《史记·商君列传》</span></span></p><p><span class="diczx1">阡陌交通，鸡犬相闻。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《桃花源记》</span></p><p><span class="cino">(3)</span> 通往坟墓的小道 <span class="encs">[the path leading to a grave]</span></p><p><span class="diczx1">哀挽青门去，新阡绛水遥。——</span><span class="diczx2">杜甫</span><span class="smcs">《敌武卫将军挽词》</span>之三</p><p><span class="cino">(4)</span> 泛指田间小路 <span class="encs">[foot path between fields]</span></p><p><span class="diczx1">驱牛向东阡。——</span><span class="diczx2">柳宗元</span><span class="smcs">《田家》</span></p><p><span class="diczx1">要什么素车白马，断送出古陌荒阡。——</span><span class="diczx2">元</span>· <span class="diczx2">关汉卿</span><span class="smcs">《窦娥冤》</span></p><p><span class="cino">(5)</span> 又如:阡陇(田间高地);阡塍(田埂)</p><p><span class="cino">(6)</span> 道路 <span class="encs">[road;path]</span></p><p><span class="diczx1">阡，路亦曰阡。——<span class="smcs">《增韵》</span></span></p><p><span class="diczx1">越陌度阡，枉用相存。——</span><span class="diczx2">曹操</span><span class="smcs">《短歌行》</span></p><p><span class="diczx1">野径既盘纡，荒阡亦交互。——</span><span class="diczx2">南朝</span><span class="diczx2">梁</span>· <span class="diczx2">沈约</span><span class="smcs">《宿东园》</span></p><p><span class="cino">(7)</span> 坟墓 <span class="encs">[grave]</span></p><p><span class="diczx1">其子修始克表于其阡。——</span><span class="diczx2">欧阳修</span><span class="smcs">《沋冈阡表》</span></p><p><span class="cino">(8)</span> 又如:阡原(墓地);阡兆(墓地)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a                     <div class="gycd"><div class="pz"><ruby><rbc><rb>阡</rb></rbc><rtc><rt>ㄑㄧㄢ</rt></rtc><rtc><rt>qiān <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/gycd/zd/3651.ogg"></a></span> </rt></rtc></ruby></div><div class="gycd-item"><span class="gc_cx">名</span><ol><li><p class="def"><span class="gc_sy">本为田间南北向的小路，后泛指田间的小路。</span><span class="gc_yy">《史记·卷五·秦本纪》：「为田开阡陌。」</span><span class="gc_yy">司马贞索隐引《风俗通》曰：「南北曰<b>阡</b>，东西曰陌。」</span><span class="gc_yy">《文选·曹植·送应氏诗二首之一》：「游子久不归，不识陌与<b>阡</b>。」</span></p></li><li><p class="def"><span class="gc_sy">墓前的道路。</span><span class="gc_yy">宋·欧阳修〈泷冈阡表〉：「始克表于其<b>阡</b>。」</span></p></li><li><p class="def"><span class="gc_sy">姓。如唐代有<b><b>阡</b></b>能。</span></p></li></ol></div></div>

                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/9621.svg">						  <p>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>倉先切，音千。【說文】路南北爲阡。【史記·秦本紀】開阡陌。【註】風俗通曰：南北曰阡，東西曰陌。河東以東西爲阡，南北爲陌。朱子曰：二說不同，後說爲正。陌之爲言百也，遂洫從，而涇涂亦從，則遂閒百畝，洫閒百夫，而涇涂爲陌。阡之爲言千也，溝澮衡，而畛道亦衡，則溝閒千畝，澮閒千夫，而畛道爲阡，阡陌之名由此而得。◎按陸機答張士然詩：迴渠繞曲陌，通波扶直阡。此以南北爲阡也。柳宗元田家詩：蓐食狥所務，驅車向東阡。此以東西爲阡也。</p><p>又姓。【正字通】唐有阡能。</p><p>又墓道也。【杜甫·故武衞將軍挽詩】新阡絳水遙。通作仟。【前漢·原涉傳】京兆尹曹氏葬茂陵，民謂其道爲京兆仟。</p><p>又與芊同。密茂貌。【楚辭·九懷】遠望兮阡眠。【謝朓·和王著八公山詩】阡眠起雜樹。【游東田詩】遠樹瞹阡阡。【註】阡，與芊同。亦作仟仟。</p><p>又【集韻】倉甸切，音蒨。義同。　【玉篇】或作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/27BAE.svg'><span class='k_txt'>𧮮</span>圱。</p>
