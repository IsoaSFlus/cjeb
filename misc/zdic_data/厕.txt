                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>厕</strong></p><p>廁、厠 <span class="dicpy">cè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cè.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从厂( <span class="dicpy">hǎn</span>)，则声。<span class="yhcs">“厂”</span>，小篆字形象山崖，可以住人。<span class="yhcs">“厂”</span>作为汉字部首之一，常与<span class="yhcs">“广”</span>( <span class="dicpy">yǎn</span>)通用，表示与房屋有关，如<span class="yhcs">“廁”</span>又写作<span class="yhcs">“厠”</span>。本义:厕所)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[toilet]</span></p><p><span class="diczx1">厠，杂也。言人杂厠在上非一也。——<span class="smcs">《释名·释宫室》</span></span></p><p><span class="diczx2">沛公</span>起如厕。——<span class="smcs">《史记·项羽本纪》</span></p><p><span class="cino">(3)</span> 又如:男厕;女厕;厕屋(厕溷。厕所);厕筹(厕简。大便后用来拭粪的小木片片)</p><p><span class="cino">(4)</span> 假借为<span class="yhcs">“侧”</span>。旁边 <span class="encs">[side]</span></p><p><span class="diczx1">上踞厠视之。——<span class="smcs">《汉书·汲黯传》</span>。注:<span class="yhcs">“厠，床边侧也。”</span></span></p><p><span class="diczx1">居</span><span class="diczx2">霸陵</span>，北临厠。——<span class="smcs">《汉书·刘向传》</span>。注:<span class="yhcs">“厠，侧近水也。”</span></p><p><span class="diczx1">上居外临厠。——<span class="smcs">《汉书·张释之传》</span>。注:<span class="yhcs">“岸之边侧也。”</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>厕</strong></p><p>厕 <span class="dicpy">cè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/cè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 杂置;参与 <span class="encs">[work in]</span>。如:厕列(置于，列入);厕足;厕迹(插足，置身)</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">si</span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/厕身">厕身</a></span><span class="cino">(1)</span> 厠 <span class="dicpy">si</span></p><p><span class="cino">(2)</span> ——如<span class="yhcs">“茅厕”</span>（ <span class="dicpy">máosi</span>）:厕所(用于口语)</p><p><span
