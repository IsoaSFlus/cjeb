                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>诞</strong></p><p><strong>（<span class="diczx3">誕</span>）</strong></p><p><span class="dicpy">dàn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dàn.mp3"></a></span> ㄉㄢˋ</span></p><ol><li>大：～命。</li><li>欺诈，虚妄：怪～。～妄。</li><li>生育，人出生：～生。～辰（生日）。</li><li>生日：寿～。华～。圣～节。</li><li>放荡：放～。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>诞</strong></p><p>誕 <span class="dicpy">dàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dàn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从言，延声。本义:说大话)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[boast]</span></p><p><span class="diczx1">诞，词诞也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">诞惟民怨。——<span class="smcs">《书·酒诰》</span></span></p><p><span class="diczx1">肆朕诞以尔东征。——<span class="smcs">《书·大诰》</span></span></p><p><span class="diczx1">言神事如迂诞。——<span class="smcs">《汉书·郊祀志上》</span>。</span><span class="diczx2">师古</span>曰:<span class="yhcs">“诞，大言也。”</span></p><p><span class="diczx1">则诞欺怪迂之文弥以益多。——<span class="smcs">《汉书·艺文志》</span></span></p><p><span class="cino">(3)</span> 又如:诞诈(诞妄奸诈);诞辞(虚诞的言辞);诞言(夸大的话)</p><p><span class="cino">(4)</span> 虚妄;荒唐 <span class="encs">[absurd;fantastic]</span></p><p><span class="diczx1">是言诞也。——<span class="smcs">《国语·楚语》</span>。注:<span class="yhcs">“虚也。”</span></span></p><p><span class="diczx1">非诞说矣。——<span class="encs">[英]</span></span><span class="diczx2">赫胥黎</span>著、 <span class="diczx2">严复</span>译<span class="smcs">《天演论》</span></p><p><span class="cino">(5)</span> 又如:诞妄(虚妄不实);诞谩(荒诞不实)</p><p><span class="cino">(6)</span> 大 <span class="encs">[great]</span></p><p><span class="diczx1">诞告万方。——<span class="smcs">《书·汤诰》</span>传:<span class="yhcs">“诞，大也。”</span></span></p><p><span class="diczx1">诞先登于岸。——<span class="smcs">《诗·大雅·皇矣》</span></span></p><p><span class="cino">(7)</span> 又如:诞命(大命，承受天命);诞章(弘大的法度典章)</p><p><span class="cino">(8)</span> 恣放，放肆 <span class="encs">[unbridled]</span>。如:荒诞(极不真实;极不近情理);诞放(狂放不拘);诞傲(放诞倨傲)</p><p><strong>词性变化</strong></p><p>◎ <strong>诞</strong></p><p>誕 <span class="dicpy">dàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dàn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 生育;出生 <span class="encs">[give birth to;be born]</span></p><p><span class="diczx1">上诞日，不纳中外之贡。——<span class="smcs">《旧唐书·德宗纪》</span></span></p><p><span class="diczx1">庆门诞瓦，寒舍得雄。——<span class="smcs">《剪灯余话》</span></span></p><p><span class="cino">(2)</span> 又如:诞辰;诞日;诞育(生育);诞圣(诞生)</p><p>◎ <strong>诞</strong></p><p>誕 <span class="dicpy">dàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dàn.mp3"></a></span> </p><p>〈名〉</p><p>生日，出生的那一天 <span class="encs">[birthday]</span>。如:华诞;寿诞</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/诞辰">诞辰</a></span><span class="crefe"><a                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/8A95.svg">						  <p>【廣韻】徒旱切【集韻】【韻會】蕩旱切【正韻】徒亶切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音袒。【說文】詞誕也。【徐曰】妄爲大言也。【廣韻】欺也。【書·無逸】乃逸乃諺旣誕。【蔡傳】誕妄。【孔傳】欺誕。【史記·扁鵲傳】先生得無誕之乎。【荀子·修身篇】易言曰誕。【說苑·尊賢篇】口銳者多誕而寡信。</p><p>又【正韻】放也。【左傳·昭元年】伯州犁曰：子姑憂子晢之欲背誕也。【註】放誕也。【晉書·羊曼傳】阮孚爲誕伯。</p><p>又【爾雅·釋詁】大也。【書·大禹謨】帝乃誕敷文德。【傳】大也。【詩·大雅】誕彌厥月。【毛傳】大也。【朱傳】發語辭。</p><p>又【字彙】闊也。【詩·國風】旄丘之葛兮，何誕之節兮。【箋】土氣緩，則葛生闊節。</p><p>又【廣韻】育也。【後漢·裴楷傳】昔文王一妻，誕致十子。【晉書·袁宏傳】誕靈物以瑞德。</p><p>又【玉篇】天子生曰降誕。【後漢·虞美人傳】誕生聖皇。【舊唐書·德宗紀】上誕日不納中外之貢。</p><p>又【廣雅】信也。</p><p>又誕馬。【唐書·儀衞志】一品鹵簿有誕馬六。【宋史·儀衞志】誕馬，散馬也。</p><p>又國名。【魏志·東夷傳】有古誕者國。</p><p>又獸名。詳訛字註。</p><p>又【正韻】杜晏切，袒去聲。亦放也。與訑同。　【集韻】亦作唌。</p>
