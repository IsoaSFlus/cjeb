                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>逼</strong></p><p>偪、畐 <span class="dicpy">bī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bī.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，畐( <span class="dicpy">fú</span>)声。本义:接近，靠近)</p><p><span class="cino">(2)</span> 同本义(仅表示空间距离的接近、靠近) <span class="encs">[approach;get close to;press on towards]</span></p><p><span class="diczx1">逼，近也。——<span class="smcs">《说文新附》</span></span></p><p><span class="diczx2">秦</span>兵逼 <span class="diczx2">淝水</span>而陈。(陈:布阵)——<span class="smcs">《资治通鉴》</span></p><p><span class="diczx1">鸡健进，逐逼之，虫已在爪下矣。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="diczx1">不可偪也。——<span class="smcs">《国语·郑语》</span></span></p><p><span class="diczx1">是故入小而不偪。——<span class="smcs">《淮南子·兵略》</span></span></p><p><span class="diczx1">逼，近也。——<span class="smcs">《小尔雅·广诂》</span></span></p><p><span class="cino">(3)</span> 又如:逼水(迫近水边);逼曙(天快亮的时候);逼战(近战)</p><p><span class="cino">(4)</span> 逼迫，即紧紧催促 <span class="encs">[compel;force]</span></p><p><span class="diczx1">又</span><span class="diczx2">荆州</span>之民附 <span class="diczx2">操</span>者，逼兵势耳。——<span class="smcs">《资治通鉴》</span></p><p><span class="diczx1">其家逼之，乃投水而死。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="diczx1">心虑人逼取。——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《黄生借书说》</span></p><p><span class="cino">(5)</span> 又如:逼主(臣子胁迫君主);逼取(强迫索取);逼索(逼取。以强力索取);逼人太甚(过分逼迫人而不留余地)</p><p><span class="cino">(6)</span> 驱逐;追赶 <span class="encs">[drive;pursue]</span></p><p><span class="diczx1">鸡健进，逐逼之。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(7)</span> 又如:逼逐(驱逐);逼邪(驱邪)</p><p><strong>词性变化</strong></p><p>◎ <strong>逼</strong> <span class="dicpy">bī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" class="diczx2">曹植</span><span class="smcs">《七启》</span></p><p><span class="cino">(2)</span> 又如:逼窄(很狭窄。同逼侧);逼侧(相迫近。即狭窄。也作逼仄);逼邻(近邻。紧邻)</p><p><span class="cino">(3)</span> 危急，紧急 <span class="encs">[urgent]</span></p><p><span class="diczx1">粮食乏绝，进退逼急。——<span class="smcs">《后汉书·董卓传》</span></span></p><p><span class="cino">(4)</span> 又如:逼切(迫切，急切);逼急(急迫)</p><p>◎ <strong>逼</strong> <span class="dicpy">bī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/bī.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 很，程度深 <span class="encs">[very]</span></p><p><span class="diczx1">山石似马，望之逼真。——<span class="smcs">《水经注·沔水》</span></span></p><p><span class="cino">(2)</span>
