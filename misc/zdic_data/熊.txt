                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>熊</strong></p><p><span class="dicpy">xióng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xióng.mp3"></a></span>  ㄒㄩㄥˊ</span></p><ol><li>哺乳动物，体大，尾短，四肢短而粗，脚掌大，能直立行走，也能攀树，种类很多，有“棕熊”、“白熊”、“黑熊”等：狗～（即“黑熊”）。～掌。～白（熊背上的脂肪，白色，珍贵食品）。～胆（熊的胆，可入药）。～包（喻无能的人，废物）。</li><li>方言，指斥责：挨了一顿～。</li><li>姓。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>熊</strong> <span class="dicpy">xióng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xióng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从能，炎省声。能，<span class="yhcs">“熊”</span>。本义:动物名)</p><p><span class="cino">(2)</span> 属于食肉目熊科(Ursidae)的一种大型笨重的哺乳动物 <span class="encs">[bear]</span></p><p><span class="diczx1">熊兽似豕，山居各蛰。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">熊虎丑，其子狗。又罴如熊，黄白文。——<span class="smcs">《尔雅》</span></span></p><p><span class="diczx1">黄能入于寝门。——<span class="smcs">《国语·晋语》</span>。字误能，按，罴即熊类之大而猛者，能拔树木，今俗谓之人熊。</span></p><p><span class="cino">(3)</span> 有长而粗的毛，退化的尾，脚掌大，既食肉，也食果实和昆虫，虽然通常笨拙迟钝，也能在短距离内迅速行动，尤其是在崎岖或陡峭的地形上。又指使人感到像熊一样的人;特指在粗暴无礼、笨拙粗野、呆滞强壮方面。如:熊罴(熊和罴。指猛兽;常比喻勇猛之士);熊罴之兆(指生儿子的兆头);熊足(熊蹯、熊掌。熊的脚掌。一种珍贵的食品);熊席(熊皮坐席)</p><p><span class="cino">(4)</span> 古代以熊皮为饰的箭靶 <span class="encs">[target decorated with bear-fur]</span></p><p><span class="diczx1">有熊侯。——<span class="smcs">《仪礼·乡射礼》</span></span></p><p><span class="cino">(5)</span> 又如:熊侯(饰以熊皮的箭靶)</p><p><span class="cino">(6)</span> 古州名 <span class="encs">[Xiong prefecture]</span></p><p><span class="diczx2">熊</span>、 <span class="diczx2">谷</span>二州在度内，不取，敦邻好也。——<span class="smcs">《新唐书》</span></p><p><span class="cino">(7)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>熊</strong> <span class="dicpy">xióng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xióng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“雄”</span>。雄性 <span class="encs">[male]</span></p><p><span class="diczx1">及赞戎律，熊武斯裁。——<span class="smcs">《北史·羊祉传》</span></span></p><p><span class="cino">(2)</span> 软弱，无能 <span class="encs">[weak;incompetent;incapable]</span>。如:熊包;人熊被人欺，马熊被人骑</p><p>◎ <strong>熊</strong> <span class="dicpy">xióng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr"                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/718A.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2A3DB.svg'><span class='k_txt'>𪏛</span>【集韻】【韻會】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>胡弓切，音雄。【說文】熊獸似豕，山居冬蟄，从能炎省。【爾雅·釋獸】熊，虎醜。【疏】醜類也。【書·禹貢】熊羆狐狸織皮。【詩·小雅】維熊維羆，男子之祥。</p><p>又赤熊。【爾雅·釋獸】魋如小熊。【註】今建平山中有此獸。俗呼爲赤熊。</p><p>又【史記·天官書】熊熊靑色有光。【山海經】槐江之山，南望崑崙。其光熊熊，其氣䰟䰟。</p><p>又山名。【書·禹貢】熊耳外方桐柏。【史記·黃帝紀】南至于江登熊湘。【註】二山名。熊卽熊耳山。</p><p>又【封禪書】南伐至于召陵，登熊山。</p><p>又【山海經】熊山有穴焉，熊之穴，恆出神人，夏啓而冬閉。</p><p>又西熊，侯國。見【史記·建元以來王子侯者年表】　又人名。【書·舜典】益拜稽首讓于朱虎熊羆。</p><p>又【史記·五帝紀】黃帝爲有熊。【韻會】有熊，國名。黃帝所都。</p><p>又【廣韻】亦姓。【左傳】賢者熊宜僚。</p><p>又複姓。【左傳】楚大夫熊率且比。</p><p>又【集韻】矣㱡切。義同。【唐韻正】熊音羽陵反。春秋宣八年，葬我小君敬嬴。公羊穀梁傳<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>作頃熊。頃音近敬，熊音近嬴。正義不得其解，乃云：一人有兩號。非矣。左傳昭七年，正義曰：張叔皮論云：賔爵下革，田鼠上騰，牛哀虎變，鯀化爲熊，久血爲燐，積灰生蠅王劭曰：古人讀熊于陵反。張叔用舊音也。熊當改入蒸韻。</p><p>又【集韻】囊來切。同能。【左傳·昭七年】晉侯夢黃能入于寢門。【釋文】熊亦作能。如字，一音奴來反。三足鼈也。詳肉部能字註。　【集韻】或作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24817.svg'><span class='k_txt'>𤠗</span>能<img class='lazy k_img' data-original='//img.zdic.net/song/cn/27C2F.svg'><span class='k_txt'>𧰯</span>。</p>
