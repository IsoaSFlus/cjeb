                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>絮</strong></p><p><span class="dicpy">xù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xù.mp3"></a></span>  ㄒㄩˋ</span></p><ol><li>棉花的纤维：棉～。被～。</li><li>古代指丝的纤维，特指熟丝。</li><li>像棉絮的东西：花～。芦～。柳～。</li><li>在衣被等物里铺棉花、丝绵等：～被子。～棉袄。</li><li>连续重复，惹人厌烦：～叨。～烦。～聒。～～叨叨。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> waste cotton, raw silk or cotton</p><p><span class="z_ts2">德语</span> Xu  (Eig, Fam)</p><p><span class="z_ts2">法语</span> ouate,duvet,ouater,rembourrer</p></div>

                    <h2 class="h2_entry"><span class="orth">絮</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>絮</strong> <span class="dicpy">xù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从糸( <span class="dicpy">mì</span>)，如声。本义:粗丝绵)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[thick silk floss]</span></p><p><span class="diczx1">絮，敝绵也。——<span class="smcs">《说文》</span>。按，好者为绵，恶者为絮。</span></p><p><span class="diczx1">絮三斤。——<span class="smcs">《汉书·高帝纪》</span>。古无木绵，以絮纳袷中，谓之装褚，曰袍。</span></p><p><span class="diczx1">剖其中，干若败絮。——</span><span class="diczx2">明</span>· <span class="diczx2">刘基</span><span class="smcs">《卖柑者言》</span></p><p><span class="diczx1">败絮其中。</span></p><p><span class="cino">(3)</span> 又如:絮衣(绵衣);絮缯(丝帛之类的总称)</p><p><span class="cino">(4)</span> 经过弹制的棉花胎 <span class="encs">[(cotton) wadding]</span></p><p><span class="diczx1">他们的婴孩没有棉衣，只好成天躺在土炕上那一堆破絮里。——</span><span class="diczx2">茅盾</span><span class="smcs">《<span class="yhcs">“战时景气”</span>宠儿——宝鸡》</span></p><p><span class="cino">(5)</span> 又如:絮帛(棉絮与布帛);絮棉;絮缕(棉絮与丝缕。泛指微细之物)</p><p><span class="cino">(6)</span> 像絮一样轻柔、洁白的东西 <span class="encs">[sth. resembling cotton]</span></p><p><span class="diczx1">无风杨柳漫天絮，不雨棠梨满地花。——</span><span class="diczx2">范成大</span><span class="smcs">《碧瓦》</span></p><p><span class="diczx1">山河破碎风飘絮。——</span><span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《过零丁洋》</span></p><p><span class="cino">(7)</span> 又如:芦絮;絮暖(指春暖絮飞之时);柳絮(柳花之飞絮)</p><p><span class="cino">(8)</span> 棉花 <span class="encs">[cotton]</span></p><p><span class="diczx1">蛮夷不蚕，采木绵为絮。——<span class="smcs">《艺文类聚》</span></span></p><p><span class="cino">(9)</span> 头上巾 <span class="encs">[scarf]</span></p><p><span class="diczx2">巴蜀</span>名头上巾为冒絮。——<span class="smcs">《续方言疏证》</span></p><p><span class="diczx1">又如:絮巾(巾絮。古代的一种头巾)</span></p><p><strong>词性变化</strong></p><p>◎ <strong>絮</strong> <span class="dicpy">xù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 在衣服、被褥里铺丝棉、棉花等物 <span class="encs">[wad with cotton]</span></p><p><span class="diczx1">明朝驿使发，一夜絮征袍。——<span class="smcs">《子夜吴歌四首》</span></span></p><p><span class="cino">(2)</span> 又如:絮衣(内铺丝棉的衣服);絮纩(棉袄，棉袍);絮袄(内充丝绵或棉絮的冬季上衣)</p><p>◎ <strong>絮</strong> <span class="dicpy">xù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xù.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 优弱寡断 <span class="encs">[undetermined]</span></p><p><span class="diczx1">时有一事，</span><span class="diczx2">富公</span>疑之，久而不决。 <span class="diczx2">韩</span>曰:公又絮。—— <span class="diczx2">清</span>· <span class="diczx2">翟灏</span><span class="smcs">《通俗编》</span></p><p><span class="cino">(2)</span> 说话啰唆 <span class="encs">[long-winded]</span></p><p><span class="diczx1">今又以言语烦琐为絮，所谓絮叨叨是也。——<span class="smcs">《通俗编·言笑》</span></span></p><p><span class="cino">(3)</span>
