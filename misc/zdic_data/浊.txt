                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>浊</strong></p><p>濁 <span class="dicpy">zhuó</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuó.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从水，蜀声。本义:浊水) 同本义 <span class="encs">[Zhuo River]</span></p><p><span class="diczx2">浊水</span>出 <span class="diczx2">齐郡</span><span class="diczx2">厉伪山</span>，东北入钜定。——<span class="smcs">《说文》</span>。今曰 <span class="diczx2">北阳水</span>。</p><p><span class="cino">(2)</span> 源出山东省益都县西南，北流注入小清河</p><p><strong>词性变化</strong></p><p>◎ <strong>浊</strong></p><p>濁 <span class="dicpy">zhuó</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuó.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 液体浑浊。与<span class="yhcs">“清”</span>相对 <span class="encs">[muddy;turbid]</span></p><p><span class="diczx1">载清载浊。——<span class="smcs">《诗·小雅·四月》</span></span></p><p><span class="diczx1">浑兮其若浊——<span class="smcs">《老子》</span></span></p><p><span class="diczx1">蝉蜕于浊秽。——<span class="smcs">《史记·屈原贾生列传》</span></span></p><p><span class="diczx1">浊浪排空。——</span><span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="cino">(2)</span> 又如:浊河(混浊的河流。特指黄河);浊酒(用糯米、黄米等酿制的酒，较混浊)</p><p><span class="cino">(3)</span> 脏，不干净 <span class="encs">[dirty]</span></p><p><span class="diczx1">雨门延风凉，洗我昏浊肌。——</span><span class="diczx2">姚合</span><span class="smcs">《过张邯郸庄》</span></p><p><span class="cino">(4)</span> 又如:浊醪(浊酒);浊污(污秽);浊秽(污浊。亦比喻丑恶鄙陋之事物);浊溷(污浊)</p><p><span class="cino">(5)</span> 庸俗 <span class="encs">[vulgar]</span></p><p><span class="diczx1">你不是凡胎浊骨， 迷本性人间受苦。——</span><span class="diczx2">马致远</span><span class="smcs">《黄粱梦》</span></p><p><span class="cino">(6)</span> 又如:浊物(俗物);浊气(俗气);浊碎(指庸常琐碎之事);浊质(凡庸的姿质)</p><p><span class="cino">(7)</span> 品行坏，卑劣 <span class="encs">[bad]</span></p><p><span class="diczx1">举世皆浊我独清，众人皆醉我独醒，是以见放。——<span class="smcs">《楚辞》</span></span></p><p><span class="cino">(8)</span> 又如:浊化(变得贪鄙、卑劣);浊俗(卑污庸俗);浊操(卑污的操行);浊气(不良风气)</p><p><span class="cino">(9)</span> 发音时声带振动的 <span class="encs">[voiced]</span>。如:浊辅音</p><p><span class="cino">(10)</span> 佛教语。指尘世 <span class="encs">[worldly]</span>。如:浊劫(佛教语。指尘世);浊界(佛教语。指尘世);浊恶世(佛教语。指五浊恶世)</p><p><span class="cino">(11)</span> 混乱 <span class="encs">[confused]</span></p><p><span class="diczx1">科条既备，民多伪态;书策稠浊百姓不足。——<span class="smcs">《战国策》</span></span></p><p><span class="cino">(12)</span> 又如:浊代(混乱的时代);浊乱(搅扰使之混乱;混乱);浊躁(紊乱浮躁)</p><p><span class="cino">(13)</span> 昏暗不明 <span class="encs">[dim]</span></p><p><span class="diczx1">浊明外景，清明内景——<span class="smcs">《荀子·解蔽》</span></span></p><p><span class="diczx1">阴虹浊太阳，前星遂沦匿。——</span><span class="diczx2">李白</span><span class="smcs">《商山四皓》</span></p><p><span class="cino">(14)</span> 又如:浊才料(糊涂;不省事)</p><p><strong>常用词组</strong></p><hr class="dichr"                           『說文解字』<hr class="dichr">
                          <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/6FC1.svg">                          <p>水。出齊郡厲嬀山，東北入鉅定。从水蜀聲。直角切</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <span class="jieshi">濁水。出齊郡</span>上文不言郡。此字葢淺人增之。<span class="jieshi">厲嬀山。東北入鉅定。</span>厲當作廣。嬀當作爲。皆字之誤。齊郡廣、見前志。後志作齊國廣。今山東靑州府益都縣、縣西南四里有廣縣故城是也。前志廣下曰。爲山、濁水所出。東北至廣饒入鉅定。水經注曰。淄水、又東北逕廣饒縣故城南。又東北馬車瀆水注之。水首受巨淀。淀卽濁水所注也。吕忱曰。濁水、一名溷水。出廣縣爲山。一名冶嶺山。東北流逕廣固城西。城在廣縣西北四里。又東北流逕堯山東。又東北流逕東陽城北。合陽水。卽長沙水也。又北逕臧氏臺西。又北逕益城西。又北流注巨淀。今北陽水源出益都縣西南九迴山。卽古爲山。東北流逕城北。又東北逕壽光縣四十里。又北入淸水泊。卽古濁水也。<span class="jieshi">从水。蜀聲。</span>直角切。三部。按濁者、淸之反也。詩曰。涇以渭濁。又曰。載淸載濁。                          </div>
