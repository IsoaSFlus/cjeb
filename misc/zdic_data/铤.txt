						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>铤</strong></p><p><strong>（<span class="diczx3">鋌</span>）</strong></p><p><span class="dicpy">tǐng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǐng.mp3"></a></span> ㄊㄧㄥˇ</span></p><ol><p>　◎ 快走的样子：～而走险。</p></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>铤</strong></p><p><strong>（<span class="diczx3">鋌</span>）</strong></p><p><span class="dicpy">dìng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dìng.mp3"></a></span> ㄉㄧㄥˋ</span></p><ol><li>古代称未经冶铸的铜铁。</li><li>箭头装入箭干的部分。</li><li>古同“<span class="diczx3">锭</span>”，专门铸成的各种形态的金银块，用以货币流通。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>铤</strong></p><p>鋌 <span class="dicpy">dìng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dìng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 铜铁矿石 <span class="encs">[copper or iron ore]</span></p><p><span class="diczx1">铤，铜铁朴也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(2)</span> 铜铁质的坯料 <span class="encs">[copper or iron stock]</span></p><p><span class="diczx1">金铤，</span><span class="diczx2">许叔重</span>注<span class="smcs">《淮南子》</span>云:<span class="yhcs">“铤者，金银铜等未成器，铸作片，名曰铤。”</span>——<span class="smcs">《一切经音义》</span></p><p><span class="cino">(3)</span> 熔铸成条块等固定形状的金银，其重数两至数十两不等 <span class="encs">[ingot]</span></p><p><span class="diczx1">旧例银每铤五十两，其直百贯。——<span class="smcs">《金史》</span></span></p><p><span class="cino">(4)</span> 锭子，纺车或纺纱机上绕纱的机件 <span class="encs">[spindle]</span></p><p><span class="diczx1">摇车转铤，而纺于竹管之上。——<span class="smcs">《天工开物》</span></span></p><p><span class="cino">(5)</span> 另见 <span class="dicpy">tǐng</span> <span <strong>铤</strong></p><p>鋌 <span class="dicpy">tǐng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǐng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从金，廷声。本义:未经冶铸的铜铁。读 <span class="dicpy">dìng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dìng.mp3"></a></span> )</p><p><span class="cino">(2)</span> 快跑的样子<span class="encs">[run rapidly]</span>。</p><p><span class="cino">(3)</span> 如:铤走(快跑);铤鹿(疾奔之鹿)</p><p><span class="cino">(4)</span> 空;尽 <span class="encs">[exhausted;empty]</span></p><p><span class="diczx1">铤，尽也。物空尽者曰铤。——<span class="smcs">《方言》</span></span></p><p><span
