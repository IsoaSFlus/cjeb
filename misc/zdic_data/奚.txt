data-src-mp3="//img.zdic.net/audio/zd/py/xī.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。据甲骨文，左为手(爪)，右为绳索捆着的人。本义:奴隶，又专指女奴)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[slave]</span></p><p><span class="diczx1">凡奚隶聚而出入者，则司牧之。——<span class="smcs">《周礼·秋官·禁暴氏》</span>。</span><span class="diczx2">孙诒让</span>正义:<span class="yhcs">“奚为女奴，隶为男奴也。”</span></p><p><span class="diczx1">酒人奄十人，女酒三十人，奚三百人。——<span class="smcs">《周礼·天官》</span></span></p><p><span class="cino">(3)</span> 又如:奚女(婢女);奚奴(女奴。今泛指奴仆);奚童(奚僮。未成年男仆);奚隶(男女奴隶)</p><p><span class="cino">(4)</span> 大腹 <span class="encs">[big belly]</span></p><p><span class="diczx1">奚，大腹也。从大，籀文系省声。——<span class="smcs">《说文》</span></span></p><p><span class="diczx2">晋</span><span class="diczx2">祁</span>奚。——<span class="smcs">《左传》</span></p><p><span class="cino">(5)</span> 又如:奚奚(大腹便便的样子)</p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>奚</strong> <span class="dicpy">xī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xī.mp3"></a></span> </p><p>〈代〉</p><p><span class="cino">(1)</span> 什么，何 <span class="encs">[what]</span></p><p><span class="diczx1">奚取焉。——</span><span class="diczx2">唐</span>· <span class="diczx2">韩愈</span><span class="smcs">《朱文公校昌黎先生集》</span></p><p><span class="diczx1">复奚疑。——</span><span class="diczx2">晋</span>· <span class="diczx2">陶渊明</span><span class="smcs">《归去来兮辞》</span></p><p><span class="diczx1">为我属地，我自济之，与</span><span class="diczx2">俄</span>奚涉焉?——<span class="smcs">《清史稿》</span></p><p><span class="cino">(2)</span> 又如:奚如(何如);奚距(何尝)</p><p><span class="cino">(3)</span> 哪个 <span class="encs">[which]</span></p><p><span class="diczx1">奚方能已之乎?——<span class="smcs">《列子》</span></span></p><p><span class="cino">(4)</span> <span class="encs">[指状态、条件、情况]</span>如何、怎样;<span class="encs">[数量、价值]</span>多少;<span class="encs">[在某种情况、时间或场合]</span> 会怎么样 <span class="encs">[how]</span></p><p><span class="diczx1">奚暇治礼义哉。——<span class="diczx1">奚喋喋为。——</span><span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="diczx1">其奚决焉。——</span><span class="diczx2">清</span>· <span class="diczx2">刘开</span><span class="smcs">《问说》</span></p><p><span class="cino">(5)</span> 因何缘故 <span class="encs">[why]</span></p><p><span class="diczx1">骨肉奚无情?——<span
