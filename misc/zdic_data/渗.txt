						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>渗</strong></p><p><strong>（<span class="diczx3">滲</span>）</strong></p><p><span class="dicpy">shèn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shèn.mp3"></a></span> ㄕㄣˋ</span></p><ol><p>　◎ 液体慢慢地透入或漏出：～透。～水。～漏。～漉（水透漏下滴）。～漓。</p></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>渗</strong></p><p>滲 <span class="dicpy">shèn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shèn.mp3"></a></span> </p><p>< 动></p><p><span class="cino">(1)</span> (形声。从水，参( <span class="dicpy">shēn</span>)声。本义:水往下渗透)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[ooze;seep]</span></p><p><span class="diczx1">渗，下漉也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">滋液渗漉。——<span class="smcs">《汉书·司马相如传》</span>。注:<span class="yhcs">“谓润泽下究。”</span></span></p><p><span class="diczx1">泽渗离而下降。——<span class="smcs">《汉书·扬雄传》</span></span></p><p><span class="diczx1">百年老屋，尘泥渗漉，雨泽下注。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="cino">(3)</span> 又如:渗沥(滴漏);渗泄(小便);渗淫(小水。指逐渐渗透进去的少量水液);渗渗(因畏惧而冷汗直流);渗涸(渗漏而干涸)。又指液体慢慢地透出或漏出</p><p><span class="diczx1">一粒粒黄豆大的冷汗珠，不断从他额头渗出来。——<span class="smcs">《鞠躬尽瘁》</span></span></p><p><span class="cino">(4)</span> 水枯竭 <span class="encs">[dry up]</span></p><p><span class="diczx1"><span class="encs">[到产之]</span>自</span><span class="diczx2">淮</span>入 <span class="diczx2">泗</span>， <span class="diczx2">泗水</span>渗，日裁行十里。——<span class="smcs">《南史》</span></p><p><span class="cino">(5)</span> 其他事物渐渐侵入 <span class="encs">[penetrate]</span></p><p><span class="diczx1">远陂春日渗，犹有水禽飞。——</span><span class="diczx2">唐</span>· <span class="diczx2">司空图</span><span class="smcs">《独望》</span></p><p><span class="cino">(6)</span> 人从空隙中挤进、穿入 <span class="encs">[squeeze in]</span></p><p><span class="diczx1">那儿正中围集着一大堆人，渗进去一看，原来也就是打诗谜的。——</span><span class="diczx2">郭沫若</span><span class="smcs">《创造十年续篇》</span></p><p><span class="cino">(7)</span> 渗合，混合 <span class="encs">[mix]</span>。如:渗和(掺合混杂);渗杂</p><p><span class="cino">(8)</span> 同<span class="yhcs">“瘮”</span>。可怕 <span class="encs">[fear]</span>。如:渗人(可怕，使人害怕);渗濑(渗渗濑濑。丑陋，凶恶)</p><p><span class="cino">(9)</span> 睡 <span class="encs">[sleep]</span></p><p><span class="diczx1">原来他方才合着桃仁杏花引子服一丸子乌金丸，躺在屋里就渗着了。——<span class="smcs">《儿女英雄传》</span></span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a
