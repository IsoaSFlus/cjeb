<p><strong>（<img class="jbjs_ico" src="/images/yico_ft.svg" title="繁体字"><span class="diczx3">數</span>）</strong></p>
<p><span class="dicpy">sh&ugrave;  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sh&ugrave;.mp3"></a></span>  ㄕㄨˋ</span></p>
<hr class="dichr" />
<ol>
	<li>表示、划分或计算出来的量：～目。～量。～词。～论（数学的一支，主要研究正整数的性质以及和它有关的规律）。～控。</li>
	<li>几，几个：～人。～日。</li>
	<li>技艺，学术：&ldquo;今夫弈之为～，小～也&rdquo;。</li>
	<li>命运，天命：天～。气～。</li>
</ol>
<p>● <strong>数</strong></p>
<p><strong>（<img class="jbjs_ico" src="/images/yico_ft.svg" title="繁体字"><span class="diczx3">數</span>）</strong></p>
<p><span class="dicpy">shǔ  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǔ.mp3"></a></span>  ㄕㄨˇ</span></p>
<hr class="dichr" />
<ol>
	<li>一个一个地计算：不可胜～。～九。</li>
	<li>比较起来突出：～得着。</li>
	<li>责备，列举过错：～落。</li>
	<li>谈论，述说：～说。～典忘祖（喻忘掉自己本来的情况，亦喻对于本国历史的无知）。</li>
</ol>

<p>數 <span class="dicpy">shǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǔ.mp3"></a></span> </p>

<p>◎ <strong>数</strong></p>
<p>數 <span class="dicpy">sh&ugrave;</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sh&ugrave;.mp3"></a></span> </p>
<p>〈名〉</p>
<p><span class="cino">(1)</span> 数目;数量 <span class="encs">[figure;number]</span></p>
<p><span class="diczx1">窃怜爱之，愿令得补黑衣之数。&mdash;&mdash;<span class="smcs">《战国策&middot;赵策》</span></span></p>
<p><span class="diczx1">五陵少年争缠头，一曲红绡不知数。&mdash;&mdash;</span><span class="diczx2">唐</span>&middot; <span class="diczx2">白居易</span><span class="smcs">《琵琶行(并序)》</span></p>
<p><span class="diczx3">又如:报数(报告数目);数计(以数字来计算);数珠儿(念珠);可被2除尽的数;自然数、整数、有理数、无理数、实数或复数;基数;单数;复数</span></p>
<p><span class="cino">(2)</span> 底细 <span class="encs">[base]</span>。</p><p><span class="diczx3">如:心里没有数;心中都有个数</span></p>
<p><span class="cino">(3)</span> 道数，方法 <span class="encs">[method]</span></p>
<p><span class="diczx1">故为国之数，务在垦草。&mdash;&mdash;<span class="smcs">《商君书》</span></span></p>
<p><span class="cino">(4)</span> 技艺，如博弈的技艺 <span class="encs">[art]</span></p>
<p><span class="diczx1">今夫弈之为数，小数也。&mdash;&mdash;<span class="smcs">《孟子&middot;告子上》</span></span></p>
<p><span class="cino">(5)</span> <span class="encs">[古]</span>∶算术，古代六艺之一 <span class="encs">[arithmetic]</span></p>
<p><span class="diczx1">三曰六艺:礼、乐、射、御、书、数。&mdash;&mdash;<span class="smcs">《周礼&middot;地官&middot;大司徒》</span></span></p>
<p><span class="cino">(6)</span> 气数;命运 <span class="encs">[destiny;fate]</span></p>
<p><span class="diczx2">卫青</span>不败由天幸， <span class="diczx2">李广</span>无功缘数奇。&mdash;&mdash; <span class="diczx2">唐</span>&middot; <span class="diczx2">王维</span><span class="smcs">《老将行》</span></p>
<p><span class="diczx1">向使三国各爱其地，</span><span class="diczx2">齐</span>人勿附于 <span class="diczx2">秦</span>，刺客不行，良将犹在，则胜负之数，存亡之理，当与 <span class="diczx2">秦</span>相较，或未易量。&mdash;&mdash; <span class="diczx2">宋</span>&middot; <span class="diczx2">苏洵</span><span class="smcs">《六国论》</span></p>
<p><span class="diczx3">又如:数奇(命运不佳;遭遇不顺);数命(命运);数尽禄终(死亡);数劫(厄运)</span></p>
<p><span class="cino">(7)</span> 策略;权术 <span class="encs">[tactics]</span></p>
<p><span class="diczx1">以计求便，以数取利。&mdash;&mdash;</span><span class="diczx2">汉</span>&middot; <span class="diczx2">王充</span><span class="smcs">《论衡》</span></p>
<p><span class="cino">(8)</span> 规律;必然性 <span class="encs">[law]</span></p>
<p><span class="diczx2">汉</span>世外戚，自 <span class="diczx2">东</span>、 <span class="diczx2">西京</span>十有余族，非徒豪横盈极，自取灾故，必于贻衅后主，以至颠败者，其数有可言焉。&mdash;&mdash;<span class="smcs">《后汉书》</span></p>
<p><span class="cino">(9)</span> 道理 <span class="encs">[reason]</span></p>
<p><span class="diczx1">夫以疏远与近爱信争，其数不胜也。&mdash;&mdash;<span class="smcs">《韩非子》</span></span></p>
<p><span class="cino">(10)</span> 礼数;仪节 <span class="encs">[rites]</span></p>
<p><span class="diczx1">今嬖宠之丧，不敢择位，而数于守适。&mdash;&mdash;<span class="smcs">《左传&middot;庄公十八年》</span></span></p>
<p><span class="cino">(11)</span> 特指方术，如占卜之类 <span class="encs">[divination]</span></p>
<p><span class="diczx1">数有所不逮，神有所不通。&mdash;&mdash;<span class="smcs">《楚辞&middot;卜居》</span></span></p>
<p><span class="diczx3">又如:数家(精于术数的人);数术(术数)</span></p>
