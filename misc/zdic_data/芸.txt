                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>芸</strong> <span class="dicpy">yún</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yún.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从艸，云声。本义:香草名，也叫<span class="yhcs">“芸香”</span>)</p><p><span class="cino">(2)</span> 同本义，多年生草本植物，其下部为木质，故又称芸香树。叶互生，羽状深裂或全裂，花黄色，香气浓郁，可入药 <span class="encs">[rue]</span></p><p><span class="diczx1">芸，芸草也，似目宿。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">芸草可以死复生。——<span class="smcs">《淮南子·王说》</span></span></p><p><span class="diczx1">阳华之芸。——<span class="smcs">《吕氏春秋·本味》</span>。注:<span class="yhcs">“芳菜也。”</span></span></p><p><span class="diczx1">芸始生。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="diczx2">元载</span>末年，造芸辉堂于私第。芸辉，香草名也。—— <span class="diczx2">唐</span>· <span class="diczx2">苏鹗</span><span class="smcs">《杜阳杂编》</span></p><p><span class="cino">(3)</span> 又如:芸帙(书籍。因芸香可驱蠹虫，书卷中多置之，故名。同<span class="yhcs">“芸编”</span>);芸窗(书斋。古人藏书用芸香避蠹虫，故借芸以称书斋);芸草(芸香);芸编(指书籍。芸，香草，置书页内可以辟蠹，故称);芸馆(书斋);芸签(书签。亦借指书籍);芸扃(藏书处)</p><p><span class="cino">(4)</span> 芸台 <span class="encs">[imperial liberary]</span>，古代藏书之所，亦指掌管图书的官署，即秘书省。如:芸局(藏书处。也指掌管图书的官署，即秘书省)</p><p><strong>词性变化</strong></p><p>◎ <strong>芸</strong> <span class="dicpy">yún</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yún.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 除掉杂草。同<span class="yhcs">“耘”</span> <span class="encs">[weed]</span></p><p><span class="diczx1">植其仗而芸。——<span class="smcs">《论语·微子》</span></span></p><p><span class="diczx1">先雨芸耨。——<span class="smcs">《管子·小匡》</span></span></p><p><span class="diczx1">父子夫妇终年耕芸。——<span class="smcs">《汉书·王莽传》</span></span></p><p><span class="cino">(2)</span> 又如:芸夫(即农夫);芸锄(芸鉏。芟刈;耕耘);芸耨(耘耨。除草;耕种)</p><p>◎ <strong>芸</strong></p><p>蕓 <span class="dicpy">yún</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yún.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 芸苔 <span class="encs">[rape]</span>，二年生草本植物，又叫<span class="yhcs">“油菜”</span>，花黄色，种子可榨油</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">yùn</span></p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/芸豆">芸豆</a></span><span class="crefe"><a href="/hans/芸香">芸香</a></span><span class="crefe"><a href="/hans/芸芸">芸芸</a></span><span class="crefe"><a href="/hans/芸芸众生">芸芸众生</a></span></div>　<p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>芸</strong> <span 						<div class="mob">【<a href="/zd/kx/?zm=申集上" target="_blank">申集上</a>】【<a href="/zd/kx/?zm=申集上&amp;bs=艸" target="_blank">艸</a>】
                              芸·康熙筆画：<a href="/zd/kx/bh/?bh=10" target="_blank">10</a>　·部外筆画：4</div>
                          <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/82B8.svg">						  <p>【廣韻】【集韻】玉分切【韻會】【正韻】于分切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>音雲。【說文】草也，似目宿。【禮·月令】芸始生。【註】芸，香草也。【爾雅翼】芸類豌豆，叢生，其葉極芳香，秋後葉閒微白如粉，南人採置席下，能去蚤蝨。今謂之七里香。【續博物志】《典略》云：芸香辟紙魚蠹，故藏書臺稱芸臺。【成公綏·芸香賦】美芸香之脩潔，合隂陽之淑淸。</p><p>又【急就篇註】芸，芸蒿也，生熟皆可啗。</p><p>又【拾遺記】芳蔬園多異菜，有菜名芸薇，紫色者最繁，一名芸芝。</p><p>又多貌。【老子·道德經】夫物芸芸，各復歸其根。</p><p>又通耘。【論語】植其杖而芸。【何晏註】除草曰芸。</p><p>又音運。【詩·小雅】裳裳者華，芸其黃矣。　【集韻】作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/26C5A.svg'><span class='k_txt'>𦱚</span>。</p><p>（蕓）【唐韻】【集韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>玉分切，音云。【玉篇】蕓薹菜。【本草註】此菜易起薹，須採其薹，則分枝必多，故名蕓薹。淮人謂之薹芥。</p><p>又蕓香草也。【杜陽雜編】元載造蕓輝堂於私第，其香出于闐國，潔白如玉，舂之爲屑以塗壁。　【集韻】或作蒷。</p><p>（蕓）
