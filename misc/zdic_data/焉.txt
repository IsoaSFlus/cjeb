                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>焉</strong></p><p><span class="dicpy">yān  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> ㄧㄢˉ</span></p><ol><li>与介词“<span class="diczx3">于</span>”加代词“<span class="diczx3">是</span>”相当：心不在～。不复出～。</li><li>乃，才：必知乱之所自起，～能治之。</li><li>文言疑问词，怎么，哪儿：且～置土石？</li><li>文言助词：又何戚～。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> thereupon, then; how? why? where?</p><p><span class="z_ts2">德语</span> inwiefern, wie</p><p><span class="z_ts2">法语</span> (employé à la fin d'une phrase)​,comment,seulement,ici,cela</p></div>

                    <h2 class="h2_entry"><span class="orth">焉</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>焉</strong> <span class="dicpy">yān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (象形。小篆字形。像鸟形。本义:焉鸟)</p><p><span class="cino">(2)</span> 鸟名 <span class="encs">[a kind of bird]</span></p><p><span class="diczx1">旦，北而徂山之曲，乃见苍翠一林，其中则楮烟墨宇，椒枥坎地，群焉胙充，飞而不举。——</span><span class="diczx2">唐</span>· <span class="diczx2">黄滔</span><span class="smcs">《唐城客梦》</span></p><p><span class="cino">(3)</span> 通<span class="yhcs">“颜”</span>( <span class="dicpy">yán</span>)。额 <span class="encs">[forehead]</span></p><p><span class="diczx1">身长七尺，面长三尺，焉广三寸，鼻目耳具，而名动天下。——<span class="smcs">《荀子·非相》</span>。</span><span class="diczx2">高亨</span>云:<span class="yhcs">“焉，盖颜之借字。”</span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>焉</strong> <span class="dicpy">yān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> </p><p>〈代〉</p><p><span class="cino">(1)</span> 表示指示，相当于<span class="yhcs">“之”</span> <span class="encs">[it]</span></p><p><span class="diczx1">草木无知，叩焉何益?——</span><span class="diczx2">明</span>· <span class="diczx2">马中锡</span><span class="smcs">《中山狼传》</span></p><p><span class="cino">(2)</span> 又如:心不在焉</p><p><span class="cino">(3)</span> 哪里或那里 <span class="encs">[where]</span></p><p><span class="diczx1">且焉置土石。——<span class="smcs">《列子·汤问》</span></span></p><p><span class="diczx1">富者不能至，而贫者至焉。(焉，代词，那里，指南海。)——</span><span class="diczx2">清</span>· <span class="diczx2">彭端淑</span><span class="smcs">《为学一首示子侄》</span></p><p><span class="cino">(4)</span> 什么 <span class="encs">[what]</span></p><p><span class="diczx1">今王公大人骨肉之亲、无故富贵、面目美好者，焉故必知哉?——<span class="smcs">《墨子》</span></span></p><p><span class="cino">(5)</span> 怎么 <span class="encs">[which]</span></p><p><span class="diczx1">食其禄，焉避其难?——<span class="smcs">《三国志》</span></span></p><p><span class="cino">(6)</span> 兼有介词<span class="yhcs">“于”</span>加代词<span class="yhcs">“此”</span>的语法功能，相当于<span class="yhcs">“于是”</span>、<span class="yhcs">“于此”</span> <span class="encs">[so]</span></p><p><span class="diczx1">二陵焉。——<span class="smcs">《左传·僖公三十三年》</span></span></p><p><span class="diczx1">不择事而问焉。——</span><span class="diczx2">清</span>· <span class="diczx2">刘开</span><span class="smcs">《问说》</span></p><p><span class="diczx1">伏焉。——</span><span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p>◎ <strong>焉</strong> <span class="dicpy">yān</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yān.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 如何 <span class="encs">[how]</span>表示承接上文，得出结论。如:焉能;焉得;焉敢;焉知;焉用</p><p><span class="cino">(2)</span> 于是，就，乃，则 <span class="encs">[then]</span>。表示两件事或数个事接连发生</p><p><span class="diczx2">始皇</span>巡 <span class="diczx2">陇西</span>、 <span class="diczx2">北地</span>，出 <span class="diczx2">鸡头山</span>，过 <span class="diczx2">回中</span>，焉作信宫 <span class="diczx2">渭南</span>。——<span 			<div class="gscdl">——《述而》子曰：“三人行，必有我师<span>焉</span>。”</div><div class="gscdr">先秦·孔子弟子及其再传弟子 《《论语》十二章》</div><div class="gscdl">予独爱莲之出淤泥而不染，濯清涟而不妖，中通外直，不蔓不枝，香远益清，亭亭净植，可远观而不可亵玩<span>焉</span>。</div><div class="gscdr">宋·周敦颐 《爱莲说》</div><div class="gscdl">沧州南一寺临河干，山门圮于河，二石兽并沉<span>焉</span>。</div><div class="gscdr">清·纪昀 《河中石兽》</div><div class="gscdl">孟子曰：“是<span>焉</span>得为大丈夫乎？子未学礼乎？丈夫之冠也，父命之；女子之嫁也，母命之，往送之门，戒之曰：‘往之女家，必敬必戒，无违夫子！’”。</div><div class="gscdr">《孟子·滕文公下》《富贵不能淫》</div><div class="gscdl">其妻献疑曰：“以君之力，曾不能损魁父之丘，如太行、王屋何？且<span>焉</span>置土石？”</div><div class="gscdr">先秦·列御寇  列子《愚公移山》</div><div class="gscdl">寒暑易节，始一返<span>焉</span>。</div><div class="gscdr">先秦·列御寇  列子《愚公移山》</div>      </div>
