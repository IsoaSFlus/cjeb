                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>状</strong></p><p>狀 <span class="dicpy">zhuàng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuàng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从犬，爿( <span class="dicpy">pán</span>)声。用具体的动物<span class="yhcs">“犬”</span>表示事物各具形态。本义:犬形)</p><p><span class="cino">(2)</span> 同本义。引申为形状，形态 <span class="encs">[form;shape]</span></p><p><span class="diczx1">状，犬形也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">且曰献状。——<span class="smcs">《左传·文公十五年》</span>。谓献其骈肋之状也。</span></p><p><span class="diczx1">凡铸金之状。——<span class="smcs">《考工记·桌氏》</span>。</span><span class="diczx2">杜</span>注:<span class="yhcs">“谓形状。”</span></p><p><span class="diczx1">以筒水灌之，始出，状极俊健。——<span class="smcs">《聊斋志异》</span></span></p><p><span class="diczx1">有双石高竦，其状若门。——</span><span class="diczx2">郦道元</span><span class="smcs">《水经注》</span></p><p><span class="diczx1">若啸呼状。——</span><span class="diczx2">明</span>· <span class="encs">[state;condition]</span></p><p><span class="diczx1">舍人弟上变，告信欲反状于</span><span class="diczx2">吕后</span>。——<span class="smcs">《史记·淮阴侯列传》</span></p><p><span class="diczx1">欲死</span><span class="diczx2">秦</span>军状。——<span class="smcs">《史记·魏公子列传》</span></p><p><span class="diczx1">以状语</span><span class="diczx2">武</span>。——<span class="smcs">《汉书·李广苏建传》</span></p><p><span class="diczx2">巡抚</span>状闻。—— <span class="diczx2">清</span>· <span class="diczx2">邵长蘅</span><span class="smcs">《青门剩稿》</span></p><p><span class="diczx2">巴陵</span>胜状。—— <span class="diczx2">宋</span>· <span class="diczx2">范仲淹</span><span class="smcs">《岳阳楼记》</span></p><p><span class="diczx1">自绘败状。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="diczx1">景状亦近。——</span><span class="diczx2">蔡元培</span><span class="smcs">《图画》</span></p><p><span class="cino">(5)</span> 又如:现状;状容(形态);状候(情况;状态)</p><p><span class="cino">(6)</span> 聘任或奖励人员的证书 <span class="encs">[certificate]</span>。如:奖状;委任状</p><p><span class="cino">(7)</span> 行状，记述死者事迹的一种文体，属于传记类 <span class="encs">[brief biography of a deceased person]</span></p><p><span class="diczx1">敢以状私于执事。——</span><span class="diczx2">柳宗元</span><span class="smcs">《段太尉逸事状》</span></p><p><span class="cino">(8)</span> 又如:状略(行状的大略);状式(行状的格式)</p><p><span class="cino">(9)</span> 诉状 <span class="encs">[plaint]</span></p><p><span class="diczx2">李</span>具状，求在狱候春发遣。—— <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(10)</span> 又如:状头(原告人);状儿(状子;起诉书);告状;状呈(即状子);状告(递状控告)</p><p><span class="cino">(11)</span> 书信 <span
