                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>军</strong></p><p><strong>（<span class="diczx3">軍</span>）</strong></p><p><span class="dicpy">jūn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jūn.mp3"></a></span> ㄐㄩㄣˉ</span></p><ol><li>武装部队：～威。～服。行（<span class="dicpy">xíng </span>）～。～功。～犬。～备。～纪。～衔。～阀。～令状。异～突起。溃不成～。</li><li>军队的编制单位，是“<span class="diczx3">师</span>”的上一级。</li><li>泛指有组织的集体：劳动大～。</li></ol>

                    <h2 class="h2_entry"><span class="orth">军</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>军</strong></p><p>軍 <span class="dicpy">jūn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jūn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。金文字形，从车，从勹( <span class="dicpy">bāo</span>，包裹)。表示用车子打包围圈的意思。古代打仗主要靠车战，驻扎时，用战车围起来形成营垒，以防敌人袭击。本义:围成营垒)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[encircle]</span></p><p><span class="diczx1">军，圜围也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">军，围也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">军于庐柳。——<span class="smcs">《国语·晋语》</span></span></p><p><span class="cino">(3)</span> 注:<span class="yhcs">“犹屯兵粮储也。”</span></p><p><span class="diczx1">疏而不可蹙，数而不可军者，在于慎。——<span class="smcs">《银雀山汉墓竹简》</span></span></p><p><span class="cino">(4)</span> 攻杀 <span class="encs">[attack and kill]</span></p><p><span class="diczx1">凡盗贼军，乡邑及家人，杀之无罪。——<span class="smcs">《周礼》</span></span></p><p><span class="cino">(5)</span> 指挥作战 <span class="encs">[command;battle]</span></p><p><span class="diczx1">祝聃射中王肩，王亦能军。——<span class="smcs">《左传·桓公五年》</span></span></p><p><span class="diczx2">建德</span>不能军，以百余骑走 <span class="diczx2">饶阳</span>。 <span class="diczx2">饶阳</span>无备，因取之。——<span class="smcs">《新唐书》</span></p><p><span class="cino">(6)</span> 驻扎 <span class="encs">[be stationed]</span>。如:军屯(指驻屯的军队)</p><p><span class="cino">(7)</span> 从军 <span class="encs">[join the army]</span></p><p><span class="diczx2">越国</span>女，尚耍白猿，教俺替爷军。—— <span class="diczx2">明</span>· <span class="diczx2">徐渭</span><span class="smcs">《雌木兰》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>军</strong></p><p>軍 <span class="dicpy">jūn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jūn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 军队 <span class="encs">[armed forces;army;troops]</span></p><p><span class="diczx1">击破</span><span class="diczx2">沛公</span>军。——<span class="smcs">《史记·项羽本纪》</span></p><p><span class="diczx1">虽入军无忿争之心。——<span class="smcs">《韩非子·解老》</span></span></p><p><span class="diczx2">亮</span>身率诸军攻 <span class="diczx2">祁山</span>。——<span class="smcs">《三国志·诸葛亮传》</span></p><p><span class="diczx1">两军人马杂遝。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="diczx2">英</span><span class="diczx2">法</span>联军。—— <span class="diczx2">清</span>· <span class="diczx2">徐珂</span><span class="smcs">《清稗类钞·战事类》</span></p><p><span class="diczx1">敌军已近寨。</span></p><p><span class="diczx1">军中发枪之号。</span></p><p><span class="cino">(2)</span>                           <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/8ECD.svg">                          <p>圜圍也。四千人爲軍。从車，从包省。軍（車），兵車也。舉云切〖注〗<img class='lazy k_img' data-original='//img.zdic.net/song/cn/215FD.svg'><span class='k_txt'>𡗽</span>，古文。</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <span class="jieshi">圜圍也。</span>於字形得圜義。於字音得圍義。凡渾䡣煇等軍聲之字皆兼取其義。<span class="jieshi">四千人爲軍。</span>王氏鳴盛說此句必譌。按唐釋玄應引字林四千人爲軍。是呂忱之誤也。許書當作萬有二千五百人爲軍。見周禮大司馬職。旅篆下云。軍之五百人爲旅。師篆下云。二千五百人爲師。旣皆偁之、則此必偁無可疑者。百人爲卒。卄有五人爲㒳。不偁者以其制於字義相遠耳。若万二千五百人以爲圜圍。乃此篆之所由製。<span class="jieshi">从包省。从車。</span>包省當作勹。勹、裹也。勹車、會意也。舉云切。十三部。<span class="jieshi">車、</span>逗。<span class="jieshi">兵車也。</span>上車各本誤軍。今正。此釋从車之意。惟車是兵車。故勹車爲軍也。                          </div>
