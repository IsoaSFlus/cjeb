                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>扣</strong> <span class="dicpy">kòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kòu.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，口声。本义:拉住，牵住)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[rein]</span></p><p><span class="diczx1">扣，牵马也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">扣马而谏。——<span class="smcs">《史记·伯夷叔齐传》</span></span></p><p><span class="diczx1">梁由靡扣</span><span class="diczx2">缪公</span>之骖。——<span class="smcs">《淮南子·泛论》</span></p><p><span class="diczx1">太子与</span><span class="diczx2">郭荣</span>扣马。——<span class="smcs">《左传·襄公十八年》</span></p><p><span class="cino">(3)</span> 又如:扣马(牵住马，不使前进。也作<span class="yhcs">“叩马”</span>);环环相扣;扣身(紧身，衣服紧贴身子)</p><p><span class="cino">(4)</span> 捆绑;拘押 <span class="encs">[bind;arrest;take in]</span>。如:扣搭(扣住，捆住);把杀人犯扣起来</p><p><span class="cino">(5)</span> 假借为<span class="yhcs">“敂”</span>(击)。敲击 <span class="encs">[knock]</span></p><p><span class="diczx1">扣之其声清扬而远闻。——<span class="smcs">《荀子·法行》</span></span></p><p><span class="diczx1">扣石垦壤。——<span class="smcs">《列子·汤问》</span></span></p><p><span class="diczx1">于乱石间择其一二扣之。——</span><span class="diczx2">宋</span>· <span class="diczx2">苏轼</span><span class="smcs">《石钟山记》</span></p><p><span class="cino">(6)</span> 又如:扣扉(敲门);扣户(敲门);扣门(敲门);扣扃(叩门，敲门);扣阍(叩击宫门)</p><p><span class="cino">(7)</span> 求教;探问 <span class="encs">[ask for advice]</span></p><p><span class="diczx1">扣其乡及姓字。——</span><span class="diczx2">明</span>· <span class="diczx2">魏禧</span><span class="smcs">《大铁椎传》</span></p><p><span class="cino">(8)</span> 又如:扣问(向人请教;询问);扣发(启发;提出意见);扣请(扣问)</p><p><span class="cino">(9)</span> 叩头，俯首向下之称 <span class="encs">[kowtow]</span>。如:扣头(叩头);扣额(犹扣头)</p><p><span class="cino">(10)</span> 截留 <span class="encs">[intercept]</span>。如:扣刻，扣克(截留财物，不按应发的全数发给);扣发(截留全部或部分不发);扣压(截留搁置)</p><p><span class="cino">(11)</span> 扣除 <span class="encs">[deduct]</span></p><p><span class="diczx1">逐日将本钱扣去。——<span class="smcs">《卖油郎独占花魁》</span></span></p><p><span class="cino">(12)</span> 又如:扣工资;扣分数;扣钱;扣抵(扣除抵偿);扣算(过分地算计)</p><p><span class="cino">(13)</span> 用钮扣固定或合拢 <span class="encs">[button up;buckle]</span>。如:把衣服扣上;把皮带扣上</p><p><strong>词性变化</strong></p><p>◎ <strong>扣</strong></p><p>釦 <span class="dicpy">kòu</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kòu.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 结 <span class="encs">[knot]</span></p><p><span class="diczx1">纽结丁香，掩过芙蓉扣。——</span><span class="diczx2">王实甫</span><span class="smcs">《西厢记》</span></p><p><span class="cino">(2)</span> 又如:系一个扣</p><p><span class="cino">(3)</span> 带扣 <span class="encs">[buckle]</span>。如:鞋扣</p><p><span class="cino">(4)</span> 钮扣 <span
