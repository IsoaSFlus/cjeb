                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>舞</strong> <span class="dicpy">wǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从舛( <span class="dicpy">chuǎn</span>)，两足相背。古舞字象人执牛尾而舞之形。本义:舞蹈)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[dance]</span></p><p><span class="diczx1">舞，动其容也。——<span class="smcs">《礼记·乐记》</span></span></p><p><span class="diczx1">乐容曰舞——</span><span class="diczx2">蔡邕</span><span class="smcs">《月令章句》</span></p><p><span class="cino">(3)</span> 又如:舞旋(旋转的舞蹈;耍弄，舞弄);舞判(装扮钟馗、判官等形象的舞蹈);芭蕾舞;歌舞(唱歌和舞蹈的合称);舞局(舞会);舞咏(舞蹈歌咏);舞头(领舞者);舞天(古代东方部族祭天之舞);舞妓(以歌舞娱人的妓女);集体舞</p><p><span class="cino">(4)</span> 钟体的顶部 <span class="encs">[top of a clock]</span></p><p><span class="diczx1">钲上谓之舞。——<span class="smcs">《考工记·凫氏》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>舞</strong> <span class="dicpy">wǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 跳舞 <span class="encs">[dance]</span></p><p><span class="diczx1">请以剑舞。——<span class="smcs">《史记·项羽本纪》</span></span></p><p><span class="diczx2">项庄</span>拔剑舞。</p><p><span class="diczx2">项伯</span>亦拔剑起舞。</p><p><span class="diczx1">留连戏蝶时时舞。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《江畔独步寻花》</span></p><p><span class="cino">(2)</span> 又如:舞马(令马按节拍舞蹈);舞钟馗(扮作钟馗舞蹈);舞跃(拜舞欢跃);舞鸡(鸡鸣起舞);舞裀(供舞蹈用的地毯);舞象(学象舞。会舞蹈的象);舞判(扮作判官或钟馗舞蹈);舞抃(飞舞跃跃)</p><p><span class="cino">(3)</span> 摇动 <span class="encs">[brandish]</span>。如:手舞双刀</p><p><span class="cino">(4)</span> 玩弄 <span class="encs">[play]</span></p><p><span class="diczx1">舞智以御人。——<span class="smcs">《史记·张汤列传》</span></span></p><p><span class="cino">(5)</span> 又如:舞旋(玩弄;折腾);舞手(耍弄手段);舞文弄法(玩弄文字，曲解法律);舞文巧诋(玩弄文字，诋毁构陷);舞弄文墨(舞文弄墨。玩弄法律条文，曲解其意)</p><p><span class="cino">(6)</span> 鼓舞，振奋 <span class="encs">[inspire]</span></p><p><span class="diczx1">敌存而惧，敌去而舞。——</span><span class="diczx2">柳宗元</span><span class="smcs">《敌戒》</span></p><p><span class="cino">(7)</span> 戏弄 <span class="encs">[make fun of]</span>。如:舞旋(耍弄;折腾)</p><p><span class="cino">(8)</span> 〈方〉∶作，搞 <span class="encs">[do]</span></p><p><span class="diczx1">这事就舞起来了。——<span class="smcs">《儒林外史》</span></span></p><p><span class="cino">(9)</span> 弄，抚弄 <span
