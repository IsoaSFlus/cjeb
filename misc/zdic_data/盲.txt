                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>盲</strong></p><p><span class="dicpy">máng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/máng.mp3"></a></span>  ㄇㄤˊ</span></p><ol><p>　◎ 瞎，看不见东西，对事物不能辨认：～从。～动。～目。～人。～区（指雷达、探照灯、胃镜等探测或观察不到的区域）。～流。～人瞎马（喻处境非常危险）。</p></ol>

                    <h2 class="h2_entry"><span class="orth">盲</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>盲</strong> <span class="dicpy">máng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/máng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从目，亡声。本义:眼睛失明)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[blind]</span></p><p><span class="diczx1">盲，目无牟子也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">目不能决黑白之色则谓之盲。——<span class="smcs">《韩非子·解老》</span></span></p><p><span class="diczx1">目不见青黄曰盲。——<span class="smcs">《论衡·别通》</span></span></p><p><span class="diczx1">五色令人目盲;五音令人耳聋。——<span class="smcs">《老子》</span></span></p><p><span class="diczx2">丁椽</span>，好士也，即使其两目盲，尚当与女，何况但眇?——<span class="smcs">《三国志·陈思王植传》</span>注</p><p><span class="cino">(3)</span> 又如:盲冥(眼睛失明);盲废(因丧失视力而致残);盲聋(眼瞎耳聋);盲聩(眼瞎耳聋);盲妹，盲女(旧时多指以卖唱为生的失明女子);盲翁(丧失视力的老人)</p><p><span class="cino">(4)</span> 昏暗 <span class="encs">[dusky]</span></p><p><span class="diczx1">列星殒坠，旦暮晦盲。——<span class="smcs">《荀子·赋》</span></span></p><p><span class="cino">(5)</span> 又如:盲盲(茫茫无所见);盲倀(晦暗不明的样子);盲云(乌云)</p><p><span class="cino">(6)</span> 形容糊涂，不明事理 <span class="encs">[muddle-headed]</span>。如:盲瞽(看不见，比喻无知或不明事理);盲明(昏庸与贤明);盲陋(谓见闻不广，学识浅陋);盲聋(比喻愚昧无知);盲书(盲目刻写;胡乱刻写);盲婚(男女双方互不了解，仅凭父母之命、媒妁之言的一种包办婚姻)</p><p><span class="cino">(7)</span> 迅疾 <span class="encs">[fast]</span></p><p><span class="diczx1">盲风至。——<span class="smcs">《礼记·月令》</span></span></p><p><span class="cino">(8)</span> 又如:盲风(疾风);盲雨(急雨;暴雨);盲风怪雨(谓非常急骤凶猛的风雨)</p><p><strong>词性变化</strong></p><p>◎ <strong>盲</strong> <span class="dicpy">máng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/máng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 使失明 <span class="encs">[blind]</span></p><p><span class="diczx2">周</span>人初以敌囚为民时，乃盲其左目以为奴征。—— <span class="diczx2">郭沫若</span><span class="smcs">《甲骨文字研究》</span></p><p><span class="cino">(2)</span> 喻不能辨识事物或事理 <span class="encs">[cannot be recognized]</span>。如:当今盲于心者皆是</p><p><span class="cino">(3)</span> 通<span class="yhcs">“望”</span>。看望 <span class="encs">[visit;see]</span></p><p><span class="diczx1">豕盲眡而交睫腥。——<span class="smcs">《周礼·天官·内饔》</span></span></p><p>◎ <strong>盲</strong> <span class="dicpy">máng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/máng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 文盲，不识字或识字很少的人 <span class="encs">[analphabet]</span>。如:扫盲</p><p><span
