                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>既</strong></p><p><span class="dicpy">jì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jì.mp3"></a></span>  ㄐㄧˋ</span></p><ol>    <li>动作已经完了：～往不咎。～而。</li>    <li>已经：～成事实。～定。</li>    <li>常与“<span class="diczx3">且</span>”、“<span class="diczx3">又</span>”连用，表示两者并列：～快又好。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> already; de facto; since; then</p><p><span class="z_ts2">德语</span> schon, seit</p><p><span class="z_ts2">法语</span> puisque,étant donné que,aussi... que,et... et...,déjà</p></div>

                    <h2 class="h2_entry"><span class="orth">既</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>既</strong> <span class="dicpy">jì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。甲骨文字形，左边是食器的形状，右边象一人吃罢而掉转身体将要离开的样子。本义:吃罢，吃过)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[eat up]</span></p><p><span class="diczx1">既，小食也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">罗振玉</span>曰:<span class="yhcs">“即，象人就食;既，象人食既。许训既为小食，义与形不协矣。”</span></p><p><span class="diczx1">君既食。——<span class="smcs">《礼记·玉藻》</span></span></p><p><span class="diczx1">不拜既爵。——<span class="smcs">《义礼·乡饮酒礼》</span></span></p><p><span class="cino">(3)</span> 食尽。指日全食或月全食 <span class="encs">[eclipse]</span></p><p><span class="diczx1">秋，七月，壬辰，朔，日有食之，既。——<span class="smcs">《春秋·桓公三年》</span>。</span><span class="diczx2">杜预</span>注:<span class="yhcs">“既，尽也。”</span></p><p><span class="diczx1">其合相当如袭辟者，日既是也。——</span><span class="diczx2">汉</span>· <span class="diczx2">王充</span><span class="smcs">《论衡》</span></p><p><span class="cino">(4)</span> 完毕;完了 <span class="encs">[complete;end]</span></p><p><span class="diczx1">既而曰。——<span class="smcs">《公羊传·宣公元年》</span>。注:<span class="yhcs">“事毕也。”</span></span></p><p><span class="diczx1">既者，尽也。有继之辞也。——<span class="smcs">《谷梁传》</span></span></p><p><span class="diczx1">既月。——<span class="smcs">《虞书》</span></span></p><p><span class="diczx1">董泽之蒲可胜既乎。——<span class="smcs">《左传·宣公十二年》</span></span></p><p><span class="diczx1">薮泽肆既。——<span class="smcs">《国语·周语》</span></span></p><p><span class="diczx1">吾与汝既其文，未既其实。——<span class="smcs">《庄子·应帝王》</span></span></p><p><span class="cino">(5)</span> 又如:既济(六十四卦之。离下坎上。意指万事皆济);既冠之后(成年之后。古代男子二十岁行加冠礼，表示已成年)</p><p><strong>词性变化</strong></p><p>◎ <strong>既</strong> <span class="dicpy">jì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jì.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 已经 <span class="encs">[already]</span></p><p><span class="diczx1">既来之，则安之。——<span class="smcs">《论语·季氏》</span></span></p><p><span class="diczx1">既祖，取道。(祭过路神，就要上路。祖，临行祭路神，引申为饯行送别。)——<span class="smcs">《战国策·燕策》</span></span></p><p><span class="diczx1">既克。——<span class="smcs">《左传·庄公十年》</span></span></p><p><span class="diczx1">既其出。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="diczx1">既加冠。——</span><span class="diczx2">明</span>· <span class="diczx2">宋濂</span><span class="smcs">《送东阳马生序》</span></p><p><span class="diczx1">既入宫中，举天下所贡…遍试之，无出其右者。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="diczx1">既泣之三日。——</span><span class="diczx2">清</span>· <span class="diczx2">龚自珍</span><span class="smcs">《病梅馆记》</span></p><p><span class="cino">(2)</span> 又如:既成(已经完成);既位(已就其位);既醉以酒，既饱以德(宾客称主人优厚待客的客气话)</p><p><span class="cino">(3)</span> 不久;随即 <span class="encs">[soon]</span></p><p><span class="diczx2">楚成王</span>以商臣为太子，既欲置公子职。——<span class="smcs">《韩非子·内储说下》</span></p><p><span class="diczx1">昔</span><span class="diczx2">李斯</span>与 <span class="diczx2">包丘子</span>俱事 <span class="diczx2">荀卿</span>，既而 <span class="diczx2">李斯</span>入 <span class="diczx2">秦</span>。——<span class="smcs">《盐铁论·毁学》</span></p><p>◎                           <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/65E3_2517.svg"  /><span class="swbf">(旣)</span><span class="ywyy">小食也。</span><span class="swdz">此與口部嘰音義皆同。玉藻、少儀作禨。假借字也。引伸之義爲盡也已也。如春秋日有食之旣。周本紀東西周皆入於秦。周旣不祀。正與小食相反。此如亂訓治、徂訓存。旣者、終也。終則有始。小食則必盡。盡則復生。</span><span class="ywyy">从皀。旡聲。</span><span class="swdz">居未切。十五部。</span><span class="ywyy">論語曰。不使勝食旣。</span><span class="swdz">鄉黨篇文。此引經說假借也。論語以旣爲气。如商書以<img class='lazy k_img' data-original='//img.zdic.net/song/cn/216BD.svg'><span class='k_txt'>𡚽</span>爲好、詩以夃爲姑之類。今論語作氣。气氣古今字。作氣、葢魯論也。許偁、葢古文論語也。或云。謂不使肉勝於食。但小小食之。說固可通。然古人之文。云不使勝則巳足。不必贅此字。</span><span class="xh"></span></p>                          </div>
