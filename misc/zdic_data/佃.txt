data-src-mp3="//img.zdic.net/audio/zd/py/diàn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从人，田声。<span class="yhcs">“田”</span>又兼表字义。①( <span class="dicpy">tián</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tián.mp3"></a></span> )本义:耕种土地 ②引申义:租种田地)</p><p><span class="cino">(2)</span> 同引申义，也指向地主或在古代向官府租种土地 <span class="encs">[rent land from a landlord]</span>。如:承佃(旧社会农民被迫向地主租种土地);佃人(租种官府或地主田地的农民)</p><p><span class="cino">(3)</span> 〈方〉∶租借;租赁 <span class="encs">[rent]</span>。如:佃屋;佃铺面</p><p><span class="cino">(4)</span> 通<span class="yhcs">“畋( <span class="dicpy">tián</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tián.mp3"></a></span> )”</span>。畋猎，猎取禽兽 <span class="encs">[hunt]</span></p><p><span class="diczx1">作结绳而为罔罟，以佃以渔，盖取诸离。——<span class="smcs">《易·系辞下》</span></span></p><p><span class="diczx1">则又</span><span class="diczx2">成王</span>之所佃。——<span class="smcs">《诗·小雅·信南山》</span>笺</p><p><span class="diczx1">今观宰司之取士也，有似于司原之佃也。——<span class="smcs">《潜夫论·贤难》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>佃</strong> <span class="dicpy">diàn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/diàn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 佃户;向地主租地的农户 <span class="encs">[tenant]</span>。如:东佃(地主和佃户的合称)</p><p><span class="cino">(2)</span> 田租 <span class="encs">[land rent]</span>。如:佃科(官田收租的章程)</p><p><span class="cino">(3)</span> 契约 <span class="encs">[lease]</span>。如:撤佃，退佃(地主收回租给农民的土地)</p><p><span class="cino">(4)</span> 另见 <span class="dicpy">tián</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tián.mp3"></a></span> </p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a href="/hans/佃户">佃户</a></span><span class="crefe"><a href="/hans/佃农">佃农</a></span><span class="crefe"><a href="/hans/佃契">佃契</a></span><span class="crefe"><a href="/hans/佃租">佃租</a></span></div>　<p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>佃</strong> <span class="dicpy">tián</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tián.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span>
