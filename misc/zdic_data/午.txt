						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>午</strong></p><p><span class="dicpy">wǔ  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> ㄨˇ</span></p><ol><li>地支的第七位，属马。</li><li>用于计时：～时（白天十一点到一点）。～间。～饭。～睡。～休。～夜（半夜、子夜）。</li><li>古同“<span class="diczx3">忤</span>”、“<span class="diczx3">迕</span>”，逆，背。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>午</strong> <span class="dicpy">wǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (象形。本义:御马索)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[rope that use to manage horse]</span></p><p><span class="diczx1"><span class="encs">[甲骨文<span class="yhcs">“午”</span>字]</span>疑当是索形，殆驭马之辔也。——</span><span class="diczx2">郭沫若</span><span class="smcs">《甲骨文字研究》</span></p><p><span class="cino">(3)</span> 地支的第七位 <span class="encs">[the seventh of the twelve Earthly Branches]</span></p><p><span class="cino">(4)</span> 与天干相配，用以纪年</p><p><span class="diczx1">太岁在午曰敦羊。——<span class="smcs">《尔雅》</span></span></p><p><span class="cino">(5)</span> 用以纪月。即农历五月</p><p><span class="diczx1">午，五月阴气午逆阳，冒地而出。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(6)</span> 用以纪日</p><p><span class="diczx1">庚午之日，日始有谪。——<span class="smcs">《左传》</span></span></p><p><span class="cino">(7)</span> 用以纪时，即十一时至十三时;正午，白天十二时 <span class="encs">[noon]</span></p><p><span class="diczx1">午战于彼。——<span class="smcs">《左传·僖公三十二年》</span></span></p><p><span class="diczx1">时已过午。——<span class="smcs">《徐霞客游记·游黄山记》</span></span></p><p><span class="diczx1">过午已昏。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="cino">(8)</span> 又如:午枕(午睡);午牌(午时);午晌(晌午);午炮(正午的号炮);午火(正午烈日);午天(中午);午斋(中午的斋食);午暑(中午的暑气);午漏(午时滴漏);午际(午初);午上(中午前);午潮(正午的潮水);午茶(午饭后的茶水)</p><p><span class="cino">(9)</span> 十二肖属马，因亦为马的代称 <span class="encs">[horse]</span>。如:午日三公(马)</p><p><span class="cino">(10)</span> 干支逢五日午。亦特指五月初五 <span class="encs">[five]</span>。如:端午;重午;午月(五月);午节(端五节)</p><p><span class="cino">(11)</span> 古人以十二支配方位，午为正南，因以为南方的代称 <span class="encs">[southern]</span>。如:午上(南方的上空)</p><p><strong>词性变化</strong></p><p>◎ <strong>午</strong> <span class="dicpy">wǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 纵横相交 <span class="encs">[crisscross]</span></p><p><span class="diczx1">度尺而午。——<span class="smcs">《仪礼》</span>。</span><span class="diczx2">郑玄</span>注:<span class="yhcs">“一纵一横曰午。”</span></p><p><span class="cino">(2)</span> 又如:午午(交错杂沓的样子);午道(纵横交贯的要道);午贯(十字形交叉贯穿);午割(交叉切割)</p><p>◎ <strong>午</strong> <span class="dicpy">wǔ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wǔ.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“仵”</span>。违反 <span class="encs">[violate]</span></p><p><span class="diczx1">午其众以伐有道，求得当欲，不以其所。——<span class="smcs">《礼记·哀公问》</span></span></p><p><span class="diczx1">朝臣舛午。——<span class="smcs">《汉书·刘向传》</span></span></p><p><span class="cino">(2)</span> 通<span class="yhcs">“迕”</span>。抵触，相遇 <span class="encs">[conflict;meet with]</span></p><p><span class="diczx1">视可午其军、取其将…若是则为利者不攻也。——<span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/5348.svg">						  <p>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>疑古切，音五。【說文】牾也。五月隂氣午逆陽，冒地而出也。【徐曰】五月陽極隂生。仵者，正衝之也。</p><p>又辰名。【爾雅·釋天】太歲在午曰敦牂。【淮南子·時則訓】斗五月指午。</p><p>又【廣韻】交也。【韻會】一縱一橫曰旁午，猶言交橫也。【儀禮·特牲饋食】心舌皆去，本末午割之。【註】縱橫割也。【禮·內則】男角女羈。【註】午達曰羈。【疏】度尺而午，令女剪髮，留其頂上，縱橫各一，相通達也。【前漢·霍光傳】使者旁午。【註】旁午，分布也。</p><p>又舛午，違背也。見【前漢·劉向傳】。</p><p>又【前漢·劉向傳】水旱飢蝝，蠭午<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>起。【註】猶雜沓也。</p><p>又【段成式詩】良人爲漬木瓜水，遮却紅腮交午痕。</p><p>又【韻會】馬屬午。晉姓司馬，因攺司馬官爲典午。</p>
