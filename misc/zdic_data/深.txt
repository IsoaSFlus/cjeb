                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>深</strong></p><p><span class="dicpy">shēn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shēn.mp3"></a></span>  ㄕㄣˉ</span></p><ol><li>从表面到底或从外面到里面距离大，与“<span class="diczx3">浅</span>”相对：～水。～山。～邃。～渊。～壑。～海。～耕。～呼吸。～藏若虚（把珍贵的东西深藏起来，好像没有一样，喻人有知识才能但不在人前表现）。～居简出。</li><li>从表面到底的距离：～度。～浅。。水～三尺。</li><li>久，时间长：～夜。～秋。年～日久。</li><li>程度高的：～思。～知。～交。～造。～谈。～省（<span class="dicpy">xǐng </span>）（深刻的警悟。亦作“深醒”）。～究。～奥。～切。～沉（ａ．形容程度深，如“暮色～～”；ｂ．声音低沉，如“～～的哀鸣”；ｃ．思想感情不外露，如“他为人～～，叫人难以捉摸”）。～谋远虑。</li><li>颜色浓：～色。～红。</li></ol>

                    <h2 class="h2_entry"><span class="orth">深</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>深</strong> <span class="dicpy">shēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shēn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从水，深声( <span class="dicpy">shēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shēn.mp3"></a></span> )。本是河流名。深浅的深，本作<span class="yhcs">“深”</span>。本义:水名)</p><p><span class="cino">(2)</span> 古水名 <span class="encs">[Shen River]</span>。即今湘水支流之一的潇水。今潇水上源至江华县一段仍称深水</p><p><span class="diczx1">深，</span><span class="diczx2">深水</span>。出 <span class="diczx2">桂阳</span> <span class="diczx2">南平</span>，西入 <span class="diczx2">营</span>道。——<span class="smcs">《说文》</span></p><p><span class="cino">(3)</span> 从上到下或从外到内的距离 <span class="encs">[depth]</span></p><p><span class="diczx1">穷高极远，而测深厚。——<span class="smcs">《礼记·乐记》</span>。注:<span class="yhcs">“深厚，山川也。”</span></span></p><p><span class="diczx1">测土深。——<span class="smcs">《周礼·大司徒》</span></span></p><p><span class="diczx1">深四尺。——<span class="smcs">《仪礼·觐礼》</span></span></p><p><span class="diczx1">问其深，则好游者不能穷也。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《游褒禅山记》</span></p><p><span class="diczx1">山下皆石穴罅，不知其浅深。——</span><span class="diczx2">宋</span>· <span class="diczx2"> 苏</span>轼<span class="smcs">《石钟山记》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>深</strong> <span class="dicpy">shēn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shēn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 水深。与<span class="yhcs">“浅”</span>相对 <span class="encs">[(water)deep]</span></p><p><span class="diczx1">深矣远矣。——<span class="smcs">《老子》</span>。注:<span class="yhcs">“深不可测也。”</span></span></p><p><span class="diczx1">深蒲。——<span class="smcs">《周礼·醢人》</span>。</span><span class="diczx2">司农</span>注:<span class="yhcs">“蒲蒻入水深，故曰深蒲。”</span></p><p><span class="diczx2">郦元</span>以为下临深潭，微风鼓浪，水石相击，声如洪钟。—— <span class="diczx2">宋</span>· <span class="diczx2"> 苏</span>轼<span class="smcs">《石钟山记》</span></p><p><span class="diczx1">先茔在杭，江广河深，势难归葬。——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《祭妹文》</span></p><p><span class="cino">(2)</span> 又如:深潭(深水池);深广(指水、山谷等深邃而广阔);深堑(深沟);深隍(深而无水的护城壕);深沦(深水);深泓(深潭);深泥(很深的泥泞);深池(深的护城河)</p><p><span                         <div class="content definitions snr">
						<div class="mob">【<a href="/zd/sw/?swju=卷十一">卷十一</a>】【<a href="/zd/sw/?swju=卷十一&amp;swbs=水">水</a>】</div>
                                                    <div class="swnr">
                          『說文解字』<hr class="dichr">
                          <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/6DF1.svg">                          <p>水。出桂陽南平，西入營道。从水罙聲。式針切〖注〗<img class='lazy k_img' data-original='//img.zdic.net/song/cn/23E27.svg'><span class='k_txt'>𣸧</span>，古文。</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/6DF1_3410.svg"  /><span class="swbf">(深)</span><span class="ywyy">深水。出桂陽南平。西入營道。</span><span class="swdz">桂陽郡南平、零陵郡營道、二志同。今湖南桂陽州藍山縣縣東五里有南平城。水經曰。深水出桂陽盧聚。西北過零陵營道縣、營浦縣、泉陵縣。至燕室。邪入於湘。酈云。桂陽縣本<img class='lazy k_img' data-original='//img.zdic.net/song/cn/23729.svg'><span class='k_txt'>𣜩</span>桂陽郡。後割屬始興縣。有盧溪。盧聚山在南平縣之南、九疑山之東。玉裁謂盧聚山在南平之南。經舉其遠源。許舉其近源。洭出盧聚。南流入海。深出盧聚。西北流入湘、以入江。是分馳不同也。湘水篇經、注皆不言深水。葢吕忱言深水導源盧溪。西入營水。亂流營波。同注湘津。故湘水篇言營不言深耳。今深營二水源委未聞。漢營道、營浦縣皆氏於水。以字林訂說文、則當作入營。不必有道字。泉陵縣卽今湖南永州府零陵縣。今瀟水合諸水於此入湘。深水、營水在其中也。</span><span class="ywyy">从水。<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2594D.svg'><span class='k_txt'>𥥍</span>聲。</span><span class="swdz">式針切。七部。按此無深淺一訓者、許意深淺字當作<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2594D.svg'><span class='k_txt'>𥥍</span>。詳<img class='lazy k_img' data-original='//img.zdic.net/song/cn/2594D.svg'><span class='k_txt'>𥥍</span>下。</span><span class="xh"></span></p>                          </div>
