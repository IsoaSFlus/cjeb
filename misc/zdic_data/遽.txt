						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>遽</strong></p><p><span class="dicpy">jù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> ㄐㄩˋ</span></p><ol><li>急，仓猝：～尔（突然）。～死。～然。匆～。</li><li>惊惧、慌张：～色。惶～。</li><li>古代报信的快马或驿车：乘～而至。</li><li>遂，就：“塘有万穴，塞其一，鱼何～无由出？”</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>遽</strong> <span class="dicpy">jù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，表示与行走有关，本义:送信的快车或快马)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[post car or horse]</span></p><p><span class="diczx1">遽，传也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">以待达穷者与遽令。——<span class="smcs">《周礼·大仆》</span></span></p><p><span class="diczx1">掌邦国传遽之小事。——<span class="smcs">《周礼·行夫》</span>。注:<span class="yhcs">“若今时乘传骑驿而使者也。”</span></span></p><p><span class="diczx1">且使遽告于</span><span class="diczx2">郑</span>。——<span class="smcs">《左传·僖公三十三年》</span>。注:<span class="yhcs">“传车。”</span></p><p><span class="diczx1">士曰传遽之臣。——<span class="smcs">《礼记·玉藻》</span></span></p><p><span class="cino">(3)</span> 又如:遽人(驿卒;传达命令的人);遽驿(传车和驿马);遽传(驿车和传舍);遽委(驿站车马和储存的粮食)</p><p><span class="cino">(4)</span> 乡名 <span class="encs">[Ju village]</span>。在今河北省元氏县附近</p><p><strong>词性变化</strong></p><p>◎ <strong>遽</strong> <span class="dicpy">jù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 立刻;马上 <span class="encs">[immediately;at once;in no time]</span></p><p><span class="diczx1">遽兴姜戎。——<span class="smcs">《左传·僖公三十二》</span></span></p><p><span class="diczx1">北亦未敢遽轻吾国。——</span><span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《指南录后序》</span></p><p><span class="diczx1">遽扑之。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="diczx1">遽以不变名之。——<span class="encs">[英]</span></span><span class="diczx2">赫胥黎</span>著、 <span class="diczx2">严复</span>译<span class="smcs">《天演论》</span></p><p><span class="cino">(2)</span> 又如:遽诺(马上承诺、答应);遽理(马上或很快处理);遽即(立即)</p><p><span class="cino">(3)</span> 就，竟 <span class="encs">[then]</span></p><p><span class="diczx1">岂遽善游哉。——<span class="smcs">《吕氏春秋·察今》</span></span></p><p><span class="diczx1">此何遽不能为福乎?——<span class="smcs">《淮南子·人间训》</span></span></p><p><span class="diczx1">遽惊谢曰。——</span><span class="diczx2">清</span>· <span class="diczx2">周容</span><span class="smcs">《芋老人传》</span></p><p><span class="diczx1">遽躬蹈之。——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《祭妹文》</span></p><p>◎ <strong>遽</strong> <span class="dicpy">jù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jù.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 惊慌 <span class="encs">[frightened]</span></p><p><span class="cino">(2)</span> 又如:遽惶(惶遽，惊慌);遽惕(惶恐戒惧);遽戚(敬畏的样子);遽容(恐惧的脸色)</p><p><span class="cino">(3)</span> 十分贫困，窘迫 <span class="encs">[very poor]</span></p><p><span class="diczx1">远近久承公名，令于是大遽，不敢移公。——<span class="smcs">《世说新语》</span></span></p><p><span class="cino">(4)</span> 急忙，匆忙 <span class="encs">[hastily;hurriedly]</span></p><p><span class="diczx1">遽数之不能终其物。——<span class="smcs">《礼记·儒行》</span></span></p><p><span class="diczx1">公惧，遽见之。——<span class="smcs">《国语·晋语》</span></span></p><p><span class="diczx1">遽契其舟。——<span class="smcs">《吕氏春秋·察今》</span></span></p><p><span class="diczx1">遽曰:<span class="yhcs">“为两郎僮，孰若为一郎僮耶?”</span>——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span
