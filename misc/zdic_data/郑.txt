                          <h2 class="h2_entry"><span class="orth">郑</span><span class="span dictname"> 基本解释</span></h2>
                        </div>
                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>郑</strong></p><p><strong>（<span class="diczx3">鄭</span>）</strong></p><p><span class="dicpy">zhèng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> ㄓㄥˋ</span></p><ol><li>中国周代诸侯国名，在今河南省新郑县一带：～人买履（讽喻那些只相信教条，不顾客观实际的人）。</li><li>姓。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> state in today's Henan; surname</p><p><span class="z_ts2">德语</span> Familienname  (Eig)​,Zheng  (Eig, Fam)</p><p><span class="z_ts2">法语</span> sérieux,(abrév.)​ Zhengzhou,(nom de famille)​</p></div>

                    <h2 class="h2_entry"><span class="orth">郑</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>郑</strong></p><p>鄭 <span class="dicpy">zhèng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从邑，奠声。从邑，与地名、邦域有关。本义:周代诸侯国名，姬姓)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[Zheng state]</span></p><p><span class="diczx1">郑，</span><span class="diczx2">京兆</span>县， <span class="diczx2">周厉王</span>子 <span class="diczx2">友</span>所封。从邑，奠声。 <span class="diczx2">宗周</span>之灭， <span class="diczx2">郑</span>徙 <span class="diczx2">缯</span>洧之上，今 <span class="diczx2">新郑</span>是也。——<span class="smcs">《说文》</span></p><p><span class="diczx1">至于</span><span class="diczx2">南郑</span>。——<span class="smcs">《穆天子传》</span>。此旧郑也。</p><p><span class="diczx2">郑穆公</span>使视客馆。——<span class="smcs">《左传·僖公三十三年》</span></p><p><span class="cino">(3)</span> 周西都畿内地。周宣王封季弟友(桓公)于此。在今陕西华县境。其后犬戎杀周幽王，桓公死之，其子武公与晋文侯定平王于东都，武公迁居东都畿内，都新郑，即春秋之郑国。战国时为韩所灭。又如:郑声(郑国音乐);郑风(<span class="smcs">《诗》</span>十五国风之一);郑国渠(古代关中平原的人工灌溉渠)</p><p><span class="cino">(4)</span> 原指春秋战国时郑国的民间音乐，后与所谓雅乐对举，指与雅乐相背的<span class="yhcs">“淫邪之音”</span> <span class="encs">[wanton music]</span></p><p><span class="diczx1">改华服以就紫，变雅音而入郑。——<span class="smcs">《惟皇诫德赋》</span></span></p><p><span class="cino">(5)</span> 又如:郑音(本指春秋时郑国的音乐，后多指俗乐)</p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>郑</strong></p><p>鄭 <span class="dicpy">zhèng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhèng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 郑重;慎重 <span class="encs">[solemn]</span></p><p><span class="diczx1">郑，重也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">非皇天所以郑重降符命之意。——<span class="smcs">《汉书·王莽传》</span></span></p><p><span class="cino">(2)</span> 又如:郑重其事(对某件事持慎重、认真、严肃的态度)</p><p><strong>常用词组</strong></p><hr class="dichr" /><div class="cit type-xxjs"><span class="crefe"><a                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/912D.svg">						  <p>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>直正切，音㽀。國名。【釋名】鄭，町也，地多平町町然也。【正字通】鄭本西都畿內地，周宣王封其弟友，是爲鄭。桓公寄孥與賄于虢鄶，其子武公，定平王于東都，因徙其封，施舊號于新邑，是爲新鄭。今河南開封府鄭州是也。</p><p>又南鄭，縣名。今屬陝西漢中府，桓公封邑在畿內，爲西鄭，漢中在京兆之南，故稱南鄭以別之。【一統志】南鄭，古褒國附庸之邑。桓公歿，其民南奔居此，因曰南鄭。</p><p>又【廣韻】鄭重，殷勤之意。【前漢·王莽傳】非皇天所以鄭重降符命之意。</p><p>又姓。【集韻】鄭滅于韓，子孫因以國爲氏。</p>
