                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>奇</strong></p><p><span class="dicpy">qí  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/qí.mp3"></a></span>  ㄑㄧˊ</span></p><ol>    <li>特殊的，稀罕，不常见的：～闻。～迹。～志。～观。～妙。～巧。～耻大辱。</li>    <li>出人意料的，令人不测的：～兵。～计。～袭。出～制胜。</li>    <li>惊异，引以为奇：～怪。惊～。不足为～。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>奇</strong></p><p><span class="dicpy">jī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span>  ㄐㄧˉ</span></p><ol>    <li>数目不成双的，与“<span class="diczx3">偶</span>”相对：～数（不能被二整除的数，如一、三、五、七、九等，正的奇数亦称“单数”）。</li>    <li>零数：～零（不满整数的数）。～羡（赢余，积存的财物）。有～（如“八分～～”即八分多一点）。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>奇</strong> <span class="dicpy">jī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (字源见奇( <span class="dicpy">qí</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/qí.mp3"></a></span> )。本义:奇特，奇异)</p><p><span class="cino">(2)</span> 单数，偶之对 <span class="encs">[odd]</span></p><p><span class="diczx1">奇，一曰不耦。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">阳数奇。——<span class="smcs">《白虎通·嫁娶》</span></span></p><p><span class="diczx1">一算为奇。——<span class="smcs">《仪礼·乡射礼》</span></span></p><p><span class="diczx1">奇肱之国，其人一臂三目，有阴有阳。——<span class="smcs">《山海经·海外西经》</span></span></p><p><span class="diczx1">每奇日，未尝不视朝。——<span class="smcs">《资治通鉴》</span></span></p><p><span class="cino">(3)</span> 又如:奇日(奇数之日。即单日);奇左(仅有左臂);奇肱(仅有一臂。又指古代神话中的国名);奇偏(偏于一方面，片面)</p><p><span class="cino">(4)</span> 非法的 <span class="encs">[illegitimate]</span>。如:奇车(不合制度的车);奇请它比(于法律正文以外，另行请示或比附他例判案)</p><p><span class="cino">(5)</span> <span class="encs">[遭遇]</span>不顺当的 <span class="encs">[unsmooth]</span></p><p><span class="diczx2">卫青</span>不败由天幸， <span class="diczx2">李广</span>无功缘数奇。—— <span class="diczx2">王维</span><span class="smcs">《老将行》</span></p><p><span class="cino">(6)</span> 又如:奇人(坎坷不遇之人);数奇(命运不好);奇薄(数奇命薄。命运不好);奇蹇(命运不好)</p><p><span class="cino">(7)</span> <span class="encs">[中医]</span>∶药味合于单数或单味药的 <span class="encs">[odd-numbered]</span> 。如:奇方</p><p><strong>词性变化</strong></p><p>◎ <strong>奇</strong> <span class="dicpy">jī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/jī.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 余数，零头，不足整数者 <span class="encs">[remainder;fractional amount]</span></p><p><span class="diczx1">时有奇羡。——<span class="smcs">《史记·货殖列传》</span>。索隐:<span class="yhcs">“余衍也。”</span></span></p><p><span class="diczx1">操其奇赢。——<span class="smcs">《汉书·食货志》</span>。注:<span
