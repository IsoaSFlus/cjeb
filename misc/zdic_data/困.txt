                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>困</strong></p><p><strong>（<span class="diczx3">睏</span>）</strong></p><p><span class="dicpy">kùn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kùn.mp3"></a></span> ㄎㄨㄣˋ</span></p><ol><li>陷在艰难痛苦或无法摆脱的环境中：～厄（处境困苦危险）。～处（chǔ）。～居。～扰。～境。～窘。～知勉行。～兽犹斗。</li><li>穷苦、艰难：～苦。～难。</li><li>包围：～守。围～。</li><li>疲乏：～乏。～倦。～顿。</li><li>想睡，睡：～人。～觉（jiào）。</li></ol>

                    <h2 class="h2_entry"><span class="orth">困</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>困</strong> <span class="dicpy">kùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kùn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。甲骨文字形，从囗( <span class="dicpy">wéi</span>)，象房的四壁;里边是生长的树木。本义:废弃的房屋)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[tumble-down house]</span></p><p><span class="diczx1">困，故庐也。——<span class="smcs">《说文》</span></span></p><p><span class="cino">(3)</span> <span class="yhcs">“梱”</span>的本字。门槛 <span class="encs">[threshold]</span></p><p><span class="diczx1">试藉车之力而为之困。——<span class="smcs">《墨子·备城门》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>困</strong> <span class="dicpy">kùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/kùn.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 穷;贫困 <span class="encs">[poor]</span></p><p><span class="diczx1">困，穷也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">行而无资谓之乏，居而无食谓之困。——<span class="smcs">《周礼·地官·禀人》</span>疏引书传</span></p><p><span class="diczx1">四海困穷，天禄永终。——<span class="smcs">《论语》</span></span></p><p><span class="diczx1">以振穷困。——<span class="smcs">《吕氏春秋·慎大》</span></span></p><p><span class="diczx1">行李之往来，共其乏困。——<span class="smcs">《左传·僖公三十年》</span></span></p><p><span class="diczx1">岁饥民困。——<span class="smcs">《史记·宋世家》</span></span></p><p><span class="diczx1">今岁不登，已遣使节振贷困乏。——<span class="smcs">《汉书·宣帝纪》</span></span></p><p><span class="cino">(2)</span> 又如:困污(生活贫苦，地位低下);困悴(贫困愁苦);困匮(贫乏，贫困)</p><p><span class="cino">(3)</span> 困难;苦难 <span class="encs">[difficulty]</span></p><p><span class="diczx1">不以监困故而受。——<span class="smcs">《史记·魏公子列传》</span></span></p><p><span class="diczx2">楚</span>大困。——<span class="smcs">《史记·屈原贾生列传》</span></p><p><span class="diczx1">民困而不知救。——</span><span class="diczx2">明</span>· <span class="diczx2">刘基</span><span class="smcs">《卖柑者言》</span></p><p><span class="diczx1">俾困苦不可忍。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="cino">(4)</span> 又如:困穷(艰难窘迫;生活困苦贫穷);困笃(病重垂危);因踬(因厄不得升进);困学(遇到困难才去学习。指被动学习);困心衡虑(尽心竭虑，经过一番艰苦的思考);困知勉行(苦学以求知，勉力践行)</p><p><span class="cino">(5)</span> 劳倦;疲倦 <span class="encs">[tired;weary]</span></p><p><span class="diczx1">牛困人饥日已高。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《卖炭翁》</span></p><p><span
