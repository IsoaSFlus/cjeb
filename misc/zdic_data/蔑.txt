                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>蔑</strong> <span class="dicpy">miè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/miè.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (会意。从苜( <span class="dicpy">mò</span>)，从戍。<span class="yhcs">“苜”</span>是眼睛歪斜无神，<span class="yhcs">“戍”</span>是戍守人。合而表示人过于劳倦眼睛歪斜无神。本义:眼睛红肿看不清)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[(be red and swolle in eye)can not see clearly]</span></p><p><span class="diczx1">蔑，目眵也。从目，蔑声。——<span class="smcs">《说文》</span>。按，即蔑字</span></p><p><span class="diczx1">得目为蔑。——</span><span class="diczx2">宋玉</span><span class="smcs">《风赋》</span></p><p><span class="cino">(3)</span> 细小;轻微 <span class="encs">[slight]</span></p><p><span class="diczx1">视日月而知众星之蔑也。——</span><span class="diczx2">扬雄</span><span class="smcs">《法言·学行》</span></p><p><span class="cino">(4)</span> 又如:蔑如(微细;没有什么了不起);蔑贱(犹微贱)</p><p><strong>词性变化</strong></p><p>◎ <strong>蔑</strong> <span class="dicpy">miè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/miè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 轻视;轻侮。亦作<span class="yhcs">“懱”</span> <span class="encs">[disdain;despise]</span></p><p><span class="diczx1">国步蔑资。——<span class="smcs">《诗·大雅·桑柔》</span></span></p><p><span class="diczx1">岂蔑清庙，惮勅天乎?——<span class="smcs">《后汉书·班固传下》</span></span></p><p><span class="diczx1">蔑贞，凶。——<span class="smcs">《易·剥卦》</span></span></p><p><span class="diczx2">郑</span>未失 <span class="diczx2">周</span>典，王而蔑之，是不明贤也。——<span class="smcs">《国语·周语中》</span></p><p><span class="cino">(2)</span> 又如:蔑辱(鄙弃凌辱);蔑杀(犹灭杀);蔑侮(轻视欺侮);蔑弃(轻视，鄙弃)</p><p><span class="cino">(3)</span> 灭，消灭 <span class="encs">[extinguish;obliterate]</span></p><p><span class="diczx1">唐蔑字明。——<span class="smcs">《荀子·议兵》</span></span></p><p><span class="diczx1">而蔑杀其民人。——<span class="smcs">《国语·周语》</span></span></p><p>◎ <strong>蔑</strong> <span class="dicpy">miè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/miè.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 无;没有 <span class="encs">[nothing;none]</span></p><p><span class="diczx1">蔑，无也。——<span class="smcs">《小尔雅·广诂》</span></span></p><p><span class="diczx1">丧乱蔑资。——<span class="smcs">《诗·大雅·板》</span></span></p><p><span class="diczx1">吾有死而已，吾蔑从之矣。——<span class="smcs">《国语·晋语》</span></span></p><p><span class="diczx1">蔑不济也。——<span class="smcs">《左传·僖公十年》</span></span></p><p><span class="cino">(2)</span> 又如:蔑有(没有);蔑须有(莫须有，没有);蔑蔑(犹默默。无声息)</p><p>◎ <strong>蔑</strong></p><p>衊 <span class="dicpy">miè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/miè.mp3"></a></span>
