data-src-mp3="//img.zdic.net/audio/zd/py/xìng.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意兼形声。从女，从生，生亦声。本义:标志家族系统的字)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[surname]</span></p><p><span class="diczx1">姓，人所生也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx2">左氏</span>传曰:天子建德，因生以赐姓。——<span class="smcs">《春秋·隐公八年》</span>。按，神农母居 <span class="diczx2">姜水</span>， <span class="diczx2">黄帝</span>母居 <span class="diczx2">姬水</span>， <span class="diczx2">舜</span>母居 <span class="diczx2">姚虚</span>，因以为姓。故从女生。</p><p><span class="diczx1">赐姓</span><span class="diczx2">刘</span>氏，拜为郎中。——<span class="smcs">《史记·刘敬叔孙通列传》</span></p><p><span class="diczx1">世为著姓。——<span class="smcs">《后汉书·张衡传》</span></span></p><p><span class="diczx1">姓者，统其祖考之所自出，氏者，别其子孙之所自分。——</span><span class="diczx2">刘恕</span><span class="smcs">《通鉴外纪》</span></p><p><span class="cino">(3)</span> 又如:姓族(大族，望族);姓字(姓氏和名字);姓系(姓氏家族的源流系统);姓第(姓氏行第)</p><p><span class="cino">(4)</span> 命，性命 <span class="encs">[life]</span></p><p><span class="diczx1">救民之姓而不夸，行补三君而不有。——<span class="smcs">《晏子春秋》</span></span></p><p><span class="cino">(5)</span> 官吏 <span class="encs">[government officials]</span></p><p><span class="diczx1">百姓兆民。——<span class="smcs">《国语》</span>。</span><span class="diczx2">韦昭</span>注:<span class="yhcs">“百姓，百官。”</span></p><p><span class="cino">(6)</span> 平民 <span class="encs">[the common people]</span></p><p><span class="diczx1">填国家，抚百姓。——<span class="smcs">《汉书》</span></span></p><p><span class="cino">(7)</span> 子孙的通称 <span class="encs">[descendants]</span></p><p><span class="diczx1">问其姓，对曰:<span class="yhcs">“余子长矣，能奉雉而从我矣。”</span>——<span class="smcs">《左传》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>姓</strong> <span class="dicpy">xìng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xìng.mp3"></a></span> </p><p>〈动〉</p><p>命名，称呼 <span class="encs">[call]</span></p><p><span class="diczx2">项氏</span>世世为 <span class="diczx2">楚</span>将，封于 <span class="diczx2">项</span>，故姓 <span class="diczx2">项氏</span>。——<span class="smcs">《史记·项羽本纪》</span></p><p><span class="diczx1">或曰，</span><span class="diczx2">冉</span>氏尝居也，故姓是溪曰 <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/59D3.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24BE7.svg'><span class='k_txt'>𤯧</span><img class='lazy k_img' data-original='//img.zdic.net/song/cn/24BE3.svg'><span class='k_txt'>𤯣</span>【唐韻】【集韻】【韻會】【正韻】<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>息正切，音性。【說文】人所生也。【左傳·隱八年】天子建德因生以賜姓。</p><p>又孫謂之子姓。【詩·周南】振振公姓。【楚語】率其子姓，從其時享。</p><p>又晐姓，備庶媵也。【吳語】一介嫡女願執箕箒，以晐姓于王宮。</p><p>又百姓，民庶也。【書·堯典】平章百姓。</p><p>又百官族姓。【酒誥】越百姓里居。</p><p>又生子曰姓。【左傳·昭四年】庚宗之婦人，獻以雉。問其姓，對曰：余子長矣，能奉雉而從我矣。</p><p>又姓。【前漢·食貨志】臨葘人姓偉，貲五千萬。【註】姓，姓也。偉，其名。</p><p>又推律定姓。京房本姓李，推律定姓爲京氏。</p><p>又筮易得姓，陸羽不知所生，自筮得蹇之漸，乃姓陸，名羽，字鴻漸。</p><p>又【集韻】師庚切，音生。人名。【春秋·哀四年】蔡殺其大夫公孫姓。</p><p>又叶桑經切，音星。【詩·唐風】豈無他人，不如我同姓。叶上菁。　【鄭曉曰】姓，字从女生。上古八大姓，姜姬嬀姒嬴姞姚妘，皆从女。<img class='lazy k_img' data-original='//img.zdic.net/song/cn/201F7.svg'><span class='k_txt'>𠇷</span>，从人。</p><p>

                          <h2 class="h2_entry"><span class="orth">姓</span><span class="span dictname"> 《说文解字》</span></h2>
                                                  </div>
                        
                        <div class="content definitions snr">
						<div class="mob">【<a href="/zd/sw/?swju=卷十二">卷十二</a>】【<a href="/zd/sw/?swju=卷十二&amp;swbs=女">女</a>】</div>
                                                    <div class="swnr">
                          『說文解字』<hr class="dichr">
                          <img class="lazy kxtimg" data-original="//img.zdic.net/swxz/59D3.svg">                          <p>人所生也。古之神聖母，感天而生子，故稱天子。从女从生，生亦聲。《春秋傳》曰：“天子因生以賜姓。”息正切〖注〗<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24BE7.svg'><span class='k_txt'>𤯧</span>、<img class='lazy k_img' data-original='//img.zdic.net/song/cn/24BE3.svg'><span class='k_txt'>𤯣</span>，古文姓。</p>                          </div>
                                                                              <div class="swnr">
                          『說文解字注』<hr class="dichr">
                          <p><img class="lazy kxtimg" data-original="//img.zdic.net/xz/swdz/59D3_1236.svg"  /><span class="swbf">(姓)</span><span class="ywyy">人所生也。</span><span class="swdz">白虎通曰。姓者、生也。人所稟天氣所以生者也。吹律定姓。故姓有百。按詩振振公姓傳曰。公姓、公生也。不如我同姓傳曰。同姓、同祖也。昭四年左傳。問其姓。釋文云。女生曰姓。姓謂子也。定四年蔡大夫公孫生、公穀皆作公孫姓。</span><span class="ywyy">古之神聖人。母感天而生子。故偁天子。</span><span class="swdz">五經異義。詩齊魯韓、春秋公羊說聖人皆無父感天而生。左氏說聖人皆有父。謹案堯典以親九族。卽堯母慶都感赤龍而生堯。安得九族而親之。禮讖云唐五廟。知不感天而生。玄之聞也。詩言感生得無父。有父則不感生。此皆偏見之說也。商頌曰。天命玄鳥。降而生商。謂娀<img class='lazy k_img' data-original='//img.zdic.net/song/cn/25CD1.svg'><span class='k_txt'>𥳑</span>吞鳦子生契。是聖人感生見於經之明文。劉媼是漢大上皇之妻。感赤龍而生高祖。是非有父感神而生者也。且夫蒲盧之氣、嫗煦桑蟲成爲己子。况乎天氣因人之精。就而神之。反不使子賢聖乎。是則然矣。又何多怪。按此鄭君調停之說。許作異義時、從左氏說聖人皆有父。造說文、則云神聖之母感天而生。不言聖人無父。則與鄭說同矣。</span><span class="ywyy">因生㠯爲姓。从女生。</span><span class="swdz">因生以爲姓、若下文神農母居姜水因以爲姓、黃帝母居姬水因以爲姓、舜母居姚虛因以爲姓是也。感天而生者母也。故姓从女生會意。其子孫復析爲衆姓。如黃帝子二十五宗十二姓、則皆因生以爲姓也。</span><span class="ywyy">生亦聲。</span><span class="swdz">息正切。十一部。按古平聲。</span><span class="ywyy">春秋傳曰。天子因生㠯賜姓。</span><span class="swdz">大小徐本互異。由淺人不學以爲重複。故大徐本刪因生以爲姓五字。小徐刪春秋傳以下十一字。皆非也。傳者、隱八年左傳文。無駭卒。羽父請謚與族。公問族於衆仲。對曰。天子建德。因生以賜姓。胙之土而命之氏。杜曰。因其所由生以賜姓也。按人各有所由生之姓。其後氏別旣久而姓幾湮。有德者出。則天子立之令姓其正姓。若大宗然。如周語帝胙四岳國。賜姓曰姜。氏曰有吕。陳胡公不淫。故周賜之姓。命氏曰陳。飂叔安裔子董父事帝舜。帝賜之姓曰董。氏曰豢龍。葢此三者、本皆姜嬀董之子孫。故予之以其姓。又或特賜之姓。前無所承者。如史記、白虎通禹祖昌意。以薏苡生。賜姓姒氏。殷契以玄鳥子生。賜姓子氏。斯皆因生以賜姓也。必兼春秋傳之說而姓之義乃完。姒字不見於許書。葢古衹作以。古書亦有作似者。</span><span class="xh"></span></p>                          </div>
