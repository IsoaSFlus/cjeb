						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>捉</strong></p><p><span class="dicpy">zhuō  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuō.mp3"></a></span>  ㄓㄨㄛˉ</span></p><ol><li>抓，逮：～拿。～奸。～贼。～弄。捕～。捕风～影。</li><li>握：～笔。～刀（代人作文）。～襟见肘。</li></ol>

                    <h2 class="h2_entry"><span class="orth">捉</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>捉</strong> <span class="dicpy">zhuō</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuō.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，足声。本义:握，持)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[hold;grasp]</span></p><p><span class="diczx1">捉，搤也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">捉，持也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">伊便能捉杖打人。——<span class="smcs">《世说新语·方正》</span></span></p><p><span class="diczx2">叔武</span>将沐，闻君至，喜，捉发走出。——<span class="smcs">《左传》</span></p><p><span class="diczx2">孙权</span>捉 <span class="diczx2">预</span>手，涕泣而别。——<span class="smcs">《三国志》</span></p><p><span class="cino">(3)</span> 又如:捉发(手握头发);捉秤(拿秤，掌秤);捉手(握手);捉身(把握身体);捉船(撑船，划船);捉笔(执笔)</p><p><span class="cino">(4)</span> <span class="encs">[到唐代引申出]</span>捉拿，擒拿;追捕 <span class="encs">[catch;arrest]</span></p><p><span class="diczx1">暮投</span><span class="diczx2">石壕村</span>，有吏夜捉人。—— <span class="diczx2">唐</span>· <span class="diczx2">杜甫</span><span class="smcs">《石壕吏》</span></p><p><span class="diczx1">往日用钱捉私铸，今许铅锡和青铜。——</span><span class="diczx2">杜甫</span><span class="smcs">《岁晏行》</span></p><p><span class="diczx1">遍捉乡民。——<span class="smcs">《广东军务记》</span></span></p><p><span class="diczx1">不能行捉。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(5)</span> 又</p><p><span class="diczx1">迅不可捉。</span></p><p><span class="diczx1">余捉蟋蟀。——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《祭妹文》</span></p><p><span class="cino">(6)</span> 又如:捉事人(捉拿罪犯的公人);捉生替死(陷害无罪之人为有罪者抵死);捉生(活捉敌人);捉取(擒拿;捕捉)</p><p><span class="cino">(7)</span> 戏弄 <span class="encs">[make fun of]</span></p><p><span class="diczx2">张珙</span>新来，受了别人家捉。—— <span class="diczx2">董解元</span><span class="smcs">《西厢》</span></p><p><span class="cino">(8)</span> 又如:捉掐(捉弄);捉狭(捉弄);捉搦(戏弄)</p><p><span class="cino">(9)</span> 捡，拾 <span class="encs">[pick up]</span></p><p><span class="diczx1">若见女人一发在地，自言持戒，不肯捉之。——<span class="smcs">《百喻经》</span></span></p><p><span class="cino">(10)</span> 找，趁 <span class="encs">[find]</span></p><p><span class="diczx1">当日权且欢天喜地，并无他说。明日捉个空，便一径到</span><span class="diczx2">临安府</span>前叫起屈来。——<span class="smcs">《京本通俗小说》</span></p><p><span class="diczx1">你瞧，太太们老在一起，哪有我捉空儿跟他说话的机会。——</span><span class="diczx2">茅盾</span><span class="smcs">《霜叶红似二月花》</span></p><p><span class="cino">(11)</span> 又如:捉个空(找个空子);捉冷眼(趁人没看见);捉脉(堪舆家据地势，寻找所谓龙脉);捉空(趁空);捉破(抓出破绽)</p><p><span class="cino">(12)</span> 控制 <span class="encs">[control]</span></p><p><span class="diczx1">那小二不曾提防，捉脚不定，翻筋斗直跌出门外，磕碎脑后，鲜血直淌。——<span class="smcs">《醒世恒言》</span></span></p><p><span class="diczx1">原来</span><span class="diczx2">杨志</span>吃的酒少，便醒得快，爬将起来，兀自捉脚不住。——<span class="smcs">《水浒全传》</span></p><p><span class="cino">(13)</span> 又如:捉不住(控制不住);捉兵(提调军队);捉店(守护店铺)</p><p><strong>词性变化</strong></p><p>◎ <strong>捉</strong> <span class="dicpy">zhuō</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuō.mp3"></a></span>
