                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>抑</strong></p><p><span class="dicpy">yì  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> ㄧˋ</span></p><ol><li>压，压制：～制。～止。压～。～强扶弱。～扬（a.音调的高低起伏；b.沉浮；c.褒贬）。</li><li>忧闷：～郁。～塞（<span class="dicpy">sè</span>）。</li><li>文言连词（a.表选择，相当于或是、还是，如“～或”；b.表转折，相当于可是、但是，如“多则多矣，～君似鼠”）。</li><li>文言发语词：“～齐人不盟，若之何”。</li><li>古同“<span class="diczx3">噫</span>”，叹词。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>抑</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (本写作<span class="yhcs">“归”</span>。会意。小篆字形，象用手抑人使跽。本义:按，向下压)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[press down]</span></p><p><span class="diczx1">抑之欲其奥。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="cino">(3)</span> 又如:抑按(按压);抑首(按其首);抑搔(按摩抓搔);抑扬(按下与上举)</p><p><span class="cino">(4)</span> 抑制 <span class="encs">[restrain]</span></p><p><span class="diczx1">抑其兼并。——</span><span class="diczx2">清</span>· <span class="diczx2">洪亮吉</span><span class="smcs">《治平篇》</span></p><p><span class="diczx1">不可遏抑。——</span><span class="diczx2">孙文</span><span class="smcs">《黄花冈七十二烈士事略序》</span></p><p><span class="cino">(5)</span> 又如:抑勒(压制被扣);抑情(克制感情);抑引(抑制情欲而导之以善);抑末(抑制商贾);抑志(抑制自己的志向);抑枉(抑制邪辟);抑削(抑制削弱);抑挫(抑制折挫)</p><p><span class="cino">(6)</span> 贬 <span class="encs">[demote]</span></p><p><span class="diczx1">教之春秋，而为之耸善而抑恶焉。——<span class="smcs">《国语·楚语上》</span></span></p><p><span class="cino">(7)</span> 又如:抑卑(贬低);抑贬(贬低);抑退(黜退;贬退);抑损(贬低;贬损);抑黜(亦作<span class="yhcs">“抑绌”</span>。贬废;排斥)</p><p><span class="cino">(8)</span> 阻止 <span class="encs">[hold back]</span>。如:抑割(遏止舍弃);抑阻(阻挠;阻止)</p><p><span class="cino">(9)</span> 强行;强迫 <span class="encs">[force]</span></p><p><span class="diczx1">乙卯，诏诸路散青苗钱，禁抑配。——<span class="smcs">《宋史·神宗纪二》</span></span></p><p><span class="cino">(10)</span> 又如:抑配(强行分配);抑勒(强逼;压制);抑逼(强迫);抑留(强行留住);抑掠(强夺);抑买(强行购买);抑夺(剥夺;强行夺取);抑籴(强行收购粮食)</p><p><span class="cino">(11)</span> 忧郁 <span class="encs">[worry]</span>。如:抑抑(忧郁的样子);抑悒(忧郁)</p><p><strong>词性变化</strong></p><p>◎ <strong>抑</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p><连></p><p><span class="cino">(1)</span> 或是，还是 <span class="encs">[or]</span></p><p><span class="diczx1">抑其无取。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="diczx1">抑未死耶。——</span><span class="diczx2">清</span>· <span class="diczx2">全祖望</span><span
