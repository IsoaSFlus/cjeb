                    <h2 class="h2_entry"><span class="orth">辜</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>辜</strong> <span class="dicpy">gū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gū.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从辛，古声。<span class="yhcs">“辛”</span>，是受刑的意思。本义:罪行)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[crime]</span></p><p><span class="diczx1">辜，罪也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">段</span>注:<span class="yhcs">“辜本非常重罪，引申之凡有罪者皆曰辜。”</span></p><p><span class="diczx1">民之无辜。——<span class="smcs">《诗·小雅·正月》</span></span></p><p><span class="diczx1">无罪无辜。——<span class="smcs">《诗·小雅·十月之交》</span></span></p><p><span class="diczx1">天乎无辜。——<span class="smcs">《公羊传·宣公六年》</span></span></p><p><span class="diczx1">至</span><span class="diczx2">齐</span>，见辜人焉。——<span class="smcs">《庄子·则阳》</span></p><p><span class="diczx1">赏无功之人，罚不辜之民，非所谓明也。——<span class="smcs">《韩非子·说疑》</span></span></p><p><span class="diczx1">至</span><span class="diczx2">海陵</span>，如 <span class="diczx2">高沙</span>，常恐无辜死。—— <span class="diczx2">宋</span>· <span class="diczx2">文天祥</span><span class="smcs">《指南录后序》</span></p><p><span class="cino">(3)</span> 又如:死有余辜;无辜</p><p><span class="cino">(4)</span> 灾难，祸害 <span class="encs">[calamity]</span></p><p><span class="diczx1">害遍生民，辜及朽骨。——<span class="smcs">《汉书》</span></span></p><p><span class="cino">(5)</span> 通<span class="yhcs">“故”</span>( <span class="dicpy">gù</span>)。原故，原因 <span class="encs">[reason;cause]</span></p><p><span class="diczx1">般纷纷其离此尤兮，亦夫子之辜也。——<span class="smcs">《史记·屈原贾生列传》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>辜</strong> <span class="dicpy">gū</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gū.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 惩罚，治罪 <span class="encs">[punish]</span></p><p><span class="diczx1">挟目而辜。——<span class="smcs">《说苑·杂言》</span></span></p><p><span class="cino">(2)</span> 肢解，分裂肢体 <span class="encs">[dismember]</span></p><p><span class="diczx1">杀王之亲者辜之。——<span class="smcs">《周礼·秋官·掌戮》</span></span></p><p><span class="diczx1">大罪莫重辜磔于市。——<span
