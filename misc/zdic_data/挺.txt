                    <h2 class="h2_entry"><span class="orth">挺</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>挺</strong> <span class="dicpy">tǐng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǐng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，廷声。本义:拔出)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[draw]</span></p><p><span class="diczx1">挺，拔也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">挺剑而起。——<span class="smcs">《战国策·魏策》</span></span></p><p><span class="diczx1">尉剑挺。——<span class="smcs">《史记·陈涉世家》</span></span></p><p><span class="cino">(3)</span> 又如:挺举;挺挏(上下推动)</p><p><span class="cino">(4)</span> 举起 <span class="encs">[lift]</span>。如:挺脚(举步，抬脚)</p><p><span class="cino">(5)</span> 生出;生长 <span class="encs">[grow]</span></p><p><span class="diczx1">荔挺。——<span class="smcs">《吕氏春秋·仲冬》</span>。注:<span class="yhcs">“生也。”</span></span></p><p><span class="cino">(6)</span> 又如:挺茂(生长茂盛)</p><p><span class="cino">(7)</span> 动摇 <span class="encs">[vacillate]</span></p><p><span class="diczx1">不足以挺其心矣。——<span class="smcs">《吕氏春秋》</span></span></p><p><span class="cino">(8)</span> 直，伸直，撑直 <span class="encs">[straighten]</span></p><p><span class="diczx1">不复挺者。——<span class="smcs">《荀子·劝学》</span></span></p><p><span class="diczx1">曲挺纵横。——<span class="smcs">《徐霞客游记·游黄山记》</span></span></p><p><span class="cino">(9)</span> 睡 <span class="encs">[sleep]</span>。</p><p><span class="cino">(10)</span> 如:挺死尸(骂人睡得死);挺床(詈词。睡觉);挺觉(睡眠的谑词或詈词)</p><p><span class="cino">(11)</span> 使身体处于直立状态 <span class="encs">[stick out(a part of the body)]</span>。</p><p><span class="cino">(12)</span> 如:挺争(挺身争斗)</p><p><span class="cino">(13)</span> 勉强支撑 <span class="encs">[stand]</span>。</p><p><span class="cino">(14)</span> 如:挺折腰(到了顶，竭尽全力);你挺得住吗</p><p><span class="cino">(15)</span> 突出 <span class="encs">[protrude]</span>。</p><p><span class="cino">(16)</span> 如:挺着肚子;他把头挺得高高的</p><p><span class="cino">(17)</span> 忍受痛苦 <span class="encs">[sustain]</span>。</p><p><span class="cino">(18)</span> 如:挺过去;挺住</p><p><span class="cino">(19)</span> 前进 <span class="encs">[advance]</span></p><p><span class="diczx1">单身挺战，众莫能当。——<span class="smcs">《宋书》</span></span></p><p><span class="cino">(20)</span> 顶住，抵抗 <span class="encs">[resist]</span>。</p><p><span class="cino">(21)</span> 如:挺撞(顶撞);谁能挺敌;挺刑(不屈服于刑罚)</p><p><span class="cino">(22)</span> 脱身 <span class="encs">[get away]</span>。</p><p><span class="cino">(23)</span> 如:挺走(退走);挺逸(脱逃)</p><p><strong>词性变化</strong></p><p>◎ <strong>挺</strong> <span class="dicpy">tǐng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/tǐng.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 杰出;特出 <span class="encs">[outstanding]</span></p><p><span class="diczx1">自谓颇挺出，立登要路津。——</span><span class="diczx2">杜甫</span><span class="smcs">《奉赠韦左丞丈二十二韵》</span></p><p><span class="cino">(2)</span>
