                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>拽</strong> <span class="dicpy">yè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" class="yhcs">“曳”</span> <span class="encs">[drag;haul]</span>。如 :拽步(拉开脚步);拽扶(牵扶);拽扎(绷紧;捆紧)</p><p><span class="cino">(2)</span> 携带 <span class="encs">[bring]</span></p><p><span class="diczx1">(</span><span class="diczx2">盖道云</span>)便把札子取了出来 ，给大家瞧了一瞧，仍旧拽在身上。——<span class="smcs">《官场现形记》</span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>拽</strong> <span class="dicpy">yè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yè.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 短桨。一说船舷 <span class="encs">[short oar]</span></p><p><span class="diczx1">今又促装下邑，浪拽上京。——<span class="smcs">《文选·孔稚圭·北山移文》</span></span></p><p><span class="cino">(2)</span> 另见 <span class="dicpy">zhuāi;zhuài</span></p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>拽</strong> <span class="dicpy">zhuāi</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhuāi.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 〈方〉∶抛;扔 <span class="encs">[fling;hurl;throw]</span>。如:把皮球拽出去</p><p><span class="cino">(2)</span> 另见 <span class="dicpy">yè;zhuài</span></p><p><strong>基本词义</strong></p><hr class="dichr" /><p>◎ <strong>拽</strong> <span class="dicpy">zhuài</span> <span class="ptr"><a
