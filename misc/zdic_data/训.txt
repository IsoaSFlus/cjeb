                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>训</strong></p><p><strong>（<span class="diczx3">訓</span>）</strong></p><p><span class="dicpy">xùn  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xùn.mp3"></a></span> ㄒㄩㄣˋ</span></p><ol><li>教导，教诲：～诫。～蒙（教育儿童）。～迪（教诲开导）。教（<span class="dicpy">jiào </span>）～。培～。</li><li>可以作为法则的话：家～。</li><li>典式，法则：不足为～。</li><li>解释词的意义：～诂（解释古书中的字、词句的意义。亦称“～故”、“诂～”、“故～”）。～读（日文借用汉字写日语原有的词，用日语语音读汉字）。</li></ol>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>训</strong></p><p>訓 <span class="dicpy">xùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xùn.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从言，川声。本义:教育，教导)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[instruct;teach;lecture]</span></p><p><span class="diczx1">训，说教也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">大训。——<span class="smcs">《书·顾命》</span></span></p><p><span class="diczx1">告之训典。——<span class="smcs">《左传·文公六年》</span></span></p><p><span class="diczx1">是为明训。——<span class="smcs">《国语·晋语》</span>。注:<span class="yhcs">“教也。”</span></span></p><p><span class="diczx1">务材训农，通商惠工。——<span class="smcs">《左传·闵公二年》</span></span></p><p><span class="diczx1">而五人生于编伍之间，素不闻诗书之训。——</span><span class="diczx2">张溥</span><span class="smcs">《五人墓碑记》</span></p><p><span class="diczx1">训俭示康。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="cino">(3)</span> 又如:训蒙(在私塾教授小学生);训蒙教授(教幼童的私塾先生);训人(负责教育的官员或师长);训章(训示规范)</p><p><span class="cino">(4)</span> 解说，注释。用通俗的话解释词语的意义 <span class="encs">[explain]</span></p><p><span class="diczx1">但世人未之深亮训说，况文烦富，行于世者，失其旨要，故撰为<span class="smcs">《略解》</span>焉。——</span><span class="diczx2">三国</span><span class="diczx2">魏</span>· <span class="diczx2">曹操</span><span class="smcs">《孙子》</span>序</p><p><span class="cino">(5)</span> 又如:训故(同训诂);训义(解释文字的意义);训解(训释解说);训传(训解经义)</p><p><span class="cino">(6)</span> 训练;讲习 <span class="encs">[train;lecture and study]</span></p><p><span class="diczx1">有智略，能训治军旅。——</span><span class="diczx2">宋</span>· <span class="diczx2">王安石</span><span class="smcs">《举渭川兵马都监盖传等充边上任使状》</span></p><p><span class="cino">(7)</span> 又如:训戎(训练军旅);训治(训练整治)</p><p><span class="cino">(8)</span> 顺从;归顺 <span class="encs">[tame]</span></p><p><span class="diczx1">四方其训之。——<span class="smcs">《诗·大雅·抑》</span></span></p><p><span class="diczx1">于帝其训，又，是训是行。——<span class="smcs">《书·洪范》</span></span></p><p><span class="cino">(9)</span> 取名 <span class="encs">[choose a name for]</span></p><p><span class="diczx1">后志存小字，不训法名者，遵慈母之意也。——</span><span class="diczx2">宋</span>· <span class="diczx2">赞宁</span><span class="smcs">《宋高僧传》</span></p><p><strong>词性变化</strong></p><p>◎ <strong>训</strong></p><p>訓 <span class="dicpy">xùn</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/xùn.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 典式、法则 <span                           <img class="lazy kxtimg" data-original="//img.zdic.net/kxzd/8A13.svg">						  <p>〔古文〕<img class='lazy k_img' data-original='//img.zdic.net/song/cn/27965.svg'><span class='k_txt'>𧥥</span>馴【唐韻】【集韻】【韻會】許運切【正韻】吁運切，<img class='lazy k_img' data-original='//img.zdic.net/song/cn/20024.svg'><span class='k_txt'>𠀤</span>薰去聲。【說文】說敎也。【徐曰】訓者，順其意以訓之也。【正韻】誨也。【字彙】導也。【書·伊訓】伊尹乃明言烈祖之成德以訓于王。【傳】作訓以敎導太甲。【詩·大雅】四方其訓之。【正義】訓是敎誨之別名。【禮·曲禮】敎訓正俗。【疏】謂訓說義理。</p><p>又【正字通】古言可爲法也。【書·說命】學于古訓。【詩·大雅】古訓是式。【傳】古，故也。訓，道也。【傳】古訓，先王之遺典也。【左傳·文六年】告之訓典。【註】訓典，先王之書。</p><p>又【玉篇】誡也。【魏書·高允傳】臣被敕論集往世酒之敗德以爲《酒訓》。</p><p>又【廣韻】男曰敎，女曰訓。</p><p>又【博雅】順也。【書·康誥】皇天用訓厥道，付畀四方。【傳】上天用順文武之道，而付之以天下之大也。【後漢·方術傳】下使五品咸訓于嘉時。【註】訓，順也。</p><p>又詁訓，註解之別名。【爾雅·郭璞序】《爾雅》者，所以通詁訓之指歸。【疏】訓，道也。道物之貌以吿人也。【前漢·揚雄傳】不爲章句訓詁。【師古註】訓者，釋所言之理也。【字彙】如某字釋作某義，順其義以訓之也。</p><p>又官名。【周禮·夏官】訓方氏，掌道四方之政事。</p><p>又鳥名。【唐書·五行志】鵂鶹，一名訓狐。</p><p>又獸名。【神異經】檮杌，西方荒中獸。一名難訓。</p><p>又州名。【唐書·地理志】江南道有訓州。</p><p>又姓。明宣德中有敎授訓濬。</p><p>又【五音集韻】詳遵切，音旬。道也。【周禮·地官】土訓，鄭司農讀。</p><p>又叶文韻。【晉書·衞恆傳】大晉開元，弘道敷訓。天垂其象，地耀其文。</p>
