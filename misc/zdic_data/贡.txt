                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>贡</strong></p><p><strong>（<span class="diczx3">貢</span>）</strong></p><p><span class="dicpy">gòng  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gòng.mp3"></a></span> ㄍㄨㄥˋ</span></p><ol><li>献东西给上级，古代臣下或属国把物品进献给帝王：进～。纳～。～献。～品。～赋。</li><li>封建时代给朝廷荐举人才：～生（指经科举考试升入京师国子监读书的人）。～举。</li><li>姓。</li></ol>

                    <h2 class="h2_entry"><span class="orth">贡</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>贡</strong></p><p>貢 <span class="dicpy">gòng</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/gòng.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从贝，工声。本义:进贡，把物品进献给朝廷)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[offer tribute]</span></p><p><span class="diczx1">贡，献功也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">贡，上也。——<span class="smcs">《释言》</span></span></p><p><span class="diczx1">贡，献也。——<span class="smcs">《广雅》</span></span></p><p><span class="diczx1">尔贡包茅不入。——<span class="smcs">《左传·僖公四年》</span></span></p><p><span class="diczx1">以令地贡。——<span class="smcs">《周礼·大司徒》</span>。注:<span class="yhcs">“贡地所生，谓九谷。”</span></span></p><p><span class="diczx1">以任地事而令贡赋。——<span class="smcs">《周礼·小司徒》</span>。注:<span class="yhcs">“谓九谷三泽之材也。”</span></span></p><p><span class="diczx1">贡者，自治其所受田，贡其税谷。——<span class="smcs">《考工记·匠人》</span>注</span></p><p><span class="diczx1">举天下所贡蝴蝶、螳螂、油利挞、青丝额一切异状遍试之，无出其右者。——<span class="smcs">《聊斋志异·促织》</span></span></p><p><span class="cino">(3)</span> 又如:贡奉(进献物品给朝廷);贡云(宋宫中假山艮岳既成，命近山处居民以油绢巨囊，于晨间捕捉山间野云，用车驾运送艮岳，放纵假山间，以供君王观赏，名为贡云)</p><p><span class="cino">(4)</span> 示告;告诉 <span class="encs">[tell;talk]</span></p><p><span class="diczx1">六爻之义易以贡。——<span class="smcs">《易·系辞》</span>。</span><span class="diczx2">韩</span>注:<span class="yhcs">“告也。”</span></p><p><span class="cino">(5)</span> 又如:贡诚(表达真诚之心);贡实(表达诚心);贡款(贡诚)</p><p><span class="cino">(6)</span> 赏赐 <span class="encs">[award]</span></p><p><span class="diczx1">贡，赐也。——<span class="smcs">《尔雅·释诂》</span></span></p><p><span class="diczx1">端木赐，字子贡。——<span class="smcs">《史记·仲尼弟子传》</span></span></p><p><span class="cino">(7)</span> 又如:贡土(古代帝王分封诸侯时，按封地方位取一色土授之，供其立社设坛)</p><p><span class="cino">(8)</span> 推荐 <span class="encs">[某人]</span> <span class="encs">[recommend]</span></p><p><span class="diczx1">举人贡士。——<span class="smcs">《后汉书·章帝纪》</span></span></p><p><span class="diczx1">尝勤苦学文，迨今十年，始获一贡。——</span><span class="diczx2">唐</span>· <span class="diczx2">白居易</span><span class="smcs">《与陈给事书》</span></p><p><span class="cino">(9)</span> 又如:贡人(由府、州、县推荐到朝廷的人士);贡士(清代会试被录取者);贡官(科举时代主持考务的官员);贡察(举荐);贡贤(荐举贤人);贡选(州、郡向中央举荐选拔的人材)</p><p><span class="cino">(10)</span> 成为贡生。亦指贡生 <span class="encs">[become scholar]</span>。如:贡元(对贡生的尊称);贡行(贡生);贡监(明、清时生员入国子监读书的贡生和监生)</p><p><strong>词性变化</strong></p><p>◎ <strong>贡</strong></p><p>貢 <span class="dicpy">gòng</span> <span class="ptr"><a class="audio_play_button
