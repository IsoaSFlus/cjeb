						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>速</strong></p><p><span class="dicpy">sù  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sù.mp3"></a></span>  ㄙㄨˋ</span></p><ol><li>快：～记。～效。～印。～写。～成。飞～。神～。</li><li>速度：慢～。超～。加～。风～。</li><li>邀请，招致：不～之客。</li></ol>

                    <h2 class="h2_entry"><span class="orth">速</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>速</strong> <span class="dicpy">sù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sù.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> (形声。从辵( <span class="dicpy">chuò</span>)，束声。本义:速度快)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[fast;quick;rapid]</span></p><p><span class="diczx1">速，疾也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">不微至，无以为戚速也。——<span class="smcs">《周礼·考工记·总目》</span></span></p><p><span class="cino">(3)</span> 字亦作<span class="yhcs">“遬”</span></p><p><span class="diczx1">轻则僄遬。——<span class="smcs">《荀子·议兵》</span></span></p><p><span class="diczx1">则莫能以速中。——<span class="smcs">《周礼·考工记·弓人》</span></span></p><p><span class="diczx1">王速出令。——<span class="smcs">《孟子·梁惠王下》</span></span></p><p><span class="diczx1">子济而陈，迟速唯命。——<span class="smcs">《左传·僖公三十三年》</span></span></p><p><span class="diczx1">知幸与不幸，则其读书也必专，而其归书也必速。——</span><span class="diczx2">清</span>· <span class="diczx2">袁枚</span><span class="smcs">《黄生借书说》</span></p><p><span class="diczx1">今不速往，恐为</span><span class="diczx2">操</span>所先。——<span class="smcs">《资治通鉴》</span></p><p><span class="cino">(4)</span> 又如:收效甚速;速速(非常快速);速捷(快速敏捷);速战速决(用最快的速度发动战争，以达到预期的目的;比喻凡事必须进行得快，才能迅速结束);速食店(快餐店)</p><p><span class="cino">(5)</span> 急迫;紧急 <span class="encs">[urgent;pressing]</span>。如:速刻(即刻;立刻);速帖(加急请帖);速忙(急忙，赶快);速煞(极快，非常迅速);速件(急件)</p><p><strong>词性变化</strong></p><p>◎ <strong>速</strong> <span class="dicpy">sù</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/sù.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 加速，加快 <span class="encs">[quicken;speed up]</span></p><p><span class="diczx1">交语速装束，络绎如浮云。——<span class="smcs">《玉台新咏·古诗为焦仲卿妻作》</span></span></p><p><span class="cino">(2)</span> 又如:速熟(使农作物生长期短、成熟较快)</p><p><span class="cino">(3)</span> 邀请 <span class="encs">[invite]</span></p><p><span class="diczx1">主人速宾。——<span class="smcs">《仪礼·乡饮酒礼》</span></span></p><p><span class="diczx1">何以速我狱。——<span class="smcs">《诗·召南·行露》</span></span></p><p><span class="diczx1">有不速之客三人来，敬之终吉。——<span class="smcs">《易·需》</span></span></p><p><span class="cino">(4)</span> 又如:速客(请客);速帖(请帖)</p><p><span class="cino">(5)</span> 招致 <span class="encs">[incur]</span></p><p><span class="diczx1">君子多欲则贪慕富贵，枉道速祸。——</span><span class="diczx2">宋</span>· <span class="diczx2">司马光</span><span class="smcs">《训俭示康》</span></p><p><span class="diczx1">至</span><span class="diczx2">丹</span>以 <span class="diczx2">荆卿</span>为计，始速祸焉。—— <span class="diczx2">宋</span>· <span class="diczx2">苏洵</span><span class="smcs">《六国论》</span></p><p><span class="cino">(6)</span> 又如:速辜(招来罪过);速福(召福);速祸(招来祸害);速罪(招致罪祸)</p><p><span class="cino">(7)</span> 催促 <span class="encs">[urge]</span></p><p><span
