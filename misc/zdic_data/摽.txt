                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>摽</strong></p><p><span class="dicpy">biào  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biào.mp3"></a></span>  ㄅㄧㄠˋ</span></p><ol><li>紧紧地捆绑物体使其相连：椅子腿活动了，用铁丝～住。</li><li>比着：～着劲干。</li><li>用胳膊紧紧地钩住：他俩～着胳膊走。</li><li>亲近，依附（多含贬义）：他们～在一块儿。</li><li>落：～梅。</li><li>打，击：“长木之毙，无不～也”。</li><li>捶胸的样子：“静言思之，寤辟有～”。</li></ol><p><strong>其它字义</strong></p><hr class="dichr" /><p>● <strong>摽</strong></p><p><span class="dicpy">biāo  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biāo.mp3"></a></span>  ㄅㄧㄠˉ</span></p><ol><li>挥之使动：“曹子～剑而去之”。</li><li>高扬：“～然若秋云之远”。</li><li>古同“<span class="diczx3">标</span>”，标榜。</li></ol>

                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>摽</strong> <span class="dicpy">biāo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biāo.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 挥之使去 <span class="encs">[dismiss with a hand gesture]</span></p><p><span class="diczx1">摽使者出诸大门之外。——<span class="smcs">《孟子》</span></span></p><p><span class="cino">(2)</span> 抛弃 <span class="encs">[get rid of;abandon]</span></p><p><span class="diczx1">已盟，</span><span class="diczx2">曹子</span>摽剑而去之。——<span class="smcs">《公羊传·庄公十三年》</span></p><p><span class="cino">(3)</span> 另见 <span class="dicpy">biào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biào.mp3"></a></span> </p><p><strong>其它字义</strong></p><hr class="dichr" /><p>◎ <strong>摽</strong> <span class="dicpy">biào</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biào.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> 捶胸;击 <span class="encs">[strike]</span></p><p><span class="diczx1">摽，击也。从手，票声。字亦作抛。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">寤辟有摽。——<span class="smcs">《诗·柏舟》</span></span></p><p><span class="diczx1">长木之斃，无不摽也。——<span class="smcs">《左传·哀公十二年》</span></span></p><p><span class="cino">(2)</span> 又如:摽辟(捶胸)</p><p><span class="cino">(3)</span> 落下 <span class="encs">[fall]</span></p><p><span class="diczx1">摽有梅，其实七兮。——<span class="smcs">《诗·召南·摽有梅》</span></span></p><p><span class="cino">(4)</span> 又如:摽落(落下);摽梅(梅子成熟而落下，比喻女子应当结婚的年龄)</p><p><span class="cino">(5)</span> 用胳膊紧紧地钩住 <span class="encs">[lock in arms]</span>。如:两个人摽着胳膊走</p><p><span class="cino">(6)</span> 捆绑或互相缠绕在一起 <span class="encs">[fasten tightly to sth.]</span>。如:把口袋摽在车架子上。引申为结合。如:大伙摽成了一股劲;他们老摽在一起</p><p><span class="cino">(7)</span> 另见 <span class="dicpy">biāo</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/biāo.mp3"></a></span>
