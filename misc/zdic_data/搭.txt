                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>搭</strong> <span class="dicpy">dā</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dā.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (形声。从手，荅声。本义:打，击)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[strike]</span></p><p><span class="diczx1">常于</span><span class="diczx2">并州市</span>搭杀人者。——<span class="smcs">《北齐书·神武纪》</span></p><p><span class="cino">(3)</span> 又如:搭面(抽打面部);扑搭;搭墩(跌坐)</p><p><span class="cino">(4)</span> 披;挂;戴 <span class="encs">[hang over]</span></p><p><span class="diczx1">行者的眼乖，见他屋檐下，有一条搭衣的绳子。——<span class="smcs">《西游记》</span></span></p><p><span class="cino">(5)</span> 又如:把衣服搭在篙子上;肩搭道衣;软帽搭在头上</p><p><span class="cino">(6)</span> 答腔;答话;交谈 <span class="encs">[answer]</span>。如:搭茬;搭岔儿(对答)</p><p><span class="cino">(7)</span> 架设 <span class="encs">[put up;erect;build]</span></p><p><span class="diczx1">又在西花厅上，搭了一座菊花假山。——<span class="smcs">《老残游记》</span></span></p><p><span class="cino">(8)</span> 又如:搭舞台;搭浮桥;搭厂(搭棚;搭台)</p><p><span class="cino">(9)</span> 搁;接触 <span class="encs">[touch;come into contact]</span></p><p><span class="diczx1">他两个搭上手，却才赌斗。——<span class="smcs">《西游记》</span></span></p><p><span class="cino">(10)</span> 又如:两根电线搭上了;勾肩搭背</p><p><span class="cino">(11)</span> 凑上;加上 <span class="encs">[add;throw in more]</span>。如:把这些钱搭上就够了;搭嘴(吃零食);搭头(附带的;凑数的)</p><p><span class="cino">(12)</span> 乘 <span class="encs">[飞机、轮船…]</span> 去 <span class="encs">[travel by]</span>。如:搭飞机;搭轮船;搭他们的车走</p><p><span class="cino">(13)</span> 勾搭，勾引 <span class="encs">[seduce]</span></p><p><span class="diczx2">阎婆惜</span>自从和那 <span class="diczx2">小张三</span>两个搭上，并无半点情分在这 <span class="diczx2">宋江</span>身上。——<span class="smcs">《水浒全传》</span></p><p><span class="cino">(14)</span> 参加，加入 <span class="encs">[join]</span>。如:搭对(做对手);搭犋(方言。合作使用畜力)</p><p><span class="cino">(15)</span> 钩 <span class="encs">[hook;secure with a hook]</span></p><p><span class="diczx1">叫轿夫歇下轿子，拨去葱管，搭起轿帘。——<span class="smcs">《水浒传》</span></span></p><p><strong>词性变化</strong></p><p>◎ <strong>搭</strong> <span class="dicpy">dā</span> <span class="diczx2">宋</span>· <span class="diczx2">林逋</span><span class="smcs">《深居杂兴》</span></p><p>◎ <strong>搭</strong> <span class="dicpy">dā</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/dā.mp3"></a></span> </p><p>〈量〉</p><p>块，处 <span class="encs">[piece]</span></p><p><span class="diczx1">珍哥的脸就如三月的花园，一搭青，一搭紫，一搭绿，一搭红。——<span
