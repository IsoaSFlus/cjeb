                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>支</strong></p><p><span class="dicpy">zhī  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span>  ㄓˉ</span></p><ol><li>撑持，伸出，竖起：～撑。～援。体力不～。</li><li>受得住：乐不可～。</li><li>领款或付款：～付。</li><li>调度，指使：～使。～应。～着儿。</li><li>附属于总体的一个部分：总～。～流。～离。</li><li>〔地～〕历法中用的十二个字：子丑寅卯辰巳午未申酉戌亥。</li><li>量词：一～笔。</li></ol>                        
<div class="enbox"><p><span class="z_ts2">英语</span> disperse, pay; support; branch</p><p><span class="z_ts2">德语</span> Radikal Nr. 65 = Zweig, stützen, aufrichten, Abzweigung  (S)​,Zähleinheitswort  für länglichen, stockartigen Gegenstand (z.B. Stift, Pinsel, Essstäbchen)​  (Zähl)</p><p><span class="z_ts2">法语</span> branche,rameau,appuyer,soutenir,payer,(classificateur pour les objets longs, les troupes, les chansons, les ampoules électriques)​,(nom de famille)​</p></div>

                    <h2 class="h2_entry"><span class="orth">支</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>支</strong> <span class="dicpy">zhī</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zhī.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (会意。小篆字形，上面是<span class="yhcs">“竹”</span>的一半，下面是手(又)。<span class="smcs">《说文》</span>:<span class="yhcs">“从手持半竹。”</span>本义:去枝的竹子)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[branchless bamboo]</span></p><p><span class="diczx1">支，去竹之枝也。从手持半竹。——<span class="smcs">《说文》</span>。</span><span class="diczx2">桂馥</span>义证:<span class="yhcs">“疑作去枝之竹也。”</span></p><p><span class="cino">(3)</span> 枝条。后作<span class="yhcs">“枝”</span> <span class="encs">[branch]</span></p><p><span class="diczx1">芄兰之支。——<span class="smcs">《诗·卫风·芄兰》</span></span></p><p><span class="diczx1">支叶茂接。——<span class="smcs">《汉书》</span></span></p><p><span class="diczx1">以畅其支。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《柳河东集》</span></p><p><span class="cino">(4)</span> 又如:支调(枝节);支蘖(枝条，旁生树枝);支辅(辅助枝叶)</p><p><span class="cino">(5)</span> 分支 <span class="encs">[branch;offshoot]</span></p><p><span class="diczx1">封弟子功臣自为支辅。——<span class="smcs">《史记·李斯列传》</span></span></p><p><span class="diczx1">中小支。——<span class="smcs">《后汉书·吕布传》</span>。注:<span class="yhcs">“谓胡也。即今之戟旁曲支。”</span></span></p><p><span class="cino">(6)</span> 复杂整体中的一部分。如:支店;支军，支兵(主力部队以外的军队)</p><p><span class="cino">(7)</span> 支流</p><p><span class="diczx1">分一支为</span><span class="diczx2">南江</span>。—— <span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《过小孤山大孤山》</span></p><p><span class="diczx1">不啻小支。——<span class="encs">[英]</span></span><span class="diczx2">赫胥黎</span>著、 <span class="diczx2">严复</span>译<span class="smcs">《天演论》</span></p><p><span class="cino">(8)</span> 又如:支川</p><p><span class="cino">(9)</span> 祖宗支系</p><p><span class="diczx2">文王</span>孙子，本支百世。——<span class="smcs">《诗·大雅·文王》</span></p><p><span class="cino">(10)</span> 又如:支胄(支流、宗族的派系分支);支胤(后代子孙);支析(指分家);支别(分支;歧出)</p><p><span class="cino">(11)</span> 支，线及棉纱等的粗细单位 <span class="encs">[count]</span></p><p><span class="cino">(12)</span> 地支的简称 <span class="encs">[Earthly Branches]</span></p><p><span class="cino">(13)</span> 通<span class="yhcs">“肢”</span><span class="encs">[limbs]</span></p><p><span class="diczx1">而畅于四支。——<span class="smcs">《易·坤》</span></span></p><p><span class="diczx1">尽其四支动。——<span class="smcs">《管子·小匡》</span></span></p><p><span
