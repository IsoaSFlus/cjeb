						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>卧</strong></p><p><span class="dicpy">wò  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wò.mp3"></a></span> ㄨㄛˋ</span></p><ol>    <li>睡倒，躺或趴：～倒。～铺。～室。～底（埋伏下来做内应，如“～～传中”）。～槽。～龙（喻隐居或尚未露头角的杰出人才）。～薪尝胆（形容刻苦自励，志图恢复）。藏龙～虎（喻潜藏着人才）。</li>    <li>把去壳的鸡蛋放到开水里煮：～个鸡子儿。</li></ol>

                    <h2 class="h2_entry"><span class="orth">卧</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>卧</strong></p><p>臥 <span class="dicpy">wò</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/wò.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。从人臣。<span class="yhcs">“臣”</span>是竖立的眼睛。本义:人伏在几案上休息，眼睛呈竖立形)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[lie prone upon]</span></p><p><span class="diczx1">坐而言，不应，隐几而卧。——<span class="smcs">《孟子》</span>。</span><span class="diczx2">焦循</span>正义:<span class="yhcs">“卧与寝异，寝于床，卧于几。统言之则不别。”</span></p><p><span class="diczx1">一生伏案卧。——</span><span class="diczx2">清</span>· <span class="diczx2">方苞</span><span class="smcs">《左忠毅公逸事》</span></p><p><span class="cino">(3)</span> 躺;卧而不眠 <span class="encs">[lie]</span></p><p><span class="diczx1">弛然而卧。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《捕蛇者说》</span></p><p><span class="diczx1">溪头卧剥莲蓬。——</span><span class="diczx2">宋</span>· <span class="diczx2">辛弃疾</span><span class="smcs">《清平乐·村居》</span></p><p><span class="diczx1">卧听风吹雨。——</span><span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《十一月四日风雨大作》</span></p><p><span class="diczx1">僵卧孤村不自哀。——</span><span class="diczx2">宋</span>· <span class="diczx2">陆游</span><span class="smcs">《示儿》</span></p><p><span class="diczx1">或立或卧。——</span><span class="diczx2">清</span>· <span class="diczx2">薛福成</span><span class="smcs">《观巴黎油画记》</span></p><p><span class="cino">(4)</span> 又如:</p><p><span class="cino">(5)</span> 卧冰(卧于冰上。古孝子卧冰事母的故事);</p><p><span class="cino">(6)</span> 卧碑(规矩。原指明清两代在明伦堂立的石碑，上面刻着约束生员的条文);</p><p><span class="cino">(7)</span> 卧佛(横卧的佛像。即释迦的涅槃像);</p><p><span class="cino">(8)</span> 卧治(躺着治事。指无为而治);</p><p><span class="cino">(9)</span> 卧理(躺着理事。同<span class="yhcs">“卧治”</span>);</p><p><span class="cino">(10)</span> 卧苫枕块(卧于草垫上，以土块作枕头。指人子守父母之丧而言。同<span class="yhcs">“寝苫枕块”</span>);</p><p><span class="cino">(11)</span> 卧甲(不解甲而卧。有枕戈待敌之意)</p><p><span class="cino">(12)</span> 眠，睡 <span class="encs">[sleep]</span></p><p><span class="diczx1">暮卧则梦闻。——<span class="smcs">《论衡·订鬼》</span></span></p><p><span class="cino">(13)</span> 又如:卧不安席(睡不安宁。形容心事重重);卧处(指睡卧起居;寝卧之所);卧起(寝卧和起身。多指日常生活诸事);卧榻之侧(床铺旁边。比喻距离很近或在自己的管辖范围之内)</p><p><span class="cino">(14)</span> 倒伏;横陈 <span class="encs">[fall]</span></p><p><span class="diczx1">长桥卧波。——</span><span class="diczx2">唐</span>· <span class="diczx2">杜牧</span><span class="smcs">《阿房宫赋》</span></p><p><span class="diczx1">横卧一楫。——</span><span class="diczx2">明</span>· <span class="diczx2">魏学洢</span><span class="smcs">《核舟记》</span></p><p><span class="cino">(15)</span> 又如:卧辙(卧于道路上阻止行车。后用作挽留去职官吏的用语);卧骨(谓死于沙场);卧翻(将牲兽翻倒)</p><p><span class="cino">(16)</span> 舍弃 <span class="encs">[abandon]</span>。如:卧鼓(将战鼓弃置不用，使之废卧地上，表示时局清平无战事)</p><p><span class="cino">(17)</span> 燠，暖 <span class="encs">[heat]</span></p><p><span class="diczx1">屈木为棬，以张生绢袋子，滤热乳，著瓦瓶子中卧之。——<span class="smcs">《齐民要术》</span></span></p><p><span class="cino">(18)</span> 又方言，专指用开水煮熟鸡蛋。如:卧个鸡子儿</p><p><span class="cino">(19)</span> 隐居 <span class="encs">[live in the seclution]</span>。如:卧云(卧于高山云海深处。比喻隐居);卧隐(隐居);卧龙(喻隐居或尚未崭露头角的杰出人才);卧龙凤雏(指诸葛亮与庞统。后引申指未为世用的英才)</p><p><span class="cino">(20)</span> 伏藏 <span                           『說文解字注』<hr class="dichr">
                          <span class="jieshi">伏也。</span>伏大徐作休。誤。臥與寑異。寑於牀。論語寢不尸是也。臥於几。孟子隱几而臥是也。臥於几。故曰伏。尸篆下曰象臥之形是也。此析言之耳。統言之則不別。故宀部曰寑者臥也。曲禮云。寢毋伏。則謂寢於牀者毋得俯伏也。引伸爲凡休息之偁。<span class="jieshi">从人臣。取其伏也。</span>臣下曰。象屈服之形。故以人臣會意。吾貨切。十七部。<span class="jieshi">凡臥之屬皆从臥。</span>                          </div>
