                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>仄</strong> <span class="dicpy">zè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zè.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (会意。小篆字形象人侧身在山崖洞穴里的形状。本义:倾斜)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[oblique]</span></p><p><span class="diczx1">仄，侧倾也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">行山者仄輮。——<span class="smcs">《周礼·考工记·车人》</span></span></p><p><span class="diczx1">众畏其口，见之仄目。——<span class="smcs">《汉书·息夫躬传》</span></span></p><p><span class="diczx1">日极则仄，月满则亏。——<span class="smcs">《管子·白心》</span></span></p><p><span class="diczx1">每旦视朝，日仄乃罢。——<span class="smcs">《后汉书·光武纪》</span></span></p><p><span class="cino">(3)</span> 又如:仄歪(方言。倾斜摇晃);仄日(夕阳，斜阳);仄席(不敢正座，侧身坐在席上。形容忧心的样子);仄起(倾斜突起);仄目(斜着眼看);仄步(行走歪斜貌)</p><p><span class="cino">(4)</span> 内心不安 <span class="encs">[worried]</span></p><p><span class="diczx1">衷怀歉仄，自叹无缘。——<span class="smcs">《红楼梦》</span></span></p><p><span class="cino">(5)</span> 又如:仄悚(恐惧不安);仄兀(歉疚不安)</p><p><span class="cino">(6)</span> 置 <span class="encs">[place oneself]</span>。如:仄身(置身)</p><p><span class="cino">(7)</span> 〈方〉∶仰，抬起 <span class="encs">[raise]</span></p><p><span class="diczx1">她仄起头儿，眨巴着眼睛瞅</span><span class="diczx2">江涛</span>。—— <span class="diczx2">梁斌</span><span class="smcs">《红旗谱》</span></p><p><strong>词性变化</strong><hr class="dichr" /></p><p>◎ <strong>仄</strong> <span class="dicpy">zè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zè.mp3"></a></span> </p><p>〈形〉</p><p><span class="cino">(1)</span> 狭窄 <span class="encs">[cramped;narrow]</span></p><p><span class="diczx1">险道倾仄，且驰且射。——<span class="smcs">《汉书》</span></span></p><p><span class="cino">(2)</span> 又如:人多地仄;仄小(狭小，窄小);仄陋(狭窄简陋);仄狭(狭窄);仄径(狭窄的小路);仄道(狭窄的小路);仄仄(狭窄)</p><p><span class="cino">(3)</span> 社会地位或政治地位低下的 <span class="encs">[humble]</span></p><p><span class="diczx1">明明扬仄陋。——<span class="smcs">《书·尧典》</span></span></p><p><span class="cino">(4)</span> 又如:仄微(卑贱，社会地位低)</p><p>◎ <strong>仄</strong> <span class="dicpy">zè</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/zè.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 通<span class="yhcs">“侧”</span>( <span class="dicpy">cè</span>)。旁边 <span class="encs">[side]</span></p><p><span class="diczx1">若子之材，可优游都城而取卿相，何必勒功</span><span class="diczx2">昆山</span>之仄。——<span class="smcs">《汉书·段会宗传》</span></p><p><span class="diczx1">行山者仄輮。——<span class="smcs">《周礼·考工记·车人》</span></span></p><p><span class="diczx1">罢退外亲及旁仄素餐之人。——<span class="smcs">《汉书·鲍宣传》</span>。</span><span class="diczx2">颜师古</span>云:<span class="yhcs">“仄，古侧字也。”</span></p><p><span class="cino">(2)</span> 又如:仄足(侧足);仄室(侧室)</p><p><span class="cino">(3)</span> 仄声，古汉语四声中上、去、入三声的总称 <span class="encs">[oblique
