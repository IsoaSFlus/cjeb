                    <h2 class="h2_entry"><span class="orth">役</span><span class="span dictname"> 详细解释</span></h2>
                  </div>
                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>役</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈动〉</p><p><span class="cino">(1)</span> (古文作伇。会意。甲骨文字形，象人持殳( <span class="dicpy">shū</span>)击人。本义:服兵役，戍守边疆)</p><p><span class="cino">(2)</span> 同本义 <span class="encs">[serve in the army]</span></p><p><span class="diczx1">役，戍边也。执殳巡行也。——<span class="smcs">《说文》</span></span></p><p><span class="diczx1">若起野役。——<span class="smcs">《周礼·遂人》</span></span></p><p><span class="diczx1">听政役以比居州长。——<span class="smcs">《周礼·小宰》</span></span></p><p><span class="diczx1">行役之事。——<span class="smcs">《周礼·师田》</span></span></p><p><span class="diczx1">君子于役，如之何勿思。——<span class="smcs">《诗·王风》</span>。</span><span class="diczx2">郑玄</span>笺:<span class="yhcs">“行役多危难，我诚思之。”</span></p><p><span class="diczx1">弃政而役，非其任也。——<span class="smcs">《国语》</span></span></p><p><span class="cino">(3)</span> 又如:役夫(服役的人);役民(服役的人);役卒(服役的人)</p><p><span class="cino">(4)</span> 役使，驱使 <span class="encs">[use as a servant]</span></p><p><span class="diczx1">予造天役，遗大投艰于朕身。——<span class="smcs">《书·大诰》</span></span></p><p><span class="cino">(5)</span> 又如:役车(供役用的车);役徒(服工役的人);役属(使隶属于己而役使之);役养(役使;役使休养);役法(为法令所役使);役物(役使外物为我所用);役令(役使，驱遣)</p><p><strong>词性变化</strong></p><p>◎ <strong>役</strong> <span class="dicpy">yì</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/yì.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> 劳役 <span class="encs">[labor]</span></p><p><span class="diczx1">兵久不辍，民困于役。——<span class="smcs">《三国志》</span></span></p><p><span class="cino">(2)</span> 又如:役丁(服劳役者);役作(工役劳作);役所(役作的处所);役庸(用以代替力役的赋税)</p><p><span class="cino">(3)</span> 服劳役的人 <span class="encs">[laborer]</span></p><p><span class="diczx2">季</span>氏使其乘之人，以其役邑入者无征。——<span class="smcs">《左传》</span>。 <span class="diczx2">孔颖达</span>疏:役谓共官力役，帅今之丁也</p><p><span class="cino">(4)</span> 仆人 <span class="encs">[servant]</span></p><p><span class="diczx1">汝辈幼小，家贫无役，柴水之劳，何时可免?——<span class="smcs">《南史》</span></span></p><p><span class="diczx1">虽留下几户穷民，只好奔走官差，那有空役等雇。——<span class="smcs">《京本通俗小说》</span></span></p><p><span class="cino">(5)</span> 门生，弟子 <span class="encs">[follower]</span></p><p><span class="diczx1">儒家之徒董无心，墨家之役缠子，相见讲道。——<span class="smcs">《论衡》</span></span></p><p><span class="cino">(6)</span> 事情 <span class="encs">[affair]</span></p><p><span class="diczx1">为此役也，子若以君命赐之，其已。——<span class="smcs">《左传》</span>。</span><span class="diczx2">杜预</span>注:役，事也</p><p><span class="cino">(7)</span> 事件 <span class="encs">[accident]</span></p><p><span class="diczx1">鸿门之役，微项伯几殆。——</span><span class="diczx2">宋</span>· <span class="diczx2">陈亮</span><span class="smcs">《酌古论一》</span></p><p><span class="cino">(8)</span> 士卒，服兵役的人 <span class="encs">[soldier]</span></p><p><span class="diczx1">寡人帅不腆</span><span
