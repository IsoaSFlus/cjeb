                        <div class="content definitions jnr">
						<p><strong>基本字义</strong></p><hr class="dichr" /><p>● <strong>始</strong></p><p><span class="dicpy">shǐ  <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǐ.mp3"></a></span>  ㄕˇ</span></p><ol>    <li>起头，最初，与“<span class="diczx3">终</span>”相对：开～。～终。～祖。～创。周而复～。</li>    <li>才，刚才：方～。～悟（才觉悟到）。春蚕到死丝方尽，蜡炬成灰泪～干。</li></ol>

                    <h2 class="h2_entry"><span class="orth">始</span><span class="span dictname"> 详细解释</span></h2>

                  <div class="content definitions xnr">
                    <p><strong>详细字义</strong></p><hr class="dichr" /><p>◎ <strong>始</strong> <span class="dicpy">shǐ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǐ.mp3"></a></span> </p><p>〈名〉</p><p><span class="cino">(1)</span> (形声。从女，台声。本义:开头，开始)</p><p><span class="cino">(2)</span> 同本义。与<span class="yhcs">“终”</span>相对 <span class="encs">[begin;start]</span></p><p><span class="diczx1">始，女之初也。——<span class="smcs">《说文》</span>。</span><span class="diczx2">朱</span>曰:<span class="yhcs">“裁衣之始为初，草木之始为才，人身之始为首为元，筑墙之始为基，开户之始为戽，子孙之始为祖，形生之始为胎。”</span></p><p><span class="diczx1">无名天地之始。——<span class="smcs">《老子》</span>。注:<span class="yhcs">“始者，道本也。”</span></span></p><p><span class="diczx1">始求深也。——<span class="smcs">《易·恒》</span>。</span><span class="diczx2">虞</span>注:<span class="yhcs">“乾为始。”</span></p><p><span class="diczx1">君子慎始。——<span class="smcs">《礼记·经解》</span></span></p><p><span class="diczx1">礼始于冠。——<span class="smcs">《礼记·昏义》</span>。注:<span class="yhcs">“犹根也。”</span></span></p><p><span class="diczx1">天地者，生之始也。——<span class="smcs">《荀子·王制》</span></span></p><p><span class="diczx1">忧懈怠则思慎始而敬终。——</span><span class="diczx2">唐</span>· <span class="diczx2">魏征</span><span class="smcs">《谏太宗十思疏》</span></p><p><span class="diczx1">始缢即气绝。——</span><span class="diczx2">方苞</span><span class="smcs">《狱中杂记》</span></p><p><span class="diczx1">始于文艺中兴时代之</span><span class="diczx2">意大利</span>。—— <span class="diczx2">蔡元培</span><span class="smcs">《图画》</span></p><p><span class="cino">(3)</span> 又如:始卒(开始和终止);始室(始娶的妻子，元配夫人);始基(初立的基业);自始至终</p><p><span class="cino">(4)</span> 过去，从前 <span class="encs">[past]</span></p><p><span class="diczx1">公侯之子孙，必复其始。——<span class="smcs">《左传》</span></span></p><p><span class="cino">(5)</span> 根本，本源 <span class="encs">[foundation]</span></p><p><span class="diczx1">坚树在始。——<span class="smcs">《国语》</span>。</span><span class="diczx2">韦昭</span>注:<span class="yhcs">“始，根本也。”</span></p><p><span class="cino">(6)</span> 姓</p><p><strong>词性变化</strong></p><p>◎ <strong>始</strong> <span class="dicpy">shǐ</span> <span class="ptr"><a class="audio_play_button i_volume-up ptr" data-src-mp3="//img.zdic.net/audio/zd/py/shǐ.mp3"></a></span> </p><p>〈副〉</p><p><span class="cino">(1)</span> 当初，在最初的时候 <span class="encs">[at first]</span></p><p><span class="diczx1">其始太医以王命聚之，岁赋其二。——</span><span class="diczx2">唐</span>· <span class="diczx2">柳宗元</span><span class="smcs">《捕蛇者说》</span></p><p><span class="diczx1">庭中始为篱，已为墙，凡再变矣。——</span><span class="diczx2">明</span>· <span class="diczx2">归有光</span><span class="smcs">《项脊轩志》</span></p><p><span class="cino">(2)</span> 又如:始春(立春日。二十四节气之一);始愿(最初的志愿);欲善终，应善始;始初(开始，起初);始闻其语(当初听到他的话)</p><p><span class="cino">(3)</span> 曾，尝 <span class="encs">[before long]</span></p><p><span class="diczx1">以为凡是州之山水有异态者，皆我有也;而未始知西山之怪特。——</span><span class="diczx2">柳宗元</span><span class="smcs">《始得西山宴游记》</span></p><p><span class="cino">(4)</span> 刚刚;才 <span class="encs">[just， only
